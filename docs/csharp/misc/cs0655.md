## <a name="example"></a><span data-ttu-id="3c51a-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="3c51a-105">Example</span></span>  
 <span data-ttu-id="3c51a-106">Aşağıdaki örnek CS0655 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3c51a-106">The following sample generates CS0655:</span></span>  
  
```csharp  
// CS0655.cs  
using System;  
  
class MyAttribute : Attribute  
{  
    // decimal is not valid attribute parameter type  
    public decimal d = 0;  
    public int e = 0;  
}  
  
[My(d = 0)]   // CS0655  
// Try the following line instead:  
// [My(e = 0)]  
class C  
{  
    public static void Main()  
    {  
    }  
}  
```
