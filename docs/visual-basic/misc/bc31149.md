# <a name="duplicate-xml-attribute-39attributename39"></a>Yinelenen XML özniteliği &#39;attributeName&#39;
Bir XML öğesi için değişmez değer belirtilen bir öznitelik, XML öğesi değişmez değeri için önceden tanımlanmış bir özniteliğin eşleşir. XML öğesi değişmez değerlerinde öznitelik adları benzersiz olmalıdır.  
  
 **Hata Kimliği:** BC31149  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Öznitelik, XML öğesi değişmez değeri için benzersiz bir adla yeniden adlandırın.
