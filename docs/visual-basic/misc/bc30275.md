# <a name="39ltspecifiergt39-is-not-valid-on-an-interface-event-declaration"></a>&#39;&lt;belirleyici&gt; &#39; arabirimi olay bildiriminde geçerli değil
Bir `Event` bir arabirim ifadeye içerir, gibi izin verilmeyen bir anahtar sözcük `Implements`. Arabirim üyeleri yalnızca tanımlayın, bunları uygulama değil.  
  
 **Hata Kimliği:** BC30275  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Anahtar sözcüğü bildirimi deyimden kaldırın.  
  
2.  Arabirim üyeleri uyarlamasını arabirimini uygulayan bir sınıf taşıyın.
