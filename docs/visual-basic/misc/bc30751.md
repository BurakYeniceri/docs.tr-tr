# <a name="unable-to-get-type-information-for-39lttypenamegt39"></a>Tür bilgileri &#39;alınamıyor; &lt;typename&gt;&#39;
Bir tür, ortak dil çalışma zamanı tarafından yüklenmemiş başvuruldu.  
  
 **Hata Kimliği:** BC30751  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Türü için geçerli kapsamda tanımlı böylece kodunuzu yeniden yapılandırabilirsiniz.
