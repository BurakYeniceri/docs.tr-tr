# <a name="compiler-warning-level-1-cs0612"></a>Derleyici Uyarısı (düzey 1) CS0612
'member' kullanılmıyor  
  
 Sınıf Tasarımcısı sahip bir üye olarak işaretlenmiş [Kullanımdan kalktı](http://msdn.microsoft.com/en-us/05e99cd0-bda6-4f79-a890-1ca093b4b488) özniteliği. Bu üye sınıfı gelecek bir sürümünde desteklenmeyebilir anlamına gelir.  
  
 Aşağıdaki örnek, geçersiz bir üye erişme CS0612 nasıl oluşturur gösterir:  
  
```  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
