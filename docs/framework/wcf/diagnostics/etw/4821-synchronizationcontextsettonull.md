|||  
|-|-|  
|Kimlik|4821|  
|Anahtar Sözcükler|Bulma|  
|Düzey|Bilgiler|

## <a name="description"></a>Açıklama  
 Bu olay SynchronizationContext ayarladığınızda yayılan bulma işlemini başlatmadan önce DiscoveryClientChannel tarafından null.

 SynchronizationContext ayarlamak için bulma işlemini başlatmadan önce DiscoveryClientChannel tarafından null.
