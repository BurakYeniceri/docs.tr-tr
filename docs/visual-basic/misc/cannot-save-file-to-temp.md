# <a name="cannot-save-file-to-temp"></a>Dosya TEMP kaydedilemiyor.
Bir bileşenin TEMP adlı bir dizin bulunamıyor ya da sürücü veya TEMP dizinini içeren bölüm bilgileri kaydetmek için yeterli alana sahip değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  "TEMP" adlı bir dizin oluşturun ve TEMP ortam değişkenini yolunu eşit ayarlayın.  
  
2.  Gereksiz dosyaları silerek sürücüsünde olun veya başka bir bölüme TEMP dizini oluşturun ve TEMP ortam değişkenini yolunu eşit ayarlayın.
