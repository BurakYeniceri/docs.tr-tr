# <a name="39lteventnamegt39-is-not-an-event-of-39ltcontainernamegt39"></a>&#39; &lt;eventname&gt;&#39; olay &#39; değil&lt; kapsayıcı adı&gt;&#39;
Nesne için belirtilen olay bildirilmedi.  
  
 **Hata Kimliği:** BC30676  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Olayın adının yazımını denetleyin.  
  
2.  Doğru nesne erişimi denetleyin. Nesne için tam başvuruları veya kullanın `Imports` gerekirse uygun ad almak için deyimi.
