# <a name="39resume39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Resume&#39; ifadeleri geçerli değildir komut penceresi
Resume deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30714  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Resume` deyimleri hata ayıklama kodundan.
