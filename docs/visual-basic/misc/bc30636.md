# <a name="39gt39-expected"></a>&#39;&gt;&#39;Beklenen
Belirtilen kod sözdizimi yanlış. Bu hata, genellikle bir öznitelik veya XML öğesi değişmez değeri için kapanış açılı ayracı eksik olduğunu gösterir.  
  
 **Hata Kimliği:** BC30636  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanılan kod için sözdizimini denetleyin.
