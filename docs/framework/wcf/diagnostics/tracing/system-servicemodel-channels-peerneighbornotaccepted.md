# <a name="systemservicemodelchannelspeerneighbornotaccepted"></a>System.ServiceModel.Channels.PeerNeighborNotAccepted
Eş komşu kabul edilmedi.  
  
## <a name="description"></a>Açıklama  
 Bu izleme, komşu bağlantı işlenirken gerçekleşir. Zaman aşımı, bir bağlantı veya geçici ağ sorunları kabul edemez komşu, olası nedenleri eşleşmeyen güvenlik ayarlarını içerir.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Eşleşmeyen güvenlik ayarları, zamanaşımı aşıldı, komşu'nın bir bağlantı veya geçici ağ sorunları kabul etme becerisinin denetleyin.
