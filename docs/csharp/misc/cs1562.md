# <a name="compiler-error-cs1562"></a>Derleyici Hatası CS1562
Çıkış kaynağı olmadan olmalıdır/out seçeneği belirtildi  
  
 Derleme çıktı dosyası oluşturabilirsiniz, ancak çıktı dosyası adını kapsanan giriş olarak hiçbir kaynak kodu dosyası değil. Örneğin, yalnızca meta veri veya kaynak dosyasını derleyin çalışıyor olabilir.  
  
 Kullanım [/out](../../csharp/language-reference/compiler-options/out-compiler-option.md) derleyici seçeneği çıkış dosyasının adını belirtin.
