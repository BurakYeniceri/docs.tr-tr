# <a name="compiler-error-cs0112"></a><span data-ttu-id="6b2f4-102">Derleyici Hatası CS0112</span><span class="sxs-lookup"><span data-stu-id="6b2f4-102">Compiler Error CS0112</span></span>
<span data-ttu-id="6b2f4-103">Statik bir üyenin 'function' override, sanal veya soyut işaretlenemez</span><span class="sxs-lookup"><span data-stu-id="6b2f4-103">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="6b2f4-104">Kullanan herhangi bir yöntem bildirimi `override`, **sanal**, veya **soyut** anahtar sözcüğü de kullanamaz **statik** anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="6b2f4-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="6b2f4-105">Daha fazla bilgi için bkz: [yöntemleri](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="6b2f4-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="6b2f4-106">Aşağıdaki örnek CS0112 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6b2f4-106">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
