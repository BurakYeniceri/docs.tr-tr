# <a name="how-to-represent-columns-as-database-generated"></a>Nasıl yapılır: sütunlar veritabanında oluşturulan olarak temsil eder
Kullanım [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] <xref:System.Data.Linq.Mapping.ColumnAttribute.IsDbGenerated%2A> özellikte <xref:System.Data.Linq.Mapping.ColumnAttribute> veritabanında oluşturulan bir sütun olarak temsil eden bir alan veya özellik atamak özniteliği.  
  
 Kod örnekleri için bkz: <xref:System.Data.Linq.Mapping.ColumnAttribute.IsDbGenerated%2A>.  
  
### <a name="to-designate-a-field-or-property-as-representing-a-database-generated-column"></a>Bir alan veya özellik veritabanında oluşturulan bir sütunu temsil eden olarak atamak için  
  
1.  Ekleme <xref:System.Data.Linq.Mapping.ColumnAttribute.IsDbGenerated%2A> özelliğine <xref:System.Data.Linq.Mapping.ColumnAttribute> özniteliği.  
  
2.  Özellik değerini ayarlamak `true`.
