# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-using-the-ansi-auto-or-unicode-modifier"></a>.NET Compact Framework hedeflenen sürümü ANSI, otomatik veya Unicode değiştiricisi kullanımını desteklemez
.NET Compact Framework'ı çalıştığınız sürümü desteklemediği `Ansi`, `Auto`, ve `Unicode` değiştiricileri.  
  
 **Hata Kimliği:** BC30763  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değiştirici kaldırın.
