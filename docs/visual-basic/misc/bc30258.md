# <a name="classes-can-inherit-only-from-other-classes"></a>Yalnızca diğer sınıflardan devrettiği
Bir `Inherits` deyimi bir sınıf dışında bir türü temel türü belirtir.  
  
 **Hata Kimliği:** BC30258  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Temel türü zaten tanımlı bir sınıf değiştirin.
