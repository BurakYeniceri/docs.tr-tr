# <a name="integrating-with-com-applications"></a>COM Uygulamaları ile Tümleştirme
[!INCLUDE[indigo1](../../../../includes/indigo1-md.md)]dağıtılmış uygulamaları oluşturmak için zengin bir ortam sağlar. COM + barındırılan bir bileşen tabanlı uygulama mantığı önemli ölçüde yatırımınız varsa, kullanabileceğiniz [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] yeniden yazmak zorunda yerine mevcut mantığınızı genişletmek için. Bu bölüm içindeki konuları COM + ile kullanmayı açıklar [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)].  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [COM + uygulamaları ile tümleştirme genel bakış](../../../../docs/framework/wcf/feature-details/integrating-with-com-plus-applications-overview.md)  
 COM + bileşenleri bütünleştirmek nasıl ve ne zaman bir bakış sağlar.  
  
 [Nasıl yapılır: COM + hizmet modeli yapılandırma aracını kullanın](../../../../docs/framework/wcf/feature-details/how-to-use-the-com-service-model-configuration-tool.md)  
 COM + hizmet modeli yapılandırma komut satırı aracı (ComSvcConfig.exe) olarak gösterilen istediğiniz uygulama arabirimleri yapılandırmak için nasıl kullanılacağını açıklar [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] Hizmetleri.  
  
 [Nasıl yapılır: COM + hizmet ayarlarını yapılandırma](../../../../docs/framework/wcf/feature-details/how-to-configure-com-service-settings.md)  
 Bir COM + nesnesi olarak yapılandırılmasını açıklamaktadır bir [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] hizmet.  
  
 [Nasıl yapılır: COM + tümleştirme uygulaması dağıtma](../../../../docs/framework/wcf/feature-details/how-to-deploy-a-com-integration-application.md)
