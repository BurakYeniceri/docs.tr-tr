## <a name="example"></a><span data-ttu-id="1c816-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="1c816-106">Example</span></span>  
 <span data-ttu-id="1c816-107">Aşağıdaki örnek CS1642 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="1c816-107">The following sample generates CS1642.</span></span>  
  
```csharp  
// CS1642.cs

unsafe class C  
{  
   fixed int a[10];   // CS1642

    fixed int a[10];  
}

{  
   public int[] a = null;  
}  
```
