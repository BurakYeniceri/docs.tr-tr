## <a name="drawbacks-of-the-api-gateway-pattern"></a><span data-ttu-id="d8ad6-244">API ağ geçidi düzeni eksileri</span><span class="sxs-lookup"><span data-stu-id="d8ad6-244">Drawbacks of the API Gateway pattern</span></span>
