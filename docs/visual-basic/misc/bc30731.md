# <a name="39addressof39-expressions-are-not-valid-in-debug-windows"></a>&#39;AddressOf&#39; ifadeleri geçerli değildir hata ayıklama pencerelerinde
`AddressOf` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30731  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kaldırma `AddressOf` kodda hata ayıklama gelen deyimleri.
