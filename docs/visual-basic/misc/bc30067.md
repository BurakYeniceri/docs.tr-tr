# <a name="39exit-function39-is-not-valid-in-a-sub-or-property"></a>&#39; Exit işlevi &#39; bir alt veya özelliği geçerli değil
`Exit Function`görünür bir `Sub` yordamı veya `Property` yordamı. Bir `Exit` deyimi blok eşleşmelidir içinde hangi oluşur.  
  
 **Hata Kimliği:** BC30067  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değiştir `Exit Function` ile `Exit Sub` veya `Exit Property` uygun şekilde deyimi.
