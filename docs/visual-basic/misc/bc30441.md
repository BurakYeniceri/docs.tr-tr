# <a name="39catch39-must-end-with-a-matching-39end-try39"></a>&#39;Catch&#39; eşleşen ile bitmelidir &#39;son deneyin&#39;
A `Catch` deyimi, eşleşen bir olmadan kodunuzda görünüyor `End Try` deyimi. `Catch` deyimleri sonuçları, ile bir `End Try` deyimi.  
  
 **Hata Kimliği:** BC30441  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kaldırma `Catch` deyimi.  
  
2.  Ekleme bir `End Try` blok sonuçlandırmak için deyimi.
