# <a name="39end-property39-must-be-preceded-by-a-matching-39property39"></a>&#39; End özelliği &#39; eşleşen bir &#39;göre gelmelidir; Özellik &#39;
Bir `End Property` deyimi görünüyor, kodunuzda hiçbir eşleşen `Property` kendisinden bildirimi.  
  
 **Hata Kimliği:** BC30431  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `End Property` yedek ise deyimi.  
  
-   Eksik tedarik `Property` eksikse yordamı.  
  
-   Taşıma `End Property` uygun yerleştirmek için kod.
