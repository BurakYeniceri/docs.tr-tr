## <a name="delegates-and-their-role"></a><span data-ttu-id="46955-114">Temsilciler ve rolleri</span><span class="sxs-lookup"><span data-stu-id="46955-114">Delegates and Their Role</span></span>  
 <span data-ttu-id="46955-115">Temsilciler sınıflardır içinde yaygın olarak kullanılan [!INCLUDE[dnprdnshort](../../../includes/dnprdnshort-md.md)] olay işleme mekanizmaları oluşturmak için.</span><span class="sxs-lookup"><span data-stu-id="46955-115">Delegates are classes commonly used within the [!INCLUDE[dnprdnshort](../../../includes/dnprdnshort-md.md)] to build event-handling mechanisms.</span></span> <span data-ttu-id="46955-116">Temsilciler kabaca eşitlemek için yaygın olarak kullanılan işlev işaretçileri [!INCLUDE[vcprvc](../../../includes/vcprvc-md.md)] ve diğer nesne yönelimli diller.</span><span class="sxs-lookup"><span data-stu-id="46955-116">Delegates roughly equate to function pointers, commonly used in [!INCLUDE[vcprvc](../../../includes/vcprvc-md.md)] and other object-oriented languages.</span></span> <span data-ttu-id="46955-117">İşlev işaretçileri, ancak, nesne yönelimli, tür kullanımı uyumlu ve güvenli temsilciler.</span><span class="sxs-lookup"><span data-stu-id="46955-117">Unlike function pointers however, delegates are object-oriented, type-safe, and secure.</span></span> <span data-ttu-id="46955-118">Ayrıca, burada bir işlev işaretçisi yalnızca belirli bir işleve bir başvuru içeriyor bir temsilci bir nesneye başvuru oluşur ve nesne içinde bir veya daha fazla yöntemlerini başvuruyor.</span><span class="sxs-lookup"><span data-stu-id="46955-118">In addition, where a function pointer contains only a reference to a particular function, a delegate consists of a reference to an object, and references to one or more methods within the object.</span></span>  
  
 <span data-ttu-id="46955-119">Bu olay modelini kullanır, *Temsilciler* bunları işlemek için kullanılan yöntemleri olayları bağlanacak.</span><span class="sxs-lookup"><span data-stu-id="46955-119">This event model uses *delegates* to bind events to the methods that are used to handle them.</span></span> <span data-ttu-id="46955-120">Temsilci bir işleyici yöntemi belirterek olay bildirimi için kaydolmaya diğer sınıflar sağlar.</span><span class="sxs-lookup"><span data-stu-id="46955-120">The delegate enables other classes to register for event notification by specifying a handler method.</span></span> <span data-ttu-id="46955-121">Olay ortaya çıktığında temsilci ilişkili yöntemi çağırır.</span><span class="sxs-lookup"><span data-stu-id="46955-121">When the event occurs, the delegate calls the bound method.</span></span> <span data-ttu-id="46955-122">Temsilcileri tanımlama hakkında daha fazla bilgi için bkz: [olayları](../../../docs/standard/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="46955-122">For more information about how to define delegates, see [Events](../../../docs/standard/events/index.md).</span></span>  
  
 <span data-ttu-id="46955-123">Temsilciler, tek bir yöntemi veya birden çok yöntem, çok noktaya yayın başvurulan bağlanabilir.</span><span class="sxs-lookup"><span data-stu-id="46955-123">Delegates can be bound to a single method or to multiple methods, referred to as multicasting.</span></span> <span data-ttu-id="46955-124">Bir olay için bir temsilci oluştururken, sizin (veya Windows Form Tasarımcısı) genellikle bir çok noktaya yayın olay oluşturursunuz.</span><span class="sxs-lookup"><span data-stu-id="46955-124">When creating a delegate for an event, you (or the Windows Forms Designer) typically create a multicast event.</span></span> <span data-ttu-id="46955-125">Nadir bir özel durum, mantıksal olarak birden çok kez olay yinelemeniz değil (örneğin bir iletişim kutusunu görüntüleme) belirli bir yordamı sonuçlanan bir olay olabilir.</span><span class="sxs-lookup"><span data-stu-id="46955-125">A rare exception might be an event that results in a specific procedure (such as displaying a dialog box) that would not logically repeat multiple times per event.</span></span> <span data-ttu-id="46955-126">Çok noktaya yayın temsilci oluşturma hakkında daha fazla bilgi için bkz: [nasıl yapılır: temsilcileri birleştirme (çok noktaya yayın temsilcileri)](~/docs/csharp/programming-guide/delegates/how-to-combine-delegates-multicast-delegates.md).</span><span class="sxs-lookup"><span data-stu-id="46955-126">For information about how to create a multicast delegate, see [How to: Combine Delegates (Multicast Delegates)](~/docs/csharp/programming-guide/delegates/how-to-combine-delegates-multicast-delegates.md).</span></span>  
  
 <span data-ttu-id="46955-127">Çok noktaya yayın temsilci bağlı yöntemleri çağırma listesini tutar.</span><span class="sxs-lookup"><span data-stu-id="46955-127">A multicast delegate maintains an invocation list of the methods it is bound to.</span></span> <span data-ttu-id="46955-128">Çok noktaya yayını destekleyen temsilci bir <xref:System.Delegate.Combine%2A> bir yöntemi çağırma listesine eklemek için yöntem ve bir <xref:System.Delegate.Remove%2A> kaldırmaya yöntemi.</span><span class="sxs-lookup"><span data-stu-id="46955-128">The multicast delegate supports a <xref:System.Delegate.Combine%2A> method to add a method to the invocation list and a <xref:System.Delegate.Remove%2A> method to remove it.</span></span>  
  
 <span data-ttu-id="46955-129">Bir olay uygulama tarafından kaydedildiğinde denetimi bu olay için temsilci çağırarak olayını başlatır.</span><span class="sxs-lookup"><span data-stu-id="46955-129">When an event is recorded by the application, the control raises the event by invoking the delegate for that event.</span></span> <span data-ttu-id="46955-130">Temsilci sırayla ilişkili yöntemi çağırır.</span><span class="sxs-lookup"><span data-stu-id="46955-130">The delegate in turn calls the bound method.</span></span> <span data-ttu-id="46955-131">(Çok noktaya yayın temsilci) en sık karşılaşılan durumda temsilci her ilişkili yöntemini çağırma listesinde sırasıyla bir-çok bildirim sağlayan çağırır.</span><span class="sxs-lookup"><span data-stu-id="46955-131">In the most common case (a multicast delegate) the delegate calls each bound method in the invocation list in turn, which provides a one-to-many notification.</span></span> <span data-ttu-id="46955-132">Bu strateji denetim olayı bildirimi için hedef nesnelerin listesini korumak gerekmez anlamına gelir — tüm kayıt ve bildirim temsilci işler.</span><span class="sxs-lookup"><span data-stu-id="46955-132">This strategy means that the control does not need to maintain a list of target objects for event notification—the delegate handles all registration and notification.</span></span>  
  
 <span data-ttu-id="46955-133">Temsilciler çok bir bildirim izin vererek aynı yönteme bağlı birden çok olay da etkinleştirin.</span><span class="sxs-lookup"><span data-stu-id="46955-133">Delegates also enable multiple events to be bound to the same method, allowing a many-to-one notification.</span></span> <span data-ttu-id="46955-134">Örneğin, bir düğme tıklama olay ve menü komutu – tıklatma olay her ikisi de aynı şekilde bu ayrı olayları işlemek için tek bir yöntemi çağırır aynı temsilci çağırabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="46955-134">For example, a button-click event and a menu-command–click event can both invoke the same delegate, which then calls a single method to handle these separate events the same way.</span></span>  
  
 <span data-ttu-id="46955-135">Temsilciler ile kullanılan bağlama dinamik mekanizmadır: çalışma zamanında, imza, olay işleyicisinin eşleşen herhangi bir yöntemi için bir temsilci bağlanabilir.</span><span class="sxs-lookup"><span data-stu-id="46955-135">The binding mechanism used with delegates is dynamic: a delegate can be bound at run time to any method whose signature matches that of the event handler.</span></span> <span data-ttu-id="46955-136">Bu özellik ile ayarlayın veya bağlı bir koşul olarak ve bir olay işleyicisi için bir denetim dinamik olarak eklemek için bağlama yöntemini değiştirin.</span><span class="sxs-lookup"><span data-stu-id="46955-136">With this feature, you can set up or change the bound method depending on a condition and to dynamically attach an event handler to a control.</span></span>
