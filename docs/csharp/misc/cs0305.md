## <a name="example"></a><span data-ttu-id="c1abe-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="c1abe-106">Example</span></span>  
 <span data-ttu-id="c1abe-107">Aşağıdaki örnek CS0305 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="c1abe-107">The following sample generates CS0305.</span></span>  
  
```csharp

   public static void Main()

      MyList<MyClass, MyClass> list1 = new MyList<MyClass>();   // CS0305

}  
```
