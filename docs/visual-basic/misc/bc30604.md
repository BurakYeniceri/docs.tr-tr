# <a name="statement-cannot-appear-within-an-interface-body-visual-basic-error"></a>Deyimi (Visual Basic hatası) arabirim gövdesi içinde yer alamaz
Beklenmeyen dil yapısı karşılaşıldı. Varsayılır bir `End Interface` yapı eksik ve bu noktadan sonra herhangi bir kaynak kod dışında arabirimidir.  
  
 **Hata Kimliği:** BC30604  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Arabirim tanımı içinde kullanılan kod söz dizimini doğrulayın.  
  
2.  Arabirim tanımı ile biten emin olun bir `End Interface` oluşturun.
