# <a name="encoding-cannot-be-set-to-nothing"></a>Kodlama için hiçbir şey ayarlanamaz
Okuma veya bir dosyaya yazma girişimi başarısız oldu parametresi `encoding` ayarlandığından `Nothing` ancak geçerli bir değer gerektirir.  
  
 <xref:System.Text.Encoding>hangi bir dosyaya yazılırken kullanılacak kodlama belirlemek için kullanılır. Varsayılan değer UTF-8'dir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İçin geçerli bir değer sağlamanız `encoding` parametresi.
