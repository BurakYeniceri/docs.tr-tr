# <a name="ltcodegt-visual-basic"></a>&lt;kod&gt; (Visual Basic)
Metin kodu çok satırlı olduğunu gösterir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```xml  
<code>content</code>  
```  
  
#### <a name="parameters"></a>Parametreler  
 `content`  
 Kod olarak işaretlemek için metin.  
  
## <a name="remarks"></a>Açıklamalar  
 Kullanım `<code>` birden fazla satır kodu olarak göstermek için etiket. Kullanım [ \<c >](../../../visual-basic/language-reference/xmldoc/c.md) belirten bir açıklama metni kodu olarak işaretlenmelidir.  
  
 İle derleme [/doc](../../../visual-basic/reference/command-line-compiler/doc.md) bir dosyaya işlem belgesi açıklamaları için.
