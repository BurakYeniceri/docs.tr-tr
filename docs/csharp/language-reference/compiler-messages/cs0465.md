<span data-ttu-id="7a820-103">'Finalize' yönteminin sunulması yıkıcı çağrılmasını engelleyebilir.</span><span class="sxs-lookup"><span data-stu-id="7a820-103">Introducing a 'Finalize' method can interfere with destructor invocation.</span></span> <span data-ttu-id="7a820-104">Bir yıkıcı bildirmeyi mi istediniz?</span><span class="sxs-lookup"><span data-stu-id="7a820-104">Did you intend to declare a destructor?</span></span>

 <span data-ttu-id="7a820-106">Böyle bir sınıfın temel sınıf olarak kullanılıyorsa ve türetilen sınıf bir yıkıcı tanımlıyorsa, yıkıcı temel sınıfı geçersiz kılar `Finalize` yöntemini verilmiyor <xref:System.Object.Finalize%2A>.</span><span class="sxs-lookup"><span data-stu-id="7a820-106">If such a class is used as a base class and if the deriving class defines a destructor, the destructor will override the base class `Finalize` method, not <xref:System.Object.Finalize%2A>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7a820-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="7a820-107">Example</span></span>  
 <span data-ttu-id="7a820-108">Aşağıdaki örnek CS0465 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="7a820-108">The following sample generates CS0465.</span></span>  
  
```csharp  
// CS0465.cs  
// compile with: /target:library  
class A  
{  
   public virtual void Finalize() {}   // CS0465  
}  
  
// OK  
class B  
{  
   ~B() {}  
}  
```
