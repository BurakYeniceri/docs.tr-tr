## <a name="example"></a><span data-ttu-id="f8920-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="f8920-106">Example</span></span>

```csharp

using System;  
public  class Test   
{  
    public static void Main()  
    {  
        int a = 5;  
        int? b = a + null;    // CS0458  
        int? qa = 15;  
        b = qa + null;        // CS0458  
        b -= null;            // CS0458  
        int? qa2 = null;  
        b = qa2 + null;       // CS0458  
        qa2 -= null;          // CS0458  
    }  
}  
```
