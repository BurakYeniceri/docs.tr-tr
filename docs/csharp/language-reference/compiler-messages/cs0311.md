## <a name="example"></a><span data-ttu-id="0a8f2-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="0a8f2-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B{}  
class C{}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```
