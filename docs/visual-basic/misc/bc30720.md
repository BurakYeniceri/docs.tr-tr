# <a name="39on-error39-statements-are-not-valid-in-the-immediate-window"></a>&#39; hata &#39; deyimleri Komut penceresi içinde geçerli değildir
`On Error`deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30720  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `On Error` kodda hata ayıklama gelen deyimleri.
