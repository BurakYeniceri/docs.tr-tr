# <a name="39lib39-expected"></a>&#39;LIB&#39; bekleniyor
A `Declare` deyimi kullanmadan bir kitaplık adı belirtir `Lib` anahtar sözcüğü.  
  
 **Hata Kimliği:** BC30218  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme `Lib` kitaplığı adından önce anahtar sözcüğü.
