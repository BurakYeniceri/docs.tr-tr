2.  <span data-ttu-id="b3596-113">Uygulamak ve kullanmak nasıl <xref:System.ServiceModel.Description.IPolicyExportExtension?displayProperty=nameWithType> ilke onaylamalarını WSDL veri aktarma işleminden önce ilke verilerini size sunar arabirimi.</span><span class="sxs-lookup"><span data-stu-id="b3596-113">How to implement and use the <xref:System.ServiceModel.Description.IPolicyExportExtension?displayProperty=nameWithType> interface, which exposes the policy data to you prior to exporting the policy assertions in WSDL data.</span></span>
