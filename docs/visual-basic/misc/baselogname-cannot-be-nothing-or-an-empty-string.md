# <a name="baselogname-cannot-be-nothing-or-an-empty-string"></a>BaseLogName Nothing veya boş bir dize olamaz
Değeri <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A> olamaz `Nothing` ya da boş bir dize.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A> Özelliği, günlük dosyaları temel adını belirtir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ayarlama <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.BaseFileName%2A> en az bir karakter içeren bir dize özelliği.
