# <a name="39withevents39-variables-must-have-an-39as39-clause"></a>&#39;WithEvents&#39; değişkenleri olmalıdır bir &#39;olarak&#39; yan tümcesi
Değil sağladığınız bir `As` yan tümcesi anahtar sözcüğüyle `WithEvents`. Değişken olaylar oluşturabilir belirli sınıfı olarak bildirin.  
  
 **Hata Kimliği:** BC30412  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ekleme `As` olayları yükseltebilirsiniz sınıfını belirtir yan tümcesi.
