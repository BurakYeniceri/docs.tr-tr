# <a name="compiler-error-cs1622"></a><span data-ttu-id="00aa2-102">Derleyici Hatası CS1622</span><span class="sxs-lookup"><span data-stu-id="00aa2-102">Compiler Error CS1622</span></span>
<span data-ttu-id="00aa2-103">Yineleyici bir değer döndüremiyor.</span><span class="sxs-lookup"><span data-stu-id="00aa2-103">Cannot return a value from an iterator.</span></span> <span data-ttu-id="00aa2-104">Yineleme bitiş kesmeyi veya bir değer döndürmesi için verim return deyimi kullanın.</span><span class="sxs-lookup"><span data-stu-id="00aa2-104">Use the yield return statement to return a value, or yield break to end the iteration.</span></span>  
  
 <span data-ttu-id="00aa2-105">Yineleyici return deyiminin yerine verim deyimi bir değer döndüren özel bir işlevdir.</span><span class="sxs-lookup"><span data-stu-id="00aa2-105">An iterator is a special function that returns a value via the yield statement rather than the return statement.</span></span> <span data-ttu-id="00aa2-106">Daha fazla bilgi için bkz: **yineleyiciler**.</span><span class="sxs-lookup"><span data-stu-id="00aa2-106">For more information, see **iterators**.</span></span>  
  
 <span data-ttu-id="00aa2-107">Aşağıdaki örnek CS1622 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="00aa2-107">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
