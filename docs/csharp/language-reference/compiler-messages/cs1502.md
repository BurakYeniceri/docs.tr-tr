<span data-ttu-id="ae933-103">'Name' için en iyi aşırı yüklenmiş yöntemin eşleşme bazı geçersiz bağımsız değişkenlere sahiptir</span><span class="sxs-lookup"><span data-stu-id="ae933-103">The best overloaded method match for 'name' has some invalid arguments</span></span>  
  
 <span data-ttu-id="ae933-104">Yönteme geçirilen bağımsız değişken türleri bu yöntem parametre türleri eşleşmiyor Bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="ae933-104">This error occurs when the argument types being passed to the method do not match the parameter types of that method.</span></span> <span data-ttu-id="ae933-105">Çağrılan yöntem aşırı yüklüyse yöntemi aşırı yüklenmiş sürümlerinin geçirilen bağımsız değişken türleri ile eşleşen bir imzası yok.</span><span class="sxs-lookup"><span data-stu-id="ae933-105">If the called method is overloaded, then none of the overloaded versions has a signature that matches the argument types being passed.</span></span>  
  
 <span data-ttu-id="ae933-106">Bu sorunu gidermek için aşağıdakilerden birini yapın:</span><span class="sxs-lookup"><span data-stu-id="ae933-106">To resolve this problem, do one of the following:</span></span>
