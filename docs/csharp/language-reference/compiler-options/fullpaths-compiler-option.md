# <a name="-fullpaths-c-compiler-options"></a>-fullpaths (C# Derleyici Seçenekleri)
**- Fullpaths** seçeneği derleme hataları ve Uyarıları listelerken dosyanın tam yolunu belirtmek derleyicinin neden olur.  
  
## <a name="syntax"></a>Sözdizimi  
  
```console  
-fullpaths  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Varsayılan olarak, hataları ve Uyarıları derlemeden kaynaklanan bir hata bulunduğu dosya adını belirtin. **- Fullpaths** seçeneği dosyasının tam yolunu belirtmek derleyicinin neden olur.  
  
 Bu derleyici seçeneği Visual Studio'da kullanılamıyor ve programlı olarak değiştirilemez.
