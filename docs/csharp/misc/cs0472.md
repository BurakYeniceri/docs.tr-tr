<span data-ttu-id="f03be-103">'Değer2' türünde bir değer hiçbir zaman eşit 'null' türü 'Değer3' ifadesinin sonucu her zaman 'değer1' olduğundan</span><span class="sxs-lookup"><span data-stu-id="f03be-103">The result of the expression is always 'value1' since a value of type 'value2' is never equal to 'null' of type 'value3'</span></span>

## <a name="example"></a><span data-ttu-id="f03be-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="f03be-105">Example</span></span>  
 <span data-ttu-id="f03be-106">Aşağıdaki örnek CS0472 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="f03be-106">The following sample generates CS0472.</span></span>  
  
```  
public class Test  
{  
    public static int Main()  
    {  
        int i = 5;  
        int counter = 0;  
  
        // Comparison:  
        if (i == null)  // CS0472  
        // To resolve, use a valid value for i.  
            counter++;   
        return counter;  
    }  
}  
```
