# <a name="compiler-error-cs1101"></a><span data-ttu-id="a5647-102">Derleyici Hatası CS1101</span><span class="sxs-lookup"><span data-stu-id="a5647-102">Compiler Error CS1101</span></span>
<span data-ttu-id="a5647-103">'Ref' parametresi değiştiricisi 'this' ile kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="a5647-103">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="a5647-104">Zaman `this` anahtar sözcüğü değiştirir statik yönteminin ilk parametresi, derleyiciye yöntemi bir genişletme yöntemi olduğunu bildirir.</span><span class="sxs-lookup"><span data-stu-id="a5647-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="a5647-105">Diğer bir değiştiricileri gerekli veya bir genişletme yöntemi ilk parametresi izin verilmiyor.</span><span class="sxs-lookup"><span data-stu-id="a5647-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a5647-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="a5647-106">Example</span></span>  
 <span data-ttu-id="a5647-107">Aşağıdaki örnek CS1101 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a5647-107">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
