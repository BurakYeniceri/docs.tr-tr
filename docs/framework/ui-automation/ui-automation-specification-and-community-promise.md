# <a name="ui-automation-specification-and-community-promise"></a>UI Otomasyonu Belirtimi ve Topluluk Taahhüdü
UI Otomasyonu, son kullanıcılar için kullanıcı Arabirimi hakkında bilgi sağlamak ve UI standart giriş farklı yollarla işlemek için ekran okuyucular gibi yardımcı teknoloji ürünlerinin etkinleştirme masaüstünde, çoğu kullanıcı arabirimi (UI) öğeleri programlı erişim sağlar .
