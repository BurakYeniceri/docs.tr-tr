# <a name="compiler-error-cs0260"></a><span data-ttu-id="20e9b-102">Derleyici Hatası CS0260</span><span class="sxs-lookup"><span data-stu-id="20e9b-102">Compiler Error CS0260</span></span>
<span data-ttu-id="20e9b-103">Türü 'type'; bildirimi Kısmi değiştirici eksik Bu türde başka bir kısmi bildirimi var</span><span class="sxs-lookup"><span data-stu-id="20e9b-103">Missing partial modifier on declaration of type 'type'; another partial declaration of this type exists</span></span>  
  
 <span data-ttu-id="20e9b-104">Bu hata, aynı ada sahip birden çok sınıf bildirilen gösterir.</span><span class="sxs-lookup"><span data-stu-id="20e9b-104">This error indicates that you have declared multiple classes that have the same name.</span></span> <span data-ttu-id="20e9b-105">Ayrıca, en az bir ancak tüm bildirimleri içeren `partial` değiştiricisi.</span><span class="sxs-lookup"><span data-stu-id="20e9b-105">In addition, at least one but not all of the declarations contains the `partial` modifier.</span></span> <span data-ttu-id="20e9b-106">Birden fazla bölümde bir sınıf tanımlamak istiyorsanız, anahtar sözcüğünü kullanarak her bölümü bildirilmelidir `partial`.</span><span class="sxs-lookup"><span data-stu-id="20e9b-106">If you want to define a class in several parts, you must declare each part by using the keyword `partial`.</span></span>  
  
 <span data-ttu-id="20e9b-107">Bir sınıf bildirme ve yanlışlıkla aynı ad alanında başka bir yerde bildirilmiş bir parçalı sınıf aynı adı verirseniz Ayrıca bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="20e9b-107">This error also occurs if you declare a class and accidentally give it the same name as a partial class that's declared elsewhere in the same namespace.</span></span>  
  
 <span data-ttu-id="20e9b-108">Aşağıdaki örnek CS0260 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="20e9b-108">The following sample generates CS0260:</span></span>  
  
```csharp  
// CS0260.cs  
// You must mark both parts of the definition of class C   
// by using the partial keyword.  
  
// The following line causes CS0260. To resolve the error, add  
// the 'partial' keyword to the declaration.  
class C    
{  
}  
  
partial class C  
{  
}  
```
