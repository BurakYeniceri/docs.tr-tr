# <a name="39ltspecifiergt39-is-not-valid-on-a-structure-declaration"></a>&#39; &lt;belirticisi&gt;&#39; üzerinde bir yapı bildirimleri geçerli değil
Bir belirleyici gibi kullandığınız `Dim` veya `Override`, yani üzerinde geçerli değil bir `Structure` bildirimi.  
  
 **Hata Kimliği:** BC30395  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirleyici kaldırın.
