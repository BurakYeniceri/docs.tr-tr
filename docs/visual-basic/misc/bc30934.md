<span data-ttu-id="249eb-103">Karşılık gelen bir öznitelik parametre farklı bir veri türü için öznitelik bağımsız değişkeni için kullanılan bir ifadeyi değerlendirir ve Visual Basic öznitelik bağımsız değişkenleri için gerekli tür dönüştürme izin vermez.</span><span class="sxs-lookup"><span data-stu-id="249eb-103">An expression used for an attribute argument evaluates to a data type different from that of the corresponding attribute parameter, and Visual Basic does not allow the required type conversion for attribute arguments.</span></span>  
  
 <span data-ttu-id="249eb-104">Uygulandığı öğe için bir öznitelik meta veri sağlar ve derleyici derleme zamanında tüm meta verilerin oluşturmak mümkün olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="249eb-104">An attribute provides metadata for the element it is applied to, and the compiler must be able to construct all the metadata at compile time.</span></span> <span data-ttu-id="249eb-105">Bu nedenle, her özniteliği derleme zamanında sabit değerleri kullanmanız gerekir ve bu nedenle, her öznitelik bağımsız değişkeni bir derleme zamanı sabit değer değerlendirmeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="249eb-105">For this reason, every attribute must use values that are constant at compile time, and therefore every attribute argument must evaluate to a compile-time constant value.</span></span>
