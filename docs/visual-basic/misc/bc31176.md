# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-in-xml-literal"></a>Karakter &#39; karakter &#39; (0 x&lt;onaltılık&gt;) XML değişmez izin verilmiyor
Desteklenmeyen bir karakter, bir XML değişmez değer kullanılmış.  
  
 **Hata Kimliği:** BC31176  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Desteklenmeyen bir karakter kaldırın.
