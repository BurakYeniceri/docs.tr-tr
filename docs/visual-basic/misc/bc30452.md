# <a name="operator-39ltoperatornamegt39-is-not-defined-for-types-39lttype1gt39-and-39lttype2gt39"></a>İşleç &#39; &lt;operatorname&gt; &#39; türleri için tanımlı değil &#39; &lt;type1&gt; &#39; ve &#39; &lt;type2&gt;&#39;
İki değer döndürme kod öğeleri dahil geçerli olmayan bir işlemi gerçekleştirmek için bir ikili işleç kullanma girişiminde bulundunuz.  
  
 **Hata Kimliği:** BC30452  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İki öğe denetleyin ve uyumlu olduklarından emin olun.
