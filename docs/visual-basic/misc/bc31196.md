# <a name="xml-literals-and-xml-axis-properties-are-not-available-during-this-debugging-session-because-they-are-not-used-in-this-assembly"></a>Bu derlemede kullanılmadığından XML değişmez değerleri ve XML eksen özellikleri bu hata ayıklama oturumu sırasında kullanılabilir değil
XML değişmez değeri veya XML axis özelliği de başvurulmaktaydı **izleme** veya **hemen** penceresi sırasında hangi XML dosyasında hata ayıklama oturumunda [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] özellikler kullanılabilir değildir. Ya da kullanmayan XML'de bir derlemenin böyledir [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] özellikleri veya bir yayın yapıdır.  
  
 **Hata Kimliği:** BC31196  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Hata ayıklama derlemesi derlemenin kullanarak yeni bir hata ayıklama oturumu başlatın.
