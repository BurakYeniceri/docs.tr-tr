# <a name="39continue39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Devam &#39; deyimleri Komut penceresi içinde geçerli değildir
`Continue`deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30780  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Continue` hata ayıklama kodundan deyimi.
