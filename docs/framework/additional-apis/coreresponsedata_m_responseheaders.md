# <a name="coreresponsedatamresponseheaders-field"></a>CoreResponseData.m\_ResponseHeaders alan

`CoreResponseData.m_ResponseHeaders`olan bir <xref:System.Net.WebHeaderCollection> sunucu Yanıtla ilişkili üstbilgi.

## <a name="syntax"></a>Sözdizimi
  
```csharp
public WebHeaderCollection m_ResponseHeaders
```

> [!WARNING]
> Bu API, kodunuzda doğrudan kullanılmak üzere tasarlanmamıştır. Bunun yerine, kullanmanız bir <xref:System.Diagnostics.DiagnosticSource> ağ kod bağlanmanıza. Bkz: [DiagnosticSource Kullanıcı Kılavuzu](https://github.com/dotnet/corefx/blob/master/src/System.Diagnostics.DiagnosticSource/src/DiagnosticSourceUsersGuide.md).
