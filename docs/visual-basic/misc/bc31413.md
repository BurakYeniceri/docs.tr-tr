# <a name="39ltmembergt39-implicitly-defined-for-39lteventnamegt39-cannot-shadow-a-39mustoverride39-method-in-the-base-ltclassgt-39ltclassnamegt39"></a>&#39; &lt;üye&gt;&#39; için örtük olarak tanımlı &#39;&lt; eventname&gt;&#39; gölge olamaz bir &#39; MustOverride &#39; Base yöntemi &lt;sınıfı&gt; &#39;&lt; ClassName&gt;&#39;
Bir yöntem ile bildirilen şekilde belirtilen olay örtük olarak aynı ada sahip bir üye bildirir `MustOverride` değiştiricisi.  
  
 **Hata Kimliği:** BC31413  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `MustOverride` değiştirici yöntemi temel sınıf veya verin özelliği veya yöntemi benzersiz bir ad.
