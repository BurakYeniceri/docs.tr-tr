# <a name="the-remote-server-machine-does-not-exist-or-is-unavailable-visual-basic"></a>Uzak sunucu makine mevcut değil veya kullanılamıyor (Visual Basic)
Bir uzak sunucuya kullanılamıyor veya yok bağlanmaya çalıştığınız. Bu hata, ağ bağlantısının kaybolmasına neden olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ağa bağlayın ve yeniden deneyin.
