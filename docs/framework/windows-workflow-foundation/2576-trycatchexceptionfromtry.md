# <a name="2576---trycatchexceptionfromtry"></a>2576 - TryCatchExceptionFromTry
## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|Kimlik|2576|  
|Anahtar Sözcükler|WFActivities|  
|Düzey|Bilgiler|  
|Kanal|Microsoft Windows uygulaması sunucu-uygulamalar/hata ayıklama|  
  
## <a name="description"></a>Açıklama  
 TryCatch etkinlik bir özel durum yakalandı gösterir.  
  
## <a name="message"></a>İleti  
 TryCatch etkinliği '%1', '%2' türünde bir özel durum yakalandı.
