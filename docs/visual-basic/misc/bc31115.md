# <a name="39addhandler39-declaration-must-end-with-a-matching-39end-addhandler39"></a>&#39; AddHandler &#39; bildirimi eşleşen bir &#39;ile bitmelidir; Son AddHandler &#39;
Bir `AddHandler` bildirimi ile bitmelidir bir `End AddHandler` deyimi.  
  
 **Hata Kimliği:** BC31115  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Denetleyin `AddHandler` bildirimi ile biten bir `End AddHandler` deyimi.
