 <span data-ttu-id="46852-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) değişken aynı olması gerekir [statik](../../csharp/language-reference/keywords/static.md) kullanım başlatmak istediğiniz oluşturucusu olarak.</span><span class="sxs-lookup"><span data-stu-id="46852-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="46852-105">Daha fazla bilgi için bkz: [statik oluşturucular](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span><span class="sxs-lookup"><span data-stu-id="46852-105">For more information, see [Static Constructors](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span></span>

```csharp

   public static readonly int TestInt = 6;  
  
   MyClass()  
   {  
      TestInt = 11;   // CS0198, constructor is not static and readonly field is  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
