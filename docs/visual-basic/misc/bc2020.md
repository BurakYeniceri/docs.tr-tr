# <a name="the-maximum-number-of-errors-has-been-exceeded"></a>En fazla hata sayısı aşıldı
Derleyici en fazla 100 hataları sınırına ulaştı.  
  
 **Hata Kimliği:** BC2020  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Hata ayıklama ve her sırayla adresleme oluşturulan hatalarının sayısını azaltın.
