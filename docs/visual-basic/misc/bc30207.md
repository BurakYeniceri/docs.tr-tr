# <a name="39option-compare39-must-be-followed-by-39text39-or-39binary39"></a>&#39; option Compare &#39; gelmelidir &#39; Metin &#39; ya &#39; İkili &#39;
Bir `Option Compare` yanlış bir ayar veya hiçbir ayar deyimi içeriyor. Ayarları yalnızca izin verilen `Option Compare` olan `Text` ve `Binary`.  
  
 **Hata Kimliği:** BC30207  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ayar belirleyici yanlış olmadığını denetleyin.  
  
2.  Ekleyip ya da `Text` veya `Binary` için `Option Compare` deyimi; Örneğin, `Option Compare Text`.
