# <a name="constructors-cannot-implement-interface-methods"></a>Oluşturucular arabirim yöntemleri uygulayamaz
Kullanma girişiminde bulundunuz `New` geçersiz bir arabirim yöntemini uygulamak için anahtar sözcüğü.  
  
 **Hata Kimliği:** BC30550  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Uygulama arabirimleriyle `Implements` anahtar sözcüğü.
