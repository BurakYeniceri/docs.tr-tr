<span data-ttu-id="bab09-103">Lambda, tür bağımsız değişkeni 'türü' bir temsilci türü olmayan bir ifade ağacına dönüştürülemiyor.</span><span class="sxs-lookup"><span data-stu-id="bab09-103">Cannot convert lambda to an expression tree whose type argument 'type' is not a delegate type.</span></span>  
  
 <span data-ttu-id="bab09-104">Lambda ifadesi bir ifade ağacına dönüştürülürse, ifade ağacına bağımsız değişkeni için bir temsilci türü olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="bab09-104">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="bab09-105">Ayrıca, lambda ifadesi temsilci türüne dönüştürülebilir olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="bab09-105">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>

## <a name="example"></a><span data-ttu-id="bab09-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="bab09-108">Example</span></span>  
 <span data-ttu-id="bab09-109">Aşağıdaki örnek CS0835 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="bab09-109">The following example generates CS0835:</span></span>

```csharp  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.

    }  
}  
```
