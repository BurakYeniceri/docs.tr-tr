# <a name="39ltspecifiergt39-is-not-valid-on-a-withevents-declaration"></a>&#39; &lt;belirticisi&gt;&#39; WithEvents bildiriminde geçerli değil
A `Dim` deyimi içeren bir anahtar sözcük gibi `ReadOnly`, yani birlikte geçersiz `WithEvents` anahtar sözcüğü.  
  
 **Hata Kimliği:** BC30234  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Geçersiz anahtar sözcüğü kaldırmak `Dim` deyimi.
