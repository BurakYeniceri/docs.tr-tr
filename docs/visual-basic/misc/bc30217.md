# <a name="string-constant-expected"></a>Beklenen dize sabiti
Bir karakterin çift tırnak işareti (") dışında burada bir dize sabitine bağlam gerektirir oluşur. Olası bir nedeni, tırnak işaretleri dizeden eksik olmasıdır.  
  
 **Hata Kimliği:** BC30217  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Dize düzgün çift tırnak işaretleri içine emin olun.  
  
2.  Bir dize sabitine sağlayın.
