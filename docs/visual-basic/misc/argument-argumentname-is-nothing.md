# <a name="argument-39ltargumentnamegt39-is-nothing"></a>Bağımsız değişken &#39; &lt;argumentname&gt;&#39; bir şey yok
Bir ifade null bir bağımsız değişken içeriyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  İfade değişkenlerinde yazımını denetleyin. Yanlış yazılmış bir değişken adı örtük olarak sıfır olarak başlatılan sayısal bir değişken oluşturabilirsiniz.  
  
2.  Özellikle yordama diğer yordamlardan bağımsız değişken olarak geçirilen değişkenleri ifadesinde önceki işlemleri denetleyin.
