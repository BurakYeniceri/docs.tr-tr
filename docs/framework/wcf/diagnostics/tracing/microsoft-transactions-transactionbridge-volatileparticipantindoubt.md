# <a name="microsofttransactionstransactionbridgevolatileparticipantindoubt"></a>Microsoft.Transactions.TransactionBridge.VolatileParticipantInDoubt
WS-AT protokolü hizmeti tanınmayan bir geçici Katılımcısı Prepared ya da yeniden yürütme bir ileti aldı. Bir hata bildirir hareketin sonucu şüpheli olduğunu katılımcı döndürüldü.  
  
## <a name="description"></a>Açıklama  
 Yerel işlem yöneticisi unutulursa bir değişken kaydı Prepared ya da yeniden yürütme iletisi aldığında izlenen.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Olası nedenler, geç Volatile katılımcı gelen iletileri araştırın.
