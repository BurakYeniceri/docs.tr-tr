# <a name="compiler-warning-level-1-cs2014"></a>Derleyici Uyarısı (düzey 1) CS2014
Derleyici seçeneği 'eski seçeneği' artık kullanılmıyor, Lütfen bunun yerine 'yeni seçenek' kullanın  
  
 Formun derleyici seçeneği kullanım dışıdır. Bkz: [C# Derleyici Seçenekleri](../../csharp/language-reference/compiler-options/index.md) daha fazla bilgi için.
