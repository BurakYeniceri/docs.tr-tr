# <a name="internal-compiler-error-code-generator-received-malformed-input"></a>Derleyici iç hatası: Kod Oluşturucu alınan hatalı oluşturulmuş giriş
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] Derleyici başarıyla yorumlanamaz dahili olarak yazılmış dosyalarından birini.  
  
 **Hata Kimliği:** BC30011  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Hata durumunda yinelenen yeniden görmek için programı derleyin.  
  
2.  Hata tekrar oluşursa, yeniden [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] derleyici.  
  
3.  Yeniden yüklenmesinden sonra hata devam ederse, koşullar hakkında bilgi toplayın ve Microsoft Ürün Destek Hizmetleri'ne bildirin.
