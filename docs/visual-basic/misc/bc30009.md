# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-implemented-interface-39ltinterfacenamegt39"></a>Derleme &#39;için gereken başvuru; &lt;assemblyname&gt;&#39; uygulanan arabirimi &#39; içeren&lt; InterfaceName&gt;&#39;
Derleme için gerekli referans '\<assemblyname >' uygulanmış arabirimini içeren '\<InterfaceName >'. Projenize ekleyin.  
  
 Arabirim bir dinamik bağlantı kitaplığı (DLL) tanımlanan veya doğrudan projenizde başvurulmuyor derleme. [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] Derleyici arabirimi birden fazla DLL veya derleme tanımlı durumda Karışıklığı önlemek için bir başvuru gerektirir.  
  
 **Hata Kimliği:** BC30009  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Başvurulmayan DLL veya derleme adını, proje başvuruları içerir.
