## <a name="example"></a><span data-ttu-id="a637e-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="a637e-107">Example</span></span>  
 <span data-ttu-id="a637e-108">Aşağıdaki örnek CS3010 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a637e-108">The following example generates CS3010:</span></span>  
  
```  
// CS3010.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public interface I  
{  
    [CLSCompliant(false)]  
    int M();   // CS3010  
}

{  
    public int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
