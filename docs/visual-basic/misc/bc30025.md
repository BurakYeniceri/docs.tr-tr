# <a name="property-missing-39end-property39"></a>Özellik eksik &#39; End özelliği &#39;
Bir `End` deyimi sonlandırır blok eşleşmesi gerekir. A `Property` tanımı değil düzgün bir şekilde sonlandırıldı.  
  
 **Hata Kimliği:** BC30025  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme bir `End Property` özellik tanımını sonuna deyimi.
