# <a name="systemservicemodeltxcompletionstatuscompletedforsetcomplete"></a>System.ServiceModel.TxCompletionStatusCompletedForSetComplete
Belirtilen işlem için belirtilen işlem SetTransactionComplete çağrısı nedeniyle tamamlandı.  
  
## <a name="description"></a>Açıklama  
 Geçerli işlem OperationContext üzerinden tamamlamak kullanıcının açıkça karar verdiyse izlenen.
