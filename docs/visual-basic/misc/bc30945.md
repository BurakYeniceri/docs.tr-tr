# <a name="type-39lttypenamegt39-cannot-implement-interface-39ltinterfacenamegt39-because-it-declares-39lteventsignaturegt39-which-has-a-return-type"></a>Tür &#39; &lt;typename&gt; &#39; arabirimi uygulayamaz &#39; &lt;InterfaceName&gt; &#39; onu bildirdiğinden &#39; &lt;eventsignature&gt; &#39;bir dönüş türüne sahip
Bir sınıf veya yapı bir değer döndüren bir olay bildiren bir arabirim dener.  
  
 Visual Basic dönüş değerleri bildiren olayları şu anda desteklemiyor.  
  
 **Hata Kimliği:** BC30945  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Implements` deyimi sınıf veya yapı tanımı ya da uygulama farklı bir arabirim.
