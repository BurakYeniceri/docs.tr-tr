# <a name="argument-cannot-be-nothing"></a>Bağımsız değişken Nothing olamaz
Bir null değer bir değere sahip olması gereken bir bağımsız değişken belirtilmiş.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Nesne örneği sağlamadan nesneyi kullanmaya çalışmış. Böyle bir durumda kullanmak `New` örneği oluşturmak için anahtar sözcüğü.  
  
-   Değer doğru hesaplanır denetleyin.
