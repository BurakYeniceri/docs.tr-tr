# <a name="properties-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Bir modül özelliklerinde bildirilemez &#39; &lt;belirticisi&gt;&#39;
Bir belirleyici gibi kullanma girişiminde bulundunuz `MustOverride`, `NotOverridable`, veya `Overridable`, bir modül içinde bir özellikte.  
  
 **Hata Kimliği:** BC30503  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirleyici kaldırın.
