# <a name="39for39-loop-control-variable-already-in-use-by-an-enclosing-39for39-loop"></a>&#39;İçin&#39; for döngüsü denetim değişkeni bir kapsayan tarafından zaten kullanılmakta &#39;için&#39; döngü
İç içe geçmiş `For` döngüler tüm kullanmalıdır benzersiz denetim değişkenleri.  
  
 **Hata Kimliği:** BC30975  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Denetim değişken adları biriyle değiştirin.
