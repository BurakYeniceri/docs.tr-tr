# <a name="39handles39-in-modules-must-specify-a-39withevents39-variable-qualified-with-a-single-identifier"></a>&#39;İşleme&#39; modülleri belirtmelisiniz bir &#39;WithEvents&#39; değişkeni tam ile tek bir tanımlayıcı
Bir olay işleyicisi belirtmek için `Handles` deyimleri ile bildirildi bir nesne değişkeni belirtmeniz gerekir `WithEvents` anahtar sözcüğü.  
  
 **Hata Kimliği:** BC31418  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanım `WithEvents` kullanılacak değişkenleri bildirmek için değiştirici `Handles` deyimi.
