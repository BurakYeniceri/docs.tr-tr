# <a name="39lttype1gt39-cannot-override-lttype2gt-because-it-is-not-declared-39overridable39"></a>&#39;&lt;type1&gt; &#39; geçersiz kılamaz &lt;type2&gt; değil olarak bildirildiğinden &#39;Overridable&#39;
Türetilmiş bir sınıf üyesi ile işaretli olmayan bir taban sınıf üyesi geçersiz kılmaları `Overridable` değiştiricisi.  
  
 **Hata Kimliği:** BC31086  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ekleme `Overridable` geçersiz kılınan üye temel sınıf olarak değiştiricisi.  
  
2.  Kullanım `Shadows` temel sınıfı öğesinde gölge anahtar.
