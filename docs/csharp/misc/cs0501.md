# <a name="compiler-error-cs0501"></a>Derleyici Hatası CS0501
Özet, extern veya kısmi işaretlenmediğinden 'üye işlevi' gövde bildirmelidir  
  
 Soyut yöntemler uygulamaları olması gerekir.  
  
 Aşağıdaki örnek CS0501 oluşturur:  
  
```csharp  
// CS0501.cs  
// compile with: /target:library  
public class clx  
{  
   public void f();   // CS0501 declared but not defined  
   public void g() {}   // OK  
}  
```
