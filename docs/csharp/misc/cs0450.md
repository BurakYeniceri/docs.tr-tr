# <a name="compiler-error-cs0450"></a><span data-ttu-id="eb930-102">Derleyici Hatası CS0450</span><span class="sxs-lookup"><span data-stu-id="eb930-102">Compiler Error CS0450</span></span>
<span data-ttu-id="eb930-103">'Türü, parametre adı': hem bir kısıtlama sınıfı hem de 'class' veya 'struct' kısıtlaması belirtilemez</span><span class="sxs-lookup"><span data-stu-id="eb930-103">'Type Parameter Name': cannot specify both a constraint class and the 'class' or 'struct' constraint</span></span>  
  
 <span data-ttu-id="eb930-104">Tür parametresi yapı türü kısıtlaması tarafından kısıtlanan, yapı ve sınıf birbirini dışlayan kategorileri olduğundan Ayrıca belirli bir sınıf türü tarafından kısıtlı için mantıksal olarak çelişkili olur.</span><span class="sxs-lookup"><span data-stu-id="eb930-104">If a type parameter is constrained by the struct type constraint, it is logically contradictory for it to also be constrained by a specific class type, since struct and class are mutually exclusive categories.</span></span> <span data-ttu-id="eb930-105">Tür parametresi bir belirli bir sınıf türü kısıtlaması tarafından kısıtlanan, sınıf türü kısıtlaması tarafından kısıtlanan tanımı tarafından olur ve bu nedenle sınıf türü kısıtlaması belirtme gereksizdir.</span><span class="sxs-lookup"><span data-stu-id="eb930-105">If a type parameter is constrained by a specific class type constraint, then it is by definition constrained by the class type constraint, and so specifying the class type constraint is redundant.</span></span>  
  
## <a name="example"></a><span data-ttu-id="eb930-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="eb930-106">Example</span></span>  
  
```  
// CS0450.cs  
// compile with: /t:library  
public class GenericsErrors   
{  
    public class B { }  
    public class G3<T> where T : struct, B { } // CS0450  
// To resolve, use the following line instead:  
// public class G3<T> where T : B { }  
}  
```
