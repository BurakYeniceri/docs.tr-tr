<span data-ttu-id="72bc0-103">'Class' veya 'struct' kısıtlaması diğer kısıtlamalardan önce gelmelidir</span><span class="sxs-lookup"><span data-stu-id="72bc0-103">The 'class' or 'struct' constraint must come before any other constraints</span></span>  
  
 <span data-ttu-id="72bc0-104">Genel tür veya yöntem tür parametresi kısıtlamalar belirli bir sırada olmalıdır: `class` veya `struct` ilk, mevcutsa, daha sonra arabirim kısıtlamalar ve son olarak Oluşturucusu kısıtlamalar olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="72bc0-104">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="72bc0-105">Bu hatanın nedeni `class` veya `struct` ilk görünmeyen kısıtlaması.</span><span class="sxs-lookup"><span data-stu-id="72bc0-105">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="72bc0-106">Bu hatayı gidermek için kısıtlama yan tümceleri yeniden sıralar.</span><span class="sxs-lookup"><span data-stu-id="72bc0-106">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="72bc0-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="72bc0-107">Example</span></span>  
 <span data-ttu-id="72bc0-108">Aşağıdaki örnek CS0449 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="72bc0-108">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4   
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```
