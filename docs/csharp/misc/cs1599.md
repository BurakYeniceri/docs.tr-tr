# <a name="compiler-error-cs1599"></a><span data-ttu-id="5247c-102">Derleyici Hatası CS1599</span><span class="sxs-lookup"><span data-stu-id="5247c-102">Compiler Error CS1599</span></span>
<span data-ttu-id="5247c-103">Yöntem veya temsilci türü 'type' döndürülemiyor</span><span class="sxs-lookup"><span data-stu-id="5247c-103">Method or delegate cannot return type 'type'</span></span>  
  
 <span data-ttu-id="5247c-104">.NET Framework Sınıf Kitaplığı'nda, örneğin, bazı türleri <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> ve <xref:System.ArgIterator> çünkü güvensiz işlemlerini gerçekleştirmek için büyük olasılıkla kullanılabilir dönüş türleri olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="5247c-104">Some types in the .NET Framework class library, for example, <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> and <xref:System.ArgIterator> cannot be used as return types because they can potentially be used to perform unsafe operations.</span></span>  
  
 <span data-ttu-id="5247c-105">Aşağıdaki örnek CS1599 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5247c-105">The following sample generates CS1599:</span></span>  
  
```csharp  
// CS1599.cs  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public TypedReference Test1()   // CS1599  
   {  
      return null;  
   }  
  
   public ArgIterator Test2()   // CS1599  
   {  
      return null;  
   }  
}  
```
