# <a name="systemservicemodelchannelspeerflooderreceivemessagequotaexceeded"></a>System.ServiceModel.Channels.PeerFlooderReceiveMessageQuotaExceeded
İletilerin gelen alma hızı çok yüksektir.  
  
## <a name="description"></a>Açıklama  
 Bu izleme, gelen iletileri işlemek çalışılırken zaman oluşur. Bu komşusu için belirlediğiniz kotayı aştığı için bir ileti için belirli bir komşu iletilemedi. Bu durum, bu komşusu için biriktirme listesi iletileri temizlemek bir yanıt vermeyen komşu başarısız oluşur.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Mesh içinde gönderilen iletileri hızını azaltın.
