# <a name="compiler-warning-level-3-cs1717"></a><span data-ttu-id="65152-102">Derleyici Uyarısı (Düzey 3) CS1717</span><span class="sxs-lookup"><span data-stu-id="65152-102">Compiler Warning (level 3) CS1717</span></span>
<span data-ttu-id="65152-103">Atama aynı değişkene yapıldı; başka bir şey atamak mu demek istediniz?</span><span class="sxs-lookup"><span data-stu-id="65152-103">Assignment made to same variable; did you mean to assign something else?</span></span>  
  
 <span data-ttu-id="65152-104">Bir değişkenin kendisine gibi atadığınızda, bu uyarıyı oluşur `a = a`.</span><span class="sxs-lookup"><span data-stu-id="65152-104">This warning occurs when you assign a variable to itself, such as `a = a`.</span></span>  
  
 <span data-ttu-id="65152-105">Birçok yaygın hatalar, bu uyarıyı oluşturabilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="65152-105">Several common mistakes can generate this warning:</span></span>  
  
-   <span data-ttu-id="65152-106">Yazma `a = a` koşulu olarak bir **varsa** deyimi, gibi `if (a = a)`.</span><span class="sxs-lookup"><span data-stu-id="65152-106">Writing `a = a` as the condition of an **if** statement, such as `if (a = a)`.</span></span> <span data-ttu-id="65152-107">Büyük olasılıkla söyleyin yönelik `if (a == a)`, olduğu her zaman true, daha fazla, bu yazabilmesi için az ama öz olarak `if (true)`.</span><span class="sxs-lookup"><span data-stu-id="65152-107">You probably meant to say `if (a == a)`, which is always true, so you could write this more concisely as `if (true)`.</span></span>  
  
-   <span data-ttu-id="65152-108">Yanlış yazmanız.</span><span class="sxs-lookup"><span data-stu-id="65152-108">Mistyping.</span></span> <span data-ttu-id="65152-109">Büyük olasılıkla söyleyin yönelik `a = b`.</span><span class="sxs-lookup"><span data-stu-id="65152-109">You probably meant to say `a = b`.</span></span>  
  
-   <span data-ttu-id="65152-110">Parametre sahip olduğu aynı ad alanı olarak oluşturucuda kullanmıyorsa **bu** anahtar sözcüğü: büyük olasılıkla söyleyin yönelik `this.a = a`.</span><span class="sxs-lookup"><span data-stu-id="65152-110">In a constructor where the parameter has the same name as the field, not using the **this** keyword: you probably meant to say `this.a = a`.</span></span>
