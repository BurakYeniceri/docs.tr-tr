# <a name="label-39ltlabelnamegt39-is-not-defined"></a>Etiket &#39; &lt;etiketadı&gt; &#39; tanımlı değil
Bir satır etiket veya satır numarası başvurulan ancak başvuru kapsamı içinde tanımlı değil. Etiket başvurusu içeriyor yordam içinde olmalıdır.  
  
 **Hata Kimliği:** BC30132  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Satır etiket ve başvuru yordamın aynısını; bu nedenle kodu yeniden yapılandırabilirsiniz.
