# <a name="39ltdeclaration1gt39-cannot-override-39ltdeclaration2gt39-because-it-is-declared-39notoverridable39"></a>&#39; &lt;declaration1&gt;&#39; olamaz geçersiz kılma &#39;&lt; declaration2&gt;&#39; bunu bildirildiğinden &#39; NotOverridable &#39;
Aynı ada sahip bir devralınan öğe geçersiz kılmak bir yordam veya özellik bildirimi çalışır, ancak devralınan öğesi olarak belirtilen `NotOverridable`.  
  
 **Hata Kimliği:** BC30267  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `NotOverridable` devralınan öğe veya Kaldır geçersiz kılma bildirimi bildirimi anahtar sözcük.
