# <a name="39sub-new39-cannot-implement-interface-members"></a>&#39; Yeni alt &#39; arabirim üyeleri uygulayamaz
`Sub New`bir kurucusu olan ve üyeleri uygulayamaz.  
  
 **Hata Kimliği:** BC31042  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Implements` gelen deyimleri `Sub New` yordamlar.
