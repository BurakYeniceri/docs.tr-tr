# <a name="39end-set39-must-be-the-first-statement-on-a-line"></a>&#39; Son kümesi &#39; satırındaki ilk deyim olmalıdır
Bir `End Set` deyimi izleyen iki nokta üst üste (:) deyimi ayırıcı. `End Set`Kaynak satırında yegane deyim olmalıdır.  
  
 **Hata Kimliği:** BC32034  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Birden çok deyime farklı satırlarına ayırın.
