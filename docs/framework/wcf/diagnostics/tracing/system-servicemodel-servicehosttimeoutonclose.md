# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose
System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Açıklama  
 Bir ServiceHost kapatma işlemi zaman aşımına. Bu süre sonuyla bazı kanallar düzgün istemci tarafından kapatılan değil ve kesilmiş olabilir gösterir. Kapatma zaman aşımını ayarlamak için varsayılan saat değiştirin.
