# <a name="members-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Modül içindeki üyeler bildirilemez &#39; &lt;belirticisi&gt;&#39;
İçinde bir üyede geçerli olmayan bir belirleyici kullanılan bir `Module` deyimi. Modüller hiçbir zaman oluşturulabilir, devralma desteklemez ve arabirimleri uygulayamaz.  
  
 **Hata Kimliği:** BC30436  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirleyici kaldırın.
