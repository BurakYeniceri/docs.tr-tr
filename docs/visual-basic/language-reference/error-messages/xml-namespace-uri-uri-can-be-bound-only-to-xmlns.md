# <a name="xml-namespace-uri-39httpwwww3orgxml1998namespace39-can-be-bound-only-to-39xmlns39"></a>XML ad alanı URI'si &#39; http://www.w3.org/XML/1998/namespace &#39; yalnızca ilişkili &#39; xmlns &#39;
URI http://www.w3.org/XML/1998/namespace bir XML ad alanı bildirimi kullanılır. Bu URI, ayrılmış bir ad alanıdır ve bir XML ad alanı bildirimi dahil edilemez.  
  
 **Hata Kimliği:** BC31183  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML ad alanı bildirimini kaldırın veya URI http://www.w3.org/XML/1998/namespace geçerli ad alanı URI ile değiştirin.
