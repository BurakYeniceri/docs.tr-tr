 <span data-ttu-id="35969-104">A *kısıtlaması* türde bir parametre genel türü oluşturulduğunda, bu tür parametresine geçirilen her tür bağımsız değişkeni gereksinime uygular.</span><span class="sxs-lookup"><span data-stu-id="35969-104">A *constraint* on a type parameter imposes a requirement on any type argument passed to that type parameter when the generic type is created.</span></span> <span data-ttu-id="35969-105">`New` Kısıtlaması, tür bağımsız değişkeni oluşturma kodu erişebilirsiniz parametresiz bir kurucusu kullanıma gerekir belirtir.</span><span class="sxs-lookup"><span data-stu-id="35969-105">The `New` constraint specifies that the type argument must expose a parameterless constructor that the creating code can access.</span></span> <span data-ttu-id="35969-106">Nelere izin budur bir `New` bu türünün bir örneğini oluşturmak için bir bildirim deyimindeki yan tümcesi.</span><span class="sxs-lookup"><span data-stu-id="35969-106">This is what allows a `New` clause in a declaration statement to create an instance of that type.</span></span>
