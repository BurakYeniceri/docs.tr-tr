# <a name="windows-forms-controls-used-to-list-options"></a>Seçenekleri Listelemede Kullanılan Windows Forms Denetimleri
Aralarından seçim yapabileceğiniz seçeneklerin bir listesini kullanıcılara sağlamak istiyorsanız, bir Windows formu denetimlerini çeşitli ekleyebilirsiniz. Bağlı olarak ne kadar istediğiniz kullanıcılarınızın kısıtlamak giriş, ekleyebileceğiniz bir <xref:System.Windows.Forms.ListBox> denetimi, bir <xref:System.Windows.Forms.ComboBox> denetimi veya <xref:System.Windows.Forms.CheckedListBox> denetim. Gereksinimlerinize en iyi hangi denetim en belirlemek için aşağıdaki bağlantıları kullanın.

 Gereksinimleri ve kısıtlamaları, Windows formun bağlı olarak, uygun bir listeye bağlı denetim önerir.  
  
 [Nasıl yapılır: Bir Windows Forms ComboBox, ListBox veya CheckedListBox Denetiminde Belirli Öğelere Erişme](../../../../docs/framework/winforms/controls/access-specific-items-in-a-wf-combobox-listbox-or-checkedlistbox.md)
