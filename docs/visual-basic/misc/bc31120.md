# <a name="39end-raiseevent39-must-be-the-first-statement-on-a-line"></a>&#39;End RaiseEvent&#39; satırdaki ilk deyim olmalıdır
Bir `End RaiseEvent` deyimi izleyen iki nokta üst üste (:) deyimi ayırıcı. `End RaiseEvent` Kaynak satırında yegane deyim olmalıdır.  
  
 **Hata Kimliği:** BC31120  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Birden çok deyime farklı satırlarına ayırın.
