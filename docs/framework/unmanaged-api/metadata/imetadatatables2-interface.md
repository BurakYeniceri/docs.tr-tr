# <a name="imetadatatables2-interface"></a>IMetaDataTables2 Arabirimi
Genişletir [Imetadatatables](../../../../docs/framework/unmanaged-api/metadata/imetadatatables-interface.md) meta veri akışlarını ile çalışmak için yöntemler eklenecek.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[GetMetaDataStorage Yöntemi](../../../../docs/framework/unmanaged-api/metadata/imetadatatables2-getmetadatastorage-method.md)|Belirtilen bölüm içinde depolanan meta veriler içeriğini ve boyutunu alır.|  
|[GetMetaDataStreamInfo Yöntemi](../../../../docs/framework/unmanaged-api/metadata/imetadatatables2-getmetadatastreaminfo-method.md)|Adı, boyutu ve belirtilen dizindeki meta veri akışı içeriğini alır.|
