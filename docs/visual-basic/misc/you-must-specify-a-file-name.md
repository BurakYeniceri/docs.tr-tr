# <a name="you-must-specify-a-file-name"></a>Bir dosya adı belirtmeniz gerekir
Taşıma veya bir dosya adı sağladığını gerektiren bir dosya kopyalama gibi bir işlem denendi.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gerekli dosya adı sağlayın.  
  
-   Yol bir dizin ayırıcı karakteri ile bitemez emin olun (\\).
