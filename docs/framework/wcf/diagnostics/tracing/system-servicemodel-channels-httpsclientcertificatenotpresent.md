# <a name="systemservicemodelchannelshttpsclientcertificatenotpresent"></a>System.ServiceModel.Channels.HttpsClientCertificateNotPresent
İstemci sertifikası gereklidir. Hiçbir sertifika isteğinde bulundu.  
  
## <a name="description"></a>Açıklama  
 Bu izleme, HTTPS dinleyicisi bir istemci sertifikası ile ilişkilendirilmemiş bir HTTPS isteği aldığını gösterir. Dinleyici tüm HTTPS isteklerinde istemci sertifikaları gerektirmek için yapılandırıldıktan sonra dinleyiciyi istemcinin orijinallik sertifikası doğrulanamadı.
