# <a name="methods-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Bir modüldeki yöntemleri bildirilemez &#39; &lt;belirticisi&gt;&#39;
Yöntemi içinde geçerli olmayan bir belirleyici kullanılan bir `Module` deyimi. Modüller hiçbir zaman oluşturulabilir, devralma desteklemez ve arabirimleri uygulayamaz.  
  
 **Hata Kimliği:** BC30433  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirleyici kaldırın.
