# <a name="compiler-error-cs0230"></a>Derleyici Hatası CS0230
Foreach deyimi içinde türü ve tanımlayıcı her ikisi de gereklidir  
  
 A [foreach](../../csharp/language-reference/keywords/foreach-in.md) deyimi biçimi hatalı.  
  
 Aşağıdaki örnek CS0230 oluşturur:  
  
```csharp  
// CS0230.cs  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[3] {1,2,3};  
  
      foreach (int in myarray)   // CS0230  
      // try the following line instead  
      // foreach (int x in myarray)  
      {  
         Console.WriteLine(x);  
      }  
   }  
}  
```
