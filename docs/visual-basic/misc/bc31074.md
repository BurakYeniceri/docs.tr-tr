# <a name="delegate-constructor-for-type-39lttypenamegt39-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>Türü &#39;Oluşturucusu temsilci; &lt;typename&gt;&#39; Temsilciler, uygulama için gerekli bulunamıyor
Türü içeren derlemenin bozuk ya da geçersiz olabilir.  
  
 **Hata Kimliği:** BC31074  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bu türü içeren derlemenin geçerli olduğunu doğrulayın.  
  
2.  Hata durumunda yinelenen yeniden görmek için programı derleyin.  
  
3.  Hata devam ederse, çalışmanızı kaydedin ve yeniden [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].
