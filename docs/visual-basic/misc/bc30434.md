# <a name="events-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Modül içindeki olaylar bildirilemez &#39; &lt;belirticisi&gt;&#39;
İçinde bir olayın üzerinde geçersiz bir belirleyici kullanılan bir `Module` deyimi. Modüller hiçbir zaman oluşturulabilir, devralma desteklemez ve arabirimleri uygulayamaz.  
  
 **Hata Kimliği:** BC30434  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirleyici kaldırın.
