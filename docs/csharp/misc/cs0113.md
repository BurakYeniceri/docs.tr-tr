# <a name="compiler-error-cs0113"></a><span data-ttu-id="1333f-102">Derleyici Hatası CS0113</span><span class="sxs-lookup"><span data-stu-id="1333f-102">Compiler Error CS0113</span></span>
<span data-ttu-id="1333f-103">Geçersiz kılma yeni veya sanal olarak işaretlenemez üyesi 'function' işaretli</span><span class="sxs-lookup"><span data-stu-id="1333f-103">A member 'function' marked as override cannot be marked as new or virtual</span></span>  
  
 <span data-ttu-id="1333f-104">Bir yöntemle işaretlemek için birbirini dışlayan [yeni](../../csharp/language-reference/keywords/new.md) ve [geçersiz kılma](../../csharp/language-reference/keywords/override.md) anahtar sözcükler.</span><span class="sxs-lookup"><span data-stu-id="1333f-104">It is mutually exclusive to mark a method with the [new](../../csharp/language-reference/keywords/new.md) and [override](../../csharp/language-reference/keywords/override.md) keywords.</span></span>  
  
 <span data-ttu-id="1333f-105">Aşağıdaki örnek CS0113 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="1333f-105">The following sample generates CS0113:</span></span>  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
