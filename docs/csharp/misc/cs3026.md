# <a name="compiler-warning-level-1-cs3026"></a>Derleyici Uyarısı (düzey 1) CS3026
CLS uyumlu alan 'field' geçici olamaz  
  
 Geçici bir değişken CLS uyumlu olmaması gerekir.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS3026 oluşturur.  
  
```csharp  
// CS3026.cs  
[assembly:System.CLSCompliant(true)]  
public class Test  
{  
public volatile int v0 =0;   // CS3026  
      // To resolve remove the CLS-CComplient attribute.  
public static void Main() { }  
}  
```
