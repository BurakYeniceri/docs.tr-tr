# <a name="icordebugchainenum-interface1"></a>Icordebugchainenum Interface1
Icordebugenum yöntemlerini uygular ve Icordebugchain diziler numaralandırır.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[Next Yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugchainenum-next-method.md)|Belirtilen sayıda alır `ICorDebugChain` geçerli konumdan başlayarak numaralandırması örneklerden.|
