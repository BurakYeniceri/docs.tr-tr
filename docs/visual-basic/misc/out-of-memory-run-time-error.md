# <a name="out-of-memory-visual-basic-run-time-error"></a>Bellek yetersiz (Visual Basic çalışma zamanı hatası)
Daha fazla bellek bulunandan gerekli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Gereksiz uygulamaları kapatın, belgeler veya açık kaynak dosyaları.  
  
2.  Büyük modülleri ya da yordamlar küçük parçalara bölün.  
  
3.  Sonlandır-ve-bellekte kal programları ortadan kaldırır.  
  
4.  Gereksiz aygıt sürücülerini ortadan kaldırır.  
  
5.  Genel değişkenler sayısını azaltın.
