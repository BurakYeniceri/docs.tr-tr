# <a name="compiler-error-cs1940"></a><span data-ttu-id="74470-102">Derleyici Hatası CS1940</span><span class="sxs-lookup"><span data-stu-id="74470-102">Compiler Error CS1940</span></span>
<span data-ttu-id="74470-103">Sorgu kalıbının çoklu uygulamaları 'type' kaynak türü için bulunamadı.</span><span class="sxs-lookup"><span data-stu-id="74470-103">Multiple implementations of the query pattern were found for source type 'type'.</span></span> <span data-ttu-id="74470-104">'Method' çağrısı belirsiz.</span><span class="sxs-lookup"><span data-stu-id="74470-104">Ambiguous call to 'method'.</span></span>  
  
 <span data-ttu-id="74470-105">Bu hata, bir sorgu yöntemi, birden fazla uygulaması tanımlanır ve derleyici hangisinin sorgunun kullanmak en iyisidir belirsizliğinin olamaz oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="74470-105">This error is generated when multiple implementations of a query method are defined and the compiler cannot disambiguate which one is best to use for the query.</span></span> <span data-ttu-id="74470-106">Aşağıdaki örnekte, iki sürümü de `Select` çünkü her ikisi de kabul aynı imzaya sahip `int` giriş parametresi olarak ve `int` dönüş değeri olarak.</span><span class="sxs-lookup"><span data-stu-id="74470-106">In the following example, both versions of `Select` have the same signature, because they both accept one `int` as an input parameter and have `int` as a return value.</span></span>

## <a name="example"></a><span data-ttu-id="74470-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="74470-109">Example</span></span>  
 <span data-ttu-id="74470-110">Aşağıdaki kod CS1940 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="74470-110">The following code generates CS1940:</span></span>  
  
```csharp  
// cs1940.cs  
using System; //must include explicitly for types defined in 3.5  
class Test  
{  
    public delegate int Dele(int x);  
    int num = 0;  
    public int Select(Func<int, int> d)  
    {  
        return d(this.num);  
    }  
    public int Select(Dele d) // CS1940  
    {  
        return d(this.num) + 1;  
    }  
    public static void Main()  
    {  
        var q = from x in new Test()  
        select x;  
    }  
}  
```
