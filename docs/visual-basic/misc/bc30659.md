# <a name="property-or-field-39ltnamegt39-does-not-have-a-valid-attribute-type"></a>Özellik veya alan &#39; &lt;adı&gt; &#39; geçerli öznitelik türü yok
Öznitelik diğer uygun oluşturucular olsanız bile, geçerli bir öznitelik kabul etmeyen türü, bir öznitelik oluşturucunun kullanıyor olabilir.  
  
 **Hata Kimliği:** BC30659  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bu öznitelik'ın kullanımını denetleyin.
