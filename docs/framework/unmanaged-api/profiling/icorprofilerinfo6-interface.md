# <a name="icorprofilerinfo6-interface"></a>ICorProfilerInfo6 arabirimi
[.NET Framework 4.6 ve sonraki sürümlerinde desteklenen]  
  
 Öğesinin bir alt [Icorprofilerınfo5](../../../../docs/framework/unmanaged-api/profiling/icorprofilerinfo5-interface.md) verilen NGen modülü ve satır içi belirli bir yöntemin içinde tanımlanan tüm yöntemleri için bir numaralandırıcı sağlar.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[ICorProfilerInfo6::EnumNgenModuleMethodsInliningThisMethod Yöntemi](../../../../docs/framework/unmanaged-api/profiling/icorprofilerinfo6-enumngenmodulemethodsinliningthismethod-method.md)|Belirli bir NGen modüle ait ve belirli bir yöntemin gövdesine içermesinden tüm yöntemleri için bir numaralandırıcı döndürür.|
