# <a name="microsofttransactionstransactionbridgeregisterparticipantfailure"></a>Microsoft.Transactions.TransactionBridge.RegisterParticipantFailure
WS-AT protokolü hizmeti katılımcı Denetim Protokolü için kaydedilemedi.  
  
## <a name="description"></a>Açıklama  
 Geçersiz bir kayıt isteğini MSDTC algılarsa, izlenen. Bu, birden çok tamamlama kayıt isteklerini ve iç hatalar oluşabilir.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Tamamlama kaydolun için birden çok kez çalışmayın.  Ayrıca, herhangi bir işlem yapılabilir öğe olup olmadığını belirlemek için izleme iletisi içinde durum dizesi inceleyin.
