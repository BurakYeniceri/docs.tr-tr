# <a name="39sub-main39-is-declared-more-than-once-in-39ltnamespacegt39-ltmessagegt"></a>&#39; Sub Main &#39; birden çok kez söz &#39;bildirilir; &lt;ad alanı&gt;&#39;: &lt;iletisi&gt;
`Sub Main`yalnızca bir kez bir ad alanı içinde bildirilebilir.  
  
 **Hata Kimliği:** BC30738  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yalnızca tek bir olduğundan emin olun `Sub Main` projenizdeki yordamı.
