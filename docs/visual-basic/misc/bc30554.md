# <a name="39ltnamespacenamegt39-is-ambiguous"></a>&#39;&lt;NamespaceName&gt; &#39; belirsiz
Belirsiz ve bu nedenle çakışan bir adı başka bir adla sağlanmıştır. Visual Basic derleyici hiçbir çakışma çözümü kuralları yok; adlarını kendiniz ayırt etmek gerekir.  
  
 **Hata Kimliği:** BC30554  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Adı tam olarak nitelemek.
