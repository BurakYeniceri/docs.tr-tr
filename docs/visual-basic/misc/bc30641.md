# <a name="39byval39-and-39byref39-cannot-be-combined"></a>&#39; ByVal &#39; ve &#39; ByRef &#39; birleştirilemez
`ByVal` Ve `ByRef` anahtar sözcükleri karşılıklı olarak birbirini dışlar.  
  
 **Hata Kimliği:** BC30641  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirtin `ByVal` veya `ByRef`, ancak ikisini birden değil.
