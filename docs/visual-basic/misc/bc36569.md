# <a name="argument-not-specified-for-parameter-39ltparameternamegt39"></a>Bağımsız değişken parametre &#39;belirtilmemiş; &lt;parametername&gt;&#39;
Gerekli parametre için bir bağımsız değişken sağlanmadı.  
  
 **Hata Kimliği:** BC36569  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirtilen parametre için bir bağımsız değişken sağlayın.
