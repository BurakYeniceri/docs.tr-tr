# <a name="icordebugvaluebreakpoint-interface1"></a>Icordebugvaluebreakpoint Interface1
Icordebugbreakpoint arabirimi belirli değerleri erişim sağlamak için genişletir.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[GetValue Yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugvaluebreakpoint-getvalue-method.md)|Icordebugvalue nesneye bağlı kesme noktası belirleyerek nesnenin değerini temsil eden bir arabirim işaretçisi alır.|  
  
## <a name="remarks"></a>Açıklamalar  
  
> [!NOTE]
>  Bu arabirim, makineler arası veya çapraz işlem uzaktan çağrılan desteklemez.
