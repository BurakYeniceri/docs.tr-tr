# <a name="creating-xml-in-visual-basic"></a>Visual Basic'de XML Oluşturma
Visual Basic kullanmanıza olanak sağlayan *XML değişmez değerleri* kodunuzda doğrudan. XML değişmez sözdizimini temsil eden [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] nesneleri ve XML 1.0 sözdizimine benzer. Bu, kodunuzu son XML aynı yapısını olduğundan XML öğeleri, belgeler ve parçaları program aracılığıyla oluşturma kolaylaştırır.
