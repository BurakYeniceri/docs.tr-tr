# <a name="conversions-from-39lttypename1gt39-to-39lttypename2gt39-must-be-explicit"></a>Gelen dönüşümleri &#39; &lt;typename1&gt;&#39; çok &#39;&lt; TypeName2&gt;&#39; açık olması gerekir
Bu dönüştürme örtük olarak elde edilemiyor.  
  
 **Hata Kimliği:** BC30664  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanım `CType`, veya gibi belirli dönüştürme birini işlevleri `CInt` veya `CDec`.
