# <a name="xml-processing-instruction-name-39name39-is-not-valid"></a>XML işleme talimatı adı &#39;adı&#39; geçerli değil
XML işleme talimatı için sağlanan ad geçersiz.  
  
 **Hata Kimliği:** BC31174  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML işleme talimatı adına hataları düzeltin veya kaldırın.
