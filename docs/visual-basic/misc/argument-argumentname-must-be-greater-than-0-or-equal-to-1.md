# <a name="argument-39ltargumentnamegt39-must-be-greater-than-0-or-equal-to--1"></a>Bağımsız değişken &#39; &lt;argumentname&gt; &#39; -1 veya 0'dan büyük olmalıdır
Bir değişkenin değeri 0'dan küçük, ancak -1 olarak eşit değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bağımsız değişkeninin değeri geçerli aralıkta olduğundan emin olun.  
  
2.  Bağımsız değişkeni bir hesaplama sağlanırsa, hesaplama işlemleri değerlerini denetleyin.
