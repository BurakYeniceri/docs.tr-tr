# <a name="iceegengetstring-method"></a>ICeeGen::GetString Metodu
Belirtilen göreli sanal adresinde depolanan dizesini alır.  
  
 Bu yöntem artık kullanılmıyor ve kullanılmamalıdır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT GetString (  
    [in]  ULONG      RVA,   
    [out] LPWSTR     *lpString  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `RVA`  
 [in] Döndürülecek dizesinin göreli sanal adres.  
  
 `lpString`  
 [out] Döndürülen dize.
