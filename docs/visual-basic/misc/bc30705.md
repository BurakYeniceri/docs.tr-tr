# <a name="referenced-object-39ltobjectnamegt39-has-a-value-of-39nothing39"></a>Başvurulan nesne &#39; &lt;objectname&gt; &#39; değerine sahip &#39;hiçbir şey&#39;
Başvurulan nesne ayarlanmamışsa ya da değerine ayarlayın `Nothing`.  
  
 **Hata Kimliği:** BC30705  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Nesnenin adını doğru yazdığınızdan ve nesne ayarlandı doğrulayın.
