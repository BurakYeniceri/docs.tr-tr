# <a name="compiler-error-cs1514"></a>Derleyici Hatası CS1514
{bekleniyor  
  
 Derleyici bir parantezinden beklenen (`{`), bulunamadı.  
  
 Aşağıdaki örnek CS1514 oluşturur:  
  
```csharp  
// CS1514.cs  
namespace x  
// CS1514, no open curly brace  
```
