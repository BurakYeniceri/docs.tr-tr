# <a name="compiler-error-cs0466"></a><span data-ttu-id="68113-102">Derleyici Hatası CS0466</span><span class="sxs-lookup"><span data-stu-id="68113-102">Compiler Error CS0466</span></span>
<span data-ttu-id="68113-103">'method2' öğesinde olmadığından 'method1' params parametresi olmamalıdır</span><span class="sxs-lookup"><span data-stu-id="68113-103">'method1' should not have a params parameter since 'method2' does not</span></span>  
  
 <span data-ttu-id="68113-104">Kullanamazsınız `params` uygulanan arabirimi kullanmıyorsa bir sınıf üyesi parametresi.</span><span class="sxs-lookup"><span data-stu-id="68113-104">You cannot use `params` parameter on a class member if the implemented interface doesn't use it.</span></span>  
  
## <a name="example"></a><span data-ttu-id="68113-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="68113-105">Example</span></span>  
 <span data-ttu-id="68113-106">Aşağıdaki örnek CS0466 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="68113-106">The following sample generates CS0466.</span></span>  
  
```  
// CS0466.cs  
interface I  
{  
   void F1(params int[] a);  
   void F2(int[] a);  
}  
  
class C : I  
{  
   void I.F1(params int[] a) {}  
   void I.F2(params int[] a) {}   // CS0466  
   void I.F2(int[] a) {}   // OK  
  
   public static void Main()  
   {  
      I i = (I) new C();  
  
      i.F1(new int[] {1, 2} );  
      i.F2(new int[] {1, 2} );  
   }  
}  
```
