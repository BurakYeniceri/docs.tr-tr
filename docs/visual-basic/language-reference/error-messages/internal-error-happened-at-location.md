# <a name="internal-error-happened-at-ltlocationgt"></a>İç hata oluştu &lt;konumu&gt;
Bir iç hata oluştu. Gerçekleştiği satır hata iletisinde yer alır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Emin olun bu hatayı yok üretilen tarafından `Error` deyimi veya `Raise` yöntemi; bunu olduğunu değil, ileti görünen koşullarını bildirmek için Microsoft Ürün Destek Hizmetleri'ne başvurun.
