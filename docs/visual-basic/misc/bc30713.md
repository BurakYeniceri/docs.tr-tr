# <a name="39exit39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Çıkış&#39; ifadeleri geçerli değildir komut penceresi
`Exit` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30713  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Exit` deyimleri hata ayıklama kodundan.
