# <a name="compiler-error-cs1527"></a><span data-ttu-id="ef9d2-102">Derleyici Hatası CS1527</span><span class="sxs-lookup"><span data-stu-id="ef9d2-102">Compiler Error CS1527</span></span>
<span data-ttu-id="ef9d2-103">Bir ad alanında tanımlanan öğeler açıkça özel olarak bildirilemez korunan, iç korumalı veya private protected.</span><span class="sxs-lookup"><span data-stu-id="ef9d2-103">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span> 
  
 <span data-ttu-id="ef9d2-104">Bir ad alanında tür bildirimleri ya da olabilir [genel](../../csharp/language-reference/keywords/public.md) veya [iç](../../csharp/language-reference/keywords/internal.md) erişim.</span><span class="sxs-lookup"><span data-stu-id="ef9d2-104">Type declarations in a namespace can have either [public](../../csharp/language-reference/keywords/public.md) or [internal](../../csharp/language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="ef9d2-105">Hiçbir erişilebilirlik belirtilmişse **iç** varsayılandır.</span><span class="sxs-lookup"><span data-stu-id="ef9d2-105">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="ef9d2-106">Aşağıdaki örnek, CS1527 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ef9d2-106">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="ef9d2-107">Aşağıdaki örnek, ad alanı, program kodu içinde açıkça bildirildiğinde, tüm tür bildirimleri örtük olarak genel ad alanı içinde bulunduğundan CS1527 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="ef9d2-107">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4 {}  
private struct S1 {}  
```
