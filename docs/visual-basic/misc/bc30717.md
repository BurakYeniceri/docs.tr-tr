# <a name="39try39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Try &#39; deyimleri Komut penceresi içinde geçerli değildir
`Try`deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30717  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Try` kodda hata ayıklama gelen deyimleri.
