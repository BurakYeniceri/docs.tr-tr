# <a name="integrating-with-com-applications"></a>COM Uygulamaları ile Tümleştirme
Windows Communication Foundation (WCF) dağıtılmış uygulamaları oluşturmak için zengin bir ortam sağlar. COM + barındırılan bir bileşen tabanlı uygulama mantığı önemli ölçüde yatırımınız varsa, onu yeniden yazmaya gerek yerine mevcut mantığınızı genişletmek için WCF kullanabilirsiniz. Bu bölüm içindeki konuları COM + WCF ile nasıl kullanılacağını açıklar.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [COM+ Uygulamaları ile Tümleştirme Genel Bakış](../../../../docs/framework/wcf/feature-details/integrating-with-com-plus-applications-overview.md)  
 COM + bileşenleri bütünleştirmek nasıl ve ne zaman bir bakış sağlar.  
  
 [Nasıl yapılır: COM+ Hizmet Modeli Yapılandırma Aracı'nı Kullanma](../../../../docs/framework/wcf/feature-details/how-to-use-the-com-service-model-configuration-tool.md)  
 COM + hizmet modeli yapılandırma komut satırı aracı (ComSvcConfig.exe) WCF hizmetleri sunulan istediğiniz uygulama arabirimleri yapılandırmak için nasıl kullanılacağını açıklar.  
  
 [Nasıl yapılır: COM+ Hizmet Ayarlarını Yapılandırma](../../../../docs/framework/wcf/feature-details/how-to-configure-com-service-settings.md)  
 WCF hizmet olarak bir COM + nesnesi yapılandırmak açıklanmaktadır.  
  
 [Nasıl Yapılır: COM+ Tümleştirme Uygulaması Dağıtma](../../../../docs/framework/wcf/feature-details/how-to-deploy-a-com-integration-application.md)  
 COM + tümleştirme uygulaması taşınacağı açıklanmaktadır.
