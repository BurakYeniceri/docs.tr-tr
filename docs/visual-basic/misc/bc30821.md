# <a name="39ltrsetstmtgt39-is-not-declared"></a>&#39; &lt;rsetstmt&gt;&#39; bildirilmemiş olan
'\<rsetstmt >' bildirilmemiş olan. RSet deyimleri artık desteklenmiyor; Bunun yerine Microsoft.VisualBasic.RSet kullanın.  
  
 İç için çeşitli işlevleri [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] önceki sürümlerde taşınmıştır <xref:Microsoft.VisualBasic?displayProperty=nameWithType> ad alanı. Bu işlevleri tüm programlama dilleri için daha genel olarak kullanılabilir hale getirir.  
  
 **Hata Kimliği:** BC30821  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanım `RSet` işlevi `Microsoft.VisualBasic` ad alanı yerine.
