# <a name="icorprofilercallbackexceptionclrcatcherfound-method"></a>ICorProfilerCallback::ExceptionCLRCatcherFound Yöntemi

## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT ExceptionCLRCatcherFound();  
```
