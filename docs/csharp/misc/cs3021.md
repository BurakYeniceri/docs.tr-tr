 <span data-ttu-id="7d9b6-104">Bu uyarı saptanmışsa `[CLSCompliant(false)]` bir derleme düzeyi CLSCompliant özniteliği true olarak ayarlandığında olmayan bir derleme sınıfında görünür (diğer bir deyişle, satır `[assembly: CLSCompliant(true)]`).</span><span class="sxs-lookup"><span data-stu-id="7d9b6-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="7d9b6-105">Derleme kendisini CLS uyumlu bildiriyor değil olduğundan, uyumlu olmayan varsayıldığından, kendisini uyumlu olmayan, bildirmek için derleme içindeki her şey için gerek yoktur.</span><span class="sxs-lookup"><span data-stu-id="7d9b6-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="7d9b6-106">CLS uyumluluğu hakkında daha fazla bilgi için bkz: [CLS uyumlu kod yazma](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span><span class="sxs-lookup"><span data-stu-id="7d9b6-106">For more information on CLS Compliance, see [Writing CLS-Compliant Code](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span></span>  
  
 <span data-ttu-id="7d9b6-107">Bu uyarıdan kurtulmak için özniteliği kaldırın veya derleme düzeyi özniteliğini ekleyin.</span><span class="sxs-lookup"><span data-stu-id="7d9b6-107">To get rid of this warning, remove the attribute or add the assembly level attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7d9b6-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="7d9b6-108">Example</span></span>  
 <span data-ttu-id="7d9b6-109">Aşağıdaki örnek CS3021 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7d9b6-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
