# <a name="compiler-error-cs0065"></a><span data-ttu-id="32afa-102">Derleyici Hatası CS0065</span><span class="sxs-lookup"><span data-stu-id="32afa-102">Compiler Error CS0065</span></span>
<span data-ttu-id="32afa-103">'event': olay özelliği hem de sahip olmalıdır ekleme ve kaldırma erişimcisi</span><span class="sxs-lookup"><span data-stu-id="32afa-103">'event' : event property must have both add and remove accessors</span></span>  
  
 <span data-ttu-id="32afa-104">Bir alan değil bir olayı her iki erişim yöntemleri olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="32afa-104">An event that is not a field must have both access methods.</span></span>  
  
 <span data-ttu-id="32afa-105">Aşağıdaki örnek CS0065 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="32afa-105">The following sample generates CS0065:</span></span>  
  
```csharp  
// CS0065.cs  
using System;  
public delegate void Eventhandler(object sender, int e);  
public class MyClass  
{  
   public event EventHandler Click   // CS0065,  
   {  
      // to fix, uncomment the add and remove definitions  
      /*  
      add  
      {  
         Click += value;  
      }  
      remove  
      {  
         Click -= value;  
      }  
      */  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
