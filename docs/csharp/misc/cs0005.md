# <a name="compiler-error-cs0005"></a>Derleyici Hatası CS0005
Bir bağımsız değişkeni tarafından derleyici seçeneği 'compiler_option' gelmelidir  
  
 Bazı derleyici seçenekleri parametreleri gerektirir. Derleyici seçeneği tarafından gerekli bağımsız değişkenleri geçirmezseniz CS0005 oluşturulur.  
  
 Daha fazla bilgi için bkz: [C# Derleyici Seçenekleri](../../csharp/language-reference/compiler-options/index.md).
