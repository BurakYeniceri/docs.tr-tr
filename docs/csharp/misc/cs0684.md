# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="93dbc-102">Derleyici Uyarısı (düzey 1) CS0684</span><span class="sxs-lookup"><span data-stu-id="93dbc-102">Compiler Warning (level 1) CS0684</span></span>
<span data-ttu-id="93dbc-103">'işaretli 'ComImportAttribute' ile işaretlenmemiş CoClassAttribute' ile 'arabirimi' arabirimi işaretli</span><span class="sxs-lookup"><span data-stu-id="93dbc-103">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="93dbc-104">Belirtirseniz **CoClassAttribute** bir arabirimde de belirtmeniz gerekir **ComImportAttribute**.</span><span class="sxs-lookup"><span data-stu-id="93dbc-104">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="93dbc-105">Aşağıdaki örnek CS0684 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="93dbc-105">The following sample generates CS0684:</span></span>  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
