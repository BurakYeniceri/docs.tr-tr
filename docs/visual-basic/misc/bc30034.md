# <a name="bracketed-identifier-is-missing-closing-3939"></a>Köşeli parantez içindeki tanımlayıcı kapanış &#39;eksik;] &#39;
Kaçış karakterli bir adı köşeli eşleşen çiftleri bulunması gerekir.  
  
 **Hata Kimliği:** BC30034  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaçış karakterli adının sonuna bir kapanış ayracı yerleştirin.
