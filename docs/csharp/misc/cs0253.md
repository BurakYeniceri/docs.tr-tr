# <a name="compiler-warning-level-2-cs0253"></a>Derleyici Uyarısı (Düzey 2) CS0253
Olası istenmeyen başvuru karşılaştırması; değer karşılaştırması almak için 'type' türü için sağ taraftaki atayın  
  
 Derleyici başvurusu karşılaştırma yapıyor. Dize değeri karşılaştırma yapmak isterseniz, ifade sağ tarafındaki cast `type`.  
  
 Aşağıdaki örnek CS0253 oluşturur:  
  
```  
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
