# <a name="expression-is-not-a-method"></a>İfade bir yöntem değildir
Yöntem çağrısı hedefi bir yöntem değil.  
  
 **Hata Kimliği:** BC30454  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yöntem adı yanlış yazdınız değil emin olun.
