# <a name="expanded-properties-cannot-be-initialized"></a><span data-ttu-id="bce98-102">Genişletilmiş özellikler başlatılamaz</span><span class="sxs-lookup"><span data-stu-id="bce98-102">Expanded properties cannot be initialized</span></span>
<span data-ttu-id="bce98-103">Otomatik uygulanan bir özellik bildiriminden bir parçası olarak başlatılabilir, ancak genişletilmiş özellik olamaz.</span><span class="sxs-lookup"><span data-stu-id="bce98-103">An auto-implemented property can be initialized as part of its declaration, but an expanded property cannot be.</span></span>  
  
 <span data-ttu-id="bce98-104">**Hata Kimliği:** BC36714</span><span class="sxs-lookup"><span data-stu-id="bce98-104">**Error ID:** BC36714</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="bce98-105">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="bce98-105">To correct this error</span></span>  
  
-   <span data-ttu-id="bce98-106">Otomatik uygulanan özelliğini kullanın ya da özellik bildiriminden başlatma kaldırın.</span><span class="sxs-lookup"><span data-stu-id="bce98-106">Either use an auto-implemented property or remove the initialization from the property declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bce98-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="bce98-107">Example</span></span>  
 <span data-ttu-id="bce98-108">Aşağıdaki örnekler, her ikisi de otomatik uygulanan ve Özellikler genişletilmiş gösterir.</span><span class="sxs-lookup"><span data-stu-id="bce98-108">The following examples show both auto-implemented and expanded properties.</span></span> <span data-ttu-id="bce98-109">Otomatik uygulanan özellikler atamaları kullanılarak başlatılabilir veya `New` yan tümcesi, ancak genişletilmiş özellikler olamaz.</span><span class="sxs-lookup"><span data-stu-id="bce98-109">Auto-implemented properties can be initialized by using assignment or a `New` clause, but expanded properties cannot be.</span></span>  
  
```vb  
Class AutoImplementedExample  
    ' An auto-implemented property can be assigned an initial value.  
    Property IDNum As Integer = 33542  
    ' An auto-implemented property can be initialized with New.  
    Property Name As New String("Don Hall")  
End Class  
  
Class ExpandedExample  
    ' Attempting to expand an initialized auto-implemented property  
    ' causes this error.  
    'Property IDNum As Integer = 33542  
    '    Get  
    '    End Get  
    '    Set(ByVal value As Integer)  
    '    End Set  
    'End Property  
  
    ' Instead, you can assign the initial value to the backing field.  
    Private _IDNum As Integer = 33542  
    Property IDNum As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
End Class  
```
