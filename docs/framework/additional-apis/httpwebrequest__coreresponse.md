# <a name="httpwebrequestcoreresponse-field"></a>HttpWebRequest. \_CoreResponse alan

`HttpWebRequest._CoreResponse` bir nesne (ya da bir [CoreResponseData](coreresponsedata.md) veya bir <xref:System.Exception>) HTTP yanıt ayrıştırma sonucunu içeren.

## <a name="syntax"></a>Sözdizimi
  
```csharp
private object _CoreResponse
```

> [!WARNING]
> Bu API, kodunuzda doğrudan kullanılmak üzere tasarlanmamıştır. Bunun yerine, kullanmanız bir <xref:System.Diagnostics.DiagnosticSource> ağ kod bağlanmanıza. Bkz: [DiagnosticSource Kullanıcı Kılavuzu](https://github.com/dotnet/corefx/blob/master/src/System.Diagnostics.DiagnosticSource/src/DiagnosticSourceUsersGuide.md).
> 
> Microsoft hiçbir koşulda bir üretim uygulamasında bu sınıf kullanımını desteklemez.
