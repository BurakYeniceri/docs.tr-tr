# <a name="method-cannot-have-both-a-paramarray-and-optional-parameters"></a>Yöntemi bir ParamArray ve isteğe bağlı parametreleri olamaz
A `ParamArray` bağımsız değişkeni otomatik olarak isteğe bağlıdır ve yalnızca isteğe bağlı bağımsız değişkeni yordam bildiriminde olması gerekir. Tüm önceki bağımsız değişkenleri gereklidir.  
  
 **Hata Kimliği:** BC30046  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İsteğe bağlı bağımsız değişkenler bildiriminde kaldırın.
