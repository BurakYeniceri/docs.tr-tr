# <a name="xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments"></a>XML açıklama bloğu XML belgeleri yorumları uygulamasının desteklediği herhangi bir dil öğeyle ilişkili olamaz
XML açıklama bloğu herhangi bir geçerli dil öğeyle ilişkili olamaz. XML açıklama yoksayılacak. Açıklama bloğu XML etiketinde eşleşen bir dil öğe varsa, bu hata ayrıca oluşur.  
  
 **Hata Kimliği:** BC42312  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bulmak ve gereksiz XML etiketi kaldırmak veya açıkladığı language öğesi sağlayın.
