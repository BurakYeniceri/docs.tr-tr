# <a name="compiler-error-cs0517"></a>Derleyici Hatası CS0517
'class' hiçbir temel sınıf var ve temel oluşturucu çağrılamıyor  
  
 Yalnızca .NET Framework ortak dil çalışma zamanı hiçbir temel sınıf olan tek sınıftır nesne sınıfı için kaynak kodu derleme CS0517 ortaya çıkabilir.
