 <span data-ttu-id="64e61-104">Bir **ekleme** veya **kaldırmak** in anahtar sözcüğü bir [olay](../../csharp/language-reference/keywords/event.md) tanımı gövde olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="64e61-104">An **add** or **remove** keyword in an [event](../../csharp/language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="64e61-105">Daha fazla bilgi için bkz: [olayları](../../csharp/programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="64e61-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="64e61-106">Aşağıdaki örnek CS0073 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="64e61-106">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
