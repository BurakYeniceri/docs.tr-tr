# <a name="39new39-cannot-be-used-on-class-39ltclassnamegt39-because-it-contains-a-39mustoverride39-member-that-has-not-been-overridden"></a>&#39; Yeni &#39; sınıf &#39;kullanılamaz; &lt;classname&gt;&#39; içerdiği için bir &#39; MustOverride &#39; değil geçersiz kılınmış üyesi
Kullanma girişiminde bulundunuz `New` içeren bir sınıf üzerinde bir `MustOverride` değil geçersiz kılınmış üye.  
  
 **Hata Kimliği:** BC30376  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `New` deyimi.
