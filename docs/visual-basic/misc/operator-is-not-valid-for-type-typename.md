# <a name="operator-is-not-valid-for-type-39lttypenamegt39"></a>İşleç türü &#39;için geçerli değil; &lt;typename&gt;&#39;

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İşleç üzerinde çalıştırılan türleri için uygun olduğundan emin olun.
