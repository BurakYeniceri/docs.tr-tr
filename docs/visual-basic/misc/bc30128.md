# <a name="security-attribute-39ltattributenamegt39-is-not-valid-lterrorgt"></a>Güvenlik özniteliği &#39; &lt;attributename&gt; &#39; geçerli değil: &lt;hata&gt;
Ortak dil çalışma zamanı bir güvenlik bağlamı özniteliğinde kabul etmiyor.

**Hata Kimliği:** BC30128

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

1. Öznitelik bildiriminden kaldırın veya bağlamı için geçerli olan özniteliği ile değiştirin.

2. Tırnak işaretli hata iletisini inceleyin ve uygun eylemi gerçekleştirin.
