# <a name="39ltmembergt39-is-already-declared-in-this-structure"></a>&#39;&lt;üye&gt; &#39; bu yapısında zaten bildirildi

 **Hata Kimliği:** BC31421  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yabancı bildirimini kaldırın.
