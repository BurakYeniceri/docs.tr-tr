# <a name="language-integrated-axes-in-visual-basic-linq-to-xml"></a>Dil ile tümleşik eksenleri Visual Basic (LINQ-XML)
Bu bölümde XML erişim kolaylaştırmak için doğrudan Visual Basic dilinde yerleşik özellikleri açıklanmıştır. XML belgeleri için LINQ örneklerin çoğu bu tümleşik Visual Basic eksenler kullanın.  
  
## <a name="in-this-section"></a>Bu Bölümde  
  
|Konu|Açıklama|  
|-----------|-----------------|  
|[XML Alt Axis Özelliği](../../../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md)|Aşağıdaki alt erişim sağlar: bir <xref:System.Xml.Linq.XElement> nesne, bir <xref:System.Xml.Linq.XDocument> nesnesi, bir koleksiyonu <xref:System.Xml.Linq.XElement> nesneleri veya koleksiyonu <xref:System.Xml.Linq.XDocument> nesneleri. Bu eksen eşdeğerdir <xref:System.Xml.Linq.XContainer.Elements%2A> ekseni.|  
|[XML Descendant Axis Özelliği](../../../../visual-basic/language-reference/xml-axis/xml-descendant-axis-property.md)|Aşağıdaki alt öğelere erişim sağlar: bir <xref:System.Xml.Linq.XElement> nesne, bir <xref:System.Xml.Linq.XDocument> nesne ya da bir koleksiyonu <xref:System.Xml.Linq.XElement> nesneleri. Bu eksen eşdeğerdir <xref:System.Xml.Linq.XContainer.Descendants%2A> ekseni.|  
|[XML Özniteliği Axis Özelliği](../../../../visual-basic/language-reference/xml-axis/xml-attribute-axis-property.md)|Bir özniteliğin erişim sağlayan bir <xref:System.Xml.Linq.XElement> nesnesi. Bu eksen kabaca eşdeğerdir <xref:System.Xml.Linq.XElement.Attribute%2A> ekseni. Bu eksen farklıdır <xref:System.Xml.Linq.XElement.Attribute%2A> , eksen döndürdüğü özniteliğinin değeri olmayan bir <xref:System.Xml.Linq.XAttribute> nesnesi.|  
|[Extension Indexer Özelliği](../../../../visual-basic/language-reference/xml-axis/extension-indexer-property.md)|Bir koleksiyondaki tek tek öğelere erişim sağlar.|
