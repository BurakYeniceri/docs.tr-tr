# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-method-declaration"></a>&#39;Paylaşılan&#39; birleştirilemez &#39; &lt;belirticisi&gt; &#39; yöntemi bildiriminde
Birleştirme denediniz `Shared` değiştiricisi bir tanımlayıcısı ile gibi `Overridable`, `NotOverridable`, veya `MustOverride`, yani bu tür bir birlikte bir yöntem bildirimi geçerli değil.  
  
 **Hata Kimliği:** BC30501  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirleyici kaldırın.
