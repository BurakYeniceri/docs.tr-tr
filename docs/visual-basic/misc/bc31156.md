# <a name="xml-attribute-39version39-must-be-the-first-attribute-in-xml-declaration"></a>XML özniteliği &#39; sürüm &#39; XML bildirimi ilk özniteliğinde olmalıdır
`version` Bir XML belgesi değişmez değer özniteliğinin belirtilen ilk öznitelik değil. `version` Özniteliği için bir XML belgesi değişmez XML 1.0 belirtimi belirtilen ilk öznitelik olmalıdır.  
  
 **Hata Kimliği:** BC31156  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yeniden konumlandırma `version` XML belgesinin ilk öznitelik olmasını değişmez değer özniteliği.
