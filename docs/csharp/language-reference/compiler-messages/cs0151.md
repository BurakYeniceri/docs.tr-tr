## <a name="example"></a><span data-ttu-id="4f2eb-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="4f2eb-106">Example</span></span>  
 <span data-ttu-id="4f2eb-107">Bu hata, dönüştürme olduğunda veya kullanılabilir örtük dönüşümler belirsiz bir duruma neden olduğunda ortaya çıkabilir.</span><span class="sxs-lookup"><span data-stu-id="4f2eb-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="4f2eb-108">Aşağıdaki örnek CS0151 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="4f2eb-108">The following sample generates CS0151.</span></span>  
  
```csharp  
// CS0151.cs  
public class MyClass  
{  
   public static implicit operator int (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static implicit operator long (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
  
      // Compiler cannot choose between int and long  
      switch (a)   // CS0151  
      // try the following line instead  
      // switch ((int)a)  
      {  
         case 1:  
            break;  
      }  
   }  
}  
```
