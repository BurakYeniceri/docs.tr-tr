# <a name="compiler-error-cs1675"></a>Derleyici Hatası CS1675
Numaralandırmalar tür parametreleri olamaz  
  
 Bu hatayı gidermek için tür parametresi ile Kaldır `enum` bildirimi.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS1675 oluşturur:  
  
```csharp  
// CS1675.cs  
enum E<T>  // CS1675  
{  
}  
  
class CMain  
{  
    public static void Main()  
    {  
    }  
}  
```
