# <a name="customizing-insert-update-and-delete-operations"></a>Özelleştirme ekleme, güncelleştirme ve silme işlemleri

> [!NOTE]
>  Visual Studio kullanıyorsanız, kullanabileceğiniz [!INCLUDE[vs_ordesigner_long](../../../../../../includes/vs-ordesigner-long-md.md)] INSERT özelleştirmek için güncelleştirme ve silme eylemlerinin.
