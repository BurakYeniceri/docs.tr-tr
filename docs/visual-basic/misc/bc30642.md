# <a name="39optional39-and-39paramarray39-cannot-be-combined"></a>&#39; isteğe bağlı &#39; ve &#39; ParamArray &#39; birleştirilemez
`Optional` Ve `ParamArray` değiştiricileri karşılıklı olarak birbirini dışlar.  
  
 **Hata Kimliği:** BC30642  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirtin `ParamArray` veya `Optional`, ancak ikisini birden değil.
