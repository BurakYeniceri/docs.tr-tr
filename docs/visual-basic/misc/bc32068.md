# <a name="local-variables-within-generic-methods-cannot-be-declared-39static39"></a>Genel yöntemler yerel değişkenler bildirilemez &#39; Statik &#39;
Bir genel yordam içinde yerel bir değişken bildirimi belirtir `Static`.  
  
 Şu anda Visual Basic ve .NET Framework statik değişkenler ve genel yordamlar herhangi bir birleşimini desteklemez.  
  
 **Hata Kimliği:** BC32068  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Static` değişken bildirimi anahtar sözcük.
