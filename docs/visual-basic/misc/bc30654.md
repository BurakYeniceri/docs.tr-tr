# <a name="39return39-statement-in-a-function-get-or-operator-must-return-a-value"></a>&#39;Dönüş&#39; işlevi, Get veya işleci deyiminde bir değer döndürmesi gerekir
`Return` deyimler, çağıran bir yordam için bir değer döndürmek için kullanılmalıdır. Kullanamazsınız `Return` kendileri tarafından deyimleri denetlemek için program akışı.  
  
 **Hata Kimliği:** BC30654  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  İşlev veya yordam döndürebilirsiniz bir değer belirtin.  
  
2.  Kullanım `End` deyimi geçerli yordamı çıkmak program neden olacak.
