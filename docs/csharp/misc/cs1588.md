# <a name="compiler-error-cs1588"></a>Derleyici Hatası CS1588
Ortak dil çalışma zamanı dizini--'gerekçe' belirlenemiyor  
  
 Derleyici, .NET Framework ortak dil çalışma zamanı yüklü dizin alınamıyor. Bu hata, ortak dil çalışma zamanı düzgün yüklendiğini gösterir.
