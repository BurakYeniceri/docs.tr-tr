# <a name="compiler-error-cs1719"></a>Derleyici Hatası CS1719
'Dosya adı'--'gerekçe' Win32 kaynak dosyası okunurken hata oluştu  
  
 Hata verilen bir nedenle başarısız oldu Win32 kaynak dosyasını okuma girişimi genellikle bir şey ister "dosya bulunamadı" veya "erişim engellendi." Bu hatanın nedeni tarafından açıklanan sorunu düzelterek çözümlenir.
