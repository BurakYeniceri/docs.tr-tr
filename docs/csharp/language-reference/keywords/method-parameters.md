# <a name="method-parameters-c-reference"></a>Yöntem Parametreleri (C# Başvurusu)
Bir parametre bir yöntem için bildirilirse [ref](../../../csharp/language-reference/keywords/ref.md) veya [çıkışı](../../../csharp/language-reference/keywords/out.md), parametresi, kendisiyle ilişkili bir değere sahip olabilir. Bu değer yönteminde değiştirilebilir, ancak denetim arama yordamın geçtiğinde değişen değeri korunmaz. Bir yöntemin parametre anahtar sözcüğünü kullanarak, bu davranışı değiştirebilirsiniz.  
  
 Bu bölüm, yöntem parametreleri bildirirken kullanabileceğiniz anahtar sözcükleri açıklar:  
  
-   [parametreleri](../../../csharp/language-reference/keywords/params.md)  
  
-   [Ref](../../../csharp/language-reference/keywords/ref.md)  
  
-   [çıkışı](../../../csharp/language-reference/keywords/out.md)
