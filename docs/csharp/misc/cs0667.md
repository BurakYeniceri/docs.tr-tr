# <a name="compiler-error-cs0667"></a>Derleyici Hatası CS0667
Özellik 'geçersiz özelliği kullanım dışı bırakılmıştır. Lütfen bunun yerine 'geçerli özelliğini' kullanın '.  
  
 Kullanmaya çalıştığınız özellik artık kullanılmıyor. Bunun yerine geçerli özelliğini kullanmak için kodunuzu güncelleştirin.
