# <a name="local-variable-39ltvariablenamegt39-is-already-declared-in-the-current-block"></a>Yerel değişken &#39; &lt;variablename&gt;&#39; geçerli blok içinde zaten bildirildi
Aynı ada sahip iki değişken aynı bloğu içinde bildirilen (olabilen bir yordam veya bir denetim bloğu gibi `If...Then`).  
  
 **Hata Kimliği:** BC30288  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bildirimleri biri için farklı bir blok taşımak veya değişkenlerinden birinin adını değiştirin.
