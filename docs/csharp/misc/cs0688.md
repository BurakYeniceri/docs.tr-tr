# <a name="compiler-warning-level-1-cs0688"></a>Derleyici Uyarısı (düzey 1) CS0688
'method1' bir bağlantı isteği var ancak geçersiz kılar veya 'bir bağlantı isteği olmayan method2' uygular. Bir güvenlik açığı var olabilir.  
  
 Türetilmiş sınıf yöntemi üzerinde ayarlanan bağlantı isteği kolayca temel sınıf yöntemini çağırarak atlatılabilir. Güvenlik boşluğu kapatmak için temel sınıf yöntemi de bağlantı isteği kullanması gerekir. Daha fazla bilgi için bkz: [talep vs. LinkDemand](http://msdn.microsoft.com/en-us/1ab877f2-70f4-4e0d-8116-943999dfe8f5).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0688 oluşturur. Taban sınıf değiştirmeden uyarıyı çözümlemek için geçersiz kılma yönteminden güvenlik özniteliğini kaldırın. Bu güvenlik sorunu çözmek için değil.  
  
```  
// CS0688.cs  
// compile with: /W:1  
using System;  
using System.Security.Permissions;  
  
class Base   
{  
    //Uncomment the following line to close the security hole  
    //[FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")]  
    public virtual void DoScaryFileStuff()  
    {  
    }  
}  
  
class Derived: Base  
{  
    [FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")] // CS0688  
    public override void DoScaryFileStuff()  
    {  
    }  
    static void Main()  
    {  
    }  
}  
```
