# <a name="string-constants-must-end-with-a-double-quote"></a>Dize sabitleri çift tırnak işaretiyle bitmelidir
Dize sabitleri başlar ve tırnak işaretiyle bitmelidir.  
  
 **ErrorID:** BC30648  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Dize sabit değeri bir tırnak işareti (") ile biter emin olun. Diğer metin düzenleyicileri değerlerinden yapıştırmak isterseniz, geçerli bir tırnak işareti ve, "Akıllı" veya "süslü" tırnak işaretleri gibi benzer karakterden oluşan bir yapıştırılan karakter olduğundan emin olun ("veya") ya da iki tek tırnak işaretleri (").
