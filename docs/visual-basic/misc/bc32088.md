# <a name="type-arguments-unexpected"></a><span data-ttu-id="4a91b-102">Tür bağımsız değişkenleri beklenmeyen</span><span class="sxs-lookup"><span data-stu-id="4a91b-102">Type arguments unexpected</span></span>
<span data-ttu-id="4a91b-103">Bir `Implements` yan tümcesi uygulama arabirim üyesi için tür bağımsız değişkenleri temin eder.</span><span class="sxs-lookup"><span data-stu-id="4a91b-103">An `Implements` clause supplies type arguments for the interface member it is implementing.</span></span>  
  
 <span data-ttu-id="4a91b-104">`Implements` Yan tümcesi yalnızca tanımlamak üyesi uygulama ve arabirim.</span><span class="sxs-lookup"><span data-stu-id="4a91b-104">The `Implements` clause should only identify the interface and member it is implementing.</span></span> <span data-ttu-id="4a91b-105">Bunun anlamı bir genel yordam bildiriyorsanız, `Of` yan tümcesi ve tür bağımsız değişkenleri görünmelidir bildirimi ana bölümünde bir arabirim yordamı değil uyguluyorsanız yaptıkları gibi.</span><span class="sxs-lookup"><span data-stu-id="4a91b-105">This means that if you are declaring a generic procedure, the `Of` clause and the type arguments should appear in the main part of the declaration, just as they would if you were not implementing an interface procedure.</span></span>  
  
 <span data-ttu-id="4a91b-106">Aşağıdaki kod bu hata oluşmasına neden olabilir.</span><span class="sxs-lookup"><span data-stu-id="4a91b-106">The following code can generate this error.</span></span>  
  
```  
Public Interface testInterface  
    Sub testSub(Of t)()  
End Interface  
Public Class testClass  
    Implements testInterface  
    Public Sub testSub() Implements testInterface.testSub(Of t)()  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="4a91b-107">Önceki bildirimi `Implements` yan tümcesi, erişim veya yordam değiştiricileri olası eklenmesi dışında arabirim tanımı gibi görünmelidir.</span><span class="sxs-lookup"><span data-stu-id="4a91b-107">The declaration preceding the `Implements` clause should look like the interface definition, except for the possible addition of access or procedure modifiers.</span></span> <span data-ttu-id="4a91b-108">Aşağıdaki kod hata önler.</span><span class="sxs-lookup"><span data-stu-id="4a91b-108">The following code avoids the error.</span></span>  
  
```  
Public Sub testSub(Of t)() Implements testInterface.testSub  
```  
  
 <span data-ttu-id="4a91b-109">**Hata Kimliği:** BC32088</span><span class="sxs-lookup"><span data-stu-id="4a91b-109">**Error ID:** BC32088</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4a91b-110">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="4a91b-110">To correct this error</span></span>  
  
-   <span data-ttu-id="4a91b-111">Tür bağımsız değişkeni listesinden kaldırmak `Implements` yan tümcesi.</span><span class="sxs-lookup"><span data-stu-id="4a91b-111">Remove the type argument list from the `Implements` clause.</span></span>  
  
-   <span data-ttu-id="4a91b-112">Genel arabirim üyesi uyguluyorsanız, ardından türü bağımsız değişken listesi bildirimi ana bölümünde put, önceki `Implements` anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="4a91b-112">If you are implementing a generic member of the interface, then put the type argument list in the main part of the declaration, preceding the `Implements` keyword.</span></span>
