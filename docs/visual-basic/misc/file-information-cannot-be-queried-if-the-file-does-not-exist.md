# <a name="file-information-cannot-be-queried-if-the-file-does-not-exist"></a>Dosya mevcut değilse, dosya bilgisi sorgulanamıyor
Sorgulamak için bir girişimde bulunuldu bir <xref:System.IO.FileInfo> var olmayan bir dosya hakkında bilgi için nesne. Dosya adı sağlanırken yapılan bir hata nedeniyle genellikle budur.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Doğru dosya adı sağlayarak denetleyin.
