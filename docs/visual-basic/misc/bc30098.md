# <a name="39readonly39-property-39ltpropertynamegt39-cannot-be-the-target-of-an-assignment"></a>&#39;Salt okunur&#39; özelliği &#39; &lt;propertyname&gt; &#39; atama işleminin hedefi olamaz
A `ReadOnly` özelliği için değer atayan bir bağlam oluşuyor. Yalnızca yazılabilir değişkenleri, özellikleri ve dizi öğeleri yürütme sırasında atanmış değerlere sahip olabilir.  
  
 **Hata Kimliği:** BC30098  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `ReadOnly` anahtar sözcüğünün `Property` değişken bildirme deyimi veya bir değer atar deyimi kaldırın.
