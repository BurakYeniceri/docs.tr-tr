# <a name="mycomputerclock-object"></a>My.Computer.Clock Nesnesi

 Yöntemleri ve özellikleri hakkında bilgi için `My.Computer.Clock` nesne için bkz: <xref:Microsoft.VisualBasic.Devices.Clock>.
