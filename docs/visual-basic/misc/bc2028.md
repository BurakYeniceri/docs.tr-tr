# <a name="option-39ltoptiongt39-cannot-be-followed-by-3939-or-39-39-ignored"></a>seçenek &#39; &lt;seçeneği&gt;&#39; tarafından izlenemiyor &#39; + &#39; veya &#39;-&#39; yoksayıldı
Derleyici seçeneği değeriyle belirtilen `+` veya `-`; bu değerleri desteklenmiyor.  
  
 Derleyici seçenekleri ve destekledikleri değerleri hakkında daha fazla bilgi için bkz: [Visual Basic derleyici seçenekleri listelenen alfabetik olarak](../../visual-basic/reference/command-line-compiler/compiler-options-listed-alphabetically.md).  
  
 **Hata Kimliği:** BC2028  
  
## <a name="to-correct-this-error-and-use-the-option"></a>Bu hatayı düzeltmek ve seçeneğini kullanın  
  
-   Derleyici seçeneği ile herhangi bir değer belirtin.  
  
## <a name="to-correct-this-error-and-not-use-the-option"></a>Bu hatayı düzeltmek ve seçeneğini kullanmayın  
  
-   Derleyici seçeneği kaldırın.
