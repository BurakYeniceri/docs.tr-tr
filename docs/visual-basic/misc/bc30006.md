# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>Modül için gereken başvuru &#39; &lt;modulename&gt; &#39; olay tanımını içeren &#39; &lt;eventname&gt;&#39;
Modül için gereken başvuru ' <`modulename`>' olay tanımını içeren ' <`eventname`>'. Projenize ekleyin.  
  
 Olay doğrudan projenizde başvurulmuyor bir modüldeki tanımlanır. Visual Basic derleyici olay birden fazla modülde tanımlanmış durumda Karışıklığı önlemek için bir başvuru gerektirir.  
  
 **Hata Kimliği:** BC30006  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Başvurulmayan modül adı, proje başvuruları içerir.
