# <a name="imetadatafilter-interface"></a>IMetaDataFilter Arabirimi
İşaretleme ve zaten gerçekleştirilecek eylemler yinelenen önlemek için meta veri simgesi filtreleme için yöntemleri sağlar.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[Istokenmarked yöntemi](../../../../docs/framework/unmanaged-api/metadata/imetadatafilter-istokenmarked-method.md)|Belirtilen meta veri simgesi işlenen olup olmadığını belirten bir değer alır.|  
|[MarkToken yöntemi](../../../../docs/framework/unmanaged-api/metadata/imetadatafilter-marktoken-method.md)|Belirtilen meta veri simgesi işlenmiş belirten bir değer ayarlar.|  
|[UnmarkAll yöntemi](../../../../docs/framework/unmanaged-api/metadata/imetadatafilter-unmarkall-method.md)|Geçerli meta veri kapsamdaki tüm belirteçleri işleme işaretleri kaldırır.|
