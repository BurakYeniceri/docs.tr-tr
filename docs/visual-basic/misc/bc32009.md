# <a name="method-declaration-statements-must-be-the-first-on-a-logical-line"></a>Yöntem bildirim deyimleri ilk mantıksal bir satırda olmalıdır
A `Function` veya `Sub` deyimi izleyen iki nokta üst üste (:) deyimi ayırıcı. `Function`veya `Sub` , kaynak satırında yegane deyim olmalıdır.  
  
 **Hata Kimliği:** BC32009  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Birden çok deyime farklı satırlarına ayırın.
