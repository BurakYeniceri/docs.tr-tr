<span data-ttu-id="37963-103">Açık arabirim bildirimindeki ' member' arabirimi üyesi değil</span><span class="sxs-lookup"><span data-stu-id="37963-103">'member' in explicit interface declaration is not a member of interface</span></span>  
  
 <span data-ttu-id="37963-104">Açıkça bildirmek için girişimde bulunuldu bir [arabirimi](../../csharp/language-reference/keywords/interface.md) üye yok.</span><span class="sxs-lookup"><span data-stu-id="37963-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md) member that does not exist.</span></span> <span data-ttu-id="37963-105">Bildirimi silmek ya da geçerli arabirim üyesi başvurduğu şekilde değiştirin.</span><span class="sxs-lookup"><span data-stu-id="37963-105">You should either delete the declaration or change it so that it refers to a valid interface member.</span></span>  
  
 <span data-ttu-id="37963-106">Aşağıdaki örnek CS0539 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="37963-106">The following sample generates CS0539:</span></span>  
  
```csharp

   interface I  
   {  
      void m();  
   }

   {  
      void I.x()   // CS0539  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
