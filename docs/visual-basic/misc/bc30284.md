# <a name="lttype1gt-39lttypenamegt39-cannot-be-declared-39overrides39-because-it-does-not-override-a-lttype1gt-in-a-base-lttype2gt"></a>&lt;type1&gt; &#39;&lt; TypeName&gt;&#39; bildirilemez &#39; geçersiz kılmaları &#39; geçersiz olduğundan bir &lt;type1&gt; bir Base &lt;type2&gt;
A `Sub`, `Function`, veya `Property` deyimi belirtir `Overrides` zaman hiçbir türü aynı ada sahip mevcut bir temel sınıf.  
  
 **Hata Kimliği:** BC30284  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Tür adının doğru yazıldığından emin olun.  
  
2.  Gereksiz kaldırma `Overrides` anahtar sözcüğü.
