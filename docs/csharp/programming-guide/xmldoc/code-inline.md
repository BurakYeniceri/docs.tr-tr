# <a name="ltcgt-c-programming-guide"></a>&lt;c&gt; (C# programlama Kılavuzu)
## <a name="syntax"></a>Sözdizimi  
  
```xml  
<c>text</c>  
```  
  
#### <a name="parameters"></a>Parametreler  
 `text`  
 Kod olarak belirtmek istediğiniz metin.  
  
## <a name="remarks"></a>Açıklamalar  
 \<c > etiketi için bir açıklama metni kodu olarak işaretlenmelidir belirtmek için bir yol sağlar. Kullanım [ \<kodu >](../../../csharp/programming-guide/xmldoc/code.md) birden fazla satır kodu olarak belirtmek için.  
  
 İle derleme [/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) bir dosyaya işlem belgesi açıklamaları için.
