# <a name="compiler-error-cs0529"></a><span data-ttu-id="5c74d-102">Derleyici Hatası CS0529</span><span class="sxs-lookup"><span data-stu-id="5c74d-102">Compiler Error CS0529</span></span>
<span data-ttu-id="5c74d-103">Devralınan arabirimi 'interface1', 'interface2' arabirim hiyerarşisinde döngüye neden oluyor</span><span class="sxs-lookup"><span data-stu-id="5c74d-103">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="5c74d-104">Devralma listesi için bir [arabirimi](../../csharp/language-reference/keywords/interface.md) kendisine doğrudan veya dolaylı başvuru içeriyor.</span><span class="sxs-lookup"><span data-stu-id="5c74d-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="5c74d-105">Arabirim kendisinden devralır olamaz.</span><span class="sxs-lookup"><span data-stu-id="5c74d-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="5c74d-106">Aşağıdaki örnek CS0529 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5c74d-106">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
