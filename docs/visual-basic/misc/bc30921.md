# <a name="39ltderivedtypenamegt39-cannot-inherit-from-lttypegt-39ltconstructedbasetypenamegt39-because-it-expands-the-access-of-type-39ltinternaltypenamegt39-to-ltregiongt-39ltregionnamegt39"></a><span data-ttu-id="45a33-102">&#39;&lt;derivedtypename&gt; &#39; devralınmalıdır olamaz &lt;türü&gt; &#39; &lt;constructedbasetypename&gt; &#39; türü erişimini genişlettiğinden &#39; &lt; internaltypename&gt; &#39; için &lt;bölge&gt; &#39; &lt;regionname&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="45a33-102">&#39;&lt;derivedtypename&gt;&#39; cannot inherit from &lt;type&gt; &#39;&lt;constructedbasetypename&gt;&#39; because it expands the access of type &#39;&lt;internaltypename&gt;&#39; to &lt;region&gt; &#39;&lt;regionname&gt;&#39;</span></span>
<span data-ttu-id="45a33-103">Tür bağımsız değişkeni bir temel sınıf veya arabirim olarak kullanarak iç tür erişim düzeyini genişletmek bir türetilmiş sınıfta veya arabirimde çalışır.</span><span class="sxs-lookup"><span data-stu-id="45a33-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="45a33-104">Aşağıdaki kod bu hata oluşmasına neden olabilir.</span><span class="sxs-lookup"><span data-stu-id="45a33-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="45a33-105">Dışında kod `containingClass` erişmesine izin verilmiyor `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="45a33-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="45a33-106">Ancak, `derivedClass` aynı bütünleştirilmiş kodda herhangi bir koddan erişilebilir.</span><span class="sxs-lookup"><span data-stu-id="45a33-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="45a33-107">Bu nedenle, `derivedClass` devral olamaz `baseClass` onu kullanıyorsa `internalStructure` bir tür bağımsız değişkeni olarak maruz bırakabileceğinden çünkü `internalStructure` tanımlama kod bölgesini boyunca.</span><span class="sxs-lookup"><span data-stu-id="45a33-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="45a33-108">**Hata Kimliği:** BC30921</span><span class="sxs-lookup"><span data-stu-id="45a33-108">**Error ID:** BC30921</span></span>
