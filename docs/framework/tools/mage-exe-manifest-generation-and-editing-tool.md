<span data-ttu-id="cebb6-408">Aşağıdaki örnek, tüm Derlemelerle ve kaynak dosyaları geçerli dizinden doldurulan bir uygulama bildirimi oluşturur.</span><span class="sxs-lookup"><span data-stu-id="cebb6-408">The following example creates an application manifest populated with all of the assemblies and resource files from thecurrent directory.</span></span>

<span data-ttu-id="cebb6-411">Aşağıdaki örnek, Internet Explorer'da barındırılacak bir WPF uygulamasını dağıtmak için bir bildirim çiftinin nasıl oluşturulacağını gösterir.</span><span class="sxs-lookup"><span data-stu-id="cebb6-411">The following example demonstrates how to create a pair of manifests for deploying a WPF application that will be hosted in Internet Explorer.</span></span>

<span data-ttu-id="cebb6-413">Aşağıdaki örnek dağıtım bildirimini düzenleyerek kullanıcının yüklü sürümünün zorla güncelleştirilmesini sağlar.</span><span class="sxs-lookup"><span data-stu-id="cebb6-413">The following example edits the deployment manifest to force an update of the user's installed version.</span></span>
