# <a name="character-constant-must-contain-exactly-one-character"></a>Karakter sabiti tam olarak bir karakter içermelidir
Karakter sabiti ya da birden fazla karakteri içerdiğinden geçerli değil veya (tüm BITS dışında sıfır) bir null karakter.  
  
 **Hata Kimliği:** BC30004  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Karakter sabiti tek bir karakter veya null karakteri geçerli bir karakter değiştirebilirsiniz.
