# <a name="constants-must-have-a-value"></a>Sabit bir değere sahip olmalıdır
Bir sabit bir değer atamadan bildirildi.  
  
 **Hata Kimliği:** BC30438  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir değer sabite atayın.
