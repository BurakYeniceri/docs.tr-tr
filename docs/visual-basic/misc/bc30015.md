# <a name="inheriting-from-39systemltclassnamegt39-is-not-valid"></a>İçinden devralma &#39;sistem. &lt;classname&gt; &#39; geçerli değil
Bir sınıf devralınmalıdır olamaz `System.Array`, `System.Delegate`, `System.Enum`, veya `System.ValueType`.  
  
 **Hata Kimliği:** BC30015  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Inherits` deyimi veya geçerli bir taban sınıftan devralın şekilde değiştirin.
