# <a name="type-or-39with39-expected"></a><span data-ttu-id="314cb-102">Tür veya &#39;ile&#39; bekleniyor</span><span class="sxs-lookup"><span data-stu-id="314cb-102">Type or &#39;With&#39; expected</span></span>
<span data-ttu-id="314cb-103">Bir sınıf örneği bildirdiğinizde `New` anahtar sözcüğü, bir tür adı tarafından ya da gelmelidir `With`.</span><span class="sxs-lookup"><span data-stu-id="314cb-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="314cb-104">Örneğin, aşağıdaki deyimleri her bildirmek `client` örneğini olmasını `Customer` sınıfı.</span><span class="sxs-lookup"><span data-stu-id="314cb-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="314cb-105">Tür adı `Customer` izleyen `New`.</span><span class="sxs-lookup"><span data-stu-id="314cb-105">The type name `Customer` follows `New`.</span></span>  
  
```

' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="314cb-106">İle başlayarak [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)]nesneyi anonim bir türün bir örneği olarak bildirebilirsiniz, bu durumda, bir veri türü belirtmeyin.</span><span class="sxs-lookup"><span data-stu-id="314cb-106">Beginning with [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="314cb-107">Anonim türde bildirimlerden, anahtar sözcüğü `With` izleyen `New`.</span><span class="sxs-lookup"><span data-stu-id="314cb-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>
