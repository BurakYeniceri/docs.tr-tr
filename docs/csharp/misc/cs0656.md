# <a name="compiler-error-cs0656"></a>Derleyici Hatası CS0656
Üye 'object.member' eksik derleyici gerekli  
  
 Aşağıdaki sorunlardan biri bulunmaktadır:  
  
-   Ortak dil çalışma zamanı yüklemenizin bozuk.  
  
-   Ayrıca ortak dil çalışma zamanı'nda bulunan bir türü tanımlayan derleme başvurusu var. Ancak, derlemenin türü C# Derleyici bekliyor yolu tanımlı değil.  
  
 Ortak dil çalışma zamanı doğru sürümünü kullandığınızdan emin olmak için başvuruları denetleyin.
