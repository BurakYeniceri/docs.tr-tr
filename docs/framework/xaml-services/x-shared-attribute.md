# <a name="xshared-attribute"></a><span data-ttu-id="b3e62-102">x:Shared Özniteliği</span><span class="sxs-lookup"><span data-stu-id="b3e62-102">x:Shared Attribute</span></span>
<span data-ttu-id="b3e62-103">Ayarlandığında `false`, böylece istekleri öznitelikli kaynak için tüm istekler için aynı örneği paylaşmak yerine her istek için yeni bir örnek oluşturmak WPF kaynak alma davranışını değiştirir.</span><span class="sxs-lookup"><span data-stu-id="b3e62-103">When set to `false`, modifies WPF resource-retrieval behavior so that requests for the attributed resource create a new instance for each request instead of sharing the same instance for all requests.</span></span>

```xaml

  <object x:Shared="false".../>  
</ResourceDictionary>  
```
