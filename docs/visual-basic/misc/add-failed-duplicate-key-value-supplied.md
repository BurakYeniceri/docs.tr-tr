# <a name="add-failed-duplicate-key-value-supplied"></a>Ekleme başarısız. Yinelenen anahtar değeri sağlandı
Bir `Add` anahtar değeri başka bir anahtar değeri ile aynı olduğundan işlem başarısız oldu. Anahtar değerlerinin benzersiz olması gerekir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Anahtar değeri benzersiz olduğundan emin olun.
