# <a name="compiler-error-cs0415"></a><span data-ttu-id="ce702-102">Derleyici Hatası CS0415</span><span class="sxs-lookup"><span data-stu-id="ce702-102">Compiler Error CS0415</span></span>
<span data-ttu-id="ce702-103">Açık arabirim üye bildirimi olmayan bir oluşturucuda 'IndexerName' özniteliği geçerlidir</span><span class="sxs-lookup"><span data-stu-id="ce702-103">The 'IndexerName' attribute is valid only on an indexer that is not an explicit interface member declaration</span></span>  
  
 <span data-ttu-id="ce702-104">Açık arabirim uygulaması olan bir dizin oluşturucu IndexerName özniteliği kullanırsanız, bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="ce702-104">This error occurs if you use an IndexerName attribute on an indexer that was an explicit implementation of an interface.</span></span> <span data-ttu-id="ce702-105">Bu hata, arabirim adı dizin oluşturucu bildiriminden mümkünse kaldırarak kalmayabilir.</span><span class="sxs-lookup"><span data-stu-id="ce702-105">This error may be avoided by removing the interface name from the declaration of the indexer, if possible.</span></span> <span data-ttu-id="ce702-106">Daha fazla bilgi için bkz: [IndexerNameAttribute sınıfı](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="ce702-106">For more information, see the [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="ce702-107">Aşağıdaki örnek CS0415 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ce702-107">The following sample generates CS0415:</span></span>  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
