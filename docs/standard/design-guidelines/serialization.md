 <span data-ttu-id="f9100-165">**✓ DÜŞÜNÜN** seri hale getirme işlemi üzerinde tam denetimi istiyorsanız çalışma zamanı seri hale getirilebilir düzeni uygulama.</span><span class="sxs-lookup"><span data-stu-id="f9100-165">**✓ CONSIDER** implementing the Runtime Serializable Pattern if you want complete control over the serialization process.</span></span> <span data-ttu-id="f9100-166">Örneğin, verileri olarak dönüştürmek istiyorsanız seri durumdan veya serileştirilir.</span><span class="sxs-lookup"><span data-stu-id="f9100-166">For example, if you want to transform data as it gets serialized or deserialized.</span></span>  
  
 <span data-ttu-id="f9100-167">Düzen çok basit yöneliktir.</span><span class="sxs-lookup"><span data-stu-id="f9100-167">The pattern is very simple.</span></span> <span data-ttu-id="f9100-168">Yapmanız gereken tek şey uygulamak <xref:System.Runtime.Serialization.ISerializable> arabirim ve nesnenin seri durumda olduğunda kullanılan özel bir oluşturucu sağlayın.</span><span class="sxs-lookup"><span data-stu-id="f9100-168">All you need to do is implement the <xref:System.Runtime.Serialization.ISerializable> interface and provide a special constructor that is used when the object is deserialized.</span></span>  
  
 <span data-ttu-id="f9100-169">**✓ YAPMAK** korumalı serileştirme Oluşturucusu oluşturarak yazılan ve tam olarak burada aşağıdaki örnekte gösterildiği gibi adlı iki parametre sağlayabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="f9100-169">**✓ DO** make the serialization constructor protected and provide two parameters typed and named exactly as shown in the sample here.</span></span>  
  
```  
[Serializable]  
public class Person : ISerializable {  
    protected Person(SerializationInfo info, StreamingContext context) {  
        ...  
    }  
}  
```  
  
 <span data-ttu-id="f9100-170">**✓ YAPMAK** uygulamak `ISerializable` üyeleri açıkça.</span><span class="sxs-lookup"><span data-stu-id="f9100-170">**✓ DO** implement the `ISerializable` members explicitly.</span></span>  
  
 <span data-ttu-id="f9100-171">**✓ YAPMAK** bir bağlantı isteği uygulamak <xref:System.Runtime.Serialization.ISerializable.GetObjectData%2A?displayProperty=nameWithType> uygulaması.</span><span class="sxs-lookup"><span data-stu-id="f9100-171">**✓ DO** apply a link demand to <xref:System.Runtime.Serialization.ISerializable.GetObjectData%2A?displayProperty=nameWithType> implementation.</span></span> <span data-ttu-id="f9100-172">Bu, yalnızca tam olarak çekirdek ve çalışma zamanı seri hale getirici üye erişimi güvenilir sağlar.</span><span class="sxs-lookup"><span data-stu-id="f9100-172">This ensures that only fully trusted core and the Runtime Serializer have access to the member.</span></span>
