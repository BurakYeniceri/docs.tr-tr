 <span data-ttu-id="0d0a3-104">Bir sınıf, yapı veya arabirim bildirilen bir yöntemi, açık bir dönüş türüne sahip olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="0d0a3-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="0d0a3-105">Aşağıdaki örnekte, kare yöntemin dönüş değerine sahiptir. [dize](../../csharp/language-reference/keywords/string.md):</span><span class="sxs-lookup"><span data-stu-id="0d0a3-105">In the following example, the Square method has a return value of [string](../../csharp/language-reference/keywords/string.md):</span></span>

    string IntToString(int i)  
    {  
        return i.ToString();

 <span data-ttu-id="0d0a3-106">Aşağıdaki örnek, CS1520 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0d0a3-106">The following sample generates CS1520:</span></span>  
  
```csharp

   {  
   }

```
