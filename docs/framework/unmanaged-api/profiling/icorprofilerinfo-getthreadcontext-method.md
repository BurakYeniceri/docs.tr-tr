# <a name="icorprofilerinfogetthreadcontext-method"></a>ICorProfilerInfo::GetThreadContext Metodu
Şu anda belirtilen iş parçacığı ile ilişkili bağlam kimliğini alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT GetThreadContext(  
    [in]  ThreadID  threadId,  
    [out] ContextID *pContextId);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `threadId`  
 [in] İş parçacığı kimliği.  
  
 `pContextId`  
 [out] Şu anda belirtilen iş parçacığı ile ilişkili içerik kimliği için bir işaretçi. İş parçacığı şu anda ilişkili bağlam varsa, bu işlev CORPROF_E_DATAINCOMPLETE döndürür.
