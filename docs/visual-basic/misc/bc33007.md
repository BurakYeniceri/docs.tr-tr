# <a name="39end-operator39-must-be-preceded-by-a-matching-39operator39"></a>&#39;End işleci&#39; eşleşen bir gelmelidir &#39;işleci&#39;
Bir `End Operator` deyimi görünüyor eşleşme ile `Operator` kendisinden bildirimi.  
  
 **Hata Kimliği:** BC33007  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `End Operator` yedek ise deyimi.  
  
-   Eksik tedarik `Operator` eksikse yordamı.  
  
-   Taşıma `End Operator` kodu uygun bir yerinde ifadesine.
