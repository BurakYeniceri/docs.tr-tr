# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-accepts-this-number-of-arguments"></a>Aşırı yükleme çözümü başarısız oldu, çünkü Hayır erişilebilir &#39; &lt;yöntemi&gt; &#39; bu sayıda bağımsız değişken kabul eder
Aşırı yüklenmiş yöntemine bir çağrı yapılır, ancak derleyici eşleşen bir bağımsız değişken listesi olan bir yöntem bulamıyor.  
  
 **Hata Kimliği:** BC30516  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Onay, sağladığınız bağımsız değişken listesi.
