Bir erişim değeri Append modunda geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Değeri ya da değiştirin `OpenAccess.Write` veya `OpenAccess.Default`.
