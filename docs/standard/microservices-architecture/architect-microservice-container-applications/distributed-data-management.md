## <a name="challenge-4-how-to-design-communication-across-microservice-boundaries"></a><span data-ttu-id="c992b-176">Sınama \#4: mikro hizmet sınırları boyunca iletişimi tasarlamak nasıl</span><span class="sxs-lookup"><span data-stu-id="c992b-176">Challenge \#4: How to design communication across microservice boundaries</span></span>
