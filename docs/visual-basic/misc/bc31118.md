# <a name="39end-addhandler39-must-be-the-first-statement-on-a-line"></a>&#39; Son AddHandler &#39; satırındaki ilk deyim olmalıdır
Bir `End AddHandler` deyimi izleyen iki nokta üst üste (:) deyimi ayırıcı. `End AddHandler`Kaynak satırında yegane deyim olmalıdır.  
  
 **Hata Kimliği:** BC31118  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Birden çok deyime farklı satırlarına ayırın.
