# <a name="accessing-application-settings-visual-basic"></a>Uygulama ayarları (Visual Basic) erişme
Bu bölüm açıklayan konuları içerir `My.Settings` nesne ve görevleri, gerçekleştirmenize olanak sağlar.  
  
## <a name="mysettings"></a>My.Settings  
 Özelliklerini `My.Settings` nesne uygulamanızın ayarlarına erişim sağlar. Ayarlar eklemek veya kaldırmak için kullanın **ayarları** bölmesinde **Proje Tasarımcısı**.  
  
 Yöntemlerinin `My.Settings` nesne geçerli kullanıcı ayarlarını kaydetmek veya son kaydedilen değerlerini kullanıcı ayarlarına döndürmek izin.
