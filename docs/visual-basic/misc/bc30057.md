# <a name="too-many-arguments-to-39ltprocedurenamegt39"></a>Çok fazla bağımsız değişken &#39; &lt;procedurename&gt;&#39;
Bir yordam çağrısında sağlanan bağımsız değişken sayısı yordam için tanımlanan bağımsız değişken sayısı aşıyor.  
  
 **Hata Kimliği:** BC30057  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yordamı tanımı başvurun ve ek bağımsız değişkenler çağrısından kaldırın.
