# <a name="compiler-error-cs0516"></a>Derleyici Hatası CS0516
Oluşturucu 'Oluşturucusu' kendisini çağırabilirsiniz değil  
  
 Bir program yinelemeli olarak çağrı oluşturucular olamaz.  
  
 Aşağıdaki örnek CS0516 oluşturur:  
  
```csharp  
// CS0516.cs  
namespace x  
{  
   public class clx  
   {  
      public clx() : this()   // CS0516, delete "this()"  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
