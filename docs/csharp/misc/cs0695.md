# <a name="compiler-error-cs0695"></a><span data-ttu-id="234f2-102">Derleyici Hatası CS0695</span><span class="sxs-lookup"><span data-stu-id="234f2-102">Compiler Error CS0695</span></span>
<span data-ttu-id="234f2-103">Bazı tür parametresi değiştirmeleri için bütünleştirin çünkü 'genel tür' hem ' genel ' ve 'genel arabirimlerinden' uygulayamaz</span><span class="sxs-lookup"><span data-stu-id="234f2-103">'generic type' cannot implement both 'generic interface' and 'generic interface' because they may unify for some type parameter substitutions</span></span>  
  
 <span data-ttu-id="234f2-104">Genel bir sınıf aynı genel arabiriminin birden fazla parametrelemeyi uygular ve iki arabirimleri aynı yapacağı bir tür parametre değiştirme vardır, bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="234f2-104">This error occurs when a generic class implements more than one parameterization of the same generic interface, and there exists a type parameter substitution which would make the two interfaces identical.</span></span> <span data-ttu-id="234f2-105">Bu hatayı önlemek için yalnızca arabirimlerinden biri, uygulama veya çakışmayı önlemek için tür parametreleri değiştirin.</span><span class="sxs-lookup"><span data-stu-id="234f2-105">To avoid this error, implement only one of the interfaces, or change the type parameters to avoid the conflict.</span></span>  
  
 <span data-ttu-id="234f2-106">Aşağıdaki örnek CS0695 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="234f2-106">The following sample generates CS0695:</span></span>  
  
```csharp  
// CS0695.cs  
// compile with: /target:library  
  
interface I<T>  
{  
}  
  
class G<T1, T2> : I<T1>, I<T2>  // CS0695  
{  
}  
```
