| İŞLETİM SİSTEMİ | Visual Studio tercih | Visual Studio Code tercih | Bir komut satırı tercih |
| -- |------------------------|--------------------------|-----------------------------|-------------------------|

| Linux | Yok | [VSCode ve Ionide kullanmaya başlama](get-started-vscode.md) | [.NET Core CLI ile çalışmaya başlama](get-started-command-line.md) |

Genel olarak, kalan iyi olan başlamak için belirli yolu yoktur.  F # şeyleri görmek için makinenizde kullanmak için tüm yolları çalışırken öneririz en iyi!
