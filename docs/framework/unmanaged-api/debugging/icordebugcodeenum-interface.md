# <a name="icordebugcodeenum-interface1"></a>Icordebugcodeenum Interface1

|Yöntem|Açıklama|  
|------------|-----------------|  
|[Next Yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugcodeenum-next-method.md)|Belirtilen sayıda alır `ICorDebugCode` geçerli konumdan başlayarak numaralandırması örneklerden.|
