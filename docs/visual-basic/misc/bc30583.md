# <a name="39ltmethodnamegt39-cannot-be-implemented-more-than-once"></a>&#39;&lt;MethodName&gt; &#39; birden çok kez uygulanmadı
Zaten uygulanan bir yöntem uygulamak denediniz.  
  
 **Hata Kimliği:** BC30583  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Doğru yazıldığından emin olmak için yöntem adı denetleyin.  
  
2.  Yöntemin gereksiz ise, ek bildirimini kaldırın.
