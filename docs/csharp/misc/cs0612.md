# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="58c6a-102">Derleyici Uyarısı (düzey 1) CS0612</span><span class="sxs-lookup"><span data-stu-id="58c6a-102">Compiler Warning (level 1) CS0612</span></span>
<span data-ttu-id="58c6a-103">'member' kullanılmıyor</span><span class="sxs-lookup"><span data-stu-id="58c6a-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="58c6a-104">Sınıf Tasarımcısı sahip bir üye olarak işaretlenmiş [Kullanımdan kalktı](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) özniteliği.</span><span class="sxs-lookup"><span data-stu-id="58c6a-104">The class designer marked a member with the [Obsolete](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute.</span></span> <span data-ttu-id="58c6a-105">Bu üye sınıfı gelecek bir sürümünde desteklenmeyebilir anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="58c6a-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="58c6a-106">Aşağıdaki örnek, geçersiz bir üye erişme CS0612 nasıl oluşturur gösterir:</span><span class="sxs-lookup"><span data-stu-id="58c6a-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
