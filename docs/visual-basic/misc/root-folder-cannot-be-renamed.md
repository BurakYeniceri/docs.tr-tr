# <a name="root-folder-cannot-be-renamed"></a>Kök klasör adı değiştirilemez
Bir kök dizini yeniden adlandırmak için girişimde bulunuldu. Bu yanlış ayrıştırılmış dosya yolu sonucu olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yolun doğru olduğundan emin olmak için kontrol edin.
