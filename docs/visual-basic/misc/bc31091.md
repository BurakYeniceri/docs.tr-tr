# <a name="import-of-type-39lttypenamegt39-from-assembly-or-module-39ltnamegt39-failed"></a>İçeri aktarma türü &#39; &lt;typename&gt; &#39; derlemesi veya modülünden &#39; &lt;adı&gt; &#39; başarısız oldu
Tüm başvurulan kitaplıkları birbiriyle uyumlu olduğundan emin olun.  
  
 **Hata Kimliği:** BC31091  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Başvurulan tür içeren dosyaları bozuk olmadığını doğrulayın.
