# <a name="compiler-error-cs1019"></a><span data-ttu-id="07d05-102">Derleyici Hatası CS1019</span><span class="sxs-lookup"><span data-stu-id="07d05-102">Compiler Error CS1019</span></span>
<span data-ttu-id="07d05-103">Aşırı yüklenebilir birli işleç bekleniyor</span><span class="sxs-lookup"><span data-stu-id="07d05-103">Overloadable unary operator expected</span></span>  
  
 <span data-ttu-id="07d05-104">Bir şey, fazla yüklenmiş tekli işleç bildirilmiş, ancak işleci eksik veya yanlış konumda imzayı gibi görünüyor.</span><span class="sxs-lookup"><span data-stu-id="07d05-104">Something that looks like an overloaded unary operator has been declared, but the operator is missing or is in the wrong location in the signature.</span></span>  
  
 <span data-ttu-id="07d05-105">A *birli işleç* tek bir işlenen üzerinde çalışır işlecidir.</span><span class="sxs-lookup"><span data-stu-id="07d05-105">A *unary operator* is an operator that operates on a single operand.</span></span> <span data-ttu-id="07d05-106">Örneğin, `++` birli işleç.</span><span class="sxs-lookup"><span data-stu-id="07d05-106">For example, `++` is a unary operator.</span></span> <span data-ttu-id="07d05-107">Kullanarak bazı birli işleçler aşırı yükleyebilirler `operator` anahtar sözcüğü ve işlecin üzerinde işlem türünde tek bir parametre belirterek.</span><span class="sxs-lookup"><span data-stu-id="07d05-107">You can overload some unary operators by using the `operator` keyword and specifying a single parameter of the type that the operator operates on.</span></span> <span data-ttu-id="07d05-108">Örneğin, işlecin aşırı yüklemek istiyorsanız `++` kullanıcı tanımlı bir sınıf için `Temperature` yazabileceğiniz böylece `Temperature++`, bu yolla tanımlayabilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="07d05-108">For example, if you want to overload the operator `++` for a user-defined class `Temperature` so that you can write `Temperature++`, you can define it in this way:</span></span>  
  
```csharp  
public static  Temperature operator ++ (Temperature temp)  
{  
    temp.Degrees++;  
    return temp;  
}  
```  
  
 <span data-ttu-id="07d05-109">Bu hata iletisini aldığınızda işleci eksik veya yanlış konumda imzayı dışında aşırı yüklenmiş birli operatör gibi görünen bir şey bildirilen.</span><span class="sxs-lookup"><span data-stu-id="07d05-109">When you receive this error, you have declared something that looks like an overloaded unary operator, except that the operator itself is missing or is in the wrong location in the signature.</span></span> <span data-ttu-id="07d05-110">Kaldırırsanız `++` önceki örnekte imzasının CS1019 oluşturacaktır.</span><span class="sxs-lookup"><span data-stu-id="07d05-110">If you remove the `++` from the signature in the previous example, you will generate CS1019.</span></span>  
  
 <span data-ttu-id="07d05-111">Aşağıdaki kod CS1019 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="07d05-111">The following code generates CS1019:</span></span>  
  
```csharp  
// CS1019.cs  
public class ii  
{  
   int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class a  
{  
    public int i;  
// Generates CS1019: "ii" is not a unary operator.  
   public static a operator ii(a aa)     
  
   // Use the following line instead:  
   //public static a operator ++(a aa)  
   {  
      aa.i++;  
      return aa;   
   }  
  
   public static void Main()  
   {  
   }  
}  
```
