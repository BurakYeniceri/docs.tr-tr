# <a name="39sub-main39-was-not-found-in-39ltnamegt39"></a>&#39;Sub Main&#39; bulunamadı &#39; &lt;adı&gt;&#39;
`Sub Main` olan eksik veya yanlış konumu belirtilmedi için.  
  
 **Hata Kimliği:** BC30420  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Eksik tedarik `Sub Main` deyimi veya varsa, uygun konuma kod. Daha fazla bilgi için `Sub Main`, bkz: [Visual Basic'de ana yordam](../../../visual-basic/programming-guide/program-structure/main-procedure.md).  
  
2.  Projenin başlangıç nesnesinde konumunu belirtin **başlangıç formu** kutusunun **Proje Tasarımcısı**.
