<span data-ttu-id="ad2f5-103">Uzantı yöntemindeki tür parametrelerinin veri türleri '\<methodname >' tanımlanan '\<typename >' birden çok tür mümkün olduğundan bu bağımsız değişkenlerden çıkarsanamıyor.</span><span class="sxs-lookup"><span data-stu-id="ad2f5-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="ad2f5-104">Verileri belirten türlerini açıkça bu hatayı düzeltmek.</span><span class="sxs-lookup"><span data-stu-id="ad2f5-104">Specifying the data type(s) explicitly might correct this error.</span></span>  
  
 <span data-ttu-id="ad2f5-105">Tür çıkarımı genel genişletme yöntemi çağrıda tür parametresi (veya parametreleri) türü (veya türleri) belirlemek için kullanmak üzere bir girişiminde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="ad2f5-105">An attempt has been made to use type inference to determine the type (or types) of the type parameter (or parameters) in a call to a generic extension method.</span></span> <span data-ttu-id="ad2f5-106">Bir veya daha fazla tür parametreleri için birden fazla olası veri türü derleyici bulur ve bu hata bildirir.</span><span class="sxs-lookup"><span data-stu-id="ad2f5-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>

 <span data-ttu-id="ad2f5-108">Aşağıdaki kod hata gösterir.</span><span class="sxs-lookup"><span data-stu-id="ad2f5-108">The following code demonstrates the error.</span></span>

    Sub Main()  
  
        Dim caller As New Class1  
        '' Not valid.  
        'caller.targetExtension(1, "2")  
  
    End Sub

    Sub targetExtension(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)

    End Class
