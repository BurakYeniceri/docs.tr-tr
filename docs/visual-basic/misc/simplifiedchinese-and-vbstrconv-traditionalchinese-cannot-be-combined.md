# <a name="simplifiedchinese-and-vbstrconvtraditionalchinese-cannot-be-combined"></a>SimplifiedChinese ve VbStrConv.TraditionalChinese birleştirilemez
Uygulamanızı birleştirmek çalışıyor `VbStrConv` numaralandırma üyeleri `SimplifiedChinese` ve `TraditionalChinese`, hangi karşılıklı olarak birbirini dışlar.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ya da kaldırma `VbStrConv.SimplifiedChinese` veya `VbStrConv.TraditionalChinese`.
