# <a name="compiler-error-cs0021"></a><span data-ttu-id="bfe90-102">Derleyici Hatası CS0021</span><span class="sxs-lookup"><span data-stu-id="bfe90-102">Compiler Error CS0021</span></span>

 <span data-ttu-id="bfe90-104">Aracılığıyla bir dizin oluşturucu desteği olmayan bir veri türünde bir değer erişmek için bir girişimde bulunuldu [dizin oluşturucular](../../csharp/programming-guide/indexers/index.md).</span><span class="sxs-lookup"><span data-stu-id="bfe90-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../../csharp/programming-guide/indexers/index.md).</span></span>

## <a name="example"></a><span data-ttu-id="bfe90-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="bfe90-108">Example</span></span>  
 <span data-ttu-id="bfe90-109">Bu dosya bir .dll dosyasını derler — ile `DefaultMember` özniteliği geçersiz kılınan çıkışı — hata oluşturmak için.</span><span class="sxs-lookup"><span data-stu-id="bfe90-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="bfe90-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="bfe90-110">Example</span></span>

```csharp  
{  
    public static void Main()

        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```
