# <a name="copying-document-fragments"></a>Belge parçaları kopyalama
Oluşturabileceğiniz bir **XmlDocumentFragment** düğümünü ve ardından düğümler altındaki ekleyin. Zaman **XmlDocumentFragment** ile eklenen **InsertNode** yöntemi, **XmlDocumentFragment** düğümü değil kopyalanır, ancak alt düğümlerinden XML'de eklenir Belge nesne modeli (DOM).
