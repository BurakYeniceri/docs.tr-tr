# <a name="xml-comment-cannot-be-applied-more-than-once-on-a-partial-lttypegt"></a>XML açıklama uygulanamaz birden çok kez üzerinde kısmi &lt;türü&gt;
XML açıklama uygulanamaz birden çok kez üzerinde kısmi \<türü >. XML açıklamaları bu \<türü > göz ardı edilir.  
  
 Bir XML açıklama bloğu kısmi türü yalnızca bir kısmını yer alabilir.  
  
 Bir XML açıklama bloğu kısmi bir türde birden fazla bölümü görünüyorsa, bu uyarı için her açıklama bloğu oluşturulur ve en üst düzey XML açıklama yoksayılacak.  
  
 **Hata Kimliği:** BC42314  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yabancı açıklama blokları kaldırın.
