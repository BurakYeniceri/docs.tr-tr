# <a name="shared-39sub-new39-cannot-have-any-parameters"></a>Paylaşılan &#39; Yeni alt &#39; herhangi bir parametre sahip olamaz
Bir parametre veya parametreleri bir paylaşılan dahil `Sub New` deyimi geçerli değil.  
  
 **Hata Kimliği:** BC30479  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Parametresi veya parametrelerini kaldırın.
