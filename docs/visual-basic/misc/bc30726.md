# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>&#39; İle &#39; bağlamları ve deyimleri hata ayıklama pencerelerinde geçerli değildir
`With`deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30726  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `With` kodda hata ayıklama gelen deyimleri.
