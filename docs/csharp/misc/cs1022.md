# <a name="compiler-error-cs1022"></a>Derleyici Hatası CS1022
Türü veya ad alanı tanımını ya da dosya sonu bekleniyor  
  
 Kaynak kodu dosyasının eşleşen küme parantezleri kümesi yok.  
  
 Aşağıdaki örnek CS1022 oluşturur:  
  
```  
// CS1022.cs  
namespace x  
{  
}  
}   // CS1022  
```
