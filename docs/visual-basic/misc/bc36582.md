# <a name="too-many-arguments-to-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Genişletme yöntemi için çok fazla bağımsız değişken &#39; &lt;methodname&gt; &#39; tanımlanan &#39; &lt;typename&gt;&#39;
Bir genişletme yöntemi çağrısına sağlanan bağımsız değişken sayısı yöntemi için tanımlanan bağımsız değişken sayısı aşıyor.  
  
 **Hata Kimliği:** BC36582  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Genişletme yöntemi tanımını başvurun ve ek bağımsız değişkenler çağrısından kaldırın.
