# <a name="compiler-error-cs1519"></a><span data-ttu-id="788a9-102">Derleyici Hatası CS1519</span><span class="sxs-lookup"><span data-stu-id="788a9-102">Compiler Error CS1519</span></span>
<span data-ttu-id="788a9-103">Geçersiz belirteç sınıf, yapı veya arabirim üye bildiriminde ' belirteci'</span><span class="sxs-lookup"><span data-stu-id="788a9-103">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="788a9-104">Bu hata, bir belirteç ait bir konumda karşılaşıldığında oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="788a9-104">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="788a9-105">A *belirteci* bir anahtar sözcük; bir tanımlayıcı (sınıf, yapı, yöntemi ve benzeri adı); bir dize, karakter veya 108 gibi sayısal sabit değer "Hello", veya 'Bir'; veya bir işleç veya noktalama işaretçisi gibi `==` veya `;`.</span><span class="sxs-lookup"><span data-stu-id="788a9-105">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="788a9-106">Tüm [sınıfı](../../../csharp/language-reference/keywords/class.md), yapı ya da geçersiz değiştiriciler türü önce içerir arabirimi üye bildirimi bu hata üretir.</span><span class="sxs-lookup"><span data-stu-id="788a9-106">Any [class](../../../csharp/language-reference/keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="788a9-107">Hatayı düzeltmek için geçersiz değiştiricileri kaldırın.</span><span class="sxs-lookup"><span data-stu-id="788a9-107">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="788a9-108">Geçerli olduğu yerlerde olmayan konumlarda belirteçleri yerleştirildiğinden aşağıdaki örnek beş yerde CS1519 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="788a9-108">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42   
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;   
// Generates CS1519 because of "checked" on void method:  
    checked void f4();     
  
// Generates CS1519 because of "num":  
    void f5(int a num){}        
  
// Generates CS1519 because of namespace inside class:  
    namespace;             
  
}  
```
