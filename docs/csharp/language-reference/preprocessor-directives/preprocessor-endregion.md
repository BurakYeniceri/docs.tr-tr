# <a name="endregion-c-reference"></a>#endregion (C# Başvurusu)
`#endregion`sonunu işaretler bir [#region](../../../csharp/language-reference/preprocessor-directives/preprocessor-region.md) bloğu. Örneğin:  
  
```csharp
#region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
