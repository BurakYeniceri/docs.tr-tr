### <a name="to-freeze-a-column-using-the-designer"></a><span data-ttu-id="ffac6-115">Tasarımcı kullanarak bir sütunu dondurmak için</span><span class="sxs-lookup"><span data-stu-id="ffac6-115">To freeze a column using the designer</span></span>  
  
1.  <span data-ttu-id="ffac6-116">Akıllı etiket karakteri tıklayın (![akıllı etiket karakteri](../../../../docs/framework/winforms/controls/media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) sağ üst köşesindeki <xref:System.Windows.Forms.DataGridView> denetlemek ve ardından **Edit Columns**.</span><span class="sxs-lookup"><span data-stu-id="ffac6-116">Click the smart tag glyph (![Smart Tag Glyph](../../../../docs/framework/winforms/controls/media/vs-winformsmttagglyph.gif "VS_WinFormSmtTagGlyph")) on the upper-right corner of the <xref:System.Windows.Forms.DataGridView> control, and then select **Edit Columns**.</span></span>  
  
2.  <span data-ttu-id="ffac6-117">Bir sütun seçin **seçili sütun** listesi.</span><span class="sxs-lookup"><span data-stu-id="ffac6-117">Select a column from the **Selected Columns** list.</span></span>  
  
3.  <span data-ttu-id="ffac6-118">İçinde **sütun özellikleri** kılavuzunu, ayarlamak <xref:System.Windows.Forms.DataGridViewColumn.Frozen%2A> özelliğine `true`.</span><span class="sxs-lookup"><span data-stu-id="ffac6-118">In the **Column Properties** grid, set the <xref:System.Windows.Forms.DataGridViewColumn.Frozen%2A> property to `true`.</span></span>
