# <a name="this-system-does-not-contain-support-for-the-locale-specified"></a>Bu sistem, belirtilen yerel ayar desteği içermiyor.
Kullanmaya çalıştığınız `VbStrConv` numaralandırma, ancak sisteminizi belirtilen bölge desteklemez.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Yerel ayar desteklemek için makine yükseltin veya uygulamayı desteklediği bir makinede çalıştırın.
