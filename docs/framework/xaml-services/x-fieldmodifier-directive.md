# <a name="xfieldmodifier-directive"></a><span data-ttu-id="9db44-102">x:FieldModifier Yönergesi</span><span class="sxs-lookup"><span data-stu-id="9db44-102">x:FieldModifier Directive</span></span>
<span data-ttu-id="9db44-103">Böylece adlandırılmış nesne başvuruları için alanları ile tanımlanmış XAML derleme davranışını değiştiren <xref:System.Reflection.TypeAttributes.Public?displayProperty=nameWithType> yerine erişim <xref:System.Reflection.TypeAttributes.NotPublic?displayProperty=nameWithType> varsayılan davranışı.</span><span class="sxs-lookup"><span data-stu-id="9db44-103">Modifies XAML compilation behavior so that fields for named object references are defined with <xref:System.Reflection.TypeAttributes.Public?displayProperty=nameWithType> access instead of the <xref:System.Reflection.TypeAttributes.NotPublic?displayProperty=nameWithType> default behavior.</span></span>
