# <a name="compiler-error-cs1509"></a>Derleyici Hatası CS1509
Başvurulan dosya 'dosya' bağımlılığı derleme değil; Kullan ' / addmodule' bunun yerine seçeneği  
  
 Çıktı dosyası (çıktı dosyası 1), üretilen kullanılan bir derlemede [/target: Module](../../csharp/language-reference/compiler-options/target-module-compiler-option.md) (bir derleme bildirimi yok), için belirtilen [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md). Bu nedenle, bir derlemeyi geçerli program derleme yerine sonuna ekleme, çıktı dosyasına 1 meta veri bilgileri geçerli program için derlemesine eklenir.
