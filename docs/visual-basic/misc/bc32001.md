# <a name="39ltkeywordgt39-is-not-valid-within-a-module"></a>&#39;&lt;anahtar sözcüğü&gt; &#39; modülü içinde geçerli değil
Bir anahtar sözcük ilgili sınıfın örneklerine gibi `Me` veya `MyBase`, içinde bir modül kullanılır. Modülleri devralma veya örnekleri gerekmez.  
  
 **Hata Kimliği:** BC32001  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Anahtar sözcüğünü kullanarak kod sınıf örneklerini içeriyorsa, bu, bir sınıf uygulama başına taşıyın.  
  
-   Anahtar sözcüğünü kullanarak kod modülüne geçerliyse, geçersiz anahtar sözcüğü kaldırın.
