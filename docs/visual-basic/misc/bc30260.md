# <a name="39ltnamegt39-is-already-declared-as-39ltdeclarationgt39-in-this-ltdeclarationspacegt"></a>&#39; &lt;adı&gt;&#39; olarak bildirilen &#39;&lt; bildirim&gt;&#39; bu &lt;declarationspace&gt;
İki programlama öğeleri (bir sınıf, modül, arabirim veya yapısı) aynı bildirim alanında aynı ada sahip bildirilir.  
  
 **Hata Kimliği:** BC30260  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bildirimleri birini farklı bildirim alanına taşıyın veya bildirilen öğeler birinin adını değiştirin.
