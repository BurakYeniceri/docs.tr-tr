# <a name="expression-too-complex"></a>İfade çok karmaşık
Kayan nokta ifade çok fazla iç içe alt ifadelerin içerir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İfade hata oluşmasını önlemek için gereken sayıda ayrı ifadelere bölün.
