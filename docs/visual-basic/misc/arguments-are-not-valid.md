# <a name="arguments-are-not-valid"></a>Bağımsız değişkenler geçerli değil
Geçersiz bağımsız değişkenler, bir işlev veya alt yordama geçirilen.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Neden geçersiz olduğunu belirlemek için bağımsız değişkenler denetleyin.  
  
2.  Bağımsız değişkenler hesaplamalar girilirse, hesaplamaları işlem değerlerini denetleyin.
