# <a name="microsofttransactionstransactionbridgecoordinatorstatemachinefinished"></a>Microsoft.Transactions.TransactionBridge.CoordinatorStateMachineFinished
Bir düzenleyici listelemesi için durum makinesinin tamamlanmış duruma geçti.  
  
## <a name="description"></a>Açıklama  
 Yerel işlem yöneticisi üstün düzenleyici listelemesi 2pc işleme tamamlandı düşündüğü, izlenen. Kaydı için sonuç, kabul edilen veya iptal edildi veya Forgotten olabilir. Yerel işlem yöneticisi salt okunur hazırlama sırasında oylarının, ayrıca izleneceğini.
