# <a name="39join39-expected"></a>&#39; Birleştirme &#39; Beklenen
A `Group Join` yan tümcesi olmadan belirtilmedi bir `Join` anahtar sözcüğü.  
  
 **Hata Kimliği:** BC36631  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ekleme `Join` anahtar `Group Join` yan tümcesi, aşağıdaki örnekte gösterildiği gibi:  
  
```vb  
Dim query = From var1 In collection1 _  
            Join var2 In collection2 _  
            Group Join var3 In collection3 _  
            On var2.ID Equals var3 Into Matches = Group _  
            On var1 Equals var2.ID _  
            Select JoinID = var1, var2.Name, Matches  
```
