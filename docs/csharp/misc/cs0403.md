## <a name="example"></a><span data-ttu-id="15904-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="15904-108">Example</span></span>  
 <span data-ttu-id="15904-109">Aşağıdaki örnek CS0403 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="15904-109">The following sample generates CS0403.</span></span>  
  
```csharp  
// CS0403.cs  
// compile with: /target:library  
class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class   
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
