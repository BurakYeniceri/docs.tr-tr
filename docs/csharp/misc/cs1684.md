 <span data-ttu-id="bb306-104">Bu hataya neden bir ad alanı içinde bir başvuruya yazacaktır türüne başvuran içinde ikinci bir ad alanı var, ancak türü yok.</span><span class="sxs-lookup"><span data-stu-id="bb306-104">This error can be caused by a reference inside one namespace referring to a type that it says exists inside a second namespace, but the type does not exist.</span></span> <span data-ttu-id="bb306-105">Örneğin, bu tür mydll.dll diyor `A` yourdll.dll içinde var ancak böyle bir tür yourdll.dll içinde yok.</span><span class="sxs-lookup"><span data-stu-id="bb306-105">For example, mydll.dll says that type `A` exists inside yourdll.dll, but no such type exists inside yourdll.dll.</span></span> <span data-ttu-id="bb306-106">Bu hatanın olası bir neden olan kullanmakta olduğunuz yourdll.dll sürümü çok eski olduğundan ve `A` henüz tanımlanmamış.</span><span class="sxs-lookup"><span data-stu-id="bb306-106">One possible cause of this error is that the version of yourdll.dll you are using is too old and `A` has not yet been defined.</span></span>

```csharp  
   public void Test() {}

```  
  
## <a name="example"></a><span data-ttu-id="bb306-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="bb306-109">Example</span></span>
