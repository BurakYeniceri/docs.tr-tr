# <a name="compiler-error-cs0077"></a><span data-ttu-id="d0ee9-102">Derleyici Hatası CS0077</span><span class="sxs-lookup"><span data-stu-id="d0ee9-102">Compiler Error CS0077</span></span>
<span data-ttu-id="d0ee9-103">Bir başvuru türü veya boş değer atanabilir tür işleci kullanılması gerekir ('int' olduğu bir null değer türü).</span><span class="sxs-lookup"><span data-stu-id="d0ee9-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="d0ee9-104">[Olarak](../../csharp/language-reference/keywords/as.md) işleci geçirildi bir [değer türü](../../csharp/language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="d0ee9-104">The [as](../../csharp/language-reference/keywords/as.md) operator was passed a [value type](../../csharp/language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="d0ee9-105">Çünkü `as` döndürebilir [null](../../csharp/language-reference/keywords/null.md), yalnızca geçirilebilen [başvuru türleri](../../csharp/language-reference/keywords/reference-types.md) veya boş değer atanabilir tür.</span><span class="sxs-lookup"><span data-stu-id="d0ee9-105">Because `as` can return [null](../../csharp/language-reference/keywords/null.md), it can only be passed [reference types](../../csharp/language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="d0ee9-106">Boş değer atanabilir türler hakkında daha fazla bilgi için bkz: [boş değer atanabilir türler](../../csharp/programming-guide/nullable-types/index.md).</span><span class="sxs-lookup"><span data-stu-id="d0ee9-106">For more information about nullable types, see [Nullable Types](../../csharp/programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="d0ee9-107">Aşağıdaki örnek CS0077 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="d0ee9-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
