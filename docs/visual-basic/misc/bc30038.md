# <a name="option-strict-on-prohibits-operands-of-type-object-for-operator-39ltoperatornamegt39"></a>Option Strict On yasaklar operatöre nesne türündeki işlenenler &#39; &lt;operatorname&gt;&#39;
Nesne değişkenleri için tanımlanan yalnızca işleçleri `Is` ve `TypeOf...Is`. Zaman `Option Strict` olan `On`, tüm işlenenleri verilen işleci için tanımlanan veri türlerinden birinde olmalıdır.  
  
 **Hata Kimliği:** BC30038  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Uygun tür dönüştürme işlevleri gibi kullandığınız `CInt` veya `CStr`, işlenen işleci için tanımlanan veri türlerine dönüştürmek için.
