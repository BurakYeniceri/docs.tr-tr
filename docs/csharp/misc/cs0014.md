# <a name="compiler-error-cs0014"></a>Derleyici Hatası CS0014
Gerekli dosya 'dosyası bulunamadı  
  
 Bir dosya derleyici tarafından gerekli ancak sistem üzerinde değil. Yolun doğru olduğundan emin olun. Visual Studio sistem dosyası dosyasıysa yüklemenizi onarın veya kaldırın ve Visual Studio tamamen yeniden gerekebilir.
