<span data-ttu-id="3c95e-103">Sabit değeri 'değeri' bir 'type' dönüştürülemiyor.</span><span class="sxs-lookup"><span data-stu-id="3c95e-103">Constant value 'value' cannot be converted to a 'type'.</span></span> <span data-ttu-id="3c95e-104">('unchecked' sözdizimi geçersiz kılmak için kullanın)</span><span class="sxs-lookup"><span data-stu-id="3c95e-104">(use 'unchecked' syntax to override)</span></span>  
  
 <span data-ttu-id="3c95e-105">Değeri, türü depolanamıyor bir değişkene bir değer atamak için girişimde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="3c95e-105">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="3c95e-106">Daha fazla bilgi için bkz: [türleri](../../csharp/programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="3c95e-106">For more information, see [Types](../../csharp/programming-guide/types/index.md).</span></span>  
  
 <span data-ttu-id="3c95e-107">Aşağıdaki örnek CS0031 checked ve unchecked bağlamlarda oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3c95e-107">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>  
  
```csharp  
// CS0031.cs  
namespace CS0031  
{  
   public class a  
   {  
      public static void Main()  
      {  
         int num = (int)2147483648M; //CS0031  
         // Try using a larger numeric type instead:  
         // long num = (long)2147483648M; //CS0031  
  
         const decimal d = -10M; // Decimal literal  
            unchecked  
            {  
                const byte b = (byte)d; // CS0031  
                // For small values try using a signed byte instead:  
                // const sbyte b = (sbyte)d;  
            }  
      }  
   }  
}  
```
