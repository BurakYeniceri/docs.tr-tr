# <a name="cast-from-type-39lttypename1gt39-to-type-39lttypename2gt39-is-not-valid"></a>Türünden cast &#39; &lt;typename1&gt; &#39; yazmak için &#39; &lt;typename2&gt; &#39; geçerli değil
Bir değişkenin bir türden diğerine dönüştürün girişimi geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Türleri dönüştürme söz konusu kontrol edin ve anlamlı bir türe dönüştürmek çalışıyorsunuz emin olun.
