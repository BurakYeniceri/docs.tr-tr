# <a name="compiler-error-cs0152"></a><span data-ttu-id="f480b-102">Derleyici Hatası CS0152</span><span class="sxs-lookup"><span data-stu-id="f480b-102">Compiler Error CS0152</span></span>
<span data-ttu-id="f480b-103">Etiket 'etiket' bu switch deyiminde zaten</span><span class="sxs-lookup"><span data-stu-id="f480b-103">The label 'label' already occurs in this switch statement</span></span>  
  
 <span data-ttu-id="f480b-104">Bir etiket yinelenir bir [geçiş](../../csharp/language-reference/keywords/switch.md) deyimi.</span><span class="sxs-lookup"><span data-stu-id="f480b-104">A label was repeated in a [switch](../../csharp/language-reference/keywords/switch.md) statement.</span></span> <span data-ttu-id="f480b-105">Daha fazla bilgi için bkz: [geçiş](../../csharp/language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="f480b-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="f480b-106">Aşağıdaki örnek CS0152 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f480b-106">The following sample generates CS0152:</span></span>  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
