# <a name="compiler-error-cs0447"></a><span data-ttu-id="97677-102">Derleyici Hatası CS0447</span><span class="sxs-lookup"><span data-stu-id="97677-102">Compiler Error CS0447</span></span>
<span data-ttu-id="97677-103">Tür parametreleri yalnızca üzerinde tür bağımsız değişkenleri öznitelikler kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="97677-103">Attributes cannot be used on type arguments, only on type parameters</span></span>  
  
 <span data-ttu-id="97677-104">Bu hata, bir çağrı deyiminde bir tür bağımsız değişkeni bir öznitelik uygulamak oluşur.</span><span class="sxs-lookup"><span data-stu-id="97677-104">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="97677-105">Bir sınıf veya yöntemin bildirimi deyiminde aşağıdaki gibi bir tür parametresi bir öznitelik uygulamak için kullanılabilir:</span><span class="sxs-lookup"><span data-stu-id="97677-105">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>  
  
```csharp  
class C<[some attribute] T> {…}  
```  
  
 <span data-ttu-id="97677-106">Aşağıdaki kod satırını bu hatasına neden olur.</span><span class="sxs-lookup"><span data-stu-id="97677-106">The following line of code will generate this error.</span></span> <span data-ttu-id="97677-107">Varsayılır sınıfı `C`, kod, önceki satırdaki tanımlı olan adlı bir statik yöntem `MyStaticMethod`.</span><span class="sxs-lookup"><span data-stu-id="97677-107">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>  
  
```csharp  
C<[some attribute] T>.MyStaticMethod();  
```  
  
## <a name="example"></a><span data-ttu-id="97677-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="97677-108">Example</span></span>  
 <span data-ttu-id="97677-109">Aşağıdaki kod hata CS0447 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="97677-109">The following code generates error CS0447.</span></span>  
  
```csharp  
// CS0447.cs  
using System;  
namespace Test41  
{  
    public interface I<A>   
    {  
        void Meth<B>();  
    }  
    public class B : I<int>   
    {  
        void I<[Test] int>.Meth<X>() { }  // CS0447  
    }  
}  
```
