# <a name="service-call-duration"></a>Hizmet: Çağrı Süresi
Sayaç adı: Çağrı süresi  
  
## <a name="description"></a>Açıklama  
 Bu hizmet yapılan çağrıların ortalama süresi. Ortalama süresi temel üzerinde bu denklemi hesaplanır: (N1-N0)/(D1-D0).
