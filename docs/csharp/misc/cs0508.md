# <a name="compiler-error-cs0508"></a><span data-ttu-id="19ac1-102">Derleyici Hatası CS0508</span><span class="sxs-lookup"><span data-stu-id="19ac1-102">Compiler Error CS0508</span></span>
<span data-ttu-id="19ac1-103">'Türü: 1': dönüş türü, 'Türü geçersiz kılınan üye 'Üye adı' eşleştirmek için 2' olmalıdır</span><span class="sxs-lookup"><span data-stu-id="19ac1-103">'Type 1': return type must be 'Type 2' to match overridden member 'Member Name'</span></span>  
  
 <span data-ttu-id="19ac1-104">Bir yöntemi geçersiz kılma dönüş türünü değiştirmek için girişimde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="19ac1-104">An attempt was made to change the return type in a method override.</span></span> <span data-ttu-id="19ac1-105">Bu hatayı gidermek için iki yöntem de aynı dönüş türü bildirme emin olun.</span><span class="sxs-lookup"><span data-stu-id="19ac1-105">To resolve this error, make sure both methods declare the same return type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="19ac1-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="19ac1-106">Example</span></span>  
 <span data-ttu-id="19ac1-107">Aşağıdaki örnek CS0508 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="19ac1-107">The following sample generates CS0508.</span></span>  
  
```  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
