## <a name="example"></a><span data-ttu-id="2fdbe-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="2fdbe-107">Example</span></span>  
 <span data-ttu-id="2fdbe-108">Aşağıdaki kod CS0754 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="2fdbe-108">The following code generates CS0754:</span></span>

    public interface IF  
    {  
        void Part();  
    }

        }  
    }
