# <a name="the-39ltkeywordgt39-keyword-is-used-to-overload-inherited-members-do-not-use-the-39ltkeywordgt39-keyword-when-overloading-39sub-new39"></a>&#39; &lt;anahtar sözcüğü&gt;&#39; kullanmayın; anahtar sözcüğü devralınan aşırı üyelerine kullanılan &#39;&lt; anahtar sözcüğü&gt;&#39; aşırı yüklendiğinde anahtar sözcüğü &#39; Yeni alt &#39;
Bir Oluşturucu ile bildirilmiş `Overloads` anahtar sözcüğü.  
  
 Visual Basic devralarak veya oluşturucular aşırı desteklemez.  
  
 **Hata Kimliği:** BC32040  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Overloads` tüm Oluşturucusu bildirimleri anahtar sözcük.
