# <a name="expected-matching-closing-single-quote-for-xml-attribute-value"></a>XML öznitelik değeri için tek bir teklif kapatma eşleşen bekleniyor
Kapatma eksik tek tırnak ile ayrılmış bir öznitelik değeri tırnak işareti karakteri.  
  
 **Hata Kimliği:** BC31163  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Öznitelik değeri sınırlandırmak için kapanış tek tırnak işareti karakteri (') ekleyin.
