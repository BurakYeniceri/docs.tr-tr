# <a name="systemservicemodelcommunicationobjectclosefailed"></a>System.ServiceModel.CommunicationObjectCloseFailed
İletişim nesnesi kapatılamadı.  
  
## <a name="description"></a>Açıklama  
 Close yöntemi yürütülmesi sırasında özel durum oluştu çünkü iletişim nesnesi kapatılamadı. İzleme yakındaki başka bir özel durumla ilgili ayrıntılar içerebilir.
