# <a name="39ltname1gt39-conflicts-with-a-lttypegt-by-the-same-name-defined-in-39ltname2gt39"></a>&#39; &lt;Ad1&gt;&#39; çakışıyor bir &lt;türü&gt; tarafından tanımlanan aynı adı &#39;&lt; ad2&gt;&#39;
Bir ad, aynı ada sahip bir türü ile çakışıyor karşılaşıldı.  
  
 **Hata Kimliği:** BC31073  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Benzersiz bir ad bildirin.  
  
2.  Çakışan üye adı köşeli ayraç kullanın.
