# <a name="transacted-operations-in-doubt-per-second"></a>Şüpheli Uygulanan İşlem/Saniye
Sayaç adı: Şüpheli uygulanan işlem / saniye.  
  
## <a name="description"></a>Açıklama  
 Bir saniye içinde şüpheli sonucu bu hizmette işlem işlemlerinin sayısı.  
  
 Bu sayaç performans sayacı türüdür [PERF_COUNTER_COUNTER](https://go.microsoft.com/fwlink/?LinkID=94649), değeri aşağıdaki formül kullanılarak hesaplanır.  
  
 (1 - N 0 N) / ((D 1 - D 0) / F)
