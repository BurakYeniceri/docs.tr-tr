# <a name="39460---trackingvaluenotserializable"></a>39460 - TrackingValueNotSerializable
## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|Kimlik|39460|  
|Anahtar Sözcükler|WFTracking|  
|Düzey|Uyarı|  
|Kanal|Microsoft Windows uygulaması sunucu-uygulamalar/hata ayıklama|  
  
## <a name="description"></a>Açıklama  
 Ayıklanan bağımsız değişken/değişken izleme kaydındaki seri hale getirilebilir olmadığını gösterir.  
  
## <a name="message"></a>İleti  
 Ayıklanan bağımsız değişken/değişkeni '%1' seri hale getirilebilir değil.
