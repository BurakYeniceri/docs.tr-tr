# <a name="properties-with-no-required-parameters-cannot-be-declared-39default39"></a>Zorunlu Parametreler özelliklerle bildirilemez &#39; varsayılan &#39;
Bir özelliği Gerekli Parametreler olmadan ancak ile belirtilen `Default` değiştiricisi.  
  
 **Hata Kimliği:** BC31048  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Varsayılan özellikler için gerekli parametreler bildirin.  
  
-   Kaldırma `Default` özelliği tanımından değiştiricisi.
