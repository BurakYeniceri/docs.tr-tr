# <a name="39lttypenamegt39-must-be-declared-39mustinherit39-because-it-contains-methods-declared-39mustoverride39"></a>&#39;&lt;TypeName&gt; &#39; bildirilmelidir &#39;MustInherit&#39; bildirilen yöntemler içerdiğinden &#39;MustOverride&#39;
İçeren türleri `MustOverride` üyeleri bildirilmelidir `MustInherit`.  
  
 **Hata Kimliği:** BC31411  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme `MustInherit` değiştiricisi türü.
