# <a name="compiler-error-cs0583"></a>Derleyici Hatası CS0583
Derleyici iç hatası. Derleyicide bir iç hata oluştu. Bu sorunu geçici olarak çözmek için aşağıda basitleştirme veya programın konumları yakın değiştirme deneyin listelenen. Listenin konumlarda iç hata gerçekleştiği noktasına daha yakın. / Errorreport seçeneğini kullanarak bu gibi hataları Microsoft'a bildirilebilir.  
  
 Lütfen hata raporu dosya oluştur ' / bugreport' seçeneği ve kişi raporlama atanan sorununuzu rapora gönderin.  
  
 Kullanım [/bugreport](../../csharp/language-reference/compiler-options/bugreport-compiler-option.md) sorun yakalamak için derleyici seçeneği.
