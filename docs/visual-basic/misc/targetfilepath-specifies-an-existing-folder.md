# <a name="targetfilepath-specifies-an-existing-folder"></a>Var olan bir klasörü TargetFilePath belirtir
Parametre `TargetFilePath` varolan bir dizin belirtiyor, ancak yeni dizinin adını belirtmeniz gerekir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değişiklik `TargetFilePath` yeni yerine var olan, bir dizini belirtmek için.
