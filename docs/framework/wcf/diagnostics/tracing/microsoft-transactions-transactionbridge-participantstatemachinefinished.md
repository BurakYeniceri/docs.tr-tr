# <a name="microsofttransactionstransactionbridgeparticipantstatemachinefinished"></a>Microsoft.Transactions.TransactionBridge.ParticipantStateMachineFinished

 Bağımlı bir katılımcı listesi 2pc işlemeyi tamamlandığında izlenen. Kaydı için sonuç, kabul edilen veya iptal edildi olabilir. Herhangi bir katılımcı ReadOnly hazırlama sırasında oylarının, ayrıca izleneceğini.
