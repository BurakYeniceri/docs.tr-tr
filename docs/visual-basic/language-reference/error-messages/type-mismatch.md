# <a name="type-mismatch-visual-basic"></a>Tür uyumsuzluğu (Visual Basic)
Bir değeri, geçerli olmayan bir şekilde başka bir türüne dönüştürme girişiminde bulunuldu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Atama geçerli olduğundan emin olmak için kontrol edin.  
  
2.  Tek bir özellik ya da değer bekliyor. bir yordamı bir nesne geçemedi emin olun.  
  
3.  Bir ifade beklenirken bir modül veya proje adı kullanmadı emin olun.
