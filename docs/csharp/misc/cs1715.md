# <a name="compiler-error-cs1715"></a><span data-ttu-id="b512f-102">Derleyici Hatası CS1715</span><span class="sxs-lookup"><span data-stu-id="b512f-102">Compiler Error CS1715</span></span>
<span data-ttu-id="b512f-103">'Type1': türü, 'Type2' olmalıdır geçersiz kılınan üye 'MemberName' eşleştirmek için</span><span class="sxs-lookup"><span data-stu-id="b512f-103">'Type1': type must be 'Type2' to match overridden member 'MemberName'</span></span>  
  
 <span data-ttu-id="b512f-104">Bu hata aynıdır [derleyici hatası CS0508](../../csharp/misc/cs0508.md)dışında CS0508 artık yalnızca dönüş türleri CS1715 'yerine dönüş türleri' özellikleri ve yalnızca 'türleri' sahip dizin oluşturucular uygularken olan yöntemler için de geçerlidir.</span><span class="sxs-lookup"><span data-stu-id="b512f-104">This error is the same as [Compiler Error CS0508](../../csharp/misc/cs0508.md), except that CS0508 now only applies to methods that have return types, while CS1715 applies to properties and indexers that only have 'types' instead of 'return types'.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b512f-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="b512f-105">Example</span></span>  
 <span data-ttu-id="b512f-106">Aşağıdaki kod CS1715 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="b512f-106">The following code generates CS1715.</span></span>  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
