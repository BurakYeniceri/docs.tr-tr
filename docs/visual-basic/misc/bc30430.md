# <a name="39end-function39-must-be-preceded-by-a-matching-39function39"></a>&#39; End işlevi &#39; eşleşen bir &#39;göre gelmelidir; İşlev &#39;
Bir `End Function` deyimi görünüyor, kodunuzda hiçbir eşleşen `Function` , önceki yordamı tanımı.  
  
 **Hata Kimliği:** BC30430  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kaldırma `End Function` yedek ise deyimi.  
  
2.  Eksik tedarik `Function` eksikse yordamı.  
  
3.  Taşıma `End Function` uygun yerleştirmek için kod.
