# <a name="iassemblynameclone-method"></a>IAssemblyName::Clone Yöntemi
Bu basit bir kopyasını oluşturur [Iassemblyname](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-interface.md) nesnesi.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT Clone (  
    [out] IAssemblyName **pName  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `pName`  
 [out] Bu döndürülen kopyasını `IAssemblyName` nesnesi.
