# <a name="permission-denied-visual-basic"></a>İzin reddedildi (Visual Basic)
Yazma korumalı bir diske yazma veya kilitli bir dosyayı erişmek için girişimde bulunuldu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Yazma korumalı bir dosyayı açmaya dosyasının yazma korumasını özniteliğini değiştirin.  
  
2.  Başka bir işlem dosyayı kilitlemiş değil emin olun ve diğer işlem bırakması kadar dosyayı açmak için bekleyin.  
  
3.  Kayıt defterine erişim için kullanıcı izinlerini bu tür kayıt defteri erişimi içerdiğini denetleyin.
