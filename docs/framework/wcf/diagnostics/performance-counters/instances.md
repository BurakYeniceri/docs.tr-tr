# <a name="instances"></a>Örnekler
Sayaç adı: örnekleri.  
  
## <a name="description"></a>Açıklama  
 Hizmet şu anda içeren örneği bağlamları sayısı.  
  
 Çoğu zaman, örnek bağlamları sayısı örnek sayısı için aynıdır. Ancak, aşağıdaki senaryolar bu kural için özel değildir.  
  
-   Bir hizmet yöntemini çağıran <xref:System.ServiceModel.Dispatcher.IInstanceProvider.ReleaseInstance%2A> yöntemi açıkça.  
  
-   A <xref:System.ServiceModel.ReleaseInstanceMode> uygulanan bir <xref:System.ServiceModel.OperationBehaviorAttribute> örneği.
