# <a name="delegate-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>Arabirim içindeki temsilci bildirilemez &#39; &lt;belirticisi&gt;&#39;
Geçersiz değiştirici bir arabirim, bildirilen bir temsilci ile belirtildi.  
  
 **Hata Kimliği:** BC31068  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Değiştiriciler gibi kaldırmak `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, ve `Overridable`.
