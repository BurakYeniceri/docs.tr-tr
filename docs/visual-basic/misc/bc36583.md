# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-omitted-argument"></a>Parametre &#39; &lt;parametername&gt;&#39; genişletme yöntemi &#39;&lt; MethodName&gt;&#39; tanımlıysa &#39;&lt; TypeName&gt;&#39; eşleşen bir bağımsız değişken zaten atlandı
Bir genişletme yöntemi bir yordam çağrısına konuma göre bağımsız değişken atlar ve ada göre bağımsız değişken temin eder. Örneğin, aşağıdaki için uzantı metodu çağırma `ABC` ilk parametre için bir bağımsız değişken atlar `Y`, ada göre sağlar.  
  
```  
<Extension()> _  
Public Sub ABC(ByVal X As Integer, Optional ByVal Y As Byte = 0, _  
               Optional ByVal Z As Byte = 0)  
End Sub  
' . . .  
' Calling extension method ABC.  
Dim number As Integer  
' Not valid.  
' number.ABC(, 4, Y:=5)  
```  
  
 **Hata Kimliği:** BC36583  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Konuma göre bağımsız değişken sağlayın veya bu atlar virgülle kaldırın.
