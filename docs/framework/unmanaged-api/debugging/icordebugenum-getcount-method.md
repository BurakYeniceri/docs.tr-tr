# <a name="icordebugenumgetcount-method"></a>ICorDebugEnum::GetCount Metodu
Sabit listede öğe sayısını alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT GetCount (  
    [out] ULONG *pcelt  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `pcelt`  
 [out] Listedeki öğe sayısı için bir işaretçi.
