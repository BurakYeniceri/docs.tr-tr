# <a name="expected-matching-closing-double-quote-for-xml-attribute-value"></a>XML öznitelik değeri için çift tırnak kapatma eşleşen bekleniyor
Kapatma eksik çift tırnak işareti ayrılmış bir öznitelik değeri tırnak işareti karakteri.  
  
 **Hata Kimliği:** BC31164  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Öznitelik değeri sınırlandırmak için kapama çift tırnak işareti karakteri ekleyin.
