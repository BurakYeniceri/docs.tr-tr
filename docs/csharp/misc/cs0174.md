# <a name="compiler-error-cs0174"></a>Derleyici Hatası CS0174
Bir taban sınıfı için 'base' başvuru gereklidir  
  
 Bu hata, yalnızca .NET Framework ortak dil çalışma zamanı için kaynak kodunu zaman derleme oluşur `System.Object` hiçbir temel sınıf olan tek sınıf olan sınıf.
