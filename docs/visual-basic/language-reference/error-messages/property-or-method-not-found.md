# <a name="property-or-method-not-found"></a>Özellik veya yöntem bulunamadı
Başvurulan nesne yöntemi veya nesne özelliği tanımlı değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Nesne adı yanlış. Hangi özellikleri ve yöntemleri bir nesne için tanımlanan görmek için Nesne Tarayıcısı görüntüler. Kullanılabilir özellikler ve yöntemler listesini görüntülemek için uygun nesne kitaplığı seçin.
