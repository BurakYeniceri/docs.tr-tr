# <a name="the-value-of-ltargumentnamegt-must-be-greater-than-or-equal-to-1000"></a>Değeri &lt;argumentname&gt; 1000 veya daha büyük olmalıdır
Değeri <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> 1000 eşit veya daha büyük olmalıdır.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> Özelliği, yeni bir günlük dosyası oluşturmadan önce bir günlük dosyasında izin verilen bayt sayısını belirtir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ayarlama <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.MaxFileSize%2A> 1000 eşit veya daha büyük bir sayıya özelliği.
