# <a name="attribute-39ltattributenamegt39-cannot-be-applied-to-a-method-with-optional-parameters"></a>Öznitelik &#39; &lt;attributename&gt; &#39; isteğe bağlı parametreleri olan bir yöntem uygulanamaz
Öznitelik, yalnızca gerekli bağımsız veya bağımsız değişkenler kullanan yöntemleri kullanılabilir.  
  
 **Hata Kimliği:** BC30645  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  İsteğe bağlı parametreler olmadan yöntemi tanımlayın.  
  
2.  İsteğe bağlı parametreleri olan yöntemleri kullanılabilir bir özniteliğini kullanın.  
  
3.  Kullanılabilir özel bir öznitelik bu bağlamda tanımlayın.
