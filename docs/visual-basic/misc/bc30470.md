# <a name="39myclass39-cannot-be-used-outside-of-a-class"></a>&#39;MyClass&#39; bir sınıf dışında kullanılamaz
Kullandığınız `MyClass` anahtar sözcüğü bir sınıf yapısı dışında.  
  
 **Hata Kimliği:** BC30470  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Taşıma `MyClass` bildirimi sınıfı yapınızı içine.
