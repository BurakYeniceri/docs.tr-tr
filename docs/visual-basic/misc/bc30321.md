# <a name="39case39-cannot-follow-a-39case-else39-in-the-same-39select39-statement"></a>&#39;Servis talebi&#39; izleyemez bir &#39;Case Else&#39; aynı &#39;seçin&#39; deyimi
A `Case Else` deyimi tanıtır ilk değeri için eşleşme bulunamazsa, yürütülecek deyimleri `Case`. A `Case` deyimi, sonra bulundu bir `Case Else` aynı `Select` bloğu.  
  
 **Hata Kimliği:** BC30321  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Taşıma `Case Else` sonra uygun konuma `Case` deyimi.
