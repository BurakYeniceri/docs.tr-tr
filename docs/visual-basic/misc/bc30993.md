## <a name="example"></a><span data-ttu-id="c7f1f-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="c7f1f-110">Example</span></span>  
 <span data-ttu-id="c7f1f-111">Aşağıdaki sınıf üç özellik tanımları içerir: biri `TotalItems` ve iki `Item`, hangi aşırı yüklenmiş.</span><span class="sxs-lookup"><span data-stu-id="c7f1f-111">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="c7f1f-112">`TotalItems` Özelliği hiç bağımsız değişken gerektirir ve bir nesne başlatma listesinde aşağıdaki bildiriminde gösterildiği gibi başlatılabilir.</span><span class="sxs-lookup"><span data-stu-id="c7f1f-112">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
```  
  
 <span data-ttu-id="c7f1f-113">`Item` Özelliği aşırı yüklenmiş ve her aşırı bir bağımsız değişken gerektiriyor.</span><span class="sxs-lookup"><span data-stu-id="c7f1f-113">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="c7f1f-114">Bu nedenle, `Item` bir nesne Başlatıcı listesinde yer alamaz.</span><span class="sxs-lookup"><span data-stu-id="c7f1f-114">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="c7f1f-115">Bu hatayı önlemek için başlatma `Item` nesne Başlatıcı dışında özelliği.</span><span class="sxs-lookup"><span data-stu-id="c7f1f-115">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>  
  
```  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
coinCollection.Item(1) = aCoinObject  
```
