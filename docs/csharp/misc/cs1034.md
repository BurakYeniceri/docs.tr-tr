# <a name="compiler-error-cs1034"></a>Derleyici Hatası CS1034
Derleyici sınırı aşıldı: Satır 'numara' karakteri aşamaz  
  
 Bir satır üzerinde izin verilen karakter sayısı için üst sınıra 16,777,214 ' dir.
