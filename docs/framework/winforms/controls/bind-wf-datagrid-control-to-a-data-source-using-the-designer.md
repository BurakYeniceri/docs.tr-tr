### <a name="to-data-bind-the-datagrid-control-to-a-single-table-in-the-designer"></a><span data-ttu-id="4f998-124">Veri DataGrid denetimini Tasarımcısı'nda tek bir tabloya bağlamak için</span><span class="sxs-lookup"><span data-stu-id="4f998-124">To data-bind the DataGrid control to a single table in the designer</span></span>  
  
1.  <span data-ttu-id="4f998-125">Denetimin ayarlamak <xref:System.Windows.Forms.DataGrid.DataSource%2A> özelliği bağlamak istediğiniz veri öğeleri içeren nesne.</span><span class="sxs-lookup"><span data-stu-id="4f998-125">Set the control's <xref:System.Windows.Forms.DataGrid.DataSource%2A> property to the object containing the data items you want to bind to.</span></span>  
  
2.  <span data-ttu-id="4f998-126">Veri kaynağı bir veri kümesi ise <xref:System.Windows.Forms.DataGrid.DataMember%2A> özelliğini bağlamak için tablonun adı.</span><span class="sxs-lookup"><span data-stu-id="4f998-126">If the data source is a dataset, set the <xref:System.Windows.Forms.DataGrid.DataMember%2A> property to the name of the table to bind to.</span></span>  
  
3.  <span data-ttu-id="4f998-127">Veri kaynağı bir veri kümesi ya da bir veri kümesi tabloya dayalı bir veri görünümü ise, veri kümesini doldurmak üzere formuna kodu ekleyin.</span><span class="sxs-lookup"><span data-stu-id="4f998-127">If the data source is a dataset or a data view based on a dataset table, add code to the form to fill the dataset.</span></span>

    ```vb  
    sqlDataAdapter1.Fill(DsCategories1)  
    ```  
  
    ```csharp  
    sqlDataAdapter1.Fill(DsCategories1);  
    ```  
  
    ```cpp  
    sqlDataAdapter1->Fill(dsCategories1);  
    ```  
  
4.  <span data-ttu-id="4f998-130">(İsteğe bağlı) Sütun stilleri ve uygun tablo stilleri kılavuzuna ekleyin.</span><span class="sxs-lookup"><span data-stu-id="4f998-130">(Optional) Add the appropriate table styles and column styles to the grid.</span></span>
