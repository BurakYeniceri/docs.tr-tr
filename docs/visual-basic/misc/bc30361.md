# <a name="39ltproperty1gt39-and-39ltproperty2gt39-cannot-overload-each-other-because-only-one-is-declared-39default39"></a>&#39; &lt;property1&gt;&#39; ve &#39;&lt; property2&gt;&#39; yalnızca biri bildirildiğinden birbirini tekrar yükleyemez &#39; varsayılan &#39;
Bir özellik belirtiyorsa `Default`, o adı aşırı yüklenmiş tüm özellikleri de belirtmeniz gerekir `Default`.  
  
 **Hata Kimliği:** BC30361  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Aşırı yüklenmiş tüm özellikler bildirildiğinden emin olun `Default`.
