# <a name="other-control-structures-visual-basic"></a>Diğer Denetim Yapıları (Visual Basic)
Visual Basic yardımcı denetim yapıları bir kaynağını atma veya bir nesne başvurusu yinelemek zorunda sayısını azaltmak sağlar.  
  
## <a name="usingend-using-construction"></a>Kullanarak... Son yapı kullanma  
 `Using...End Using` Yapım kurar içinde yaptığınız deyimi blok SQL bağlantısı gibi bir kaynağın kullanın. İsteğe bağlı olarak ile kaynak elde edebilirsiniz `Using` deyimi. Çıktığınızda `Using` bloğu, Visual Basic böylece diğer kod kullanmak kullanılabilir kaynak otomatik olarak siler. Kaynak, yerel ve atılabilir olmalıdır. Daha fazla bilgi için bkz: [kullanarak deyimi](../../../../visual-basic/language-reference/statements/using-statement.md).  
  
## <a name="withend-with-construction"></a>İle... Yapı ile bitmelidir  
 `With...End With` Yapım nesne başvurusu bir kez belirtmenize olanak sağlar ve bir dizi üyelerine erişim deyimi çalıştırın. Bu, kodunuzu basitleştiren ve Visual Basic başvurusu eriştiği her deyim için yeniden oluşturmak sahip olmadığından performansı. Daha fazla bilgi için bkz: [ile... End With deyimi](../../../../visual-basic/language-reference/statements/with-end-with-statement.md).
