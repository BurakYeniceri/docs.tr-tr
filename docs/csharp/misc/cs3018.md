# <a name="compiler-warning-level-1-cs3018"></a>Derleyici Uyarısı (düzey 1) CS3018
CLS uyumlu olmayan türü 'type' üyesi olduğu için 'type' CLS ile uyumlu işaretlenemez  
  
 Bu uyarı bir iç içe geçmiş sınıf kümesine CLSCompliant özniteliğiyle saptanmışsa `true` kümesine CLSCompliant özniteliği bir sınıf üyesi bildirilen olarak bildirilen `false`. CLS uyumlu olmayan bir dış sınıf üyesi ise, bir iç içe geçmiş sınıf CLS uyumlu olamayacağından bu, izin verilmiyor. Bu uyarıyı çözmek için iç içe geçmiş sınıftan CLSCompliant özniteliğini kaldırmak veya ondan değiştirmek `true` için `false`. CLS uyumluluğu hakkında daha fazla bilgi için bkz: [CLS uyumlu kod yazma](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) ve [dil bağımsızlığı ve dilden bağımsız bileşenler](https://msdn.microsoft.com/library/12a7a7h3).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS3018 oluşturur.  
  
```  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
