# <a name="compiler-error-cs0010"></a>Derleyici Hatası CS0010
Beklenmeyen önemli hatası--'error'.  
  
 Bu hata, tamamen beklenmedik derleme durdurmak için oluştuğunda üretilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Başka bir projeye yeniden derleyin. Aynı hata alırsanız, Visual Studio yeniden yüklemeyi deneyin. Bu hatayı alırsanız, hata raporunu Microsoft'a gönderin.
