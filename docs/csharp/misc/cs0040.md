# <a name="compiler-error-cs0040"></a>Derleyici Hatası CS0040
Hata ayıklama bilgileri dosyası oluşturulurken beklenmeyen hata — 'gerekçe'  
  
 Bu hatayı kullanılırken oluşabilir [/debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) derleyici seçeneği ve derleyici .pdb dosyasına yazamıyor olduğunu gösterir. Bu hata için olası çözümler, Visual Studio yeniden yüklemeyi, derleyici bir dosya veya dizin yazma erişimi olduğundan emin olduktan veya/Debug ile derleme değil içerir.
