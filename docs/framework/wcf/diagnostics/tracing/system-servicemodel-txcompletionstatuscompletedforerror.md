# <a name="systemservicemodeltxcompletionstatuscompletedforerror"></a>System.ServiceModel.TxCompletionStatusCompletedForError
Belirtilen işlem için belirtilen işlem, bir işlenmemiş yürütme özel durum nedeniyle tamamlandı.  
  
## <a name="description"></a>Açıklama  
 Geçerli işlemi tamamlamak için girişimi sırasında bir hata oluştuğunda izlenen. Bir yanıt önce aşması veya hataya çağırana gönderilir.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Özel durum iletisi ve tıklatılabilir tüm öğeler için izlenen iletisini inceleyin.
