# <a name="compiler-error-cs1061"></a><span data-ttu-id="76a51-102">Derleyici Hatası CS1061</span><span class="sxs-lookup"><span data-stu-id="76a51-102">Compiler Error CS1061</span></span>
<span data-ttu-id="76a51-103">'type', 'member' için tanım içermiyor ve hiçbir uzantı metodu 'name ' type' türünde ilk bağımsız değişken kabul' bulunamadı (kullanarak bir eksik yönergesi veya bir derleme başvurusu?).</span><span class="sxs-lookup"><span data-stu-id="76a51-103">'type' does not contain a definition for 'member' and no extension method 'name' accepting a first argument of type 'type' could be found (are you missing a using directive or an assembly reference?).</span></span>  
  
 <span data-ttu-id="76a51-104">Bir yöntem çağrısı veya var olmayan bir sınıf üyesine erişme çalıştığınızda bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="76a51-104">This error occurs when you try to call a method or access a class member that does not exist.</span></span>  
  
## <a name="example"></a><span data-ttu-id="76a51-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="76a51-105">Example</span></span>  
 <span data-ttu-id="76a51-106">Aşağıdaki örnek CS1061 oluşturur çünkü `TestClass1` sahip olmayan bir `DisplaySomething` yöntemi.</span><span class="sxs-lookup"><span data-stu-id="76a51-106">The following example generates CS1061 because `TestClass1` does not have a `DisplaySomething` method.</span></span> <span data-ttu-id="76a51-107">Çağrılan yöntemine sahip `WriteSomething`.</span><span class="sxs-lookup"><span data-stu-id="76a51-107">It does have a method that is called `WriteSomething`.</span></span> <span data-ttu-id="76a51-108">Belki de, ne yazmak bu kaynak kodu yazarı istediğinizi belirtir.</span><span class="sxs-lookup"><span data-stu-id="76a51-108">Perhaps that is what the author of this source code meant to write.</span></span>  
  
```csharp  
// cs1061.cs  
public class TestClass1  
{  
    // TestClass1 has one method, called WriteSomething.  
    public void WriteSomething(string s)  
    {  
        System.Console.WriteLine(s);  
    }  
}  
  
public class TestClass2  
{  
    // TestClass2 has one method, called DisplaySomething.  
    public void DisplaySomething(string s)  
    {  
        System.Console.WriteLine(s);  
    }  
}  
  
public class TestTheClasses  
{  
    public static void Main()  
    {  
        TestClass1 tc1 = new TestClass1();  
        TestClass2 tc2 = new TestClass2();  
        // The following call fails because TestClass1 does not have   
        // a method called DisplaySomething.  
        tc1.DisplaySomething("Hello");      // CS1061  
  
        // To correct the error, change the method call to either   
        // tc1.WriteSomething or tc2.DisplaySomething.  
        tc1.WriteSomething("Hello from TestClass1");  
        tc2.DisplaySomething("Hello from TestClass2");  
    }  
}  
```
