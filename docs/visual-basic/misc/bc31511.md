# <a name="attribute-member-39ltmembernamegt39-cannot-be-the-target-of-an-assignment-because-it-is-not-declared-39public39"></a>Öznitelik üye &#39; &lt;membername&gt; &#39; değil olarak bildirildiğinden atama işleminin hedefi olamaz &#39;ortak&#39;
Özel üyesi bir özniteliğin bir değer atamak için girişimde bulunuldu.  
  
 **Hata Kimliği:** BC31511  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Atama kaldırın.  
  
2.  Geliştirilmiş özel öznitelikleri kullanılıyorsa öznitelik üyesinin erişim değiştiricisi değiştirmek `Public`.
