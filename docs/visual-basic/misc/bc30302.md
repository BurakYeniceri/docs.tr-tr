# <a name="type-character-39ltcharactergt39-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>Tür karakteri &#39; &lt;karakter&gt;&#39; bir açık tür bildirimiyle kullanılamaz
Bir tür karakteri gibi kullandığınız `$`, açıkça belirtmek için bir tür.  
  
 **Hata Kimliği:** BC30302  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tür karakteri veya kaldırın `As <Type>` bildirimi from yan tümcesi.
