# <a name="microsofttransactionstransactionbridgeregistrationcoordinatorresponseinvalidmetadata"></a>Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorResponseInvalidMetadata
WS-Atomic işlem protokolü hizmeti geçersiz ya da uyumsuz meta verilerle bir uç nokta başvurusu içeriyor Koordinatörü RegisterResponse iletisi aldı.  
  
## <a name="description"></a>Açıklama  
 İzlenen yerel işlem yöneticisi, kendi üst düzey işlem yöneticisi ile kaydetme dener ve üst RegisterResponse iletisi içinde geçersiz bir adres döndürür.
