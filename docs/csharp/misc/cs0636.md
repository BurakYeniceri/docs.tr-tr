# <a name="compiler-error-cs0636"></a><span data-ttu-id="2000b-102">Derleyici Hatası CS0636</span><span class="sxs-lookup"><span data-stu-id="2000b-102">Compiler Error CS0636</span></span>
<span data-ttu-id="2000b-103">FieldOffset özniteliği yalnızca StructLayout(LayoutKind.Explicit) ile işaretlenmiş türlerinin üyelerinde yerleştirilebilir</span><span class="sxs-lookup"><span data-stu-id="2000b-103">The FieldOffset attribute can only be placed on members of types marked with the StructLayout(LayoutKind.Explicit)</span></span>  
  
 <span data-ttu-id="2000b-104">Kullanmalısınız **StructLayout(LayoutKind.Explicit)** ile işaretlenen herhangi bir üye içeriyorsa, yapı bildiriminde özniteliği **FieldOffset** özniteliği.</span><span class="sxs-lookup"><span data-stu-id="2000b-104">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="2000b-105">Daha fazla bilgi için bkz: [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span><span class="sxs-lookup"><span data-stu-id="2000b-105">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>  
  
 <span data-ttu-id="2000b-106">Aşağıdaki örnek CS0636 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="2000b-106">The following sample generates CS0636:</span></span>  
  
```csharp  
// CS0636.cs  
using System;  
using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]  
struct Worksheet  
{  
   [FieldOffset(4)]public int i;   // CS0636   
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
