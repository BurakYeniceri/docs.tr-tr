# <a name="39sub-new39-cannot-handle-events"></a>&#39;Sub New&#39; olayları işleyemiyor
Birleştirilecek çalıştı `Sub New` ile `Handles`, geçersiz. Kullanım `Handles` anahtar sözcüğü bir nesne değişkeni tarafından başlatılan olayları işlemek için neden bir yordam bildiriminin sonundaki kullanarak bildirilen `WithEvents` anahtar sözcüğü.  
  
 **Hata Kimliği:** BC30497  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kullanmayın `New` bu bağlamda.
