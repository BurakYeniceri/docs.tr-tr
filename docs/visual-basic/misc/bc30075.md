# <a name="named-arguments-are-not-valid-as-array-subscripts"></a>Adlandırılmış bağımsız değişkenler dizi indisi olarak geçerli değil
Bir dizi dizini bir yordam bağımsız değişkenini örneğin adıyla geçirmesi sözdizimini kullanarak belirtilen `Array(Index := 10)`. Bu sözdiziminin dizi indisleri için geçerli değil.  
  
 **Hata Kimliği:** BC30075  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Dizi dizini değişken veya sabit bir normal ifade kullanan `Array(10)`.
