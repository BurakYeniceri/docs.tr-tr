# <a name="compiler-error-cs0176"></a><span data-ttu-id="9c302-102">Derleyici Hatası CS0176</span><span class="sxs-lookup"><span data-stu-id="9c302-102">Compiler Error CS0176</span></span>
<span data-ttu-id="9c302-103">Statik üye 'member' bir örnek başvurusuyla erişilemez; Bunun yerine bir tür adıyla nitelemek</span><span class="sxs-lookup"><span data-stu-id="9c302-103">Static member 'member' cannot be accessed with an instance reference; qualify it with a type name instead</span></span>  
  
 <span data-ttu-id="9c302-104">Yalnızca bir sınıf adı nitelemek için kullanılan bir [statik](../../csharp/language-reference/keywords/static.md) değişken; bir örnek adı, bir niteleyici olamaz.</span><span class="sxs-lookup"><span data-stu-id="9c302-104">Only a class name can be used to qualify a [static](../../csharp/language-reference/keywords/static.md) variable; an instance name cannot be a qualifier.</span></span> <span data-ttu-id="9c302-105">Daha fazla bilgi için bkz: [statik sınıflar ve statik sınıf üyeleri](../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span><span class="sxs-lookup"><span data-stu-id="9c302-105">For more information, see [Static Classes and Static Class Members](../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span></span>  
  
 <span data-ttu-id="9c302-106">Aşağıdaki örnek CS0176 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="9c302-106">The following sample generates CS0176:</span></span>  
  
```csharp  
// CS0176.cs  
public class MyClass2  
{  
    public static int num;  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        MyClass2 mc2 = new MyClass2();  
        int i = mc2.num;   // CS0176  
        // try the following line instead  
        // int i = MyClass2.num;  
    }  
}  
```
