# <a name="branching-out-of-a-39finally39-is-not-valid"></a>İşyeri dışında dallanma bir &#39; Son olarak &#39; geçerli değil
A `GoTo` deyimi içinde bir `Finally` engelleme dalları blok dışında. İçinde veya dışında dal için geçerli değil bir `Catch` veya `Finally` bloğu.  
  
 **Hata Kimliği:** BC30101  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `GoTo` deyimi ve program mantığı karar veya döngü denetim yapıları ile uygulayın.
