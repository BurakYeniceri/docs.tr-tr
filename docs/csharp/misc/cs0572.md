# <a name="compiler-error-cs0572"></a><span data-ttu-id="ef356-102">Derleyici Hatası CS0572</span><span class="sxs-lookup"><span data-stu-id="ef356-102">Compiler Error CS0572</span></span>
<span data-ttu-id="ef356-103">'type': bir ifade; yoluyla bir türü başvuramaz. Bunun yerine 'path_to_type' deneyin</span><span class="sxs-lookup"><span data-stu-id="ef356-103">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="ef356-104">Bu durumda izin verilmiyor bir tanımlayıcı aracılığıyla bir sınıf üyesi erişmek için girişimde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="ef356-104">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="ef356-105">Aşağıdaki örnek CS0572 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ef356-105">The following sample generates CS0572:</span></span>  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
