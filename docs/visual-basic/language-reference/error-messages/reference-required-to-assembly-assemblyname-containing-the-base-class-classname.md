# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-base-class-39ltclassnamegt39"></a>Derleme &#39;için gereken başvuru; &lt;assemblyname&gt;&#39; temel sınıfı &#39; içeren&lt; ClassName&gt;&#39;
Derleme için gerekli referans '\<assemblyname >' temel sınıfını içeren\<classname >'. Projenize ekleyin.  
  
 Sınıf bir dinamik bağlantı kitaplığı (DLL) tanımlanan veya doğrudan projenizde başvurulmuyor derleme. [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] Derleyici sınıfı birden fazla DLL veya derleme tanımlı durumda Karışıklığı önlemek için bir başvuru gerektirir.  
  
 **Hata Kimliği:** BC30007  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Başvurulmayan DLL veya derleme adını, proje başvuruları içerir.
