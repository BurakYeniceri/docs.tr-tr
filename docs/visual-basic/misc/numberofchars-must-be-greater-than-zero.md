# <a name="numberofchars-must-be-greater-than-zero"></a>NumberOfChars sıfırdan büyük olmalıdır
Kullanırken `PeekChars` yöntemi `TextFieldParser` nesnesi sağlamanız bir `NumberOfChars` değerinden daha büyük değer `0`.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değişiklik `NumberOfChars` değerinden daha büyük bir değere `0`.
