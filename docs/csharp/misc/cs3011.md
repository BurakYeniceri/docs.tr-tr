# <a name="compiler-warning-level-1-cs3011"></a>Derleyici Uyarısı (düzey 1) CS3011
'member': yalnızca CLS uyumlu üyeler Özet olabilir  
  
 Sınıf üyesine her ikisi de olamaz [soyut](../../csharp/language-reference/keywords/abstract.md) ve uyumlu olmayan ortak dil belirtimi (CLS) sahip. CLS sınıf üyelerinin tamamı uygulanan belirtir. CLS uyumluluğu hakkında daha fazla bilgi için bkz: [CLS uyumlu kod yazma](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) ve [dil bağımsızlığı ve dilden bağımsız bileşenler](../../../docs/standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS3011 oluşturur:  
  
```  
// CS3011.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public abstract class I  
{  
    [CLSCompliant(false)]  
    public abstract int M();   // CS3011  
  
    // OK  
    [CLSCompliant(false)]  
    public void M2()  
    {  
    }  
}  
  
public class C : I  
{  
    public override int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
