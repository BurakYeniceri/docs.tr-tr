# <a name="39end-operator39-must-be-the-first-statement-on-a-line"></a>&#39;End işleci&#39; satırdaki ilk deyim olmalıdır
Bir `End Operator` deyimi izleyen iki nokta (`:`) deyimi ayırıcı. `End Operator` Kaynak satırında yegane deyim olmalıdır.  
  
 **Hata Kimliği:** BC33006  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Birden çok deyime farklı satırlarına ayırın.
