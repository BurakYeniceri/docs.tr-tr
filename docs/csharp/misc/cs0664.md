<span data-ttu-id="5182c-103">Değişmez değeri çift türünde örtük olarak 'type'; türüne dönüştürülemez. Bu tür bir hazır değer oluşturmak için bir 'soneki' sonekini kullanın</span><span class="sxs-lookup"><span data-stu-id="5182c-103">Literal of type double cannot be implicitly converted to type 'type'; use an 'suffix' suffix to create a literal of this type</span></span>  
  
 <span data-ttu-id="5182c-104">Atama tamamlanamadı; yönerge düzeltmek için bir son eki kullanın.</span><span class="sxs-lookup"><span data-stu-id="5182c-104">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="5182c-105">Her tür için belge türü için karşılık gelen sonekinin tanımlar.</span><span class="sxs-lookup"><span data-stu-id="5182c-105">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="5182c-106">Dönüşümler hakkında daha fazla bilgi için bkz: [atama ve tür dönüşümleri](../../csharp/programming-guide/types/casting-and-type-conversions.md).</span><span class="sxs-lookup"><span data-stu-id="5182c-106">For more information on conversions, see [Casting and Type Conversions](../../csharp/programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="5182c-107">Aşağıdaki örnek CS0664 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5182c-107">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
