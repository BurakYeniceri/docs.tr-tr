# <a name="compiler-error-cs0031"></a><span data-ttu-id="c2db8-102">Derleyici Hatası CS0031</span><span class="sxs-lookup"><span data-stu-id="c2db8-102">Compiler Error CS0031</span></span>
<span data-ttu-id="c2db8-103">'Value' sabit değeri bir 'type' olarak dönüştürülemez.</span><span class="sxs-lookup"><span data-stu-id="c2db8-103">Constant value 'value' cannot be converted to a 'type'.</span></span> <span data-ttu-id="c2db8-104">(geçersiz kılmak için 'unchecked' sözdizimini kullanın)</span><span class="sxs-lookup"><span data-stu-id="c2db8-104">(use 'unchecked' syntax to override)</span></span>  
  
 <span data-ttu-id="c2db8-105">Türü değeri depolanamıyor bir değişkene bir değer atamak için girişimde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="c2db8-105">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="c2db8-106">Daha fazla bilgi için [türleri](../../csharp/programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="c2db8-106">For more information, see [Types](../../csharp/programming-guide/types/index.md).</span></span>  
  
 <span data-ttu-id="c2db8-107">Aşağıdaki örnek CS0031 checked ve unchecked bağlamlarda oluşturur:</span><span class="sxs-lookup"><span data-stu-id="c2db8-107">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>  
  
```csharp  
// CS0031.cs  
namespace CS0031  
{  
   public class a  
   {  
      public static void Main()  
      {  
         int num = (int)2147483648M; //CS0031  
         // Try using a larger numeric type instead:  
         // long num = (long)2147483648M; //CS0031  
  
         const decimal d = -10M; // Decimal literal  
            unchecked  
            {  
                const byte b = (byte)d; // CS0031  
                // For small values try using a signed byte instead:  
                // const sbyte b = (sbyte)d;  
            }  
      }  
   }  
}  
```
