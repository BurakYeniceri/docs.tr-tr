# <a name="compiler-warning-level-1-cs3004"></a>Derleyici Uyarısı (düzey 1) CS3004
Karışık ve ayrıştırılmış Unicode karakterleri CLS uyumlu değildir.  
  
 Yalnızca oluşan UNICODE karakterler izin verilen [genel](../../csharp/language-reference/keywords/public.md), [korumalı](../../csharp/language-reference/keywords/protected.md), veya `protected internal` ortak dil belirtimi (CLS) ile uyumlu olması için tanımlayıcılar. CLS uyumluluğu hakkında daha fazla bilgi için bkz. [dil bağımsızlığı ve dilden bağımsız bileşenler](~/docs/standard/language-independence.md).
