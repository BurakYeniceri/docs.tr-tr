# <a name="unable-to-include-xml-fragment-39ltelementpathgt39-of-file-39ltcommentfilegt39"></a>XML parçası &#39;eklenemiyor; &lt;elementpath'de&gt;&#39; &#39; dosya&lt; commentfile&gt;&#39;
Bir `<include>` bir XML açıklama etiketinde var olmayan bir öğe için bir XPath ifadesi içeriyor veya dosya hatalı biçimlendirilmiş XML içeriyor.  
  
 **Hata Kimliği:** BC42320  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  XML dosyasında geçerli olduğunu doğrulayın.  
  
2.  XML parçası dosyada var olduğunu doğrulayın.
