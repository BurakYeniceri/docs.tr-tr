# <a name="array-initializer-has-too-few-dimensions"></a>Dizi Başlatıcı çok az boyuta sahiptir
Başlatıcı yeterli boyutlar, dizi için sağladığınız değil.  
  
 **Hata Kimliği:** BC30565  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaç tane boyutları gerekli olduğunu belirlemek için dizi Başlatıcı denetleyin.  
  
-   Eksik bağımsız değişken veya bağımsız değişkenleri sağlayın.
