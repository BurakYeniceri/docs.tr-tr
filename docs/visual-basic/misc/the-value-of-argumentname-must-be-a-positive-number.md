# <a name="the-value-of-ltargumentnamegt-must-be-a-positive-number"></a>Değeri &lt;argumentname&gt; pozitif bir sayı olmalıdır
Değeri <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> özelliği sıfırdan büyük olmalıdır.  
  
 <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> Özelliği iletileri günlük dosyasına yazılabilir önce bu boş disk alanı miktarını bayt cinsinden gerekli belirtir  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ayarlama <xref:Microsoft.VisualBasic.Logging.FileLogTraceListener.ReserveDiskSpace%2A> özelliğini pozitif bir sayı.
