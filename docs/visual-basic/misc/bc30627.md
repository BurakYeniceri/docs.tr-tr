# <a name="39option39-statements-must-precede-any-declarations-or-39imports39-statements"></a>&#39; seçeneği &#39; deyimleri tüm bildirimler gelmelidir veya &#39; içeri aktarmalar &#39; deyimleri
`Option`kaynak kodunuz ilk açıklama olmayan deyimlerinde deyimleri olmalıdır.  
  
 **Hata Kimliği:** BC30627  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Taşıma `Option` hemen önce kaynak üstüne deyimleri kod `Imports` ve `Namespace` deyimleri.
