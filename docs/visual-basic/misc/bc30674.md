# <a name="39end-synclock39-must-be-preceded-by-a-matching-39synclock39"></a>&#39;End SyncLock&#39; eşleşen bir gelmelidir &#39;SyncLock&#39;
`SyncLock` blokları ile başlar `SyncLock` anahtar sözcüğü ve ile biten `End``SyncLock` oluşturun.  
  
 **Hata Kimliği:** BC30674  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Olduğundan emin olun `SyncLock` bloğu ile başlayıp bir `SyncLock` deyimi.
