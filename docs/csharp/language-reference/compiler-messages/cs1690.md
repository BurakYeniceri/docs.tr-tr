# <a name="compiler-warning-level-1-cs1690"></a><span data-ttu-id="5800e-102">Derleyici Uyarısı (düzey 1) CS1690</span><span class="sxs-lookup"><span data-stu-id="5800e-102">Compiler Warning (level 1) CS1690</span></span>
<span data-ttu-id="5800e-103">Başvuruya göre sınıfının bir alan olduğundan, 'member' üzerindeki bir üyeye erişme çalışma zamanı özel durumuna neden olabilir</span><span class="sxs-lookup"><span data-stu-id="5800e-103">Accessing a member on 'member' may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="5800e-104">Bu uyarı bir yöntemi, özelliği veya dizin oluşturucu türeyen bir sınıf üyesi çağırmak çalıştığınızda oluşur <xref:System.MarshalByRefObject>, ve bir değer türü üyesidir.</span><span class="sxs-lookup"><span data-stu-id="5800e-104">This warning occurs when you try to call a method, property, or indexer on a member of a class that derives from <xref:System.MarshalByRefObject>, and the member is a value type.</span></span> <span data-ttu-id="5800e-105">Öğesinden devralan nesneler `MarshalByRefObject` genellikle başvuruya göre uygulama etki alanı sıralanması için tasarlanmıştır.</span><span class="sxs-lookup"><span data-stu-id="5800e-105">Objects that inherit from `MarshalByRefObject` are typically intended to be marshaled by reference across an application domain.</span></span> <span data-ttu-id="5800e-106">Herhangi bir kod herhangi bir zamanda doğrudan böyle bir nesnenin değer türü üyesi uygulama etki alanı erişmeye çalışırsa, bir çalışma zamanı özel durum meydana gelir.</span><span class="sxs-lookup"><span data-stu-id="5800e-106">If any code ever attempts to directly access the value-type member of such an object across an application domain, a runtime exception will occur.</span></span> <span data-ttu-id="5800e-107">Uyarıyı çözmek için ilk üye yerel bir değişkene kopyalayın ve bu değişken üzerindeki yöntemini çağırın.</span><span class="sxs-lookup"><span data-stu-id="5800e-107">To resolve the warning, first copy the member into a local variable and call the method on that variable.</span></span>  
  
 <span data-ttu-id="5800e-108">Aşağıdaki örnek CS1690 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5800e-108">The following sample generates CS1690:</span></span>  
  
```csharp  
// CS1690.cs  
using System;  
  
class WarningCS1690: MarshalByRefObject  
{  
   int i = 5;  
  
   public static void Main()   
   {  
     WarningCS1690 e = new WarningCS1690();  
     e.i.ToString();   // CS1690  
  
     // OK  
     int i = e.i;  
     i.ToString();  
     e.i = i;  
   }  
}  
```
