# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-latebinding"></a>.NET Compact Framework hedeflenen sürümü geç bağlamayı desteklemiyor
.NET Compact Framework'ı birlikte çalıştığınız sürümü geç bağlamayı desteklemiyor.  
  
 **Hata Kimliği:** BC30762  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  İşlevler, alt öğeler veya özellikleri nesnesi olarak bildirilen bir değişken çağrılması kaçının.  
  
2.  Nesne değişkeni bir dizi kullanmaktan kaçının.  
  
3.  Nesne değişkenleri ile sözlük üye erişimi ifadeleri kullanmaktan kaçının.
