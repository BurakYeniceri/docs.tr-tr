<span data-ttu-id="b7921-103">'Türü: 1': dönüş türü, 'Türü geçersiz kılınan üye 'Üye adı' eşleştirmek için 2' olmalıdır</span><span class="sxs-lookup"><span data-stu-id="b7921-103">'Type 1': return type must be 'Type 2' to match overridden member 'Member Name'</span></span>

## <a name="example"></a><span data-ttu-id="b7921-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="b7921-106">Example</span></span>  
 <span data-ttu-id="b7921-107">Aşağıdaki örnek CS0508 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="b7921-107">The following sample generates CS0508.</span></span>  
  
```csharp  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
