 <span data-ttu-id="66628-104">Kullanırsanız, bu hata oluşur. bir `in`, `ref`, veya `out` adsız yöntem bloğunun içine parametresi.</span><span class="sxs-lookup"><span data-stu-id="66628-104">This error occurs if you use an `in`, `ref`, or `out` parameter inside an anonymous method block.</span></span> <span data-ttu-id="66628-105">Bu hatayı önlemek için yerel bir değişken veya diğer bir yapı kullanın.</span><span class="sxs-lookup"><span data-stu-id="66628-105">To avoid this error, use a local variable or some other construct.</span></span>  
  
 <span data-ttu-id="66628-106">Aşağıdaki örnek CS1628 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="66628-106">The following sample generates CS1628:</span></span>

```csharp  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }

  {  
  
  }  
}  
```
