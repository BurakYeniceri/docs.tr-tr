# <a name="39ltmethodnamegt39-cannot-override-a-method-that-has-been-shadowed"></a>&#39; &lt;methodname&gt;&#39; gölgeli bir yöntemi geçersiz kılamaz
Başka bir üye shadows yöntemi geçersiz kılmak bir yöntem çalıştı.  
  
 **Hata Kimliği:** BC31406  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanım `Shadows` değiştiricisi gölgeli üyeleri geçersiz kılın.
