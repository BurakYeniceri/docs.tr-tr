# <a name="event-39lteventnamegt39-event-specified-by-the-39defaultevent39-attribute-is-not-a-publicly-accessible-event-for-this-class"></a>Olay &#39; &lt;eventname&gt; &#39; tarafından belirtilen olay &#39;DefaultEvent&#39; özniteliği bu sınıfın genel olarak erişilebilir bir olay değil
<xref:System.ComponentModel.DefaultEventAttribute> Özniteliği öznitelik uygulanan sınıfında genel olarak erişilebilir olayın adı belirtmeniz gerekir.  
  
 **Hata Kimliği:** BC30770  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Genel olarak erişilebilir bir olay sınıfı tanımlayan emin olun.  
  
2.  Sınıfında olayın adı tarafından belirtilen adıyla eşleştiğinden emin olun <xref:System.ComponentModel.DefaultEventAttribute> özniteliği.
