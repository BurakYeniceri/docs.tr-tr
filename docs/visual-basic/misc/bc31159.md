# <a name="expected-closing-39gt39-for-embedded-expression"></a>Kapatma beklenen &#39;%&gt; &#39; katıştırılmış ifadesi için
XML değişmez değeri katıştırılmış bir ifadede gerekli kapanış eksik `%>` karakter.  
  
 **Hata Kimliği:** BC31159  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gerekli kapanış eklemek `%>` katıştırılmış ifade sonuna karakter.
