# <a name="compiler-error-cs0236"></a><span data-ttu-id="ec702-102">Derleyici Hatası CS0236</span><span class="sxs-lookup"><span data-stu-id="ec702-102">Compiler Error CS0236</span></span>
<span data-ttu-id="ec702-103">Alan başlatıcı statik olmayan alanı, yöntemi veya özelliği 'field' başvuramaz.</span><span class="sxs-lookup"><span data-stu-id="ec702-103">A field initializer cannot reference the nonstatic field, method, or property 'field'</span></span>  
  
 <span data-ttu-id="ec702-104">Örnek alanı, diğer örneği alanları dışında bir yöntem başlatmak için kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="ec702-104">Instance fields cannot be used to initialize other instance fields outside a method.</span></span> <span data-ttu-id="ec702-105">Bir yöntem dışındaki bir değişkenin başlatmak çalışıyorsanız, sınıf oluşturucu içinde başlatma gerçekleştirmektir.</span><span class="sxs-lookup"><span data-stu-id="ec702-105">If you are trying to initialize a variable outside a method, consider performing the initialization inside the class constructor.</span></span> <span data-ttu-id="ec702-106">Daha fazla bilgi için bkz: [yöntemleri](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="ec702-106">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="ec702-107">Aşağıdaki örnek CS0236 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ec702-107">The following sample generates CS0236:</span></span>  
  
```csharp  
// CS0236.cs  
public class MyClass  
{  
   public int i = 5;  
   public int j = i;  // CS0236  
   public int k;      // initialize in constructor  
  
   MyClass()  
   {  
      k = i;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
