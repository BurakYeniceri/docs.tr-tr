# <a name="file-ltfilenamegt-could-not-be-found"></a>Dosya &lt;filename&gt; bulunamadı
Belirtilen dosya bulunamadı. Bu, hatalı dosya yolu sağlayan bir sonucu olabilir.  
  
 **Hata Kimliği:** BC2001  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yol ve dosya adının doğru olduğundan emin olun.
