# <a name="replacements-too-long"></a>Değiştirilenler çok uzun
İzin verilen uzunluğu aşıyor yenisini belirttiniz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Uzunluk sınırlaması için bileşen belgelerine bakın.
