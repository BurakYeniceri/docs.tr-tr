# <a name="39exit-property39-is-not-valid-in-a-function-or-sub"></a>&#39;Exit özelliği&#39; bir işlevi veya alt geçerli değil
`Exit Property` görünür bir `Function` yordamı veya `Sub` yordamı. Bir `Exit` deyimi blok eşleşmelidir içinde hangi oluşur.  
  
 **Hata Kimliği:** BC30066  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değiştir `Exit Property` ile `Exit Function` veya `Exit Sub` uygun şekilde deyimi.
