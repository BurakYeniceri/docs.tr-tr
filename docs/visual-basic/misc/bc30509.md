# <a name="ltspecifier1gt-lttypegt-cannot-inherit-from-a-ltspecifier2gt-lttypegt-because-it-expands-the-access-of-the-base-lttypegt"></a>&lt;specifier1&gt; &lt;türü&gt; devralınmalıdır olamaz bir &lt;specifier2&gt; &lt;türü&gt; taban erişimini genişlettiğinden &lt;türü&gt;
İşaretlenmiş bir taban sınıftan devralın genel bir sınıf içeriyor denediniz `Private` veya `Friend`.  
  
 **Hata Kimliği:** BC30509  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Taban sınıf bildirme `Public` veya türetilen sınıf bildirme `Private` veya `Friend`.
