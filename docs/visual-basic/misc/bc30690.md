# <a name="structure-39ltstructurenamegt39-cannot-be-indexed-because-it-has-no-default-property"></a>Yapı &#39; &lt;structurename&gt;&#39; varsayılan bir özellik olduğundan dizin oluşturulamıyor
Varsayılan özellik sözdizimi varsayılan özellik yok bir yapı ile kullanmak için girişimde bulunuldu.  
  
 **Hata Kimliği:** BC30690  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yapısı özelliklerine erişmek için standart özellik sözdizimini kullanın.
