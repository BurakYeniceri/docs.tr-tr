# <a name="compiler-error-cs0075"></a><span data-ttu-id="d21fb-102">Derleyici Hatası CS0075</span><span class="sxs-lookup"><span data-stu-id="d21fb-102">Compiler Error CS0075</span></span>
<span data-ttu-id="d21fb-103">Eksi değerde dönüştürme yapmak için değeri parantez içine almalısınız</span><span class="sxs-lookup"><span data-stu-id="d21fb-103">To cast a negative value, you must enclose the value in parentheses</span></span>  
  
 <span data-ttu-id="d21fb-104">Ardından, önceden tanımlanmış bir türü tanımlayan bir anahtar sözcüğü kullanılarak atama, parantez gerekmez.</span><span class="sxs-lookup"><span data-stu-id="d21fb-104">If you are casting using a keyword that identifies a predefined type, then you do not need parentheses.</span></span> <span data-ttu-id="d21fb-105">Aksi takdirde, çünkü parantezler yerleştirmelidir (x) – y atama ifadesi değil değerlendirilir.</span><span class="sxs-lookup"><span data-stu-id="d21fb-105">Otherwise, you must put the parentheses because (x) –y will not be considered a cast expression.</span></span> <span data-ttu-id="d21fb-106">C# belirtiminden 7.6.6 bölümünde:</span><span class="sxs-lookup"><span data-stu-id="d21fb-106">From the C# Specification, Section 7.6.6:</span></span>  
  
 <span data-ttu-id="d21fb-107">*Takip eden, varsa Kesinleştirme kuraldan x ve y (x) y tanımlayıcılardır (x)(y), ve (x)(-y) olan cast-ifadeleri (x) ancak-y değil, bile x bir türü tanımlar. X (örneğin, int) önceden tanımlanmış bir türü tanımlayan bir anahtar sözcüğü (böyle bir anahtar sözcük büyük olasılıkla bir ifade kendisi tarafından çözümlenemediğinden) ancak ardından tüm dört formları cast ifadeleri tutulur.*</span><span class="sxs-lookup"><span data-stu-id="d21fb-107">*From the disambiguation rule it follows that, if x and y are identifiers, (x)y, (x)(y), and (x)(-y) are cast-expressions, but (x)-y is not, even if x identifies a type. However, if x is a keyword that identifies a predefined type (such as int), then all four forms are cast-expressions (because such a keyword could not possibly be an expression by itself).*</span></span>  
  
 <span data-ttu-id="d21fb-108">Aşağıdaki kod CS0075 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="d21fb-108">The following code generates CS0075:</span></span>  
  
```csharp  
// CS0075  
namespace MyNamespace  
{  
    enum MyEnum { }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            // To fix the error, place the negative  
            // values below in parentheses  
            int i = (System.Int32) - 4; //CS0075  
            MyEnum e = (MyEnum) - 1;    //CS0075  
            System.Console.WriteLine(i); //to avoid warning  
            System.Console.WriteLine(e); //to avoid warning  
        }  
    }  
}  
```
