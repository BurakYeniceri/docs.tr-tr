# <a name="cannot-rename-with-different-drive"></a>Farklı bir sürücüyle yeniden adlandırılamıyor
`Rename` Deyimi, dosya yolunda bir sürücü harfi belirtemezsiniz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sürücü harfi dosya yolundan kaldırın.
