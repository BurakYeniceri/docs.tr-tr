# <a name="systemservicemodelchannelsincompatibleexistingtransportmanager"></a>System.ServiceModel.Channels.IncompatibleExistingTransportManager
Belirtilen URI için mevcut bir uyumsuz aktarım yöneticisi bulunamadı.  
  
## <a name="description"></a>Açıklama  
 Bu bilgi izleme eşleşen bir kayıt için açıldı, ancak varolan aktarım Yöneticisi'ni uyumsuz dinleyici bulundu gösterir. Dinleyici başka bir aktarım Yöneticisi'ni kullanın ve başarısız olursa bir özel durum dener.
