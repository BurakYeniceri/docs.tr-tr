# <a name="expression-is-of-type-39lttypenamegt39-which-is-not-a-collection-type"></a>İfade türüdür &#39; &lt;typename&gt;&#39;, bir koleksiyon türü değil
Belirtilen grup değişkeni bir `For Each` deyimi bir koleksiyon nesnesi veya bir dizi değil ve türünü uygulamıyor <xref:System.Collections.IEnumerable> arabirimi. Tür Visual Basic koleksiyon tasarım desenini destek veya uygulama <xref:System.Collections.IEnumerable>.  
  
 **Hata Kimliği:** BC32023  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ya da desteklediğinden Visual Basic koleksiyon tasarım veya uygulayan bir sınıf türünde olmasını Grup değişkeni bildirme <xref:System.Collections.IEnumerable>.
