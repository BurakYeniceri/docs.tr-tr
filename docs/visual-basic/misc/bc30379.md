# <a name="39catch39-cannot-appear-after-39finally39-within-a-39try39-statement"></a>&#39; Catch &#39; sonra bulunamaz &#39; Son olarak &#39; içinde bir &#39; Try &#39; deyimi
A `Catch` deyimi görünür koddan sonra `Finally` bitiş bir `Try` deyimi bloğu. `Catch`içinde yer almalıdır bir `Try...Catch...Finally` deyimi bloğu.  
  
 **Hata Kimliği:** BC30379  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Taşıma `Catch` kodu daha uygun bir yerde ifadesine.
