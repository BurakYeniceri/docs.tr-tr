# <a name="attribute-39ltattributenamegt39-is-not-valid-lterrorgt"></a>Öznitelik &#39; &lt;attributename&gt;&#39; geçerli değil: &lt;hata&gt;
Ortak dil çalışma zamanı özniteliği geçerli bağlamda kabul etmiyor.  
  
 **Hata Kimliği:** BC30127  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Öznitelik bildiriminden kaldırın veya bağlamı için geçerli olan özniteliği ile değiştirin.  
  
2.  Hata iletisini inceleyin ve uygun eylemi gerçekleştirin.
