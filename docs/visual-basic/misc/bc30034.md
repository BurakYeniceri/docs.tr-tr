# <a name="bracketed-identifier-is-missing-closing-3939"></a>Kapanış köşeli parantez içindeki tanımlayıcısı eksik &#39;]&#39;
Kaçış karakterli bir adı köşeli eşleşen çiftleri bulunması gerekir.  
  
 **Hata Kimliği:** BC30034  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaçış karakterli adının sonuna bir kapanış ayracı yerleştirin.
