# <a name="numeric-and-comparison-operators"></a>Sayısal ve Karşılaştırma işleçleri
Aritmetik ve Karşılaştırma işleçleri ortak dil çalışma zamanı (CLR) gibi beklendiği gibi çalışır:  
  
-   SQL modulus işleci kayan nokta sayıları üzerinde desteklemez.  
  
-   SQL denetlenmeyen aritmetik desteklemez.  
  
-   SQL'de yinelenemez ve bu nedenle, desteklenmez ifadelerde kullandığınızda artırma ve azaltma işleçleri yan etkileri neden.  
  
## <a name="supported-operators"></a>Desteklenen işleçleri  
 [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] Aşağıdaki işleçleri destekler.  
  
-   Temel aritmetik işleçler:  
  
    -   `+`  
  
    -   `-` (çıkarma)  
  
    -   `*`  
  
    -   `/`  
  
    -   Visual Basic tamsayı bölme (`\`)  
  
    -   `%` (Visual Basic `Mod`)  
  
    -   `<<`  
  
    -   `>>`  
  
    -   `-` (tekli değilleme)  
  
-   Temel Karşılaştırma işleçleri:  
  
    -   Visual Basic `=` ve C# `==`  
  
    -   Visual Basic `<>` ve C# `!=`  
  
    -   Visual Basic `Is/IsNot`  
  
    -   `<`  
  
    -   `<=`  
  
    -   `>`  
  
    -   `>=`
