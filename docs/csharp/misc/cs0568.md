# <a name="compiler-error-cs0568"></a><span data-ttu-id="66030-102">Derleyici Hatası CS0568</span><span class="sxs-lookup"><span data-stu-id="66030-102">Compiler Error CS0568</span></span>
<span data-ttu-id="66030-103">Yapılar açık parametresiz oluşturucular içeremez.</span><span class="sxs-lookup"><span data-stu-id="66030-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="66030-104">Her [yapısı](../../csharp/language-reference/keywords/struct.md) sıfır nesnesini başlatır varsayılan bir oluşturucu zaten sahip.</span><span class="sxs-lookup"><span data-stu-id="66030-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a default constructor that initializes the object to zero.</span></span> <span data-ttu-id="66030-105">Bu nedenle, bir yapı için oluşturabileceğiniz oluşturucular bir veya daha fazla parametre almalıdır.</span><span class="sxs-lookup"><span data-stu-id="66030-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="66030-106">Aşağıdaki örnek CS0568 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="66030-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
