# <a name="line-ltnumbergt-cannot-be-read-because-it-exceeds-the-maximum-line-size"></a>Satır &lt;numarası&gt; en büyük satır boyutunu aştığından okunamıyor.
En büyük satır boyutunu aştığından satırı okunamıyor. Genellikle bu dosyadaki başlangıç tırnak işareti, ancak hiçbir kapanış tırnak işareti olduğu anlamına gelir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kapanış tırnak işareti dosyaya ekleyin.
