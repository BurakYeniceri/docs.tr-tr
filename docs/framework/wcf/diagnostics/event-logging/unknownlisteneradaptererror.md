# <a name="unknownlisteneradaptererror"></a>UnknownListenerAdapterError
Kimliği: 161  
  
 Önem derecesi: hata  
  
 Kategori: SMSvcHost  
  
## <a name="description"></a>Açıklama  
 Bu olay, bir Windows İşlem Etkinleştirme Hizmeti (WAS) bildirim işlenirken beklenmeyen bir hata Dinleyici Bağdaştırıcısı oluştuğunu gösterir. İşlem sonlandırıldı. Olay protokolü, özel durum, işlem adı ve işlem kimliği listeler
