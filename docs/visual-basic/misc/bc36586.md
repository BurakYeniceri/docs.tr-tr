# <a name="argument-not-specified-for-parameter-39ltparameternamegt39-of-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Bağımsız değişken parametre &#39;belirtilmemiş; &lt;parametername&gt;&#39; genişletme yöntemi &#39;&lt; MethodName&gt;&#39; tanımlıysa &#39;&lt; TypeName&gt;&#39;
Bir genişletme yöntemi, gerekli parametre için bir bağımsız değişken sağlanmadı.  
  
 **Hata Kimliği:** BC36586  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirtilen parametre için bir bağımsız değişken sağlayın.
