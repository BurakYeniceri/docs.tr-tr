# <a name="39end-raiseevent39-must-be-preceded-by-a-matching-39raiseevent39-declaration"></a>&#39; Son RaiseEvent &#39; eşleşen bir &#39;göre gelmelidir; RaiseEvent &#39; bildirimi
Bir `End RaiseEvent` deyimi oluştu karşılık gelen olmadan `RaiseEvent` deyimi. `End RaiseEvent`karşılık gelen tarafından gelmelidir `RaiseEvent` deyimi.  
  
 **Hata Kimliği:** BC31126  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Denetleyin önceki `RaiseEvent` deyimi geçerli olduğundan ve doğru yazılmış.
