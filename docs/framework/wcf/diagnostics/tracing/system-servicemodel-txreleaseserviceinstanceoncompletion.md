# <a name="systemservicemodeltxreleaseserviceinstanceoncompletion"></a>System.ServiceModel.TxReleaseServiceInstanceOnCompletion
Hizmet örneği ReleaseServiceInstanceOnTransactionComplete ServiceBehaviorAttribute ayarlanmış olduğu için '{0}' işlem tamamlanma yayımlanan true.  
  
## <a name="description"></a>Açıklama  
 Mevcut hizmet örneğinin yayınlandığı veya geçerli etkin işlem tamamlandığında nedeniyle atıldı ve ReleaseServiceInstanceOnTransactionComplete ayarlanmış izlenen `true`.
