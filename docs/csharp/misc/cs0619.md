# <a name="compiler-error-cs0619"></a><span data-ttu-id="614de-102">Derleyici Hatası CS0619</span><span class="sxs-lookup"><span data-stu-id="614de-102">Compiler Error CS0619</span></span>
<span data-ttu-id="614de-103">'member' artık kullanılmıyor: 'text'</span><span class="sxs-lookup"><span data-stu-id="614de-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="614de-104">İle işaretlenmiş bir sınıf üyesinin [Kullanımdan kalktı](https://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) özniteliği sağlayacak şekilde sınıf üyesine başvuru olduğunda bir hata verilir.</span><span class="sxs-lookup"><span data-stu-id="614de-104">A class member was marked with the [Obsolete](https://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="614de-105">Aşağıdaki örnek, CS0619 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="614de-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
