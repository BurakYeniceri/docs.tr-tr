# <a name="compiler-error-cs1567"></a>Derleyici Hatası CS1567
Win32 kaynak oluşturulurken hata oluştu: 'dosya'  
  
 Ya da kullanılan derlemeniz [/win32icon](../../../csharp/language-reference/compiler-options/win32icon-compiler-option.md) derleyici seçeneği ya [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md), kaynak bilgilerini içeren bir dosya oluşturmak derleyicinin neden olur, ancak derleyici erişemedi disk alanı yetersiz ya da başka bir hata nedeniyle dosyası oluşturun.  
  
 Dosya oluşturma sorunu gidermek erişemiyorsanız kullanabileceğinizi [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md), kaynak bilgilerini içeren bir dosyayı oluşturmaz.
