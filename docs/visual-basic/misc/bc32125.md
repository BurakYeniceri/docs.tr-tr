# <a name="cannot-implement-39ltinterfacename1gtltmembernamegt39-because-its-implementation-could-conflict-with-the-implementation-for-39ltinterfacename2gtltmembernamegt39-for-some-type-arguments"></a><span data-ttu-id="2114b-102">Olamaz Uygula &#39; &lt;interfacename1&gt;.&lt; membername&gt;&#39; uygulaması için uygulama &#39; çakışabilecek&lt; interfacename2&gt;.&lt; membername&gt;&#39; bazı tür bağımsız değişkenleri için</span><span class="sxs-lookup"><span data-stu-id="2114b-102">Cannot implement &#39;&lt;interfacename1&gt;.&lt;membername&gt;&#39; because its implementation could conflict with the implementation for &#39;&lt;interfacename2&gt;.&lt;membername&gt;&#39; for some type arguments</span></span>
<span data-ttu-id="2114b-103">Bir sınıf biri diğerinden devralır, birden fazla genel arabirimini uygular ve arabirim üyesini iki uygulamaları belirli tür bağımsız değişkenleri değerlerini çakışabilecek.</span><span class="sxs-lookup"><span data-stu-id="2114b-103">A class implements more than one generic interface, one of which inherits from another, and two implementations of an interface member could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="2114b-104">Aşağıdaki deyimleri bu hata oluşmasına neden olabilir.</span><span class="sxs-lookup"><span data-stu-id="2114b-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface iFace1(Of t)  
    Sub testSub()  
End Interface  
Public Interface iFace2(Of u)  
    Inherits iFace1(Of u)  
End Interface  
Public Class testClass(Of y, z)  
    Implements iFace1(Of y), iFace2(Of z)  
    Public Sub testSuby() Implements iFace1(Of y).testSub  
    End Sub  
    Public Sub testSubz() Implements iFace1(Of z).testSub  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="2114b-105">Çünkü `iFace2` devraldığı `iFace1` kendi türü parametresini kullanarak (`u`), `testClass` iki sürümü uygulamak `iFace1.testSub` aynı tür bağımsız değişkeni için geçirilmiş varsa aynı imzaları ile `y` ve `z`.</span><span class="sxs-lookup"><span data-stu-id="2114b-105">Because `iFace2` inherits from `iFace1` using its own type parameter (`u`), `testClass` would implement two versions of `iFace1.testSub` with identical signatures if the same type argument were passed to `y` and `z`.</span></span> <span data-ttu-id="2114b-106">Bu bir belirsizlik erişmek için hangi sürümü hakkında üretir.</span><span class="sxs-lookup"><span data-stu-id="2114b-106">This would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="2114b-107">**Hata Kimliği:** BC32125</span><span class="sxs-lookup"><span data-stu-id="2114b-107">**Error ID:** BC32125</span></span>
