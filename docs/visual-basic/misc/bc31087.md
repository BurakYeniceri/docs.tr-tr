# <a name="array-modifiers-cannot-be-specified-on-both-a-variable-and-its-type"></a>Dizi değiştiriciler hem bir değişkeni hem de türü belirtilemez.
Dizi değiştiriciler değişkeni hem oluşan bir dizi belirten türünü yok.  
  
 **Hata Kimliği:** BC31087  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Dizi değiştiricisi tür belirteci kaldırın.
