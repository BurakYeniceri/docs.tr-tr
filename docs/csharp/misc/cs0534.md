# <a name="compiler-error-cs0534"></a><span data-ttu-id="5972e-102">Derleyici Hatası CS0534</span><span class="sxs-lookup"><span data-stu-id="5972e-102">Compiler Error CS0534</span></span>
<span data-ttu-id="5972e-103">'function1' devralınan soyut üye 'function2' uygulamıyor</span><span class="sxs-lookup"><span data-stu-id="5972e-103">'function1' does not implement inherited abstract member 'function2'</span></span>  
  
 <span data-ttu-id="5972e-104">Bir sınıfın tüm uygulamak için gerekli olan [soyut](../../csharp/language-reference/keywords/abstract.md) sınıfı olmadığı sürece taban sınıfı üyeleri de soyut.</span><span class="sxs-lookup"><span data-stu-id="5972e-104">A class is required to implement all the [abstract](../../csharp/language-reference/keywords/abstract.md) members in the base class, unless the class is also abstract.</span></span>  
  
 <span data-ttu-id="5972e-105">Aşağıdaki örnek CS0534 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5972e-105">The following sample generates CS0534:</span></span>  
  
```csharp  
// CS0534.cs  
namespace x  
{  
   abstract public class clx  
   {  
      public abstract void f();  
   }  
  
   public class cly : clx   // CS0534, no override for clx::f  
   {  
      // uncomment the following sample override to resolve CS0534  
      // override public void f()  
      // {  
      // }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
