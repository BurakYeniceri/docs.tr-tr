# <a name="compiler-error-cs0008"></a>Derleyici Hatası CS0008
Meta veri dosyası 'dosya' okunurken beklenmeyen hata: 'description'  
  
 DLL meta verileri almak için başarıyla açıldı, ancak bozuk sağlayacak şekilde verileri buradan okunamadı. Daha fazla bilgi için bkz: [/Reference (C# Derleyici Seçenekleri)](../../csharp/language-reference/compiler-options/reference-compiler-option.md).
