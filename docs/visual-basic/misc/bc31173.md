# <a name="missing-required-white-space"></a>Gerekli boşluk eksik
XML değişmez değer olmadan gereken ayırıcı alanı belirtilmedi. Bir XML öğesi değişmez değeri sonraki öznitelikleri arasındaki boşluğu gereken ayırıcı alanı örnektir.  
  
 **Hata Kimliği:** BC31173  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gerekli alan ayırıcı ekleyin.
