# <a name="specifiers-and-attributes-are-not-valid-on-39namespace39-statements"></a>Tanımlayıcıları ve öznitelikleri üzerinde geçersiz &#39; Namespace &#39; deyimleri
Ad alanları program blokları ve programlama öğeleri hiyerarşik organizasyonu için kullanılır. Bunlar, herhangi bir değiştiricileri veya öznitelikleri kazanmaz.  
  
 **Hata Kimliği:** BC32026  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tüm anahtar sözcükleri kaldırmak ve öznitelik bloklarından `Namespace` deyimi.
