# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-can-be-called-without-a-narrowing-conversion-lterrorgt"></a>Aşırı yükleme çözümü başarısız oldu, çünkü Hayır erişilebilir &#39; &lt;yöntemi&gt; &#39; daraltma dönüşümü olmadan çağrılabilir: &lt;hata&gt;
Aşırı yüklenmiş yöntemine bir çağrı yapılır, ancak derleyici daraltma dönüşümü olmadan adlı bir yöntemi bulunamıyor. Daraltma dönüşümü tam olarak bazı olası değerleri tutun mümkün olmayabilir veri türü için bir değer değiştirir.  
  
 **Hata Kimliği:** BC30519  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirtin `Option Strict Off`.
