 <span data-ttu-id="e34e0-105">İlk parametre yöntemin genişlettiği hangi veri türü belirttiğinden parametresiz bir genişletme yöntemi geçerli değil.</span><span class="sxs-lookup"><span data-stu-id="e34e0-105">An extension method without parameters is not valid because the first parameter specifies which data type the method extends.</span></span> <span data-ttu-id="e34e0-106">İlk parametre yöntemi çağırır veri türü örneğine bağlanır.</span><span class="sxs-lookup"><span data-stu-id="e34e0-106">The first parameter is bound to the instance of the data type that invokes the method.</span></span>

## <a name="example"></a><span data-ttu-id="e34e0-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="e34e0-110">Example</span></span>  
 <span data-ttu-id="e34e0-111">Aşağıdaki örnekte ilk parametre belirten `Print` yöntemin genişlettiği `String` veri türü.</span><span class="sxs-lookup"><span data-stu-id="e34e0-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>  
  
```  
<Extension()> _  
Public Sub Print (ByVal str As String)  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="e34e0-112">Ne zaman genişletme yöntemi çağrıldığında aşağıdaki gibi parametre `str` yönteminde bağlı `greeting`, örneği `String` çağırır `Print`.</span><span class="sxs-lookup"><span data-stu-id="e34e0-112">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="e34e0-113">Derleyici kullanacağı `greeting` genişletme yöntemi için bağımsız değişken olarak `Print`.</span><span class="sxs-lookup"><span data-stu-id="e34e0-113">The compiler will use `greeting` as the argument to extension method `Print`.</span></span>

```  
greeting.Print()  
```
