# <a name="unable-to-read-fixed-width-fields-because-fieldwidths-is-nothing-or-empty"></a>FieldWidths Nothing olduğundan sabit genişlikli alanlar okunamıyor veya boş
`TextFieldParser` Sabit genişlikli alanları çünkü okuyamıyor `FieldWidths` özelliği ayarlanmış `Nothing` veya boş.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ayarlama `FieldWidths` geçerli bir değer.
