# <a name="this-system-does-not-contain-support-for-the-simplifiedchinese-locale"></a>Bu sistem SimplifiedChinese yerel ayar desteği içermiyor
Kullanmaya çalıştığınız `VbStrConv.SimplifiedChinese` numaralandırma üyesi ancak sisteminizi Basitleştirilmiş Çince yerel ayar desteklemez.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Basitleştirilmiş Çince yerel ayar desteklemek için makine yükseltin veya uygulamayı yerel destekleyen bir makinede çalıştırın.
