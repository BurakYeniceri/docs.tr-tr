# <a name="how-to-remove-autogenerated-columns-from-a-windows-forms-datagridview-control"></a>Nasıl yapılır: Otomatik Oluşturulan Sütunları Windows Forms DataGridView Denetiminden Kaldırma
Olduğunda, <xref:System.Windows.Forms.DataGridView> denetim sütunlarını veri kaynağından verileri temel alan için otomatik olarak ayarlandığında, seçmeli olarak bazı sütunlar atlayabilirsiniz. Bunu çağırarak yapmak <xref:System.Windows.Forms.DataGridViewColumnCollection.Remove%2A> yöntemi <xref:System.Windows.Forms.DataGridView.Columns%2A> koleksiyonu. Ayarlayarak görünümünde sütunları alternatif olarak, gizleyebilirsiniz <xref:System.Windows.Forms.DataGridViewColumn.Visible%2A> özelliğine `false`. Belirli koşullar gizli sütunları görüntülemek istediğinizde ya da görüntülemeden sütunları verilere erişim gerektiğinde bu teknik yararlıdır.
