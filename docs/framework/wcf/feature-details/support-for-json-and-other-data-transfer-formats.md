# <a name="support-for-json-and-other-data-transfer-formats"></a>JSON ve Diğer Veri Aktarma Biçimleri için Destek
JSON (JavaScript nesne gösterimi) olan küçük miktarda veri istemci tarayıcıları ve AJAX (zaman uyumsuz JavaScript ve XML) arasında hızlı yönlendiricilerin değişimini bir verimli veri kodlama biçimi-Web Hizmetleri etkin.

 [Bağımsız JSON Seri Hale Getirme](../../../../docs/framework/wcf/feature-details/stand-alone-json-serialization.md)  
 Bağımsız JSON seri hale getirme açıklar.  
  
 [Nasıl yapılır: JSON Verilerini Seri Hale Getrime ve Seri Halden Çıkarma](../../../../docs/framework/wcf/feature-details/how-to-serialize-and-deserialize-json-data.md)  
 JSON olarak kodlanmış veri .NET türü nesneleri seri hale getirmek ve geri kullanarak .NET türleri örneğine verileri JSON biçiminde seri durumdan nasıl oluşturulduğunu gösteren <xref:System.Runtime.Serialization.Json.DataContractJsonSerializer>.
