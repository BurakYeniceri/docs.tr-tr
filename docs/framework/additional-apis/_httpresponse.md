# <a name="httpwebrequesthttpresponse-field"></a>HttpWebRequest. \_HttpResponse alan

`HttpWebRequest._HttpResponse` olan bir <xref:System.Net.HttpWebResponse> bir HTTP istek HTTP yanıt ayrıntılarını içeren. Bu olabilir `null` bir HTTP yanıt alınana kadar.

## <a name="syntax"></a>Sözdizimi
  
```csharp  
internal HttpWebResponse _HttpResponse
```

> [!WARNING]
> `HttpWebRequest._HttpResponse` Alandır iç ve kodunuzda doğrudan kullanılmak üzere yüksetlmesi.
> 
> Microsoft hiçbir koşulda bir üretim uygulamasında bu alan kullanımını desteklemez.

## <a name="requirements"></a>Gereksinimler

**Namespace:** <xref:System.Net>

**Derleme:** sisteminde (System.dll)

**.NET framework sürümleri:** 2.0 sürümünden itibaren kullanılabilir.
