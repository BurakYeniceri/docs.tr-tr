# <a name="iapartmentcallback-interface"></a>IApartmentCallback Arabirimi
Geri aramalar bir grup içinde sağlama yöntemleri sağlar. Bir *grup* aynı iş parçacığı gerekliliklerini paylaşmak nesneleri için bir işlem içinde mantıksal bir kapsayıcısıdır.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[DoCallback Yöntemi](../../../../docs/framework/unmanaged-api/hosting/iapartmentcallback-docallback-method.md)|Belirtilen işlev bir grup içinde yürütür.|
