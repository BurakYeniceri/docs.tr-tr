# <a name="xml-comment-type-parameter-must-have-a-39name39-attribute"></a>XML açıklama türü parametresi olmalıdır bir &#39;adı&#39; özniteliği
XML açıklama tür parametresi bir 'name' özniteliği olmalıdır. XML açıklama yoksayılacak.  
  
 XML açıklama etiketi gerektiren bir `name` yok veya eksik öznitelik.  
  
 **Hata Kimliği:** BC42318  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme ya da gerekli tamamlamak `name` özniteliği etiketi.
