# <a name="end-of-expression-expected"></a>İfade sonu bekleniyor
Aksi halde tam bir deyim sonunda fazladan karakterler bulundu.  
  
 **Hata Kimliği:** BC30710  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Nesne veya deyimi söz dizimini doğrulayın.
