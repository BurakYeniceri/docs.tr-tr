### <a name="to-add-a-table-to-the-datagrid-control-in-the-designer"></a><span data-ttu-id="2fd1c-116">Bir Tablo Tasarımcısı DataGrid denetiminde eklemek için</span><span class="sxs-lookup"><span data-stu-id="2fd1c-116">To add a table to the DataGrid control in the designer</span></span>  
  
1.  <span data-ttu-id="2fd1c-117">Tablodaki verileri görüntülemek için önce bağlamanız gerekir <xref:System.Windows.Forms.DataGrid> denetlemek için bir veri kümesi.</span><span class="sxs-lookup"><span data-stu-id="2fd1c-117">In order to display data in the table, you must first bind the <xref:System.Windows.Forms.DataGrid> control to a dataset.</span></span> <span data-ttu-id="2fd1c-118">Daha fazla bilgi için bkz: [nasıl yapılır: Windows Forms DataGrid denetimini veri kaynağı kullanarak bir tasarımcı bağlama](../../../../docs/framework/winforms/controls/bind-wf-datagrid-control-to-a-data-source-using-the-designer.md).</span><span class="sxs-lookup"><span data-stu-id="2fd1c-118">For more information, see [How to: Bind the Windows Forms DataGrid Control to a Data Source Using the Designer](../../../../docs/framework/winforms/controls/bind-wf-datagrid-control-to-a-data-source-using-the-designer.md).</span></span>  
  
2.  <span data-ttu-id="2fd1c-119">Seçin <xref:System.Windows.Forms.DataGrid> denetimin <xref:System.Windows.Forms.DataGrid.TableStyles%2A> Özellikleri penceresinde özelliği ve üç nokta düğmesini (![VisualStudioEllipsesButton ekran görüntüsü](../../../../docs/framework/winforms/media/vbellipsesbutton.png "vbEllipsesButton")) yanındaki görüntülenecek özelliği **DataGridTableStyle Koleksiyonu Düzenleyicisi**.</span><span class="sxs-lookup"><span data-stu-id="2fd1c-119">Select the <xref:System.Windows.Forms.DataGrid> control's <xref:System.Windows.Forms.DataGrid.TableStyles%2A> property in the Properties window, and then click the ellipsis button (![VisualStudioEllipsesButton screenshot](../../../../docs/framework/winforms/media/vbellipsesbutton.png "vbEllipsesButton")) next to the property to display the **DataGridTableStyle Collection Editor**.</span></span>  
  
3.  <span data-ttu-id="2fd1c-120">Koleksiyon Düzenleyicisi'nde tıklatın **Ekle** tablo stili eklemek için.</span><span class="sxs-lookup"><span data-stu-id="2fd1c-120">In the collection editor, click **Add** to insert a table style.</span></span>  
  
4.  <span data-ttu-id="2fd1c-121">Tıklatın **Tamam** koleksiyon Düzenleyicisi'ni kapatın ve yanındaki üç nokta düğmesine tıklayarak yeniden <xref:System.Windows.Forms.DataGrid.TableStyles%2A> özelliği.</span><span class="sxs-lookup"><span data-stu-id="2fd1c-121">Click **OK** to close the collection editor, and then reopen it by clicking the ellipsis button next to the <xref:System.Windows.Forms.DataGrid.TableStyles%2A> property.</span></span>  
  
     <span data-ttu-id="2fd1c-122">Koleksiyon Düzenleyicisi'ni yeniden açtığınızda denetime bağlı herhangi bir veri tabloları için aşağı açılan listesinde görünecektir <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> tablo stili özelliği.</span><span class="sxs-lookup"><span data-stu-id="2fd1c-122">When you reopen the collection editor, any data tables bound to the control will appear in the drop-down list for the <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> property of the table style.</span></span>  
  
5.  <span data-ttu-id="2fd1c-123">İçinde **üyeleri** kutusunu Koleksiyonu Düzenleyicisi tablo stilini tıklatın.</span><span class="sxs-lookup"><span data-stu-id="2fd1c-123">In the **Members** box of the collection editor, click the table style.</span></span>  
  
6.  <span data-ttu-id="2fd1c-124">İçinde **özellikleri** select Koleksiyonu Düzenleyicisi kutusunun <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> görüntülemek istediğiniz tablo için değer.</span><span class="sxs-lookup"><span data-stu-id="2fd1c-124">In the **Properties** box of the collection editor, select the <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> value for the table you want to display.</span></span>
