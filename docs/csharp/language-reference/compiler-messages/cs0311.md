 <span data-ttu-id="b49d1-105">Genel tür parametresi bir kısıtlama uygulandığında örtük bir kimlik veya başvuru dönüştürme kısıtlama türü için sabit bağımsız değişkenden mevcut olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="b49d1-105">When a constraint is applied to a generic type parameter, an implicit identity or reference conversion must exist from the concrete argument to the type of the constraint.</span></span>

## <a name="example"></a><span data-ttu-id="b49d1-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="b49d1-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B{}  
class C{}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```
