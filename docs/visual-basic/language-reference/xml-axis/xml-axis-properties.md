# <a name="xml-axis-properties-visual-basic"></a>XML Eksen Özellikleri (Visual Basic)
Bu bölümdeki konular, Visual Basic'te XML eksen özellikleri söz dizimi belge. XML eksen özellikleri XML kodunuzda doğrudan erişmek kolaylaştırır.  
  
## <a name="in-this-section"></a>Bu Bölümde  
  
|Konu|Açıklama|  
|-----------|-----------------|  
|[XML Özniteliği Axis Özelliği](../../../visual-basic/language-reference/xml-axis/xml-attribute-axis-property.md)|Özniteliklerini erişim açıklar bir <xref:System.Xml.Linq.XElement> nesnesi.|  
|[XML Alt Axis Özelliği](../../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md)|Öğenin alt öğelerine erişme açıklar bir <xref:System.Xml.Linq.XElement> nesnesi.|  
|[XML Descendant Axis Özelliği](../../../visual-basic/language-reference/xml-axis/xml-descendant-axis-property.md)|Alt öğeleri erişim açıklar bir <xref:System.Xml.Linq.XElement> nesnesi.|  
|[Extension Indexer Özelliği](../../../visual-basic/language-reference/xml-axis/extension-indexer-property.md)|Ayrı ayrı öğeler koleksiyonunda erişim açıklar <xref:System.Xml.Linq.XElement> veya <xref:System.Xml.Linq.XAttribute> nesneleri.|  
|[XML Value Özelliği](../../../visual-basic/language-reference/xml-axis/xml-value-property.md)|Bir ilk öğesinin değeri erişim açıklar <xref:System.Xml.Linq.XElement> veya <xref:System.Xml.Linq.XAttribute> nesneleri.|
