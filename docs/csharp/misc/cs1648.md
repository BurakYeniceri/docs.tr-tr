# <a name="compiler-error-cs1648"></a><span data-ttu-id="aa1eb-102">Derleyici Hatası CS1648</span><span class="sxs-lookup"><span data-stu-id="aa1eb-102">Compiler Error CS1648</span></span>
<span data-ttu-id="aa1eb-103">Salt okunur alan 'tanımlayıcısı' üyeleri değiştirilemez (oluşturucu veya değişken başlatıcı hariç)</span><span class="sxs-lookup"><span data-stu-id="aa1eb-103">Members of readonly field 'identifier' cannot be modified (except in a constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="aa1eb-104">Burada buna değiştirilmesine izin verilmez, salt okunur bir alanı üyesi değiştirmeye çalıştığında bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="aa1eb-104">This error occurs when you attempt to modify a member of a field which is readonly where it is not allowed to be modified.</span></span> <span data-ttu-id="aa1eb-105">Bu hatayı gidermek için salt okunur alan oluşturucu veya değişken Başlatıcısı atamaları sınırlamak veya readonly anahtar alanı bildiriminden kaldırın.</span><span class="sxs-lookup"><span data-stu-id="aa1eb-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the readonly keyword from the declaration of the field.</span></span>  
  
 <span data-ttu-id="aa1eb-106">Aşağıdaki örnek CS1648 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="aa1eb-106">The following sample generates CS1648:</span></span>  
  
```csharp  
// CS1648.cs  
public struct Inner  
  {  
    public int i;  
  }  
  
class Outer  
{    
  public readonly Inner inner = new Inner();  
}  
  
class D  
{  
   static void Main()  
   {  
      Outer outer = new Outer();  
      outer.inner.i = 1;  // CS1648  
   }  
}  
```
