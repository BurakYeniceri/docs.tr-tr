# <a name="compiler-error-cs0218"></a>Derleyici Hatası CS0218
Türü ('type') işleci true ve false işleci bildirimlerini içermelidir  
  
 Kullanıcı tanımlı bir tür için bir işleci tanımlama ve operatör olarak kullanmaya çalıştığınızda bir işleç kısa devre oluşturur, kullanıcı tanımlı işleci işleci true ve false tanımlanan işleci olması gerekir. İşleçler hakkında daha fazla bilgi için kısa devre oluşturur, bkz: [& & işleci](../../csharp/language-reference/operators/conditional-and-operator.md) ve [&#124; &#124; İşleç](../../csharp/language-reference/operators/conditional-or-operator.md).  
  
 Aşağıdaki örnek CS0218 oluşturur:  
  
```  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
