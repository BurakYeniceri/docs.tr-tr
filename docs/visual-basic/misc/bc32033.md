# <a name="39end-get39-must-be-the-first-statement-on-a-line"></a>&#39;End Get&#39; satırdaki ilk deyim olmalıdır
Bir `End Get` deyimi izleyen iki nokta üst üste (:) deyimi ayırıcı. `End Get` Kaynak satırında yegane deyim olmalıdır.  
  
 **Hata Kimliği:** BC32033  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Birden çok deyime farklı satırlarına ayırın.
