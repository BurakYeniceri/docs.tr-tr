 <span data-ttu-id="a0dfc-104">İçinde bir [ad alanı](../../../csharp/language-reference/keywords/namespace.md), derleyici yalnızca sınıflar, yapılar, birleşimler, numaralandırmalar, arabirimler ve temsilciler kabul eder.</span><span class="sxs-lookup"><span data-stu-id="a0dfc-104">Inside a [namespace](../../../csharp/language-reference/keywords/namespace.md), the compiler only accepts classes, structures, unions, enumerations, interfaces, and delegates.</span></span> <span data-ttu-id="a0dfc-105">Bu hata genellikle C# ' ta yöntemleri ve değişkenleri gerekir olarak bildirilmiş ve bir yapı veya sınıfı içinde tanımlanan olduğunu unuttunuz geliştiriciler C/C++ arka plan gelen tarafından oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="a0dfc-105">This error is often generated by developers from a C/C++ background who forget that in C#, methods and variables must be declared and defined within a struct or class.</span></span> <span data-ttu-id="a0dfc-106">Daha fazla bilgi için bkz: [C# programının genel yapısı](../../../csharp/programming-guide/inside-a-program/general-structure-of-a-csharp-program.md).</span><span class="sxs-lookup"><span data-stu-id="a0dfc-106">For more information, see [General Structure of a C# Program](../../../csharp/programming-guide/inside-a-program/general-structure-of-a-csharp-program.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="a0dfc-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="a0dfc-107">Example</span></span>  
 <span data-ttu-id="a0dfc-108">Aşağıdaki örnek CS0116 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a0dfc-108">The following sample generates CS0116:</span></span>  
  
```csharp  
// CS0116.cs  
namespace x  
{  
    using System;  
  
    // method must be in class/struct  
    void Method(string str) // CS0116  
    {  
        Console.WriteLine(str);  
    }  
    // To fix the error, you must  
    // enclose a method in a class:  
    class Program  
    {  
        void Method2(string str)  
        {  
            Console.WriteLine(str);  
        }  
    }  
}  
```
