# <a name="statement-cannot-appear-within-a-property-body"></a>Özellik gövdesi içinde deyimi yer alamaz
Özellik gövdesi içinde deyimi yer alamaz. Kabul özellik sonu.  
  
 Beklenmeyen bir deyimi içinde bir özellik karşılaşıldı.  
  
 **Hata Kimliği:** BC30634  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Özelliğinizi doğru söz dizimini kullanır ve ile sona erer doğrulayın bir `End``Property` oluşturun.
