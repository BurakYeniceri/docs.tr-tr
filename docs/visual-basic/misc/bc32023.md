# <a name="expression-is-of-type-39lttypenamegt39-which-is-not-a-collection-type"></a>İfade türüdür &#39; &lt;typename&gt;&#39;, koleksiyon türü değil
Belirtilen grup değişkeni bir `For Each` deyimi bir koleksiyon nesnesi veya bir dizi değil ve türünü uygulamayan <xref:System.Collections.IEnumerable> arabirimi. Türü Visual Basic koleksiyon tasarım deseni destek veya uygulama <xref:System.Collections.IEnumerable>.  
  
 **Hata Kimliği:** BC32023  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sınıf türü ya da desteklediğini uygular ve Visual Basic koleksiyon tasarım olmasını Grup değişkeni bildirme <xref:System.Collections.IEnumerable>.
