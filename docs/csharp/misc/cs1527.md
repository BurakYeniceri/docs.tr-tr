# <a name="compiler-error-cs1527"></a><span data-ttu-id="b4918-102">Derleyici Hatası CS1527</span><span class="sxs-lookup"><span data-stu-id="b4918-102">Compiler Error CS1527</span></span>
<span data-ttu-id="b4918-103">Bir ad alanında tanımlanan öğe açıkça özel olarak bildirilemez korumalı, iç korumalı veya özel korumalı.</span><span class="sxs-lookup"><span data-stu-id="b4918-103">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span> 
  
 <span data-ttu-id="b4918-104">Bir ad alanı türü bildirimlerinde ya da olabilir [ortak](../../csharp/language-reference/keywords/public.md) veya [iç](../../csharp/language-reference/keywords/internal.md) erişim.</span><span class="sxs-lookup"><span data-stu-id="b4918-104">Type declarations in a namespace can have either [public](../../csharp/language-reference/keywords/public.md) or [internal](../../csharp/language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="b4918-105">Hiçbir erişilebilirlik belirtilirse, **iç** varsayılandır.</span><span class="sxs-lookup"><span data-stu-id="b4918-105">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="b4918-106">Aşağıdaki örnek CS1527 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b4918-106">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="b4918-107">Hiçbir ad alanı, program kodu içinde açıkça bildirilmişse tüm tür bildirimleri örtük olarak genel ad alanı içinde bulunan çünkü aşağıdaki örnek CS1527 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="b4918-107">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4{}  
private struct S1{}  
```
