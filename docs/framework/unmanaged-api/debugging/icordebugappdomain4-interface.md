# <a name="icordebugappdomain4-interface"></a>ICorDebugAppDomain4 Arabirimi
Mantıksal olarak yönetilen bir nesnenin COM aranabilir sarmalayıcısı almak için Icordebugappdomain arabirimi genişletir.  
  
## <a name="method"></a>Yöntem  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[GetObjectForCCW yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugappdomain4-getobjectforccw-method.md)|COM aranabilir sarmalayıcısı (saat) işaretçi yönetilen bir nesne alır.|
