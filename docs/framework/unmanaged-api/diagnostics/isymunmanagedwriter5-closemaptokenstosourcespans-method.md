# <a name="isymunmanagedwriter5closemaptokenstosourcespans-method"></a>ISymUnmanagedWriter5::CloseMapTokensToSourceSpans Yöntemi
Belirteç kaynak aralık bilgi eşleme için özel özel veri bölümü kapatın. Kapalı olduğu sonra daha fazla eşleme bilgi eklenebilir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```idl  
HRESULT CloseMapTokensToSourceSpans();  
```  
  
## <a name="return-value"></a>Dönüş Değeri  
 Döndürür `HRESULT`.  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** CorSym.idl, CorSym.h
