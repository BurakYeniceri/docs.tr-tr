# <a name="textfieldparser-does-not-support-delimiters-that-contain-endline-characters"></a>TextFieldParser endline karakterden ayırıcıları desteklemiyor
Bir endline karakter içeren bir sınırlayıcı belirtildi, ancak `TextFieldParser` endline karakterden ayırıcıları desteklemiyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Doğru sınırlayıcı sağlayın.
