# <a name="internal-error-visual-basic"></a>İç hata (Visual Basic)
Visual Basic'te bir iç hatası oluştu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Emin olun bu hatayı yok üretilen tarafından `Error` deyimi veya `Raise` yöntemi. Olduğu sürece, ileti görünen koşullarını bildirmek için Microsoft Ürün Destek Hizmetleri'ne başvurun.
