# <a name="39goto39-statements-are-not-valid-in-the-immediate-window"></a>&#39; GoTo &#39; deyimleri Komut penceresi içinde geçerli değildir
`GoTo` Deyimi dallanma gerçekleştirir ve hata ayıklama bağlamda izin verilmiyor.  
  
 **Hata Kimliği:** BC30133  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sorun değil bir `GoTo` deyiminde **hemen** penceresi.
