# <a name="39on-error39-statements-are-not-valid-within-39using39-statements"></a>&#39;Hatada&#39; deyimleri içinde geçerli değildir &#39;kullanma&#39; deyimleri
Bir `On Error` deyimi görünür içinde bir `Using` deyimi ancak bu bağlamda geçerli değil.  
  
 **Hata Kimliği:** BC36013  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yapılandırılmış hata işleme, aşağıdaki gibi kullanan bir `Try…Catch` bloğunda yerini `On Error` deyimi.
