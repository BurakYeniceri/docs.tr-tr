# <a name="projectjson-and-visual-studio-2015-with-net-core"></a>Project.JSON ve .NET Core içeren Visual Studio 2015

Belge sürümünden önce 7 Mart güncelleştirme PDF dosyası ve belgeleri deposunda bir dalın kullanılabilir.

En iyi önceki belgelerin PDF dosyaları kaynağıdır:

* [.NET core - project.json PDF ve Visual Studio 2015](https://github.com/dotnet/docs/blob/project.json/net-core-project-json.pdf)
* [ASP.NET Core - project.json PDF ve Visual Studio 2015](https://github.com/aspnet/Docs/blob/master/aspnetcore/common/_static/aspnet-core-project-json.pdf)
