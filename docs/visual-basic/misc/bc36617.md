# <a name="aggregate-function-name-cannot-be-used-with-a-type-character"></a>Toplama işlevi adı türü karakteri ile kullanılamaz
Bir toplama işlevinin adının sonuna bir tür karakteri eklenmiştir.  
  
 **Hata Kimliği:** BC36617  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Tür karakteri toplama işlevi adından kaldırın.
