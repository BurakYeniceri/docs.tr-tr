# <a name="event-39lteventnamegt39-cannot-be-handled-because-it-is-not-accessible-from-39ltnamegt39"></a>Olay &#39; &lt;eventname&gt;&#39; erişilebilir olduğundan işlenemiyor &#39;&lt; ad&gt;&#39;
Erişilemez bir olay işleme çalışıldı. Örneğin, bir `Handles` değişkeni paylaşılan, olayları işleme yöntemi de paylaşılması gerekir.  
  
 **Hata Kimliği:** BC30585  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Olay erişilebilir olduğundan emin olun.
