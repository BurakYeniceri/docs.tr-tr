# <a name="39inherits39-statement-must-precede-all-declarations-in-a-class"></a>&#39;Devralınan&#39; deyimi içinde bir sınıfın tüm bildirimler gelmelidir
`Inherits` Deyimleri sınıf tanımları başında olması gerekir.  
  
 **Hata Kimliği:** BC30683  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Taşıma `Inherits` deyimini sınıfınıza üstüne.
