# <a name="hosting-interfaces"></a>Barındırma Arabirimleri
Bu bölümde yönetilmeyen arabirimler açıklanmaktadır konakları ortak dil çalışma zamanı (CLR) uygulamalarına tümleştirmeleri için kullanabilirsiniz.  
  
 .NET Framework sürüm 2.0 barındırma arabirimleri .NET Framework sürüm 1.0 ve 1.1 arabirimleri yerini alır. Arabirimlerin iki küme arasındaki önemli farklılıklar vardır. Bunları karıştırma beklenmeyen davranışlara neden olabilir ve önerilmez.  
  
 .NET Framework sürüm 3.0 ve 3.5, .NET Framework sürüm 2.0 barındırma arabirimleri kullanın ve barındırma hiçbir özellikleri tanıtır değil.  
  
 [!INCLUDE[net_v40_long](../../../../includes/net-v40-long-md.md)] Barındırma arabirimleri .NET Framework 2.0 arabirimleri yerini alır.
