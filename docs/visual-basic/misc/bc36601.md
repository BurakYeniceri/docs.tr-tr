# <a name="type-characters-cannot-be-used-in-range-variable-declarations"></a>Aralık değişkeni bildirimlerinde türü karakterler kullanılamaz
LINQ sorgusu aralık değişkeni bildiriminde adının sonunda bir tür karakteri eklenmiştir.  
  
 **Hata Kimliği:** BC36601  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Tür karakteri aralık değişkeni bildiriminden kaldırın.
