# <a name="39addhandler39-is-already-declared"></a>&#39;AddHandler&#39; zaten bildirildi
Birden fazla `AddHandler` bildirimi özel olay bildiriminde görüntülenir. Bir `AddHandler` bildirimi olay işleyicisi eklemek için kullanılan yordamı bildirir.  
  
 **Hata Kimliği:** BC31127  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yedek kaldırmak `AddHandler` deyimi.
