# <a name="xml-literals-and-the-xml-10-specification-visual-basic"></a>XML Değişmez Değerleri ve XML 1.0 Belirtimi (Visual Basic)
XML değişmez değer sözdiziminde [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] Genişletilebilir İşaretleme Dili (XML) 1.0 belirtimi çoğunu destekler. XML 1.0 belirtimi hakkında daha fazla ayrıntı için bkz: [Genişletilebilir İşaretleme Dili (XML) 1.0](http://go.microsoft.com/fwlink/?LinkId=73927) W3C Web sitesinde.  
  
## <a name="what-visual-basic-does-not-support"></a>Ne Visual Basic desteklemez  
  
-   Değişmez değer XML belge türü tanımı (DTD) içeremez.  
  
-   Bir XML belgesi değişmez değeri bir XML belgesi bildirimi ile başlamalıdır.  
  
-   XML değişmez bir satıra birden fazla 65.535 karakter içeremez.  
  
-   XML ad alanı önekleri, öğe adları ve öznitelik adları birden fazla 1024 karakter içeremez.  
  
## <a name="extra-features-that-visual-basic-supports"></a>Visual Basic destekler ek özellikler  
  
-   Belge ve öğe değişmez değerlerine izin katıştırılmış ifade sözdizimi geçerli XML değil.
