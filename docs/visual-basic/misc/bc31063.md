# <a name="39set39-method-cannot-have-more-than-one-parameter"></a>&#39; Set &#39; yöntemi, birden fazla parametresi olamaz
Bir özellik için birden fazla parametre tanımlanmış `Set` yordamlar.  
  
 **Hata Kimliği:** BC31063  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Herhangi bir ek parametre özellikten kaldırmak `Set` yordamlar.
