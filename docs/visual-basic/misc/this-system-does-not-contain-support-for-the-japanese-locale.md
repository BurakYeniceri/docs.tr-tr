# <a name="this-system-does-not-contain-support-for-the-japanese-locale"></a>Bu sistem Japonca yerel ayar desteği içermiyor
Kullanmaya çalıştığınız `VbStrConv.Japanese` numaralandırma üyesi ancak sisteminizi Japonca yerel ayarını desteklemez.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Japonca yerel ayarını desteklemek için makine yükseltin veya uygulamayı yerel destekleyen bir makinede çalıştırın.
