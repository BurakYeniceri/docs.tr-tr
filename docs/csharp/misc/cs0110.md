# <a name="compiler-error-cs0110"></a><span data-ttu-id="7dfcf-102">Derleyici Hatası CS0110</span><span class="sxs-lookup"><span data-stu-id="7dfcf-102">Compiler Error CS0110</span></span>
<span data-ttu-id="7dfcf-103">'Const bildiriminin' sabit değer değerlendirmesi döngüsel başvuru içeriyor</span><span class="sxs-lookup"><span data-stu-id="7dfcf-103">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="7dfcf-104">Bildirimi bir [const](../../csharp/language-reference/keywords/const.md) değişkeni (`a`) başka bir const değişkeni başvuramaz (`b`) ayrıca başvuran (`a`).</span><span class="sxs-lookup"><span data-stu-id="7dfcf-104">The declaration of a [const](../../csharp/language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="7dfcf-105">Aşağıdaki örnek CS0110 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7dfcf-105">The following sample generates CS0110:</span></span>  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
