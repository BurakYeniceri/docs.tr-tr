# <a name="compiler-error-cs0726"></a>Derleyici Hatası CS0726
'biçim belirticisi' geçerli bir biçim belirleyici değil  
  
 Bu hata ayıklayıcıda oluşur. Hata ayıklayıcı windows birine bir değişken adı yazdığınızda, virgül, ardından biçim belirteci ile izleyebilirsiniz. Örnekler: `myInt, h` veya `myString,nq`. Derleyici tanımaz olduğunda bu hata ortaya [C# içindeki Biçim belirticileri](/visualstudio/debugger/format-specifiers-in-csharp).
