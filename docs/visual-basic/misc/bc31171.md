# <a name="character-sequence-39--39-is-not-allowed-in-an-xml-comment"></a>Karakter dizisi &#39;--&#39; bir XML açıklama izin verilmiyor
Bir XML açıklama değişmez değeri bir karakter dizisi içeriyor '--', desteklenmiyor.  
  
 **Hata Kimliği:** BC31171  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Karakter dizisi Kaldır '--' den XML açıklama değişmez değeri.
