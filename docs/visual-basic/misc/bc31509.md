# <a name="unable-to-open-resource-file-39ltfilenamegt39-lterrorgt"></a>Kaynak dosya &#39;açılamıyor; &lt;filename&gt;&#39;: &lt;hata&gt;
Belirtilen kaynak dosyası açılırken bir hata oluştu.  
  
 **Hata Kimliği:** BC31509  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Belirtilen ad ve konum doğru olduğundan emin olun.  
  
2.  Dosya bozuk veya bozuk olmadığını doğrulayın.
