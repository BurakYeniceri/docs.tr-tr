# <a name="xml-comment-cannot-appear-within-a-method-or-a-property"></a>XML açıklama bir yöntem veya bir özellik içinde yer alamaz
Bir XML açıklama bir yöntem veya bir özellik içinde yer alamaz. XML açıklama yoksayılacak. XML açıklama blokları yöntemleri ya da özellikleri içinde yer alamaz.  
  
 **Hata Kimliği:** BC42303  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Açıklama bloğunu kaldırın veya daha uygun bir konuma taşıyın.
