# <a name="39as39-comma-or-3939-expected"></a>&#39; As &#39; virgül veya &#39;) &#39; Beklenen
Sağlanmış tür parametresi eksik bir `As`, virgülle (`,`), veya parantez kapatma ( `)` ).  
  
 **Hata Kimliği:** BC32100  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Eksik tedarik `As`, virgül veya kapanış parantezi.
