# <a name="division-by-zero-visual-basic-error"></a>Sıfır (Visual Basic hatası) bölme
Bölen olarak kullanılan bir ifade değeri sıfır olur.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  İfade değişkenlerde yazımını denetleyin. Yanlış yazılmış bir değişken örtük olarak sıfır olarak başlatılan sayısal bir değişken oluşturabilirsiniz.  
  
2.  Önceki işlemlerini ifade değişkeninde özellikle yordama diğer yordamlardan bağımsız değişken olarak geçirilen denetleyin.
