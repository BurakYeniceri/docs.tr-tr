# <a name="implementing-class-39ltclassnamegt39-for-interface-ltinterfacenamegt-cannot-be-found"></a>Uygulama sınıfı &#39; &lt;classname&gt;&#39; arabirimin &lt;InterfaceName&gt; bulunamıyor
Birlikte çalışma derlemesindeki uygulayan bir sınıf kullanılabilir değil.  
  
 **Hata Kimliği:** BC31094  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Tür kitaplığı COM nesnesi için geçerli olduğunu doğrulayın.  
  
2.  El ile birlikte çalışma bütünleştirilmiş oluşturmak için tür kitaplığı içeri Aktarıcı (Tlbimp.exe) kullanın ve ardından bu birlikte çalışma derlemesine başvuru projenizden ekleyin.
