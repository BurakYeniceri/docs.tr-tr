# <a name="39readonly39-and-39writeonly39-cannot-be-combined"></a>&#39;Salt okunur&#39; ve &#39;WriteOnly&#39; birleştirilemez
`ReadOnly` Ve `WriteOnly` değiştiricileri karşılıklı olarak birbirini dışlar.  
  
 **Hata Kimliği:** BC30635  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirtin `ReadOnly` veya `WriteOnly`, ancak ikisini birden değil.
