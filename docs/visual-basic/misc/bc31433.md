# <a name="method-39ltmethodnamegt39-cannot-be-declared-39partial39-because-only-one-method-39ltmethodnamegt39-can-be-marked-39partial39"></a>Yöntem &#39; &lt;methodname&gt; &#39; bildirilemez &#39;kısmi&#39; çünkü tek bir yöntem &#39; &lt;methodname&gt; &#39; işaretlenebilir &#39;kısmi&#39;
Aynı ada ve imzaya sahip iki kısmi yöntemi bildirimleri olamaz.  
  
 **Hata Kimliği:** BC31433  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Adı veya kısmi yöntem bildirimleri birinin imza değiştirin.
