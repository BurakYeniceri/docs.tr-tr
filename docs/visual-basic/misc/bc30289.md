# <a name="statement-cannot-appear-within-a-method-body"></a>Bir yöntem gövdesinde deyimi yer alamaz
Bir yordam başka bir yordam bildirimi gibi geçersiz bir ifade içerir. Yordam bildirimleri iç içe olamaz.  
  
 **Hata Kimliği:** BC30289  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yordam bildirimini kaldırın veya kapsayan yordamı taşıyın.
