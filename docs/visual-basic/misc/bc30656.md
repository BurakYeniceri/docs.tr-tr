# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>Alan &#39; &lt;fieldname&gt; &#39; türü desteklenmiyor
Bir alan Visual Studio tarafından temsil edilen bir türü olarak bildirilmiş.  
  
 **Hata Kimliği:** BC30656  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Alan desteklenen bir tür olarak tanımlayın.  
  
2.  Alan içindeki verileri tanımlayan yeni türü geliştirin.
