# <a name="field-or-property-39ltmembernamegt39-is-not-found"></a>Alan veya özellik &#39; &lt;membername&gt; &#39; bulunamadı
Bir tanımsız alan veya özellik özniteliğini başvuruluyor.  
  
 **Hata Kimliği:** BC30661  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Alan veya özellik yazımını denetleyin.  
  
2.  Projenizi doğru öznitelik sınıfı erişebilir ve gerekirse proje başvuruları ekleme olduğunu doğrulayın.
