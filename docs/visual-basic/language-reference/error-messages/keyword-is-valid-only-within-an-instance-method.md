# <a name="39ltkeywordgt39-is-valid-only-within-an-instance-method"></a>&#39;&lt;anahtar sözcüğü&gt; &#39; yalnızca bir örnek yöntemi içinde geçerli değil
`Me`, `MyClass`, Ve `MyBase` anahtar sözcükleri belirli bir sınıfın örneklerine bakın. Bunları paylaşılan içinde kullanamazsınız `Function` veya `Sub` yordamı.  
  
 **Hata Kimliği:** BC30043  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Anahtar sözcüğü yordamdan kaldırın veya kaldırın `Shared` yordamı bildirimi anahtar sözcük.
