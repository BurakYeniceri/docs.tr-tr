# <a name="argument-39access39-is-not-valid-valid-values-for-input-mode-are-39openaccessread39-and-39openaccessdefault39"></a>Bağımsız değişken &#39; Erişim &#39; geçerli değil. Giriş modu için geçerli değerler şunlardır #39;OpenAccess.Read &#39; ve &#39;OpenAccess.Default &#39;
Bir erişim değeri geçerli değil `Input` modu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değeri ya da değiştirin `OpenAccess.Read` veya `OpenAccess.Default`.
