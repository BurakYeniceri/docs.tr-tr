# <a name="compiler-error-cs0171"></a><span data-ttu-id="9e2bf-102">Derleyici Hatası CS0171</span><span class="sxs-lookup"><span data-stu-id="9e2bf-102">Compiler Error CS0171</span></span>
<span data-ttu-id="9e2bf-103">Denetim çağırana döndürülmeden önce yedekleme alanını otomatik olarak uygulanan bir özellik 'name' için tam olarak atanması gerekir.</span><span class="sxs-lookup"><span data-stu-id="9e2bf-103">Backing field for automatically implemented property 'name' must be fully assigned before control is returned to the caller.</span></span> <span data-ttu-id="9e2bf-104">Varsayılan Oluşturucu oluşturucu başlatıcı üzerinden çağırmayı düşünün.</span><span class="sxs-lookup"><span data-stu-id="9e2bf-104">Consider calling the default constructor from a constructor initializer.</span></span>  
  
 <span data-ttu-id="9e2bf-105">Oluşturucu içinde bir [yapısı](../../csharp/language-reference/keywords/struct.md) yapısı tüm alanlarda başlatması gerekir.</span><span class="sxs-lookup"><span data-stu-id="9e2bf-105">A constructor in a [struct](../../csharp/language-reference/keywords/struct.md) must initialize all fields in the struct.</span></span> <span data-ttu-id="9e2bf-106">Daha fazla bilgi için bkz: [oluşturucular](../../csharp/programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="9e2bf-106">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="9e2bf-107">Aşağıdaki örnek CS0171 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="9e2bf-107">The following sample generates CS0171:</span></span>  
  
```csharp  
// CS0171.cs

{  
   MyStruct(int initField)   // CS0171  
   {  
      // i = initField;      // uncomment this line to resolve this error  
   }  
   public int i;  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyStruct aStruct = new MyStruct();  
   }  
}  
```
