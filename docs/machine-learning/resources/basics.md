# <a name="machine-learning-basics"></a>Machine learning temelleri

Bu makale makine öğrenimine giriş vermek için temel bazı kaynaklar sunar.

## <a name="what-is-machine-learning"></a>Machine learning nedir?

Machine learning bilgisayarların gelecekteki davranışları, sonuçları ve eğilimleri tahmin etmek için var olan verileri kullanmasına izin veren bir veri bilimi tekniğidir. Machine learning kullanarak bilgisayarları açıkça programlanmış olmadan öğrenin.

Öngörüleri veya tahminleri machine learning'in uygulamaları ve cihazları daha akıllı hale getirir. Çevrimiçi alışveriş yaparken makine öğrenimi ne, satın aldıklarınızı temel alarak beğenebileceğiniz diğer ürünleri önermede yardımcı olur. Kredi kartınız makineden geçirilirken makine öğrenme bir veritabanı için işlem hareketlerinin karşılaştırır ve sahtekarlığı algılamaya yardımcı olur. Bir yer robot vakum temizleyici vacuums, machine learning, iş yapılır karar yardımcı olur.
