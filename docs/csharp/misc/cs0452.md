<span data-ttu-id="7128f-103">'Type name' bir başvuru türü 'parametre adı' parametresi kullanılabilmesi için genel türünde veya yönteminde 'tanımlayıcısını genel ' türünde olmalıdır</span><span class="sxs-lookup"><span data-stu-id="7128f-103">The type 'type name' must be a reference type in order to use it as parameter 'parameter name' in the generic type or method 'identifier of generic'</span></span>  
  
 <span data-ttu-id="7128f-104">Bir değer türü gibi başarılı olduğunda bu hata oluşur. bir `struct` veya `int` genel tür veya bir başvuru türü kısıtlaması olan bir yönteme parametre olarak.</span><span class="sxs-lookup"><span data-stu-id="7128f-104">This error occurs when you pass a value type such as a `struct` or `int` as a parameter to a generic type or method that has a reference type constraint.</span></span>
