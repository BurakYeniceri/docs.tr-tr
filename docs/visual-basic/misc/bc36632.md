# <a name="nullable-parameters-must-specify-a-type"></a>Null atanabilir parametrelere bir türü belirtmeniz gerekir
Bir parametre null olabilir bir tür değiştiricisi (?) içerir, ancak bir türü belirtmiyor.  
  
 **Hata Kimliği:** BC36632  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Boş değer atanabilir parametre bildirirken bir tür belirtin.
