# <a name="39ltfilenamegt39-cannot-be-referenced-because-it-is-not-an-assembly"></a>&#39; &lt;filename&gt;&#39; derleme olmadığından başvurulamaz
Bir .NET derlemesi olmayan bir dosya için bir başvuru yapıldı.  
  
 **Hata Kimliği:** BC31519  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Derlemenin konumunu ve adını doğrulayın.  
  
2.  Derleme zarar görmüş veya bozuk olmadığını doğrulayın.
