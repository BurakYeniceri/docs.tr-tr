# <a name="compiler-error-cs0719"></a><span data-ttu-id="ff9a2-102">Derleyici Hatası CS0719</span><span class="sxs-lookup"><span data-stu-id="ff9a2-102">Compiler Error CS0719</span></span>
<span data-ttu-id="ff9a2-103">'type': dizi öğeleri statik türü olamaz</span><span class="sxs-lookup"><span data-stu-id="ff9a2-103">'type': array elements cannot be of static type</span></span>  
  
 <span data-ttu-id="ff9a2-104">Statik türünde bir dizi dizi öğeleri örneklerdir, ancak statik türler örneklerini oluşturmak mümkün değil beri mantıklı değildir.</span><span class="sxs-lookup"><span data-stu-id="ff9a2-104">An array of static type does not make sense since array elements are instances, but it is not possible to create instances of static types.</span></span>  
  
 <span data-ttu-id="ff9a2-105">Aşağıdaki örnek CS0719 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ff9a2-105">The following sample generates CS0719:</span></span>  
  
```csharp  
// CS0719.cs  
public static class SC  
{  
   public static void F()  
   {  
   }  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC[] sca = new SC[10];  // CS0719  
   }  
}  
```
