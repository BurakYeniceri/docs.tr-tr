 <span data-ttu-id="a0617-107">Aşağıdaki örnek CS0017 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a0617-107">The following sample generates CS0017:</span></span>

```csharp  
{  
   static public void Main()

   }

{  
   public static void Main()   // CS0017, delete one Main or use /main

   }

```
