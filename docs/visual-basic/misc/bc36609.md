# <a name="query-expressions-are-not-valid-in-debug-windows"></a>Sorgu ifadeleri hata ayıklama pencerelerinde geçerli değildir
LINQ sorgusu girilen **izleme** veya **hemen** penceresi bir hata ayıklama oturumu sırasında. Sorguları hata ayıklama pencerelerinde kullanılabilir değil.  
  
 **Hata Kimliği:** BC36609
