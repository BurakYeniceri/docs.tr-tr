# <a name="39ltelementnamegt39-is-not-a-method-parameter"></a>&#39; &lt;elementname&gt;&#39; bir yöntem parametre değil
Bir yordam çağrısı yöntemi tarafından tanımlanmamış bir ek bağımsız değişken sağlar.  
  
 **Hata Kimliği:** BC36568  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gereksiz bağımsız değişken bağımsız değişken listesinden kaldırır.
