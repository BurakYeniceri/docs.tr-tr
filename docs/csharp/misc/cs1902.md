# <a name="compiler-error-cs1902"></a>Derleyici Hatası CS1902
Geçersiz'option 'seçeneği için/Debug; tam veya pdbonly olmalıdır  
  
 Geçersiz bir seçenek geçirilmedi [/debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) derleyici seçeneği.  
  
 Aşağıdaki örnek CS1902 oluşturur:  
  
```csharp  
// CS1902.cs  
// compile with: /debug:x  
// CS1902 expected  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
