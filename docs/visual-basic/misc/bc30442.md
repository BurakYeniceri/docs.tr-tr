# <a name="39finally39-must-end-with-a-matching-39end-try39"></a>&#39;Son olarak&#39; eşleşen ile bitmelidir &#39;son deneyin&#39;
A `Finally` deyimi, eşleşen bir olmadan kodunuzda görünüyor `End Try` deyimi. `Finally` deyimleri sonuçları, ile bir `End Try` deyimi.  
  
 **Hata Kimliği:** BC30442  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kaldırma `Finally` deyimi.  
  
2.  Ekleme bir `End Try` blok sonuçlandırmak için deyimi.
