# <a name="xml-comment-must-be-the-first-statement-on-a-line"></a>XML açıklama satırdaki ilk deyim olmalıdır
XML açıklama satırdaki ilk deyim olmalıdır. XML açıklama yoksayılacak.  
  
 Bir XML açıklama verilmeyen başka bir dil öğesi sonra yerleştirilir.  
  
 **Hata Kimliği:** BC42302  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML açıklama yeni bir satıra taşıyın.
