# <a name="xml-namespace-prefix-39prefix39-is-not-defined"></a>XML ad alanı öneki &#39;önek&#39; tanımlı değil
Tanımlanmamış bir XML ad alanı öneki, sabit bir XML veya bir XML axis özelliği belirtilmedi.  
  
 **Hata Kimliği:** BC31148  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML ad alanı önekini kullanarak tanımlayan `Imports` deyimi.  
  
 -veya-  
  
-   XML değişmez değer için XML ad alanı öneki veya önceden tanımlanmış bir XML ad alanı öneki için XML axis özelliği değiştirin.
