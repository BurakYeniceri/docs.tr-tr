 <span data-ttu-id="46375-107">Bu hatayı önlemek için tanımlayıcı bildirimini bulun, doğru Yazımı denetleyin ve bildirimi ayrı bir derlemede ise, uygun derleme başvurusu olduğundan emin olun.</span><span class="sxs-lookup"><span data-stu-id="46375-107">To avoid this error, locate the declaration of the identifier, verify the correct spelling, and if the declaration is in a separate assembly, make sure that you have the appropriate assembly reference.</span></span> <span data-ttu-id="46375-108">Tanımlayıcısını başka bir tür veya ad alanı içinde bildirilen sonra tam adını kullanın::.</span><span class="sxs-lookup"><span data-stu-id="46375-108">If the identifier is declared inside another type or namespace, use the fully-qualified name after the ::.</span></span> <span data-ttu-id="46375-109">Aşağıdaki örnek CS0400 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="46375-109">The following sample generates CS0400:</span></span>

    public static void Main()  
    {  
        // CS0400 - D could not be found

   }
