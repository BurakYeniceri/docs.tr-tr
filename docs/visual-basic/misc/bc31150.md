# <a name="end-tag-lttagnamegt-expected"></a>Bitiş etiketi &lt;/tagName&gt; bekleniyor
Bir XML öğesi değişmez değeri bitiş etiketinde karşılık gelen başlangıç etiketiyle eşleşmiyor.  
  
 **Hata Kimliği:** BC31150  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Son XML etiketi XML öğesi değişmez değeri başlangıç XML etiketi eşleşecek şekilde değiştirin.
