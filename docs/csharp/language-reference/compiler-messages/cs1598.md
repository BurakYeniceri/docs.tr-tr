# <a name="compiler-warning-level-1-cs1598"></a>Derleyici Uyarısı (düzey 1) CS1598
XML Ayrıştırıcı yüklenemedi şu nedenle: 'gerekçe'. XML belgesi dosyası 'dosya' üretilmez.  
  
 [/Doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) seçeneği belirtildi, ancak derleyici sağlanamadı bulmak ve msxml3.dll yükleyin. Msxml3.dll dosyasının yüklü ve kayıtlı olduğundan emin olun.
