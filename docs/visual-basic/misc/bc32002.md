# <a name="39ltkeywordgt39-is-valid-only-within-a-class"></a>&#39; &lt;anahtar sözcüğü&gt;&#39; yalnızca bir sınıf içinde geçerlidir
Bir anahtar sınıflarına, gibi ilgili `Me` veya `MyClass`, bir sınıf tanımı dışında kullanılır.  
  
 **Hata Kimliği:** BC32002  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Anahtar sözcüğünü kullanarak kod sınıf örneklerini içeriyorsa, bu, bir sınıf uygulama başına taşıyın.  
  
-   Anahtar sözcüğünü kullanarak kod sınıfları için geçerli değilse geçersiz anahtar sözcüğü kaldırın.
