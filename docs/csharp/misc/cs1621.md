# <a name="compiler-error-cs1621"></a>Derleyici Hatası CS1621
Verim deyimi adsız yöntem veya lambda ifadesi içinde kullanılamaz  
  
 Verim deyimi yineleyici anonim yöntemi bloğunda olamaz.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS1621 oluşturur:  
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
