# <a name="compiler-error-cs0001"></a>Derleyici Hatası CS0001
Derleyici iç hatası  
  
 Derleyici beklenmeyen sözdizimi ayrıştırmak için kendi sorunları nedeniyle başarısız olup olmadığını saptamak deneyin. Bu hatayı sürekli olarak alıyorsanız, lütfen Microsoft'a başvurun.
