# <a name="compiler-error-cs0215"></a><span data-ttu-id="8fd90-102">Derleyici Hatası CS0215</span><span class="sxs-lookup"><span data-stu-id="8fd90-102">Compiler Error CS0215</span></span>
<span data-ttu-id="8fd90-103">True veya False işlecinin dönüş türü Boole olmalıdır</span><span class="sxs-lookup"><span data-stu-id="8fd90-103">The return type of operator True or False must be bool</span></span>  
  
 <span data-ttu-id="8fd90-104">Kullanıcı tanımlı [true](../../csharp/language-reference/keywords/true.md) ve [false](../../csharp/language-reference/keywords/false.md) işleçleri dönüş türüne sahip olmalıdır [bool](../../csharp/language-reference/keywords/bool.md).</span><span class="sxs-lookup"><span data-stu-id="8fd90-104">User-defined [true](../../csharp/language-reference/keywords/true.md) and [false](../../csharp/language-reference/keywords/false.md) operators must have a return type of [bool](../../csharp/language-reference/keywords/bool.md).</span></span> <span data-ttu-id="8fd90-105">Daha fazla bilgi için bkz: [fazla yüklenebilir işleçler](../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md).</span><span class="sxs-lookup"><span data-stu-id="8fd90-105">For more information, see [Overloadable Operators](../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md).</span></span>  
  
 <span data-ttu-id="8fd90-106">Aşağıdaki örnek CS0215 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8fd90-106">The following sample generates CS0215:</span></span>  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
