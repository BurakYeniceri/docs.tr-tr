# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="7831b-102">Derleyici Uyarısı (düzey 1) CS0183</span><span class="sxs-lookup"><span data-stu-id="7831b-102">Compiler Warning (level 1) CS0183</span></span>
<span data-ttu-id="7831b-103">Verilen ifade her zaman sağlanan ('type') türü değil</span><span class="sxs-lookup"><span data-stu-id="7831b-103">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="7831b-104">Her zaman bir koşullu ifade değerlendirilirse **doğru**, sonra da bir koşullu ifade gerekmez.</span><span class="sxs-lookup"><span data-stu-id="7831b-104">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="7831b-105">Bu uyarı türünü kullanarak değerlendirmek çalıştığınızda oluşur **olan** işleci.</span><span class="sxs-lookup"><span data-stu-id="7831b-105">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="7831b-106">Değerlendirme bir değer türü ise, onay gerekli değildir.</span><span class="sxs-lookup"><span data-stu-id="7831b-106">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="7831b-107">Aşağıdaki örnek CS0183 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7831b-107">The following sample generates CS0183:</span></span>  
  
```csharp  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);   
   }  
}  
```
