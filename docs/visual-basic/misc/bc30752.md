# <a name="39on-error39-statements-are-not-valid-within-39synclock39-statements"></a>&#39; hata &#39; deyimleri içinde geçersiz &#39; SyncLock &#39; deyimleri
`On Error`deyimleri olamaz kullanılan `SyncLock` iş parçacığı eşitleme kesintiye çünkü engeller.  
  
 **Hata Kimliği:** BC30752  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Yer `On Error` deyimleri dışında `SyncLock` engeller.
