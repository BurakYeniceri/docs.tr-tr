# <a name="39inherits39-statements-must-precede-all-declarations-in-an-interface"></a>&#39;Devralınan&#39; deyimleri arabirimdeki tüm bildirimler gelmelidir
Bir `Inherits` deyimi bir sınıf tanımı ilk boş olmayan ve yorum olmayan satır olması gerekir.  
  
 **Hata Kimliği:** BC30357  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Taşıma, `Inherits` olan tüm bildirimleri önündeki şekilde deyimi.
