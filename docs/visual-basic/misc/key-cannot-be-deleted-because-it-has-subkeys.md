# <a name="key-cannot-be-deleted-because-it-has-subkeys"></a>Anahtarı alt anahtarları içerdiğinden silinemiyor
Alt anahtarları olduğundan anahtar silinemiyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bu işlemi denemeden önce belirtilen anahtarın alt anahtarlarını silin ve belirler.
