# <a name="unused-local-constant-39ltconstantgt39"></a>Kullanılmayan yerel sabit: &#39; &lt;sabit&gt;&#39;
Yerel bir sabit herhangi bir yerde kodunuzda kullanılmıyor bildirilmiş.  
  
 **Hata Kimliği:** BC42099  
  
### <a name="to-remove-this-warning"></a>Bu uyarıyı kaldırmak için  
  
1.  Bu hata, bir uyarı ve göz ardı edilebilir. Uyarıyı kaldırmak için kodunuzdan kullanılmayan sabit bildirimini kaldırın.
