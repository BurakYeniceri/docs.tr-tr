# <a name="wcf-directive-syntax"></a>WCF Yönerge Söz Dizimi
Derleyicileri yönlendirmek için Windows Communication Foundation (WCF) yönergeleri .svc dosyalarında tarafından kullanılan ayarları belirtir. Her yönergesi o yönergeye özgü (değerleri ile eşleştirilmiş) bir veya daha fazla öznitelik içerebilir. [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] yalnızca [ @ServiceHost ](../../../../../docs/framework/configure-apps/file-schema/wcf-directive/servicehost.md) yönergesi.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [@ServiceHost](../../../../../docs/framework/configure-apps/file-schema/wcf-directive/servicehost.md)  
 .Svc derleyici tarafından kullanılan sayfaya özel öznitelikleri tanımlar. Yalnızca .svc dosyalarında dahil edilebilir.
