# <a name="compiler-error-cs0743"></a><span data-ttu-id="8e7fa-102">Derleyici Hatası CS0743</span><span class="sxs-lookup"><span data-stu-id="8e7fa-102">Compiler Error CS0743</span></span>
<span data-ttu-id="8e7fa-103">Bağlamsal anahtar sözcüğü 'on' bekleniyor</span><span class="sxs-lookup"><span data-stu-id="8e7fa-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="8e7fa-104">Düzeni için bir `join` yan tümcesi `join`... `in`... `on`... `equals`, bu örnekte gösterildiği gibi:</span><span class="sxs-lookup"><span data-stu-id="8e7fa-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```

## <a name="example"></a><span data-ttu-id="8e7fa-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="8e7fa-107">Example</span></span>  
 <span data-ttu-id="8e7fa-108">Aşağıdaki kod CS0743 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8e7fa-108">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
