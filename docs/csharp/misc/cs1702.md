# <a name="compiler-warning-level-3-cs1702"></a>Derleyici Uyarısı (Düzey 3) CS1702
Derleme başvurusu "Derleme adı #1", "Derleme adı #2" ile eşleştiğini varsayarsak, çalışma zamanı İlkesi sağlamanız gerekebilir  
  
 İki derleme başvurularını farklı yapı ve/veya değil otomatik olarak birleştirecektir şekilde düzeltme numaralarına sahip. Uygulama .config dosyasına yönergeleri kullanarak Birleştirici zorlamak için çalışma zamanı İlkesi sağlamanız gerekebilir.
