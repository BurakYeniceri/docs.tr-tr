# <a name="compiler-error-cs1009"></a><span data-ttu-id="24e3e-102">Derleyici Hatası CS1009</span><span class="sxs-lookup"><span data-stu-id="24e3e-102">Compiler Error CS1009</span></span>
<span data-ttu-id="24e3e-103">Tanınmayan kaçış sırası</span><span class="sxs-lookup"><span data-stu-id="24e3e-103">Unrecognized escape sequence</span></span>  
  
 <span data-ttu-id="24e3e-104">Beklenmeyen bir karakter bir ters eğik çizgiyi takip eden (\\) içinde bir [dize](../../../csharp/language-reference/keywords/string.md).</span><span class="sxs-lookup"><span data-stu-id="24e3e-104">An unexpected character follows a backslash (\\) in a [string](../../../csharp/language-reference/keywords/string.md).</span></span> <span data-ttu-id="24e3e-105">Derleyici, geçerli kaçış karakterlerinden biri bekliyor.</span><span class="sxs-lookup"><span data-stu-id="24e3e-105">The compiler expects one of the valid escape characters.</span></span> <span data-ttu-id="24e3e-106">Daha fazla bilgi için [karakter çıkışları](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="24e3e-106">For more information, see [Character Escapes](../../../standard/base-types/character-escapes-in-regular-expressions.md).</span></span>  
  
 <span data-ttu-id="24e3e-107">Aşağıdaki örnek CS1009 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="24e3e-107">The following sample generates CS1009.</span></span>  
  
```csharp  
// CS1009-a.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string a = "\m";     
      // Try the following line instead.  
      // string a = "\t";  
   }  
}  
```  
  
 <span data-ttu-id="24e3e-108">Bu hatanın yaygın bir nedeni, aşağıdaki örnekte gösterildiği gibi bir dosya adı ters eğik çizgi karakteri kullanıyor.</span><span class="sxs-lookup"><span data-stu-id="24e3e-108">A common cause of this error is using the backslash character in a file name, as the following example shows.</span></span>  
  
```csharp  
string filename = "c:\myFolder\myFile.txt";  
```  
  
 <span data-ttu-id="24e3e-109">Bu hatayı gidermek için "\\\\" veya @-quoted aşağıdaki örnekte gösterildiği gibi değişmez dize değeri.</span><span class="sxs-lookup"><span data-stu-id="24e3e-109">To resolve this error, use "\\\\" or the @-quoted string literal, as the following example shows.</span></span>  
  
```csharp  
// CS1009-b.cs  
class MyClass  
{  
   static void Main()  
   {  
      // The following line causes CS1009.  
      string filename = "c:\myFolder\myFile.txt";     
      // Try one of the following lines instead.  
      // string filename = "c:\\myFolder\\myFile.txt";  
      // string filename = @"c:\myFolder\myFile.txt";  
   }  
}  
```
