# <a name="compiler-warning-level-2-cs1571"></a>Derleyici Uyarısı (Düzey 2) CS1571
XML açıklama 'yapı' üzerinde 'parametresi' için yinelenen parametre etiketi var  
  
 Kullanırken [/doc](../../csharp/language-reference/compiler-options/doc-compiler-option.md) derleyici seçeneği, birden fazla açıklama için aynı yöntemi parametresi bulundu. Yinelenen satırları kaldırın.  
  
 Aşağıdaki örnek CS1571 oluşturur:  
  
```csharp  
// CS1571.cs  
// compile with: /W:2 /doc:x.xml  
  
/// <summary>help text</summary>  
public class MyClass  
{  
   /// <param name='Int1'>Used to indicate status.</param>  
   /// <param name='Char1'>An initial.</param>  
   /// <param name='Int1'>Used to indicate status.</param> // CS1571  
   public static void MyMethod(int Int1, char Char1)  
   {  
   }  
  
   /// <summary>help text</summary>  
   public static void Main ()  
   {  
   }  
}  
```
