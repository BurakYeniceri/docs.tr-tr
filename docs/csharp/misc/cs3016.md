# <a name="compiler-warning-level-1-cs3016"></a>Derleyici Uyarısı (düzey 1) CS3016
Öznitelik bağımsız değişkenleri olarak diziler CLS uyumlu değil  
  
 İle ortak dil belirtimi (bir öznitelik için bir dizi geçmesine CLS) uyumlu değil. CLS uyumluluğu hakkında daha fazla bilgi için bkz: [CLS uyumlu kod yazma](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) ve [dil bağımsızlığı ve dilden bağımsız bileşenler](https://msdn.microsoft.com/library/12a7a7h3).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS3016 oluşturur:  
  
```  
// CS3016.cs  
  
using System;  
  
[assembly : CLSCompliant(true)]  
[C(new int[] {1, 2})]   // CS3016  
// try the following line instead  
// [C()]  
class C : Attribute  
{  
    public C()  
    {  
    }  
  
    public C(int[] a)  
    {  
    }  
  
    public static void Main ()  
    {  
    }  
}  
```
