# <a name="type-parameter-with-a-39structure39-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="474a2-102">Type parametresi ile bir &#39; Yapı &#39; kısıtlama kısıtlama olarak kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="474a2-102">Type parameter with a &#39;Structure&#39; constraint cannot be used as a constraint</span></span>
<span data-ttu-id="474a2-103">Tür parametresi ile bir `Structure` kısıtlaması kısıtlama olarak başka bir tür parametresi için kullanılır.</span><span class="sxs-lookup"><span data-stu-id="474a2-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="474a2-104">`Structure` Kısıtlaması kendi türü parametresine geçirilen tür bağımsız değişkeni bir değer türü olmasını gerektirir.</span><span class="sxs-lookup"><span data-stu-id="474a2-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="474a2-105">Ancak, bir değer türü uygulanan veya değiştirilemez devralınan uyguladıktan veya ondan devralan diğer tür parametresi gerektiren bir kısıtlama olarak kullanmak için anlamlı değildir.</span><span class="sxs-lookup"><span data-stu-id="474a2-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="474a2-106">Bu durum yalnızca anlamlı yorumu, her iki tür bağımsız değişkenleri tam aynı türde olmalıdır ' dir.</span><span class="sxs-lookup"><span data-stu-id="474a2-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="474a2-107">Bu durumda, genel türde yalnızca bir tür parametresi gerekir.</span><span class="sxs-lookup"><span data-stu-id="474a2-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="474a2-108">Aşağıdaki deyim bu hata oluşmasına neden olabilir.</span><span class="sxs-lookup"><span data-stu-id="474a2-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="474a2-109">Türü geçirilen `t2` nasıl uygulanacağını veya için geçirilen türü devral `t1`türü geçirilen çünkü `t1` bir değer türü olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="474a2-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="474a2-110">**Hata Kimliği:** BC32114</span><span class="sxs-lookup"><span data-stu-id="474a2-110">**Error ID:** BC32114</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="474a2-111">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="474a2-111">To correct this error</span></span>  
  
-   <span data-ttu-id="474a2-112">Kısıtlı için tür parametresi kaldırmak `Structure` diğer tür parametresi üzerinde sınırlama listesinden.</span><span class="sxs-lookup"><span data-stu-id="474a2-112">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>  
  
-   <span data-ttu-id="474a2-113">Tür parametrelerinin her ikisini de aynı değer türü gerektiriyorsa, yalnızca bir tür parametresi ile genel tür tanımlayın.</span><span class="sxs-lookup"><span data-stu-id="474a2-113">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>
