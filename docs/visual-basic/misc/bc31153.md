# <a name="required-attribute-39version39-missing-from-xml-declaration"></a>Gerekli öznitelik &#39; sürüm &#39; XML bildiriminden eksik
`version` Bir XML belgesi değişmez değeri eksik öznitelik.  
  
 **Hata Kimliği:** BC31153  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme `version` özniteliği aşağıdaki örnekte olduğu gibi XML belge bildirimi için:`<?xml version="1.0"?>`
