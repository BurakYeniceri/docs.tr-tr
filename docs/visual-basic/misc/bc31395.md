# <a name="type-characters-are-not-allowed-in-label-identifiers"></a>Tür karakterleri etiket tanımlayıcılarının izin verilmiyor
Bir deyim etiket en az bir tanımlayıcı türü karakteri içerir.  
  
 Bir bildirim etiketi geçerli bir Visual Basic adı olmalıdır. İzin verilen karakter tanımlayıcı türü karakterleri içermez (`%`, `&`, `@`, `!`, `#`, ve `$`). Bkz: [bildirilen öğe adları](../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **Hata Kimliği:** BC31395  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tür tanımlayıcısı karakteri veya karakterleri deyimi etiketinden kaldırın.
