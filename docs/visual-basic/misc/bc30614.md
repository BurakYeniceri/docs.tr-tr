# <a name="39mustoverride39-method-39ltmethodnamegt39-cannot-be-called-with-39myclass39"></a>&#39;MustOverride&#39; yöntemi &#39; &lt;methodname&gt; &#39; ile çağrılamaz &#39;MyClass&#39;
`MyClass` eşdeğer olan `Me`, ancak çağrılmakta olan yöntemin değilmiş gibi üzerindeki tüm yöntemi çağrılarını kabul edilir `NotOverridable`.  
  
 **Hata Kimliği:** BC30614  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `MustOverride` değiştiricisi, veya bir taban sınıf yönteminde bildirme ve devralır ve o sınıfın geçersiz kılar.
