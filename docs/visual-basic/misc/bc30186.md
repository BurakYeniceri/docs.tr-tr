# <a name="39using39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Kullanarak&#39; ifadeleri geçerli değildir komut penceresi
`Using` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30186  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Using` from deyimi **hemen** penceresi.
