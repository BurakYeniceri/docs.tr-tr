# <a name="39end39-statement-not-valid"></a>&#39; Bitiş &#39; deyimi geçerli değil
`End` Anahtar sözcüğü geçerli bağlamda uygun değil veya yanlış kullanılır.  
  
 **Hata Kimliği:** BC30678  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kullanım `End` tek başına bir yordam kapsamı kaybetmenize neden olacak.  
  
2.  Kullanım `End` yapısı veya özelliği gibi kod bloğunu sonlandırmak için diğer anahtar sözcükler.
