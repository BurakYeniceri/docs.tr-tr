# <a name="type-39lttypenamegt39-is-not-supported-because-it-either-directly-or-indirectly-inherits-from-itself"></a>Tür &#39; &lt;typename&gt; &#39; kendisinden doğrudan veya dolaylı olarak devraldığı için desteklenmiyor
Sınıfta veya arabirimde ya da kendi içinden veya başka bir sınıf ya da sonuçta devralan arabirimi devralır.  
  
 Visual Basic döngüsel devralma desteklemez.  
  
 **Hata Kimliği:** BC30916  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Devralma yapısı, bir taban sınıf veya herhangi diğer sınıf veya arabirim devralmıyor arabirimi üzerine inşa edilmiştir şekilde değiştirin.
