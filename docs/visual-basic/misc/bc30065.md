# <a name="39exit-sub39-is-not-valid-in-a-function-or-property"></a>&#39; Çıkış alt &#39; bir işlev veya özelliği geçerli değil
`Exit Sub`görünür bir `Function` yordamı veya `Property` yordamı. Bir `Exit` deyimi blok eşleşmelidir içinde hangi oluşur.  
  
 **Hata Kimliği:** BC30065  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değiştir `Exit Sub` ile `Exit Function` veya `Exit Property` uygun şekilde deyimi.
