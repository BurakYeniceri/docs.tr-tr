# <a name="interoperability-with-web-services-enhancements-30"></a>Web Services Enhancements 3.0 ile Birlikte Çalışabilme
Windows Communication Foundation (WCF) istemcileri ve Hizmetleri Ağustos 2004 sürümü kullanmak için WCF istemcileri ve Hizmetleri yapılandırıldığında Microsoft .NET (WSE) istemcileri ve Hizmetleri için Web Hizmetleri geliştirmeleri 3.0 ile hat düzeyinde uyumlu olan WS-adresleme belirtimi.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [Nasıl yapılır: WCF Hizmetlerini WSE 3.0 İstemcileriyle Birlikte Çalışmak için Yapılandırma](../../../../docs/framework/wcf/feature-details/how-to-configure-wcf-services-to-interoperate-with-wse-3-0-clients.md)  
 Microsoft .NET (WSE) istemci için Web Hizmetleri geliştirmeleri 3.0 ile iletişim kurabilmesi için WCF hizmetinin nasıl yapılandırılacağı açıklanmaktadır.  
  
 [Nasıl yapılır: WCF İstemcisini WSE3.0 Hizmetleriyle Çalışacak Şekilde Yapılandırma](../../../../docs/framework/wcf/feature-details/how-to-configure-a-wcf-client-to-interoperate-with-wse3-0-services.md)  
 Microsoft .NET (WSE) hizmeti için Web Hizmetleri geliştirmeleri 3.0 ile iletişim kurar bir WCF istemcisi yapılandırma konusunda ayrıntılı olarak açıklanmaktadır.
