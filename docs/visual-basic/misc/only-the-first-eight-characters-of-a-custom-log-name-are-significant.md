# <a name="only-the-first-eight-characters-of-a-custom-log-name-are-significant"></a>Özel günlük adının yalnızca ilk sekiz karakteri önemli
Olay günlüğü adları benzersizliğini denetlerken, yalnızca ilk sekiz karakter olarak kabul edilir. Bir çakışma kendi ilk sekiz karakter paylaşan olay günlüklerini neden olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Olay günlüğü ilk sekiz karakter benzersiz bir ad verin.
