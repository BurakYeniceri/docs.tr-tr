# <a name="automation-error"></a>Otomasyon hatası
Yöntem yürütme veya alma veya bir nesne değişkeninin bir özellik ayarlama bir hata oluştu. Hata nesnesi oluşturdu uygulama tarafından bildirildi.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Özelliklerini denetleyin `Err` kaynak hem de hatanın yapısını belirlemek için nesne.  
  
2.  Kullanım `On Error Resume Next` deyimi erişilirken deyimi hemen önce ve sonra hemen erişilirken deyimi hata olup olmadığını denetleyin.
