# <a name="compiler-error-cs0313"></a><span data-ttu-id="f23ba-102">Derleyici Hatası CS0313</span><span class="sxs-lookup"><span data-stu-id="f23ba-102">Compiler Error CS0313</span></span>
<span data-ttu-id="f23ba-103">' % S'türü 'type1' tür parametresi 'parametre adı' genel türünde veya yönteminde 'type2' olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="f23ba-103">The type 'type1' cannot be used as type parameter 'parameter name' in the generic type or method 'type2'.</span></span> <span data-ttu-id="f23ba-104">'Type1' boş değer atanabilir tür 'type2' kısıtlamasını karşılamıyor.</span><span class="sxs-lookup"><span data-stu-id="f23ba-104">The nullable type 'type1' does not satisfy the constraint of 'type2'.</span></span> <span data-ttu-id="f23ba-105">Boş değer atanabilir türler hiçbir arabirim kısıtlamasını gerçekleştiremiyor.</span><span class="sxs-lookup"><span data-stu-id="f23ba-105">Nullable types cannot satisfy any interface constraints.</span></span>  
  
 <span data-ttu-id="f23ba-106">Boş değer atanabilir bir tür için alamayan kendisine karşılık gelen eşdeğer değildir.</span><span class="sxs-lookup"><span data-stu-id="f23ba-106">A nullable type is not equivalent to its non-nullable counterpart.</span></span> <span data-ttu-id="f23ba-107">Aşağıdaki örnekte `ImplStruct` karşılayan `BaseInterface` kısıtlaması ancak `ImplStruct?` değil çünkü `Nullable<ImplStruct>` uygulamıyor `BaseInterface`.</span><span class="sxs-lookup"><span data-stu-id="f23ba-107">In the example that follows, `ImplStruct` satisfies the `BaseInterface` constraint but `ImplStruct?` does not because `Nullable<ImplStruct>` does not implement `BaseInterface`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f23ba-108">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="f23ba-108">To correct this error</span></span>  
  
1.  <span data-ttu-id="f23ba-109">Aşağıdaki kod, örnek olarak kullanıp, sıradan belirtmek için tek bir çözüm olan `ImplStruct` çağrısında ilk tür bağımsız değişkeni olarak `TestMethod`.</span><span class="sxs-lookup"><span data-stu-id="f23ba-109">Using the code that follows as an example, one solution is to specify an ordinary `ImplStruct` as the first type argument in the call to `TestMethod`.</span></span> <span data-ttu-id="f23ba-110">Ardından değiştirin `TestMethod` boş değer atanabilir bir sürümünü oluşturmak için `Implstruct` kendi dönüş deyimi içinde:</span><span class="sxs-lookup"><span data-stu-id="f23ba-110">Then modify `TestMethod` to create a nullable version of `Implstruct` in its return statement:</span></span>  
  
    ```csharp  
    return new Nullable<T>(t);  
    ```  
  
## <a name="example"></a><span data-ttu-id="f23ba-111">Örnek</span><span class="sxs-lookup"><span data-stu-id="f23ba-111">Example</span></span>  
 <span data-ttu-id="f23ba-112">Aşağıdaki kod CS0313 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f23ba-112">The following code generates CS0313:</span></span>  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
