# <a name="endpoint-call-duration"></a>Uç Noktası: Çağrı Süresi
Bu uç noktaya yapılan çağrıların ortalama süresi.  Ortalama süresi temel üzerinde bu denklemi hesaplanır: (N1-N0)/(D1-D0).
