# <a name="instance-members-and-39me39-cannot-be-used-within-a-lambda-expression-in-structures"></a>Örnek üyelerinin ve &#39; &#39; lambda ifadesi yapılarında içinde kullanılamaz
Gelen bir yapı içinde yapısı örneği üyesine başvuruyor veya kullanan bir lambda ifadesi tanımladığınız `Me`. Aşağıdaki kod bu iki geçersiz başvuruları gösterir.  
  
```vb  
Structure Structure1  
  
    Public InstanceMember As Integer  
  
    Public Function ExampleFun() As Integer  
        '' The error is caused by use of InstanceMember.  
        'Dim fun1 = Function() InstanceMember  
        '' The error is caused by use of Me.  
        'Dim fun2 = Function() Me.InstanceMember  
        'Return fun1()  
    End Function  
  
End Structure  
```  
  
 **Hata Kimliği:** BC36638  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Örnek üyesine yerel bir değişkene atayın ve yerel değişken deyiminizde kullanın.  
  
```vb  
Public Function ExampleFunFix() As Integer  
    Dim temp = InstanceMember  
    Dim fun1 = Function() temp  
    Return fun1()  
End Function  
```
