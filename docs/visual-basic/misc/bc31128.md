# <a name="39removehandler39-is-already-declared"></a>&#39; RemoveHandler &#39; zaten bildirildi
Birden fazla `RemoveHandler` bildirimi özel olay bildiriminde görüldü. Bir `RemoveHandler` bildirimi olay işleyicisi kaldırmak için kullanılan yordamı bildirir.  
  
 **Hata Kimliği:** BC31128  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yedek kaldırmak `RemoveHandler` deyimi.
