# <a name="compiler-error-cs1577"></a>Derleyici Hatası CS1577
Derleme oluşturma başarısız oldu: nedeni  
  
 C# Derleyici başarıyla koda derlenmiş sonra derleme derleme oluşturma parçası başarısız oldu. ALink yardımcı programı hataları belgelerine bakın ([Al.exe aracı hataları ve Uyarıları](http://msdn.microsoft.com/en-us/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)) daha fazla bilgi için.
