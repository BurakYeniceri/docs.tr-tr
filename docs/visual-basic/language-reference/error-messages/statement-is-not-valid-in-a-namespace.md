# <a name="statement-is-not-valid-in-a-namespace"></a>Deyim ad alanında geçerli değil
Bir ad alanı düzeyinde deyimi yer alamaz. Ad alanı düzeyinde izin yalnızca bildirimleri modülü, arabirim, sınıf, temsilci, numaralandırma ve yapı bildirimleri ' dir.  
  
 **Hata Kimliği:** BC30001  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Deyim bir modül, sınıf, arabirimi, yapısı, numaralandırma veya temsilci tanımı içinde bir konuma taşıyın.
