# <a name="39ltmethod1gt39-cannot-override-39ltmethod2gt39-because-it-is-a-39declare39-statement"></a>&#39; &lt;method1&gt;&#39; olamaz geçersiz kılma &#39;&lt; method2&gt;&#39; çünkü bir &#39; Declare &#39; deyimi
Temsilci ile bildirildi ve temel sınıfın adını üzerinde geçersiz kılma girişiminde bir `Declare` deyimi.  
  
 **Hata Kimliği:** BC30474  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Geçersiz kılınan üye olmayan biçimde değiştirme bir `Declare` deyimi.  
  
2.  Bu yöntemi geçersiz kılın çalışmayın.
