# <a name="interface-39ltinterfacenamegt39-is-not-implemented-by-this-class"></a>Arabirim &#39; &lt;InterfaceName&gt; &#39; Bu sınıf tarafından uygulanmaz.
Sınıf veya yapı uygulamayan bir arabirim üye uygulamak Bu sınıf veya yapı üyesi çalışır.  
  
 **Hata Kimliği:** BC31035  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme bir `Implements` deyiminin başında sınıfı veya onun uygun arabirimini uygulayan şekilde yapısı.  
  
-   Kaldırma `Implements` anahtar sözcüğü üyeden bu hataya neden olur.
