# <a name="compiler-warning-level-1-cs0197"></a>Derleyici Uyarısı (düzey 1) CS0197
Başvuruya göre sınıfının bir alan olduğundan 'bağımsız değişken' ref veya out olarak geçirme ya da adresinin alınması çalışma zamanı özel durumuna neden olabilir  
  
 Türetilen, doğrudan veya dolaylı olarak, herhangi bir sınıf gelen <xref:System.MarshalByRefObject> bir başvuruya göre sınıftır. Böyle bir sınıfın işlem ve makine sınırları içinde başvuruya göre sıralanabilir. Bu nedenle, bu sınıfın örnekleri, uzak nesnelere proxy'leri olabilir. Proxy nesnesi olarak alanı geçiremezsiniz [ref](../../csharp/language-reference/keywords/ref.md) veya [çıkışı](../../csharp/language-reference/keywords/out.md). Bu nedenle, bu tür bir sınıf olarak alanlarının geçiremezsiniz `ref` veya `out`örnek sürece [bu](../../csharp/language-reference/keywords/this.md), bir proxy nesnesi olamaz.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0197 oluşturur.  
  
```  
// CS0197.cs  
// compile with: /W:1  
class X : System.MarshalByRefObject  
{  
   public int i;  
}  
  
class M  
{  
   public int i;  
   static void AddSeventeen(ref int i)  
   {  
      i += 17;  
   }  
  
   static void Main()  
   {  
      X x = new X();  
      x.i = 12;  
      AddSeventeen(ref x.i);   // CS0197  
  
      // OK  
      M m = new M();  
      m.i = 12;  
      AddSeventeen(ref m.i);  
   }  
}  
```
