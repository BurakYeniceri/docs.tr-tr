## <a name="remarks"></a><span data-ttu-id="b60ed-178">Açıklamalar</span><span class="sxs-lookup"><span data-stu-id="b60ed-178">Remarks</span></span>  
 <span data-ttu-id="b60ed-179">`<Assembly>` Öğesi, bir derlemede tüm türleri için çalışma zamanı İlkesi tanımlar.</span><span class="sxs-lookup"><span data-stu-id="b60ed-179">The `<Assembly>` element defines runtime policy for all the types in an assembly.</span></span> <span data-ttu-id="b60ed-180">Bu farklı [ \<kitaplığı >](../../../docs/framework/net-native/library-element-net-native.md) , bir kitaplık belirtiyor, ancak çalışma zamanı yansıma ilkesini tanımlamak için ve alt öğeleri bağımlı öğesi.</span><span class="sxs-lookup"><span data-stu-id="b60ed-180">It differs from the [\<Library>](../../../docs/framework/net-native/library-element-net-native.md) element, which specifies a library but depends on its child elements to define runtime reflection policy.</span></span> <span data-ttu-id="b60ed-181">`<Assembly>` Öğesi geçerli bir derleme içindeki tüm türler için bir alt öğesi tarafından kılınmadıkça.</span><span class="sxs-lookup"><span data-stu-id="b60ed-181">The `<Assembly>` element applies to all the types in an assembly unless they are overridden by a child element.</span></span>  
  
 <span data-ttu-id="b60ed-182">Aşağıdaki örnek, nasıl çalışma zamanı İlkesi derlemeleri içindeki tüm türler uygulama paketinizi içinde atayarak uygulayabileceğiniz gösterir `Name` değerini özniteliği "* uygulama\*".</span><span class="sxs-lookup"><span data-stu-id="b60ed-182">The following example shows how you can apply runtime policy to all the types in assemblies within your app package by assigning the `Name` attribute a value of "*Application\*".</span></span> <span data-ttu-id="b60ed-183">`<Assembly>` Öğesi bir alt öğesi olması gerekir [ \<uygulama >](../../../docs/framework/net-native/application-element-net-native.md) öğesi.</span><span class="sxs-lookup"><span data-stu-id="b60ed-183">The `<Assembly>` element must be a child of the [\<Application>](../../../docs/framework/net-native/application-element-net-native.md) element.</span></span>

```xml  
  <Application>   
     <Assembly Name="*Application*" Dynamic="Required All" />   
  </Application>   
</Directives>  
```  
  
 <span data-ttu-id="b60ed-184">`Activate`, `Browse`, `Dynamic`, Ve `Serialize` öznitelikleri tüm isteğe bağlı.</span><span class="sxs-lookup"><span data-stu-id="b60ed-184">The `Activate`, `Browse`, `Dynamic`, and `Serialize` attributes are all optional.</span></span> <span data-ttu-id="b60ed-185">Ancak, `<Assembly>` öğesi bu öznitelikler en az birini içermelidir.</span><span class="sxs-lookup"><span data-stu-id="b60ed-185">However, the `<Assembly>` element must contain at least one of these attributes.</span></span>
