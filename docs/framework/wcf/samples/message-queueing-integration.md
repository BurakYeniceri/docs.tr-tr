# <a name="message-queueing-integration"></a>İleti Kuyruğa Alma Tümleştirme
Bu bölüm, Message Queuing ve Windows Communication Foundation (WCF) arasındaki tümleştirmeyi göstermek örnekleri içerir.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [Message Queuing’den Windows Communication Foundation'a](../../../../docs/framework/wcf/samples/message-queuing-to-wcf.md)  
 Message Queuing (MSMQ) uygulama MSMQ iletisine nasıl gönderebilirsiniz gösteren bir [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] hizmet.  
  
 [Özel Demux](../../../../docs/framework/wcf/samples/custom-demux.md)  
 MSMQ ileti üstbilgilerini farklı hizmet işlemlerini nasıl eşlenebilir gösterir şekilde [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] hizmetleri kullanan <xref:System.ServiceModel.MsmqIntegration.MsmqIntegrationBinding> bir hizmet işlemi için kullanılacak sınırlı değildir.  
  
 [Windows Communication Foundation'dan Message Queuing’e](../../../../docs/framework/wcf/samples/wcf-to-message-queuing.md)  
 Gösteren nasıl bir [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] uygulama bir Message Queuing (MSMQ) uygulamaya bir ileti gönderebilir.  
  
 [İleti Bağıntısı](../../../../docs/framework/wcf/samples/message-correlation.md)  
 Message Queuing (MSMQ) uygulama MSMQ iletisine nasıl gönderebilirsiniz gösteren bir [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] hizmeti ve nasıl iletileri bir istek/yanıt senaryosunda göndereni ve alıcısı uygulamalar arasında ilişkili olabilir.
