# <a name="compiler-error-cs0016"></a>Derleyici Hatası CS0016
Çıkış dosyası 'dosya' dosyasına yazılamadı — 'gerekçe'  
  
 Derleyici çıktı dosyasına yazılamadı. Yolun var olduğundan emin olmak için dosyaya denetleyin. Önceden oluşturulmuş bir dosya zaten konumundaysa, yazılabilir olduğundan ve hiçbir işlem kilit dosya üzerinde şu anda sahip olduğundan emin olun. Örneğin, oluşturmayı denediğiniz sırada yürütülebilir dosyanızın çalışmadığından emin olun.
