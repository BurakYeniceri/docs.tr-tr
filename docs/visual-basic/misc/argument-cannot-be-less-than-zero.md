# <a name="argument-cannot-be-less-than-zero"></a>Bağımsız değişkeni sıfırdan küçük olamaz
Bir değer, sıfır belirtildi, ancak bağımsız değişkeni sıfırdan küçük olamaz daha küçüktür.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Nasıl değer söz konusu, doğru olduğundan emin olmak için hesaplanır denetleyin.
