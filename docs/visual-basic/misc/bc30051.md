# <a name="paramarray-parameter-must-be-a-one-dimensional-array"></a>ParamArray parametresi tek boyutlu bir dizi olmalıdır
A `ParamArray` bağımsız değişkeni bir tek boyutlu dizi türü bildirilmelidir. Bildirim için birden fazla boyuta belirtir `ParamArray` bağımsız değişkeni.  
  
 **Hata Kimliği:** BC30051  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Parantez içindeki tüm virgül kaldırmak `ParamArray` bağımsız değişken adı.
