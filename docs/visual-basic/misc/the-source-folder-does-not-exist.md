# <a name="the-source-folder-does-not-exist"></a>Kaynak klasörü yok.
Var olmayan bir kaynak klasörü başvuruda bulunan bir işlem denendi. Bu, hatalı dosya yolu sağladığını sonucu olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Göreli yollar özellikle kullanıyorsanız, doğru yolu kullandığınızdan emin olun.
