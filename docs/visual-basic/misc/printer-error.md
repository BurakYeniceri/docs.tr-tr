# <a name="printer-error"></a>Yazıcı hatası
Yazıcıda bir hata oluştu, ancak başka hiçbir bilgi dosyası gönderilen bilgisayara döndürüldü.  
  
### <a name="to-correct-the-error"></a>Hatayı düzeltmek için  
  
-   Fiziksel olarak yazıcı inceleyin. Bilgisayar ve yazıcı arasındaki tüm bağlantıların dolu olduğundan emin olun. Çoğu yazıcılar gibi "Çevrimdışı" veya "dışında kağıt." hata bilgileri için bir görüntü sağlar.
