# <a name="can39t-perform-requested-operation-visual-basic"></a>Can &#39; t istenen işlemi (Visual Basic)
Proje geçerli durumunu kılacak olduğundan bir işlem gerçekleştirilemiyor. Örneğin, program aracılığıyla çalışan kod değiştirme denediniz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kod yürütmeyi durdurmak, gerekirse değişiklikleri yapın ve işlemi yeniden deneyin.
