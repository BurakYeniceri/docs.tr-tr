# <a name="3939-requires-its-left-operand-to-have-a-type-parameter-class-or-interface-type-but-this-operand-has-the-type-39lttypegt39"></a>&#39;! &#39; gerektirir, sol işleneni bir tür parametresi, sınıf veya arabirim türüne sahip, ancak bu işlenen türü &#39; var &lt;türü&gt;&#39;
Ünlem işareti kullanan sözlük üye erişimi (`!`) yerine bir süre (`.`), yalnızca bir sınıf veya arabirimin geçerli değil.  
  
 **Hata Kimliği:** BC30103  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tanımlı sınıf veya arabirim türü değerlendiren bir ünlem işareti soldaki ifadesini değiştirin.
