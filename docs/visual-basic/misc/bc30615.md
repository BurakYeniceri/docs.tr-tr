# <a name="39end39-statement-cannot-be-used-in-class-library-projects"></a>&#39;Son&#39; deyimi Sınıf Kitaplığı projelerinde kullanılamaz
DLL'leri oluşturmak için kullanılan Sınıf Kitaplığı projelerinde kullanılmasına izin verme `End` yordamdaki kod yürütmeyi durdurmak için anahtar sözcüğü.  
  
 **Hata Kimliği:** BC30615  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Denetim yapıları gibi kullanmak `While` ve `For` program yürütme akışını denetlemek için.
