# <a name="compiler-error-cs0025"></a>Derleyici Hatası CS0025
Standart Kitaplığı dosya 'dosyası bulunamadı  
  
 Derleyici tarafından gereken bir dosya bulunamadı. Yolun doğru olduğundan ve dosyanın var olduğunu denetleyin.  
  
 Dosya bir [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] sistem dosyasını, onarım gerekebilir, [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)] yükleme veya tamamen yeniden.
