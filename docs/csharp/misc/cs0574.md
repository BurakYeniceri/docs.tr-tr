# <a name="compiler-error-cs0574"></a>Derleyici Hatası CS0574
Yok Edicisi sınıfın adı eşleşmelidir  
  
 Bir tilde (~) öncesinde sınıf adı bir yıkıcı adı olmalıdır.  
  
 Aşağıdaki örnek CS0574 oluşturur:  
  
```csharp  
// CS0574.cs  
namespace x  
{  
   public class iii  
   {  
      ~iiii()   // CS0574  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
