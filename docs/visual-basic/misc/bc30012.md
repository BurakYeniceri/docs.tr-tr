# <a name="39if39-block-must-end-with-a-matching-39end-if39"></a>&#39;#If&#39; blok eşleşen ile bitmelidir &#39;#End If&#39;
`#If` Koşullu derleme yönergesi değil. Bir `#If` blok tarafından sonlandırıldı olmayan bir `#End If` yönergesi.  
  
 **Hata Kimliği:** BC30012  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme bir `#End If` koşullu derleme blok sonuna yönergesi.
