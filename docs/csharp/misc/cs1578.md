 <span data-ttu-id="8688d-104">Sonra bir [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) yönerge, yalnızca bir dosya adı (çift tırnak işaretleri) veya tek satırlık açıklama izin verilmiyor.</span><span class="sxs-lookup"><span data-stu-id="8688d-104">After a [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) directive, only a file name (in double quotation marks) or a single-line comment is allowed.</span></span>  
  
 <span data-ttu-id="8688d-105">Aşağıdaki örnek CS1578 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8688d-105">The following sample generates CS1578:</span></span>

```csharp  
   static void Main()

      #line 101 abc.cs   // CS1578

```
