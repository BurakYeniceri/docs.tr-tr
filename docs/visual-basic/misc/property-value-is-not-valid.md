# <a name="property-value-is-not-valid"></a>Özellik değeri geçerli değil
Nesne veya denetim özelliklerinden biri, aralığın dışında bir değere ayarlandı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Özellik değeri geçerli bir ayar değiştirin.
