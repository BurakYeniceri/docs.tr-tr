# <a name="the-connectiontimeout-must-be-greater-than-0"></a>ConnectionTimeout 0'dan büyük olmalıdır
Karşıya yükleme ve indirme zaman dosyaları ile [My.Computer.Network nesnesi](../../visual-basic/language-reference/objects/my-computer-network-object.md), belirtmeniz gerekir bir `connectionTimeout` büyük `0`.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tedarik bir `connectionTimeout` daha büyük `0`.
