# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-39get39-or-39set39"></a>&#39; System.Runtime.InteropServices.DllImportAttribute &#39; uygulanamaz bir &#39; Al &#39; ya &#39; Set &#39;
`DllImportAttribute` Özniteliği için uygulandığı bir `Get` veya `Set` özellik yordamı.  
  
 **Hata Kimliği:** BC31524  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kaldırma `DllImportAttribute` gelen `Get` ve `Set` özellik yordamları.
