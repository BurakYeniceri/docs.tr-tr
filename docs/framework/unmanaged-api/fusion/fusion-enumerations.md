# <a name="fusion-enumerations"></a>Fusion Numaralandırmaları
Bu bölümde API fusion kullanan yönetilmeyen numaralandırmalar açıklanmaktadır.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [ASM_CACHE_FLAGS numaralandırması](../../../../docs/framework/unmanaged-api/fusion/asm-cache-flags-enumeration.md)  
 Tarafından temsil edilen bir derleme kaynağını gösterir [Iassemblycacheıtem](../../../../docs/framework/unmanaged-api/fusion/iassemblycacheitem-interface.md) genel derleme önbelleğinde.  
  
 [ASM_CMP_FLAGS numaralandırması](../../../../docs/framework/unmanaged-api/fusion/asm-cmp-flags-enumeration.md)

 [Asm_dısplay_flags numaralandırması](../../../../docs/framework/unmanaged-api/fusion/asm-display-flags-enumeration.md)

 [ASM_NAME numaralandırması](../../../../docs/framework/unmanaged-api/fusion/asm-name-enumeration.md)
