# <a name="39raiseevent39-declaration-must-end-with-a-matching-39end-raiseevent39"></a>&#39;RaiseEvent&#39; bildirimi eşleşen ile bitmelidir &#39;End RaiseEvent&#39;
A `RaiseEvent` bildirimi ile bitmelidir bir `End RaiseEvent` deyimi.  
  
 **Hata Kimliği:** BC31117  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Denetleyin `RaiseEvent` bildirimi ile biten bir `End RaiseEvent` deyimi.
