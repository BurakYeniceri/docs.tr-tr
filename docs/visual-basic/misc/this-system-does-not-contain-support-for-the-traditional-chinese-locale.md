# <a name="this-system-does-not-contain-support-for-the-traditional-chinese-locale"></a>Bu sistem, Geleneksel Çince yerel ayar desteği içermiyor.
Kullanmaya çalıştığınız `VbStrConv.TraditionalChinese` numaralandırma üyesi ancak sisteminizi Geleneksel Çince yerel ayar desteklemez.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Geleneksel Çince yerel ayar desteklemek için makine yükseltin veya uygulamayı yerel destekleyen bir makinede çalıştırın.
