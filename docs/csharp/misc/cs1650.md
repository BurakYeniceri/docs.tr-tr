# <a name="compiler-error-cs1650"></a><span data-ttu-id="2d381-102">Derleyici Hatası CS1650</span><span class="sxs-lookup"><span data-stu-id="2d381-102">Compiler Error CS1650</span></span>
<span data-ttu-id="2d381-103">Statik salt okunur alan 'tanımlayıcısı' alanlarının (statik oluşturucu veya değişken başlatıcı içinde olması dışında) atanamaz</span><span class="sxs-lookup"><span data-stu-id="2d381-103">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="2d381-104">Bu hata, bir üyesi olduğu, değiştirilmesine izin verilmiyor, salt okunur ve statik bir alanı değiştirme girişimi oluşur.</span><span class="sxs-lookup"><span data-stu-id="2d381-104">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="2d381-105">Bu hatayı gidermek için sınırlandırmak salt okunur alan oluşturucu veya değişken Başlatıcısı atamaları ya da kaldırma `readonly` alan bildirimi anahtar sözcük.</span><span class="sxs-lookup"><span data-stu-id="2d381-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```
