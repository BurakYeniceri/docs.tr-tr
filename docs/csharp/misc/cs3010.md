## <a name="example"></a><span data-ttu-id="f6889-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="f6889-107">Example</span></span>  
 <span data-ttu-id="f6889-108">Aşağıdaki örnek CS3010 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f6889-108">The following example generates CS3010:</span></span>  
  
```csharp

public interface I  
{  
    [CLSCompliant(false)]  
    int M();   // CS3010

{  
    public int M()  
    {  
        return 1;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
