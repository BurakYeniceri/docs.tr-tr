## <a name="example"></a><span data-ttu-id="a9967-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="a9967-106">Example</span></span>  
 <span data-ttu-id="a9967-107">Aşağıdaki örnek CS0540 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="a9967-107">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
