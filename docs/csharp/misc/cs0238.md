# <a name="compiler-error-cs0238"></a><span data-ttu-id="52816-102">Derleyici Hatası CS0238</span><span class="sxs-lookup"><span data-stu-id="52816-102">Compiler Error CS0238</span></span>
<span data-ttu-id="52816-103">bir geçersiz kılma olmadığından 'member' korumalı olamaz</span><span class="sxs-lookup"><span data-stu-id="52816-103">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="52816-104">[korumalı](../../csharp/language-reference/keywords/sealed.md) de ile işaretlenmemiş üyesi kullanılan [geçersiz kılma](../../csharp/language-reference/keywords/override.md).</span><span class="sxs-lookup"><span data-stu-id="52816-104">[sealed](../../csharp/language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../../csharp/language-reference/keywords/override.md).</span></span> <span data-ttu-id="52816-105">Daha fazla bilgi için bkz: [devralma](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="52816-105">For more information, see [Inheritance](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="52816-106">Aşağıdaki örnek CS0238 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="52816-106">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
