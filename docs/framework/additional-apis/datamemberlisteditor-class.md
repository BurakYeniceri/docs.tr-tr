# <a name="datamemberlisteditor-class"></a>DataMemberListEditor sınıfı

Veri bağlama nesnelerin özelliklerini düzenlemek için bir aşağı açılan kullanıcı arabirimi sağlar (null olmayan nesneleri `DataSource` özelliği) tüm özelliklerini listeleyen tarafından `DataSource` değeri kaynak Seçilecek nesne.  
  
## <a name="syntax"></a>Sözdizimi
  
```csharp  
internal class DataMemberListEditor : UITypeEditor
```

> [!WARNING]
> `DataMemberListEditor` Sınıfı, iç ve kodunuzda doğrudan kullanılmak üzere yüksetlmesi.
> 
> Microsoft hiçbir koşulda bir üretim uygulamasında bu sınıf kullanımını desteklemez.
