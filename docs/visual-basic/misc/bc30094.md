# <a name="label-39ltlabelnamegt39-is-already-defined-in-the-current-methodmultiline-lambda"></a>Etiket &#39; &lt;etiketadı&gt;&#39; geçerli yöntem/çok satırlı lambda içinde zaten tanımlandı
Etiket adı yordam içinde yineleniyor. Bir yordam içinde tüm adlarının benzersiz olması gerekir.  
  
 **Hata Kimliği:** BC30094  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Yinelenen etiket yeniden adlandırın.
