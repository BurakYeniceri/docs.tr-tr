# <a name="microsofttransactionstransactionbridgedurableparticipantreplaywhilepreparing"></a>Microsoft.Transactions.TransactionBridge.DurableParticipantReplayWhilePreparing
WS-AT protokolü hizmeti hazırlanma yanıt vermediğinden dayanıklı katılımcı yeniden yürütme iletisi aldı. Sonuç olarak, işlem iptal edildi.  
  
## <a name="description"></a>Açıklama  
 Dayanıklı katılımcı hala hazırlama sırasında bir yeniden yürütme ileti alırsanız izlenen. Bu bu durumu için geçersiz bir ileti ve işlem iptal edecek.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Bu hatanın işlem sonucunu emin değilseniz ancak (alt TransactionManagers dahil) dayanıklı katılımcı, hatasından kurtarıldı indiate olabilir ve durumunu isteyin.
