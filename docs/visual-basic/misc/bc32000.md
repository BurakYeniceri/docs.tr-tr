# <a name="local-variable-39ltvariablenamegt39-cannot-be-referred-to-before-it-is-declared"></a>Yerel değişken &#39; &lt;variablename&gt;&#39; bildirilmeden önce başvurulamaz
Yordam içindeki bir değişkeni bildiriminden önce kullanılır.  
  
 **Hata Kimliği:** BC32000  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değişkene ilk başvurusundan önce bildirimi taşıyın.
