# <a name="39namespace39-statement-must-end-with-a-matching-39end-namespace39"></a>&#39; Namespace &#39; deyimi, eşleşen bir &#39;ile bitmelidir; Son Namespace &#39;
`Namespace`blokları ile bitmelidir bir `End Namespace` deyimi.  
  
 **Hata Kimliği:** BC30626  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme bir `End Namespace` deyimi sonuna `Namespace` bloğu.
