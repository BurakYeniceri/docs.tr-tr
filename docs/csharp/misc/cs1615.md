# <a name="compiler-error-cs1615"></a><span data-ttu-id="5deb1-102">Derleyici Hatası CS1615</span><span class="sxs-lookup"><span data-stu-id="5deb1-102">Compiler Error CS1615</span></span>
<span data-ttu-id="5deb1-103">Bağımsız değişken 'numara' with 'anahtar sözcüğü' anahtar sözcüğü geçirilmemelidir</span><span class="sxs-lookup"><span data-stu-id="5deb1-103">Argument 'number' should not be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="5deb1-104">Anahtar sözcüklerden biri `ref` veya **çıkışı** işlevi değil sürdü çağrılırken bir `ref` veya **çıkışı** bu bağımsız değişkeni için parametre.</span><span class="sxs-lookup"><span data-stu-id="5deb1-104">One of the keywords `ref` or **out** was used when the function did not take a `ref` or **out** parameter for that argument.</span></span> <span data-ttu-id="5deb1-105">Bu hatayı gidermek için yanlış anahtar sözcüğü kaldırın ve işlev bildirimi varsa eşleşen uygun anahtar kullanın.</span><span class="sxs-lookup"><span data-stu-id="5deb1-105">To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.</span></span>  
  
 <span data-ttu-id="5deb1-106">Aşağıdaki örnek CS1615 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5deb1-106">The following sample generates CS1615:</span></span>  
  
```csharp  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```
