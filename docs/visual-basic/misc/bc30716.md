# <a name="39finally39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Son olarak&#39; ifadeleri geçerli değildir komut penceresi
`Finally` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30716  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Finally` kodda hata ayıklama gelen deyimleri.
