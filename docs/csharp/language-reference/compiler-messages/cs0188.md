# <a name="compiler-error-cs0188"></a><span data-ttu-id="75760-102">Derleyici Hatası CS0188</span><span class="sxs-lookup"><span data-stu-id="75760-102">Compiler Error CS0188</span></span>
<span data-ttu-id="75760-103">Tüm alt alanlarını atanan önce 'this' nesnesi kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="75760-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="75760-104">Tüm alanlarda bir `struct` Oluşturucusu bir yöntemi çağırmak önce bir Oluşturucu tarafından atanacak olan `struct`.</span><span class="sxs-lookup"><span data-stu-id="75760-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="75760-105">Bir özellik yapı oluşturucusunda başlatılmaya çalışılırken bu hatayı görürseniz, yedekleme alanını özelliği yerine belirtmek için oluşturucu parametresini değiştirmek çözümüdür.</span><span class="sxs-lookup"><span data-stu-id="75760-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to change the constructor parameter to specify the backing field instead of the property itself.</span></span> <span data-ttu-id="75760-106">Hiçbir yedekleme alanı ve bu nedenle herhangi bir şekilde oluşturucusundan başlatılamıyor çünkü otomatik uygulanan özellikler yapılar için kaçınılmalıdır.</span><span class="sxs-lookup"><span data-stu-id="75760-106">Auto-implemented properties should be avoided in structs because they have no backing field and therefore cannot be initialized in any way from the constructor.</span></span>  
  
 <span data-ttu-id="75760-107">Daha fazla bilgi için bkz: [kullanarak yapılar](../../../csharp/programming-guide/classes-and-structs/using-structs.md).</span><span class="sxs-lookup"><span data-stu-id="75760-107">For more information, see [Using Structs](../../../csharp/programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="75760-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="75760-108">Example</span></span>  
 <span data-ttu-id="75760-109">Aşağıdaki örnek CS0188 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="75760-109">The following sample generates CS0188:</span></span>  
  
```csharp  
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
