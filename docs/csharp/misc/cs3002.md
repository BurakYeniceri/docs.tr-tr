# <a name="compiler-warning-level-1-cs3002"></a>Derleyici Uyarısı (düzey 1) CS3002
'Yönteminin' dönüş türü CLS uyumlu değil  
  
 A [ortak](../../csharp/language-reference/keywords/public.md), [korumalı](../../csharp/language-reference/keywords/protected.md), veya `protected internal` yöntemi türü ortak dil belirtimi (CLS) ile uyumlu olan bir değer döndürmesi gerekir. CLS uyumluluğu hakkında daha fazla bilgi için bkz: [CLS uyumlu kod yazma](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3) ve [dil bağımsızlığı ve dilden bağımsız bileşenler](../../../docs/standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS3002 oluşturur:  
  
```csharp  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
