# <a name="compiler-error-cs1623"></a><span data-ttu-id="ef351-102">Derleyici Hatası CS1623</span><span class="sxs-lookup"><span data-stu-id="ef351-102">Compiler Error CS1623</span></span>
<span data-ttu-id="ef351-103">Yineleyiciler ref veya out Parametreleri olamaz</span><span class="sxs-lookup"><span data-stu-id="ef351-103">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="ef351-104">İterator yöntemi sürerse, bu hata oluşur. bir `in`, `ref`, veya `out` parametresi.</span><span class="sxs-lookup"><span data-stu-id="ef351-104">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="ef351-105">Bu hatayı önlemek için kaldırmak `in`, `ref`, veya `out` yöntemi imza anahtar sözcük.</span><span class="sxs-lookup"><span data-stu-id="ef351-105">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ef351-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="ef351-106">Example</span></span>  
 <span data-ttu-id="ef351-107">Aşağıdaki örnek CS1623 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ef351-107">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
