# <a name="systemservicemodeltxcompletionstatusabortedonsessionclose"></a>System.ServiceModel.TxCompletionStatusAbortedOnSessionClose
Bu oturum kapatıldı ve TransactionAutoCompleteOnSessionClose OperationBehaviorAttribute false olarak ayarladığınızda, tamamlanmamış olduğundan belirtilen işlem durduruldu.  
  
## <a name="description"></a>Açıklama  
 Geçerli etkin oturum kapatıldı, işlem tamamlanmadı ve TransactionAutoCompleteOnSessionClose ayarlanmış izlenen `false`.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Bu izleme araştırılması gereken olası bir uygulama hatası gösterir.
