# <a name="39is39-expected"></a>&#39; olan &#39; Beklenen
A `TypeOf` işleci oluşur olmadan bir `Is` yan tümcesi.  
  
 **Hata Kimliği:** BC30224  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme bir `Is` yan tümcesi aşağıdaki `TypeOf` işleci; Örneğin, `TypeOf MyControl Is ComboBox`.
