# <a name="3939-expected"></a>&#39; (&#39; Beklenen
Bir bağımsız değişken listesi bir yordam çağrısında bir parantez ile başlamıyor (`(`) karakter.  
  
 **Hata Kimliği:** BC30199  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir açma ayracı ekleme listesi başlangıcına karakter.
