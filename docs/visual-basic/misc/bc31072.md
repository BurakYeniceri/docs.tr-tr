# <a name="warning-treated-as-error-lterrorgt"></a>Uyarı hata olarak kabul edilir: &lt;hata&gt;
Bir derlemeyi çalıştıran ile derlendiğinde bir uyarı karşılaşıldı **uyarıları hata olarak kabul** seçeneği açık.  
  
 **Hata Kimliği:** BC31072  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Uyarı düzeltin.  
  
2.  İle derleme **uyarıları hata olarak kabul** seçeneği kapalı.
