# <a name="call-duration"></a>Çağrı Süresi
Sayaç adı: Çağrı süresi  
  
## <a name="description"></a>Açıklama  
 Bu işlem yapılan çağrıların ortalama süresi. Ortalama süresi temel üzerinde bu denklemi hesaplanır: (N1-N0)/(D1-D0).  
  
> [!WARNING]
>  Zaman uyumsuz bir WCF Hizmeti üzerinde kullanıldığında çağrı süresi sayacı her zaman -1 döndürür.
