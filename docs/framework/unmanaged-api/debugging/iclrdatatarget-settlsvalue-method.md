# <a name="iclrdatatargetsettlsvalue-method"></a>ICLRDataTarget::SetTLSValue Yöntemi
Belirtilen iş parçacığının hedef işlemdeki iş parçacığı yerel depolaması (TLS) içinde bir değer ayarlar. Bu yöntem ortak dil çalışma zamanı (CLR) veri erişim Hizmetleri tarafından çağrılır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT SetTLSValue (  
    [in] ULONG32            threadID,  
    [in] ULONG32            index,  
    [in] CLRDATA_ADDRESS    value  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `threadID`  
 [in] Bir iş parçacığı hedef işlem, işletim sistemi tanımlayıcısı.  
  
 `index`  
 [in] Konumun dizini. Bu değer, belirtilen iş parçacığı yerel deposunda geçerli bir dizin olmalıdır.  
  
 `value`  
 [in] A `CLRDATA_ADDRESS` verilen TLS konuma yerleştirmek için değerini belirten değer.  
  
## <a name="remarks"></a>Açıklamalar  
 Bu yöntem, hata ayıklama uygulama yazıcı tarafından uygulanır.
