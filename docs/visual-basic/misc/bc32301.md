# <a name="project-has-been-closed"></a>Proje kapatıldı
İçinde Visual Basic derleyici bir kod modeli erişim hatası oluştu.  
  
 **Hata Kimliği:** BC32301  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Hata durumunda yinelenen yeniden görmek için programı derleyin.  
  
2.  Hata tekrar oluşursa, Visual Basic derleyici yeniden yükleyin.  
  
3.  Yeniden yüklenmesinden sonra hata devam ederse, koşullar hakkında bilgi toplayın ve Microsoft Ürün Destek Hizmetleri'ne bildirin.
