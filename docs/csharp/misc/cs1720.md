# <a name="compiler-warning-level-1-cs1720"></a><span data-ttu-id="c3ec5-102">Derleyici Uyarısı (düzey 1) CS1720</span><span class="sxs-lookup"><span data-stu-id="c3ec5-102">Compiler Warning (level 1) CS1720</span></span>
<span data-ttu-id="c3ec5-103">'Genel tür' varsayılan değeri null olduğundan ifade her zaman System.NullReferenceException özel durumuna neden olur</span><span class="sxs-lookup"><span data-stu-id="c3ec5-103">Expression will always cause a System.NullReferenceException because the default value of 'generic type' is null</span></span>  
  
 <span data-ttu-id="c3ec5-104">Varsayılan bir başvuru türü (örneğin, bir sınıf) olan bir genel tür değişken değerini içeren bir ifade yazarsanız, bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="c3ec5-104">If you write an expression involving the default of a generic type variable that is a reference type (for example, a class), this error will occur.</span></span> <span data-ttu-id="c3ec5-105">Aşağıdaki ifadeyi göz önünde bulundurun:</span><span class="sxs-lookup"><span data-stu-id="c3ec5-105">Consider the following expression:</span></span>  
  
```csharp  
default(T).ToString()  
```  
  
 <span data-ttu-id="c3ec5-106">Bu yana `T` bir başvuru türü null ve bu nedenle çalışırken uygulamak için varsayılan değeri olduğu <xref:System.Object.ToString%2A> , yönteme throw bir <xref:System.NullReferenceException>.</span><span class="sxs-lookup"><span data-stu-id="c3ec5-106">Since `T` is a reference type, its default value is null, and so attempting to apply the <xref:System.Object.ToString%2A> method to it will throw a <xref:System.NullReferenceException>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c3ec5-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="c3ec5-107">Example</span></span>  
 <span data-ttu-id="c3ec5-108">Sınıf başvuru kısıtlaması türündeki `T` sağlar `T` bir başvuru türüdür.</span><span class="sxs-lookup"><span data-stu-id="c3ec5-108">The class reference constraint on type `T` ensures that `T` is a reference type.</span></span>  
  
 <span data-ttu-id="c3ec5-109">Aşağıdaki örnek CS1720 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="c3ec5-109">The following sample generates CS1720.</span></span>  
  
```csharp  
// CS1720.cs  
using System;  
public class Tester   
{  
    public static void GenericClass<T>(T t1) where T : class   
    {  
        Console.WriteLine(default(T).ToString());  // CS1720  
    }  
    public static void Main() {}  
}  
```
