# <a name="expected-3939-at-start-of-an-embedded-expression"></a>Beklenen &#39; % = &#39; Katıştırılmış bir ifade başlangıcında
Katıştırılmış bir ifade için başlangıç tanımlayıcı `<%=`, düzgün şekilde kodlanmış değil. Bir XML öğesi değişmez değeri adında yüzde karakterini (%) kullanamayacağınızı unutmayın.  
  
 **Hata Kimliği:** BC31179  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Katıştırılmış ifade için başlangıç tanımlayıcı olarak kodlanmıştır olun `<%=`.
