<span data-ttu-id="2fea5-103">Bir geçersiz kılma yöntemi olduğundan koşullu öznitelik 'yöntemi' geçerli değil</span><span class="sxs-lookup"><span data-stu-id="2fea5-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
 <span data-ttu-id="2fea5-104">[Koşullu](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) özniteliği ile işaretlenmiş bir yöntemine izin verilmeyen [geçersiz kılma](../../csharp/language-reference/keywords/override.md) anahtar sözcüğü.</span><span class="sxs-lookup"><span data-stu-id="2fea5-104">The [Conditional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="2fea5-105">Daha fazla bilgi için bkz: [kullanımı geçersiz kılma ve yeni anahtar sözcüklerin için bilerek olduğunda](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="2fea5-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="2fea5-106">Yöntemleri geçersiz kılmak için derleyici hiçbir zaman bağlanmaz; base yöntemi yalnızca bağlar ve uygun şekilde geçersiz kılma ortak dil çalışma zamanı çağırır.</span><span class="sxs-lookup"><span data-stu-id="2fea5-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="2fea5-107">Aşağıdaki örnek CS0243 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="2fea5-107">The following sample generates CS0243:</span></span>  
  
```csharp  
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
