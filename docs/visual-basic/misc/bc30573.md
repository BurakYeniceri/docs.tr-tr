# <a name="xml-namespace-prefix-39prefix39-is-already-declared"></a>XML ad alanı öneki &#39; önek &#39; zaten bildirildi
Tarafından tanımlanan bir XML ad alanı öneki bir `Imports` deyimi eşleşen önceden tanımlanmış bir XML ad alanı öneki.  
  
 **Hata Kimliği:** BC30573  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML ad alanı öneki için benzersiz bir ad kullanın.
