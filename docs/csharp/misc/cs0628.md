# <a name="compiler-warning-level-4-cs0628"></a>Derleyici Uyarısı (düzey 4) CS0628
'member': yeni korumalı üye bildirilen korumalı sınıfında  
  
 A [korumalı](../../csharp/language-reference/keywords/sealed.md) olamaz sınıfını tanıtmak bir [korumalı](../../csharp/language-reference/keywords/protected.md) üyesi başka bir sınıf devralınmalıdır mümkün olduğundan `sealed` sınıfı ve kullanmak `protected` üyesi.  
  
 Aşağıdaki örnek CS0628 oluşturur:  
  
```  
// CS0628.cs  
// compile with: /W:4  
sealed class C  
{  
   protected int i;   // CS0628  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
