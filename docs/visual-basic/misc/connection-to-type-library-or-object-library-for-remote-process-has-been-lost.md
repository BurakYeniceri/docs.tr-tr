# <a name="connection-to-type-library-or-object-library-for-remote-process-has-been-lost"></a>Kitaplık ya da uzak işlem kitaplığının nesne türünü bağlantısı kesildi
Uzak işlemin nesne kitaplığı veya tür kitaplığı bağlantınız kesildi.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Yeniden `Application` nesne.  
  
2.  Bu Yardım konusu girdiğiniz hata iletişim kutusunda tıklatın **Tamam** görüntülemek için **başvuruları** iletişim kutusu. "Solunda eksik" sözcüğüyle kayıp referansı görüntülenir.  
  
3.  Kayıp başvuru silin.  
  
4.  İçinde **başvuruları** için iletişim kutusunda, onay kutusunu seçin `Application` nesnesi.
