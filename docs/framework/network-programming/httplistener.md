# <a name="httplistener"></a>HttpListener
<xref:System.Net.HttpListener> Sınıfı, bir program aracılığıyla denetlenen HTTP protokolü dinleyici sağlar. Dinleyici ömrü boyunca etkin olan <xref:System.Net.HttpListener> nesne ve uygulamanız içinde çalışır.  
  
## <a name="httpsys"></a>HTTP. SYS  
 <xref:System.Net.HttpListener> Sınıfı, Windows için tüm HTTP trafiğini işleyen çekirdek modu dinleyicisi HTTP.sys üstünde oluşturulur. HTTP.sys bağlantı yönetimi, bant genişliği azaltma ve Web sunucusu günlüğü sağlar. Kullanım `HttpCfg.exe` SSL sertifikalarını eklemek için aracı. Daha fazla bilgi için belgelere bakın [HttpCfg.exe](http://go.microsoft.com/fwlink/?LinkID=178284) içinde aracı [Server](http://go.microsoft.com/fwlink/?LinkID=178285) belgeleri.
