# <a name="39ltmodifiergt39-is-not-valid-on-a-method-declaration"></a>&#39; &lt;değiştiricisi&gt;&#39; bir yöntem bildirimi geçerli değil
A `Function` veya `Sub` deyimi geçersiz bir anahtar sözcük içeriyor.  
  
 **Hata Kimliği:** BC30242  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçersiz anahtar sözcüğü kaldırmak `Function` veya `Sub` deyimi.
