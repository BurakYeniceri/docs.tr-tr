# <a name="iclrstrongname2-interface"></a>ICLRStrongName2 Arabirimi
Tanımlayıcı adlar SHA-2 grubunu güvenli karma algoritması (SHA-256, SHA-384 ve SHA-512) kullanarak oluşturmanıza olanak sağlar.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[StrongNameGetPublicKeyEx yöntemi](../../../../docs/framework/unmanaged-api/hosting/strongnamegetpublickeyex-method.md)|Ortak anahtarı bir genel/özel anahtar çifti alır ve bir karma algoritması ve imza algoritmasını belirtir.|  
|[StrongNameSignatureVerificationEx2 yöntemi](../../../../docs/framework/unmanaged-api/hosting/strongnamesignatureverificationex2-method.md)|Türü kesin adlandırılmış bir derleme imzasını doğrular ve gerçek bir anahtara ECMA anahtarından bir eşleme sağlar.|
