# <a name="expected-closing-39--gt39-for-xml-comment"></a>Beklenen kapatma &#39;--&gt;&#39; XML açıklama
Bir XML açıklama değişmez değer gerekli kapanış '-->' karakteri eksik.  
  
 **Hata Kimliği:** BC31161  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gerekli kapanış '-->' karakter XML açıklama değişmez değeri sonuna ekleyin.
