# <a name="interface-members-must-be-methods-properties-events-or-type-definitions"></a>Arabirim üyeleri yöntemleri, özellikleri, olayları veya gerekir tür tanımı
Yalnızca içe türleri, yöntemleri, özellikleri ve olayları bir arabirim üyeleri olabilir.  
  
 **Hata Kimliği:** BC30602  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Öğe bir yöntemi, özelliği, olay veya numaralandırma gibi iç içe geçmiş bir tür olduğundan emin olun.
