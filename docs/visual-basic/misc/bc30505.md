# <a name="methods-or-events-that-implement-interface-members-cannot-be-declared-39shared39"></a>Yöntemleri veya arabirim üyeleri uygulayan olayları bildirilemez &#39;paylaşılan&#39;
Olarak bildirin denediniz `Shared` bir yöntemi veya arabirim üyesini uygulayan olay. Yöntemleri ve olayları bir sınıfa uygulanan olamaz belirlenmiş `Shared` veya `Private`, hariç devralınabilir olmayan bir sınıfta.  
  
 **Hata Kimliği:** BC30505  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Shared` anahtar sözcüğü.
