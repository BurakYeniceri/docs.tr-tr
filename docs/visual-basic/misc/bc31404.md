# <a name="39ltmethodnamegt39-cannot-shadow-a-method-declared-39mustoverride39"></a>&#39; &lt;methodname&gt;&#39; bildirilmiş bir yöntemi &#39; gölge olamaz MustOverride &#39;
Bir özellik veya yöntem ile `MustOverride` değiştirici ve aynı adı türetilen bir sınıfta bildirildi.  
  
 **Hata Kimliği:** BC31404  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ekleme `Overrides` geçersiz kılma özelliği veya yöntemi türetilmiş sınıfında değiştiricisi.  
  
2.  Kaldırma `MustOverride` değiştirici özelliği veya yöntemi temel sınıf.
