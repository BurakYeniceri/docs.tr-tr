# <a name="39namespace39-statements-can-occur-only-at-file-or-namespace-level"></a>&#39;Namespace&#39; deyimleri yalnızca dosya veya ad alanı düzeyinde oluşabilir
`Namespace` deyimleri sonra görünmelidir `Option` deyimleri `Imports` deyimleri ve genel öznitelikler, ancak tüm kaynak dosyanızı bildirimlerinde önce.  
  
 **Hata Kimliği:** BC30618  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Taşıma `Namespace` ad alanı bildirimini veya kaynak dosyanın en üstüne ifadesine.
