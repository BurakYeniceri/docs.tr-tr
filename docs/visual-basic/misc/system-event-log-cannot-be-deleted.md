# <a name="system-event-log-cannot-be-deleted"></a>Sistem olay günlüğünü silinemez.
Bir silinemez sistem olay günlüğünü silmek için girişimde bulunuldu. Sistem günlüğüne sistem başlatma ve donanım hataları gibi sistem olayları izler.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir uygulama veya özel günlük yerine için sistem olay günlüğüne yazma uygulamanızı depolamayı düşünün.  
  
-   Sistem olay günlüğünü silmek çalışmayın.
