# <a name="target-folder-is-a-file"></a>Hedef klasör dosyasıdır
Klasör ilgili eylemi gerçekleştirme girişimi sırasında bir dosya belirtiyorsanız.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir dosya yerine bir klasör belirtme emin olmak için sağlamış olursunuz yolunu denetleyin.
