# <a name="vbstrconvwide-and-vbstrconvnarrow-are-not-applicable-to-the-locale-specified"></a>VbStrConv.Wide ve VbStrConv.Narrow belirtilen yerel ayar için geçerli değildir
Uygulamayı kullanmaya çalışıyor `VbStrConv` numaralandırma üyeleri `Wide` veya `Narrow`, hangi olmadığı için belirtilen yerel ayar uygulanabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ya da kaldırma `VbStrConv.Wide` veya `VbStrConv.Narrow`.
