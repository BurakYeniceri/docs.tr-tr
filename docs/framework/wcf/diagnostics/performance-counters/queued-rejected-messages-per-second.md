# <a name="queued-rejected-messages-per-second"></a>Saniyede Kuyruğa Alınan Reddedilen İleti
Sayaç adı: Saniyede reddedilen iletileri kuyruğa alındı.  
  
## <a name="description"></a>Açıklama  
 Bir saniyede sıraya alınan aktarım sırasında bu hizmet tarafından reddedilen ileti sayısı.  
  
 Bu sayaç, performans sayacı türü [PERF_COUNTER_COUNTER](http://go.microsoft.com/fwlink/?LinkID=94649), değeri, aşağıdaki formül kullanılarak hesaplanır.  
  
 (1 - N 0 N) / ((D 1 - D 0) / F)
