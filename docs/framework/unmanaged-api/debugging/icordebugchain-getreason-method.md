# <a name="icordebugchaingetreason-method"></a>ICorDebugChain::GetReason Metodu
Bu arama zincirini genesis nedeni alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT GetReason (  
    [out] CorDebugChainReason *pReason  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `pReason`  
 [out] Bu arama zincirini genesis nedenini belirten CorDebugChainReason numaralandırması (Bitsel bir birleşimi) değerini gösteren bir işaretçi.
