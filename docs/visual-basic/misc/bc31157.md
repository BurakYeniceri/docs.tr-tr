# <a name="xml-attribute-39attribute139-must-appear-before-xml-attribute-39attribute239"></a>XML özniteliği &#39;öznitelik1&#39; XML özniteliği önce belirmelidir &#39;öznitelik2&#39;
Bir XML belgesi değişmez değeri öznitelikte geçersiz bir sırada belirtilir. Bir XML belgesi değişmez değeri için geçerli öznitelik sırası XML 1.0 belirtimi temel alır. Örneğin, `encoding` bir XML belgesi değişmez değer özniteliğinin hemen uymalıdır `version` özniteliği.  
  
 **Hata Kimliği:** BC31157  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML belgesi değişmez değer özniteliği sipariş XML 1.0 belirtimi içinde belirtilen yönergeleri kullanarak güncelleştirin.
