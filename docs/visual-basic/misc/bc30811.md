# <a name="39redim39-statements-can-no-longer-be-used-to-declare-array-variables"></a>&#39;ReDim&#39; deyimleri artık dizi değişkenlerini bildirmek için kullanılabilir
`ReDim` yalnızca var olan bir dizi boyutunu değiştirmek için kullanılabilir.  
  
 **Hata Kimliği:** BC30811  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bunlar bildirirken diziler boyutunu belirtin; Örneğin:  
  
    ```  
    Dim X(20) As Integer  
    ```
