# <a name="queue-dropped-messages-per-second"></a>Saniyede Kuyruğa Alınan Bırakılmış İleti
Sayaç adı: Saniye başına bırakılan iletileri kuyruğa alındı.  
  
## <a name="description"></a>Açıklama  
 Sıraya alınan aktarım sırasında bu hizmet tarafından bir saniyede bırakılan ileti sayısı.  
  
 Bu sayaç, performans sayacı türü [PERF_COUNTER_COUNTER](http://go.microsoft.com/fwlink/?LinkID=94649), değeri, aşağıdaki formül kullanılarak hesaplanır.
