# <a name="protocolrecoverycompletefailure"></a>ProtocolRecoveryCompleteFailure
Kimliği: 145  
  
 Önem derecesi: hata  
  
 Kategori: TransactionBridge  
  
## <a name="description"></a>Açıklama  
 Bu olay, WS-Atomic işlem Protokolü hizmetini başlatma ve kurtarma tamamlamak başarısız olduğunu gösterir. Sonuç olarak, WS-Atomic işlem işlevselliği devre dışı bırakıldı. Olay protokol kimliği, protokol adı, özel durum, işlem adı ve işlem kimliği listeler
