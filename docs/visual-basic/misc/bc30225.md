# <a name="39option-ltspecifiergt39-statement-can-only-appear-once-per-file"></a>&#39;Seçenek &lt;belirticisi&gt; &#39; deyimi yalnızca görünebilir kez dosya başına
`Option Compare`, `Option Explicit`, Ve `Option Strict` deyimleri yalnızca kullanılabilir bir kez aynı kaynak dosyasına.  
  
 **Hata Kimliği:** BC30225  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yinelenen kaldırma `Option` deyimi.
