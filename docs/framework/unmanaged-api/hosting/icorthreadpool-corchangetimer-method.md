# <a name="icorthreadpoolcorchangetimer-method"></a>ICorThreadpool::CorChangeTimer Yöntemi

## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT CorChangeTimer (  
    [in]  HANDLE Timer,   
    [in]  ULONG  DueTime,   
    [in]  ULONG  Period,  
    [out] BOOL*  result  
);  
```
