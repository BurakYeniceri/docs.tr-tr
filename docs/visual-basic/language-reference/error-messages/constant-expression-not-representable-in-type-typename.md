# <a name="constant-expression-not-representable-in-type-39lttypenamegt39"></a>Sabit ifade türü &#39;gösterilebilir değil; &lt;typename&gt;&#39;
Hedef türü uymayan bir sabit değerlendirmek çalıştığınız genellikle aralık taşan olduğundan.  
  
 **Hata Kimliği:** BC30439  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Hedef türü sabiti işleyebilecek bir değiştirin.
