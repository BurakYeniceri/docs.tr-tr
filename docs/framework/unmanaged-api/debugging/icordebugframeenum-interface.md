# <a name="icordebugframeenum-interface1"></a>Icordebugframeenum Interface1

## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[Next Yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugframeenum-next-method.md)|Belirtilen sayıda alır `ICorDebugFrame` geçerli konumdan başlayarak numaralandırması örneklerden.|
