# <a name="4807---discoverymessagewithinvalidreplyto"></a>4807 - DiscoveryMessageWithInvalidReplyTo
## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|Kimlik|4807|  
|Anahtar Sözcükler|Bulma|  
|Düzey|Uyarı|  
|Kanal|Microsoft Windows uygulaması sunucu-uygulamalar/hata ayıklama|  
  
## <a name="description"></a>Açıklama  
 Geçersiz bir ReplyTo adresi sahip olduğundan bulma ileti bırakıldı olduğunda bu olay yayınlanır.  
  
## <a name="message"></a>İleti  
 Bir bulma isteğine MessageID = '%1', geçersiz bir ReplyTo adresi sahip olduğundan bırakıldı.
