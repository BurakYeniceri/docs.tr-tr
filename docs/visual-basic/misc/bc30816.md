# <a name="39ltmembernamegt39-is-not-declared"></a>&#39; &lt;membername&gt;&#39; bildirilmemiş olan
'`<membername>`' bildirilmemiş olan. `Debug`Nesne işlevi `System.Diagnostics.Debug` içinde `System` derleme.  
  
 Belirtilen üye adı bulunamadı.  
  
 **Hata Kimliği:** BC30816  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Üye yazımını doğrulayın.  
  
2.  Kullanım bir `Imports` deyimi veya diğer ad alanlarında tanımlanan üyeleri tam olarak nitelemek. Örneğin, `WriteLine` işlevi içinde bildirilen `System.Diagnostics.Debug` ad alanı.
