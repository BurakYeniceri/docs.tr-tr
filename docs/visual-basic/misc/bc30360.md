# <a name="property-39ltpropertynamegt39-must-be-declared-39default39-because-it-overrides-a-default-property-on-the-base-39ltnamegt39"></a>Özellik &#39; &lt;propertyname&gt;&#39; bildirilmelidir &#39; varsayılan &#39; &#39; temel bir varsayılan özellik kıldığından&lt; ad&gt;&#39;
A `Property` kodunuzu deyiminde geçersiz kılan bir `Default` temel sınıf özelliği yapısı, veya arabirim ve bu nedenle bildirilmesi gerekir `Default`.  
  
 **Hata Kimliği:** BC30360  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Özelliği bildirme `Default`.
