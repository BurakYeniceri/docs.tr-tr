### <a name="aspnet-accessibility-improvements-in-net-framework-471"></a>.NET Framework 4.7.1 ASP.NET erişilebilirlik geliştirmeleri

|   |   |
|---|---|
|Ayrıntılar|ASP.NET .NET Framework 4.7.1 ile başlayarak, ASP.NET Web denetimleri daha iyi destek ASP.NET müşterilere Erişilebilirlik teknolojisi Visual Studio ile nasıl çalıştığıyla geliştirilmiştir.  Bunlar aşağıdaki değişiklikleri içerir:<ul><li>Değişikliklerin eksik UI erişilebilirlik desenlerini denetimlerinde, uygulama için alan Ekle iletişim kutusunda Ayrıntılar Görünümü Sihirbazı'nı veya ListView Sihirbazı'nı yapılandırma ListView iletişim gibi.</li><li>Yüksek karşıtlık modunda gibi veri çağrı cihazı alanları Düzenleyicisi görüntü iyileştirmeye yönelik değişiklikler.</li><li>Klavye gezinti iyileştirmeye yönelik değişiklikler karşılaştığında DataPager denetim, yapılandırma ObjectContext iletişim ya da veri kaynağı Yapılandırma Sihirbazı'nın veri Selction Yapılandır iletişim çağrı cihazı alanları Düzenle sihirbazdaki alanları iletişim gibi denetimleri için.</li></ul>|
|Öneri|<strong>İçinde veya dışında bu değişiklikleri kabul etmek nasıl</strong>sırada bu değişikliklerden yararlanmak Visual Studio tasarımcısı için .NET Framework 4.7.1 çalıştırmanız gerekir ya da daha sonra. Web uygulama aşağıdaki yollardan biriyle bu değişiklikleri yararlı olabilir:<ul><li>Visual Studio 2017 15.3 yükleyin veya daha sonra aşağıdaki AppContext anahtarıyla yeni erişilebilirlik özellikleri varsayılan olarak destekler.</li><li>Eski erişilebilirlik davranışları dışında ekleyerek opt <code>Switch.UseLegacyAccessibilityFeatures</code> AppContext anahtara <code>&lt;runtime&gt;</code> bölümünde devenv.exe.config dosyasında ve ayarlamak <code>false</code>, aşağıdaki örnekte gösterildiği gibi.</li></ul><pre><code class="lang-xml">&lt;?xml version=&quot;1.0&quot; encoding=&quot;utf-8&quot;?&gt;&#13;&#10;&lt;configuration&gt;&#13;&#10;&lt;runtime&gt;&#13;&#10;...&#13;&#10;&lt;!-- AppContextSwitchOverrides value attribute is in the form of &#39;key1=true/false;key2=true/false&#39;  --&gt;&#13;&#10;&lt;AppContextSwitchOverrides value=&quot;Switch.UseLegacyAccessibilityFeatures=false&quot; /&gt;&#13;&#10;...&#13;&#10;&lt;/runtime&gt;&#13;&#10;&lt;/configuration&gt;&#13;&#10;</code></pre>.NET Framework 4.7.1 hedefleyen uygulamalar veya üstü ve eski korumak istiyorsanız erişilebilirlik davranışı kabul etme eski erişilebilirlik özellikleri kullanmak için bu AppContext anahtarı açıkça ayarlayarak <code>true</code>.|
