## <a name="example"></a><span data-ttu-id="b1109-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="b1109-108">Example</span></span>  
 <span data-ttu-id="b1109-109">Bu kod örneği için bir .dll derleme bir .cpp dosyası ve bu .dll öğesini kullanan bir .cs dosyası oluşur.</span><span class="sxs-lookup"><span data-stu-id="b1109-109">This code sample consists of a .cpp file, which compiles to a .dll, and a .cs file, which uses that .dll.</span></span> <span data-ttu-id="b1109-110">Aşağıdaki kod için .dll dosyası ve .cs dosyası kod tarafından erişilecek bir özelliğini tanımlar.</span><span class="sxs-lookup"><span data-stu-id="b1109-110">The following code is for the .dll file and defines a property to be accessed by the code in the .cs file.</span></span>  
  
```cpp  
// CPP1546.cpp  
// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
