# <a name="operator-39operator39-is-not-defined-for-types-39type139-and-39type239"></a>İşleç &#39;işleci&#39; türleri için tanımlı değil &#39;type1&#39; ve &#39;type2&#39;
Türleri 'type1' ve 'type2' işleci 'işleci' tanımlı değil. 'Value' özelliği ilk öğesi dize değerini almak için kullanabileceğiniz '\<parentElement >'.  
  
 XML değişmez değeri bir ikili kullanılan veya `Boolean` karşılaştırması.  
  
 **Hata Kimliği:** BC31195  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanım `Value` özelliği için ikili değişmez değer XML veya `Boolean` karşılaştırması.
