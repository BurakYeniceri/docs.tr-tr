# <a name="static-local-variable-39ltvariablenamegt39-is-already-declared"></a>Statik yerel değişken &#39; &lt;variablename&gt;&#39; zaten bildirildi
Aynı ada sahip başka bir statik yerel değişken zaten tanımlanmış.  
  
 **Hata Kimliği:** BC31401  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Yedek statik yerel bildirimler kaldırın.  
  
2.  Her bir statik yerel değişken benzersiz bir ad verin.
