# <a name="project-39ltprojectnamegt39-makes-an-indirect-reference-to-assembly-39ltassemblynamegt39-which-contains-39ltclassnamegt39"></a>Proje &#39; &lt;projectname&gt; &#39; derlemesine dolaylı bir başvuru yapar &#39; &lt;assemblyname&gt; &#39; içeren &#39; &lt;classname&gt;&#39;
Proje '\<projectname >' derlemesine dolaylı bir başvuru yapar '\<assemblyname >' içeren '\<classname >'. Bir başvuru ekleyin '\<assemblyname >' projenize.  
  
 Bu proje Proje başvurularında yer almayan bir derlemeyi bağımlı üyeler içeriyor.  
  
 **Hata Kimliği:** BC31515  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sağ tıklayarak derlemesine başvuru ekleyin **başvuruları** içinde **Çözüm Gezgini** ve ardından **Başvuru Ekle**.
