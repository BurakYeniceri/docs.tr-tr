# <a name="compiler-error-cs0072"></a><span data-ttu-id="7e2f7-102">Derleyici Hatası CS0072</span><span class="sxs-lookup"><span data-stu-id="7e2f7-102">Compiler Error CS0072</span></span>
<span data-ttu-id="7e2f7-103">'event': geçersiz kılınamaz; 'yöntemi' bir olay değil</span><span class="sxs-lookup"><span data-stu-id="7e2f7-103">'event' : cannot override; 'method' is not an event</span></span>  
  
 <span data-ttu-id="7e2f7-104">Bir [olay](../../csharp/language-reference/keywords/event.md) yalnızca başka bir olay geçersiz kılabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="7e2f7-104">An [event](../../csharp/language-reference/keywords/event.md) can only override another event.</span></span> <span data-ttu-id="7e2f7-105">Daha fazla bilgi için bkz: [olayları](../../csharp/programming-guide/events/index.md).</span><span class="sxs-lookup"><span data-stu-id="7e2f7-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="7e2f7-106">Aşağıdaki örnek CS0072 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7e2f7-106">The following sample generates CS0072:</span></span>  
  
```csharp  
// CS0072.cs  
delegate void MyDelegate();  
  
class Test1  
{  
   public virtual event MyDelegate MyEvent;  
   public virtual void VMeth()  
   {  
   }  
  
   public void FireAway()  
   {  
      if (MyEvent != null)  
         MyEvent();  
   }  
}  
  
class Test2 : Test1  
{  
   public override event MyDelegate VMeth   // CS0072  
   // uncomment the following lines to resolve  
   // public override event MyDelegate MyEvent  
   {  
      add  
      {  
         VMeth += value;  
         // MyEvent += value;  
      }  
      remove  
      {  
         VMeth -= value;  
         // MyEvent -= value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
