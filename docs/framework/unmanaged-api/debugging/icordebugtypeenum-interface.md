# <a name="icordebugtypeenum-interface1"></a>Icordebugtypeenum Interface1

|Yöntem|Açıklama|  
|------------|-----------------|  
|[Next yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugtypeenum-next-method.md)|Belirtilen sayıda alır `ICorDebugType` geçerli konumdan başlayarak numaralandırması örneklerden.|
