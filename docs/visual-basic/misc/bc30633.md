# <a name="39set39-statement-must-end-with-a-matching-39end-set39"></a>&#39; Set &#39; deyimi, eşleşen bir &#39;ile bitmelidir; Son kümesi &#39;
`Set`özellik yordamları ile bitmelidir bir `End Set` deyimi.  
  
 **Hata Kimliği:** BC30633  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Olduğundan emin olun `Set` özellik yordamı sona erer ile bir `End Set` oluşturun.
