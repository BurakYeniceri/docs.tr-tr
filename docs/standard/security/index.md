# <a name="security-in-the-net-framework"></a>.NET Framework'te Güvenlik
Ortak dil çalışma zamanı ve .NET Framework pek çok yararlı sınıfları sağlar ve kolayca güvenli kod yazma ve sistem yöneticileri, erişebilmesi için kodu izinler özelleştirmek etkinleştirmek, geliştiricilerin Hizmetleri korunan kaynakları. Ayrıca, çalışma zamanı ve .NET Framework yararlı sınıfları ve şifreleme ve rol tabanlı güvenlik kullanımını kolaylaştırmak hizmetleri sağlar.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [Güvenlik Değişiklikleri](../../../docs/framework/security/security-changes.md)  
 .NET Framework güvenlik sistemi önemli değişiklikler açıklanmaktadır.  
  
 [Temel Güvenlik Kavramları](../../../docs/standard/security/key-security-concepts.md)  
 Bir genel bakış ortak dil çalışma zamanı güvenlik özellikleri sağlar. Bu bölümde geliştiriciler ve sistem yöneticileri için ilgilendirir.  
  
 [Rol Tabanlı Güvenlik](../../../docs/standard/security/role-based-security.md)  
 Rol tabanlı güvenlik, kodunuzda etkileşimde açıklar. Bu bölümde geliştiricilere ilgilendirir.  
  
 [Şifreleme Modeli](../../../docs/standard/security/cryptography-model.md)  
 .NET Framework tarafından sağlanan şifreleme hizmetlerine genel bakış sağlar. Bu bölümde geliştiricilere ilgilendirir.  
  
 [Güvenli Kodlama Yönergeleri](../../../docs/standard/security/secure-coding-guidelines.md)  
 Bazı güvenilir .NET Framework uygulamaları oluşturmak için en iyi uygulamaları açıklar. Bu bölümde geliştiricilere ilgilendirir.  
  
 [Yönetilmeyen Kod İçin Güvenli Kodlama Yönergeleri](../../../docs/framework/security/secure-coding-guidelines-for-unmanaged-code.md)  
 En iyi uygulamaları ve güvenlikle ilgili sorunların bazıları, yönetilmeyen kod çağrılırken açıklanmaktadır.  
  
 [Windows Identity Foundation](../../../docs/framework/security/index.md)  
 Uygulamalarınızda talep tabanlı kimlik nasıl uygulayacağınıza dair açıklar.
