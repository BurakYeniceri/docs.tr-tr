# <a name="could-not-find-library-39ltlibrarynamegt39"></a>Kitaplık &#39; bulunamıyor &lt;libraryname&gt;&#39;
Belirtilen kitaplık bulunamadı.  
  
 Komut isteminden derleme ve yanıt (.rsp) kullanarak, dosyayı açtığınızda bu sorun ortaya çıkabilir birçok derlemelere başvurur  
  
 **Hata Kimliği:** BC2017  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kitaplık adı doğru olduğundan emin olmak için kontrol edin.
