|||  
|-|-|  
|Kimlik|2576|  
|Anahtar Sözcükler|WFActivities|  
|Düzey|Bilgiler|

## <a name="description"></a>Açıklama  
 TryCatch etkinlik bir özel durum yakalandı gösterir.  
  
## <a name="message"></a>İleti  
 TryCatch etkinliği '%1', '%2' türünde bir özel durum yakalandı.
