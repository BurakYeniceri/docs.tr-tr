# <a name="endpoint-calls-faulted"></a>Uç Noktası: Hatalı Çağrılar
Sayaç adı: Hatalı çağrılar.  
  
## <a name="description"></a>Açıklama  
 Bu uç noktaya döndürmüş çağrı sayısı. İçinde [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)] uygulamalar, hizmet yöntemleri iletişim SOAP hata iletileri kullanarak hata bilgilerini işleme. SOAP, bir hizmet işlemi için meta verileri içinde yer alan ve bu nedenle istemcilerin kendi yürütme daha sağlam veya etkileşimli yapmak için kullanabileceği bir hataya sözleşmesi oluşturma ileti türlerini hatalarıdır. XML formundaki istemcilere SOAP hataları ifade olduğundan, yüksek oranda birlikte çalışabilir.
