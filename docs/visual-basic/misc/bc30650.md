# <a name="enums-must-be-declared-as-an-integral-type"></a>Numaralandırmalar bir tam sayı türü olarak bildirilmesi gerekir
Numaralandırmalara kullanabileceğiniz yalnızca geçerli türler `SByte`, `Byte`, `Short`, `UShort`, `Integer`, `UInteger`, `Long`, ve `ULong`. Diğer veri türleri kullanılabilir.  
  
 **Hata Kimliği:** BC30650  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir veri türü belirtin `SByte`, `Byte`, `Short`, `UShort`, `Integer`, `UInteger`, `Long`, veya `ULong`.
