 <span data-ttu-id="73617-104">Uygun bir oluşturucu yok kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="73617-104">There is no appropriate constructor available.</span></span> <span data-ttu-id="73617-105">Bu durum yalnızca bir değer atayarak başlatılmış yerleşik sayısal değer türleri için geçerlidir.</span><span class="sxs-lookup"><span data-stu-id="73617-105">This is the case for built-in numeric value types, which are initialized simply by assigning a value to them.</span></span>  
  
 <span data-ttu-id="73617-106">Aşağıdaki örnek CS0143 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="73617-106">The following sample generates CS0143:</span></span>  
  
```csharp  
// CS0143.cs  
class MyClass  
{  
   static public void Main ()  
   {  
      double d = new double(4.5);   // CS0143  
      // Try this line instead:  
      // double d = 4.5;  
   }  
}  
```
