# <a name="ways-to-select-a-windows-forms-button-control"></a>Windows Forms Düğme Denetimi Seçmenin Yolları
Bir Windows Forms düğme, aşağıdaki yollarla seçilebilir:  
  
-   Fare düğmesini kullanın.  
  
-   Düğmenin çağırma <xref:System.Windows.Forms.Control.Click> kodundaki olay.  
  
-   Odağı Taşı düğmesi için SEKME tuşuna basarak ve ardından Ara çubuğu veya ENTER tuşuna basarak düğmesini seçin.  
  
-   Erişim anahtarı (ALT + altı çizili harfi) düğmesine basın. Erişim anahtarları hakkında daha fazla bilgi için bkz: [nasıl yapılır: oluşturmak erişim anahtarları için Windows Forms denetimleri](../../../../docs/framework/winforms/controls/how-to-create-access-keys-for-windows-forms-controls.md).  
  
-   Düğme biçiminde "kabul" düğmesine ise, başka bir denetim odağı olsa bile ENTER tuşuna basarak düğmesini seçer — diğer denetleyen başka bir düğme, çok satırlı metin kutusu veya enter tuşuna yakalar özel bir denetim ise dışındaki.  
  
-   Düğme "İptal" düğmesini biçiminde ise, başka bir denetim odağı olsa bile ESC tuşuna basarak düğmesini seçer.  
  
-   Çağrı <xref:System.Windows.Forms.Button.PerformClick%2A> yöntemi program aracılığıyla düğmesini seçin.
