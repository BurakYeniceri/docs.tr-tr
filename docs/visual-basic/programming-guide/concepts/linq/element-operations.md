# <a name="element-operations-visual-basic"></a>Öğe işlemleri (Visual Basic)
Öğe işlemleri tek, belirli bir öğe bir dizisini döndürür.  
  
 Öğe işlemleri gerçekleştiren standart sorgu işleci yöntemleri aşağıdaki bölümde listelenmektedir.

|-----------------|-----------------|------------------------------------------|----------------------|
