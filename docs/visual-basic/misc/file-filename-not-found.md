# <a name="file-39ltfilenamegt39-not-found"></a>Dosya &#39; &lt;filename&gt; &#39; bulunamadı

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Dosya adının doğru olduğundan ve var olduğundan emin olun.
