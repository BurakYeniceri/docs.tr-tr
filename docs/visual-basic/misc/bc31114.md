# <a name="39custom-event39-must-end-with-a-matching-39end-event39"></a>&#39;Özel olay&#39; eşleşen ile bitmelidir &#39;bitiş olayı&#39;
A `Custom Event` bildirimi ile bitmelidir bir `End Event` deyimi.  
  
 **Hata Kimliği:** BC31114  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Denetleyin `Custom Event` bildirimi ile biten bir `End Event` deyimi.
