# <a name="systemservicemodelchannelstcpconnecterror"></a>System.ServiceModel.Channels.TcpConnectError
TCP bağlantı işlemi başarısız oldu.  
  
## <a name="description"></a>Açıklama  
 Bu uyarı düzeyi izleme TCP uç noktasına bağlanılamadı gösterir. Uzak uç noktada bir verilen IP adresi ve bağlantı noktasına yanıt vermiyor bu olabilir. Bu izleme sonraki varsa göz ardı edilebilir bağlanma girişiminde başka bir geçerli IP adresleri (IPv4 veya IPv6 adresleri veya belirli bir ana bilgisayar adını temsil eden diğer IP adresleri gibi) başarılı. Bu izleme içinde özel durum, hatayla ilgili ek bilgileri ortaya çıkarabilir.
