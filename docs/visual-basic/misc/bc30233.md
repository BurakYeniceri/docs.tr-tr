# <a name="39ltspecifiergt39-is-not-valid-on-a-constant-declaration"></a>&#39;&lt;belirleyici&gt; &#39; Sabit bildiriminde geçerli değil
A `Const` deyimi geçersiz bir anahtar sözcük içeriyor. A `Const` deyimi yalnızca içerebilir `Friend`, `Private`, `Protected`, `Public`, veya `Shadows` anahtar sözcükler.  
  
 **Hata Kimliği:** BC30233  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Anahtar sözcük doğru yazıldığından emin olun.  
  
2.  Geçersiz anahtar sözcüğü kaldırmak `Const` deyimi.
