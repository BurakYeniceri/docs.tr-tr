# <a name="39end-operator39-expected"></a>&#39;End işleci&#39; bekleniyor
Bir `End` deyimi sonlandırır blok eşleşmesi gerekir. Bir `Operator` yordamı tanımı değil düzgün bir şekilde sonlandırıldı.  
  
 **Hata Kimliği:** BC33005  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ekleme bir `End Operator` yordamı sonuna deyimi.
