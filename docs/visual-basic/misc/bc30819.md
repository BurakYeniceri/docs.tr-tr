# <a name="39ltmathfunction1gt39-is-not-declared"></a>&#39; &lt;mathfunction1&gt;&#39; bildirilmemiş olan
'\<mathfunction1 >' bildirilmemiş olan. İşlev System.Math sınıfına taşınmış ve şimdi çağrılır '\<mathfunction2 >'.  
  
 İç için çeşitli işlevleri [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] önceki sürümlerde taşınmıştır <xref:System.Math?displayProperty=nameWithType> ad alanı. Bu işlevleri tüm programlama dilleri için daha genel olarak kullanılabilir hale getirir.  
  
 **Hata Kimliği:** BC30819  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bildirilen yöntemleri kullanın <xref:System.Math?displayProperty=nameWithType>.
