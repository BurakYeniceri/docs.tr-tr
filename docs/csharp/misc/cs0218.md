# <a name="compiler-error-cs0218"></a><span data-ttu-id="7333c-102">Derleyici Hatası CS0218</span><span class="sxs-lookup"><span data-stu-id="7333c-102">Compiler Error CS0218</span></span>
<span data-ttu-id="7333c-103">' % S'tür ('type') true işleci ve false işleci bildirimler içermelidir</span><span class="sxs-lookup"><span data-stu-id="7333c-103">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
 <span data-ttu-id="7333c-104">Kullanıcı tanımlı bir tür için bir işleci tanımlama ve sonra operatör olarak kullanmayı deneyin bir ifadede bir kısa devre işleci, kullanıcı tanımlı işleç, işleç true ve işleç false tanımlanan olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="7333c-104">If you define an operator for a user-defined type, and then try to use the operator as a short-circuit operator, the user-defined operator must have operator true and operator false defined.</span></span> <span data-ttu-id="7333c-105">İşleçler hakkında daha fazla bilgi için kısa devre oluşturur, bkz: [& & işleci](../../csharp/language-reference/operators/conditional-and-operator.md) ve [ &#124; &#124; işleci](../../csharp/language-reference/operators/conditional-or-operator.md).</span><span class="sxs-lookup"><span data-stu-id="7333c-105">For more information on short-circuit operators, see [&& Operator](../../csharp/language-reference/operators/conditional-and-operator.md) and [&#124;&#124; Operator](../../csharp/language-reference/operators/conditional-or-operator.md).</span></span>  
  
 <span data-ttu-id="7333c-106">Aşağıdaki örnek, CS0218 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7333c-106">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
