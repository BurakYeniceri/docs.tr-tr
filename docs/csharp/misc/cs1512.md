# <a name="compiler-error-cs1512"></a>Derleyici Hatası CS1512
'Base' anahtar sözcüğü geçerli bağlamda mevcut değil  
  
 [Temel](../../csharp/language-reference/keywords/base.md) anahtar sözcüğü bir yöntemi, özelliği veya oluşturucusu dışında kullanıldı.  
  
 Aşağıdaki örnek CS1512 oluşturur:  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
