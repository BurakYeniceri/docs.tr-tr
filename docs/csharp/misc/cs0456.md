# <a name="compiler-error-cs0456"></a><span data-ttu-id="7f083-102">Derleyici Hatası CS0456</span><span class="sxs-lookup"><span data-stu-id="7f083-102">Compiler Error CS0456</span></span>
<span data-ttu-id="7f083-103">Tür parametresi 'Tür parametresi adı 1', 'struct' kısıtlaması sahiptir, böylece 'Tür parametresi adı 1' 'Tür parametresi adı için 2' kısıtlama olarak kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="7f083-103">Type parameter 'Type Parameter Name 1' has the 'struct' constraint so 'Type Parameter Name 1' cannot be used as a constraint for 'Type Parameter Name 2'</span></span>  
  
 <span data-ttu-id="7f083-104">Bu kısıtlamalar kısıtlamalar ikinci bir tür parametre olarak kullanılamaz değer türü kısıtlamaları örtük olarak korumalıdır.</span><span class="sxs-lookup"><span data-stu-id="7f083-104">Value type constraints are implicitly sealed so those constraints cannot be used as constraints on a second type parameter.</span></span> <span data-ttu-id="7f083-105">Değer türleri geçersiz kılınamaz olmasıdır.</span><span class="sxs-lookup"><span data-stu-id="7f083-105">This is because value types cannot be overridden.</span></span> <span data-ttu-id="7f083-106">Bu hatayı gidermek için bir değer türü kısıtlama doğrudan ilk tür parametresi aracılığıyla dolaylı şekilde bunun yerine ikinci tür parametresi yerleştirin.</span><span class="sxs-lookup"><span data-stu-id="7f083-106">To resolve this error, put a value type constraint directly on the second type parameter, instead of doing so indirectly by means of the first type parameter.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7f083-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="7f083-107">Example</span></span>  
 <span data-ttu-id="7f083-108">Aşağıdaki örnek CS0456 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="7f083-108">The following sample generates CS0456.</span></span>  
  
```csharp  
// CS0456.cs  
// compile with: /target:library  
public class GenericsErrors  
{  
   public class G5<T> where T : struct  
   {  
      public class N<U> where U : T {}   // CS0456  
      public class N2<U> where U : struct {}   // OK  
   }  
}  
```
