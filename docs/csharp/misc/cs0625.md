# <a name="compiler-error-cs0625"></a><span data-ttu-id="56df4-102">Derleyici Hatası CS0625</span><span class="sxs-lookup"><span data-stu-id="56df4-102">Compiler Error CS0625</span></span>
<span data-ttu-id="56df4-103">'field': StructLayout(LayoutKind.Explicit) ile işaretlenmiş örnek alanı türlerinin FieldOffset özniteliği olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="56df4-103">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="56df4-104">Ne zaman yapı işaretlenmiş açık bir **StructLayout** özniteliği, yapı tüm alanlarda olmalıdır [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) özniteliği.</span><span class="sxs-lookup"><span data-stu-id="56df4-104">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="56df4-105">Daha fazla bilgi için bkz: [StructLayoutAttribute sınıfı](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span><span class="sxs-lookup"><span data-stu-id="56df4-105">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span> 

<span data-ttu-id="56df4-106">Aşağıdaki örnek CS0625 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="56df4-106">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```
