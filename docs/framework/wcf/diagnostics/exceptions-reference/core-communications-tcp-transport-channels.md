# <a name="core-communications-tcp-transport-channels"></a>Temel İletişimler: TCP Taşıma Kanalları
Bu konuda TCP taşıma kanalları tarafından oluşturulan tüm özel durumlar listelenir.  
  
## <a name="exception-list"></a>Özel durum listesi  
  
|Kaynak kodu|Kaynak dizesi|  
|-------------------|---------------------|  
|SocketCloseReadTimeout|Belirtilen yuva uzak uç noktada bir Kapat isteği belirtilen belirlenen zaman aşımı süresi içinde yanıt vermedi. Uzak uç nokta Kapat EOF sinyal (boş) alma işlemi aldıktan sonra çağırdığını değil mümkündür. Bu işlem için ayrılan süre daha uzun bir süre bir kısmı olabilir.|
