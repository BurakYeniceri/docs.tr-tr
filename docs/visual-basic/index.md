# <a name="visual-basic-guide"></a>Visual Basic Kılavuzu

Visual Basic üretken tür kullanımı uyumlu ve nesne yönelimli uygulamaları oluşturmak için tasarlanmıştır. Visual Basic, Windows, Web ve mobil cihazları hedeflemek geliştiricilerin sağlar. Visual Basic avantajı güvenlik ve dil birlikte çalışabilirliği yazılmış programları Microsoft .NET Framework hedefleme tüm diller olduğu gibi.  
  
 Visual Basic bu nesil .NET Framework tabanlı uygulamalar oluşturmak için hızlı ve kolay bir yol vermiş yılda devam eder.  
  
 Visual Basic zaten yoksa, bir Visual Basic ücretsiz bulunur Visual Studio sürümünü elde edebilirsiniz [Visual Studio](https://aka.ms/vsdownload?utm_source=mscom&utm_campaign=msdocs) site.
