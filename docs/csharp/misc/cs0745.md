# <a name="compiler-error-cs0745"></a><span data-ttu-id="30eb6-102">Derleyici Hatası CS0745</span><span class="sxs-lookup"><span data-stu-id="30eb6-102">Compiler Error CS0745</span></span>

 <span data-ttu-id="30eb6-104">Desenini `group` yan tümcesi `group...by` bir isteğe bağlı olarak ve ardından `into`, aşağıdaki örnekte gösterildiği gibi:</span><span class="sxs-lookup"><span data-stu-id="30eb6-104">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>

## <a name="example"></a><span data-ttu-id="30eb6-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="30eb6-108">Example</span></span>  
 <span data-ttu-id="30eb6-109">Aşağıdaki kod CS0745 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="30eb6-109">The following code generates CS0745:</span></span>  
  
```csharp  
// cs0745.cs

using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };
```  
