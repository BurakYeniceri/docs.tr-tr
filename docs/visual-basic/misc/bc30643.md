# <a name="property-39ltpropertynamegt39-is-of-an-unsupported-type"></a>Özellik &#39; &lt;propertyname&gt; &#39; türü desteklenmiyor
Başvurulan bir derleme bir özelliği temsil edilemeyen bir türü içerir.  
  
 **Hata Kimliği:** BC30643  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Uygulamanızla birlikte kullanılabilen bir tür belirtin.
