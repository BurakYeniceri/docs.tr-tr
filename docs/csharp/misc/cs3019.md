# <a name="compiler-warning-level-2-cs3019"></a>Derleyici Uyarısı (Düzey 2) CS3019
Bu derleme dışından görülemediğinden olmadığından CLS uyumluluk denetimi 'türünde' gerçekleştirilmeyecek.  
  
 Bir tür veya üye sahip olduğunda bu uyarı oluşur <xref:System.CLSCompliantAttribute> özniteliği başka bir bütünleştirilmiş koddan görünür değil. Bu hatayı gidermek için öznitelik herhangi bir sınıf veya bir bütünleştirilmiş koddan görünür olmayan üyeleri kaldırın veya tür veya üye görünür yapmak. CLS uyumluluğu hakkında daha fazla bilgi için bkz. [ \<PAVE üzerinden > CLS uyumlu kod yazma](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek, CS3019 oluşturur:  
  
```csharp  
// CS3019.cs  
// compile with: /W:2  
  
using System;  
  
[assembly: CLSCompliant(true)]  
  
// To fix the error, remove the next line  
[CLSCompliant(true)]  // CS3019  
class C  
{  
    [CLSCompliant(false)]  // CS3019  
    void Foo()  
    {  
    }  
  
    static void Main()  
    {  
    }  
}  
```
