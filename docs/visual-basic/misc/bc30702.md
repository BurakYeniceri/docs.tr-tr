Dizinde tanımlanan aralığını aşıyor.  
  
 **Hata Kimliği:** BC30702  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Dizi sınırları aşılmadığını şekilde yordamı mantığını değiştirin.  
  
-   Dizi değerleri daha geniş ölçüdeki izin verecek şekilde yeniden tanımlayın.
