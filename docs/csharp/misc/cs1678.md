# <a name="compiler-error-cs1678"></a><span data-ttu-id="be18f-102">Derleyici Hatası CS1678</span><span class="sxs-lookup"><span data-stu-id="be18f-102">Compiler Error CS1678</span></span>
<span data-ttu-id="be18f-103">Parametre 'number', 'type1' türü olarak bildirilmiş ancak 'type2' olmalıdır</span><span class="sxs-lookup"><span data-stu-id="be18f-103">Parameter 'number' is declared as type 'type1' but should be 'type2'</span></span>  
  
 <span data-ttu-id="be18f-104">Anonim bir yöntem parametresi türü yönteme atama temsilci bildirimi farklı olduğunda bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="be18f-104">This error occurs when the parameter type in an anonymous method is different from the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="be18f-105">Aşağıdaki örnek CS1678 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="be18f-105">The following sample generates CS1678:</span></span>  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors   
{  
   static void Main()   
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
