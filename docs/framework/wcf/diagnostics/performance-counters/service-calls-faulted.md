# <a name="service-calls-faulted"></a>Hizmet: Hatalı Çağrılar
Sayaç adı: Hatalı çağrılar.  
  
## <a name="description"></a>Açıklama  
 Bu hizmet döndürmüş çağrı sayısı. Windows Communication Foundation (WCF) uygulamalarında hizmet yöntemleri işleme hata bilgileri SOAP hata iletileri kullanarak iletişim kurar. SOAP, bir hizmet işlemi için meta verileri içinde yer alan ve bu nedenle istemcilerin kendi yürütme daha sağlam veya etkileşimli yapmak için kullanabileceği bir hataya sözleşmesi oluşturma ileti türlerini hatalarıdır. XML formundaki istemcilere SOAP hataları ifade olduğundan, yüksek oranda birlikte çalışabilir.
