# <a name="39region39-statement-must-end-with-a-matching-39end-region39"></a>&#39;#Region&#39; deyimi, eşleşen bir ile bitmelidir &#39;#End Region&#39;
Kullanım `#Region` genişletme veya Visual Studio Kod Düzenleyicisi'nin anahat özelliğini kullanırken daraltma kod bloğu belirtmek için yönergesi. Başlangıcı ve sonu `#Region` deyimleri, aynı kod bloğunda olmalıdır.  
  
 **Hata Kimliği:** BC30681  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  INSERT `#End Region` koddan sonra uygun yerde `#Region` deyimi.
