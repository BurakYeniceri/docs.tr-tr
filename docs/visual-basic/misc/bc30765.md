# <a name="all-projects-in-a-visual-basic-solution-must-target-the-same-platform-but-the-project-you-are-trying-to-add-targets-a-platform-other-than-the-one-specified-by-your-solution"></a>Visual Basic Çözümdeki tüm projeleri aynı platformu hedeflemesi gerekir, ancak bir çözüm tarafından belirtilen dışında bir platform eklemeye çalıştığınız proje hedefler
Tüm projelerde bir [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] çözüm aynı platformu hedefleyen gerekir, ancak bir çözüm tarafından belirtilen dışında bir platform eklemekte olduğunuz proje hedefler.  
  
 **Hata Kimliği:** BC30765  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yalnızca çözümünüz tarafından belirtilen aynı platformu hedefleyen projeler ekleyin.
