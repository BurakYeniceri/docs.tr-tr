# <a name="39ltmodifiergt39-is-not-valid-on-an-event-declaration"></a>&#39;&lt;Değiştirici&gt; &#39; olay bildiriminde geçerli değil
Bir `Event` deyimi içeren bir geçersiz anahtar sözcüğü gibi `ReadOnly`.  
  
 **Hata Kimliği:** BC30243  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Geçersiz anahtar sözcüğü kaldırmak `Event` deyimi.
