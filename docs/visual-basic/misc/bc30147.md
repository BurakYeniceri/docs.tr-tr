# <a name="39return39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Dönüş&#39; ifadeleri geçerli değildir komut penceresi
`Return` Deyimi dallanma gerçekleştirir ve hata ayıklama bağlamda izin verilmiyor.  
  
 **Hata Kimliği:** BC30147  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sorun değil bir `Return` deyiminde **hemen** penceresi.
