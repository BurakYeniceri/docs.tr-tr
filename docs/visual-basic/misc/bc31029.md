# <a name="method-39ltmethodnamegt39-cannot-handle-event-39lteventnamegt39-because-they-do-not-have-the-same-signature"></a>Yöntem &#39; &lt;methodname&gt; &#39; olay işleyemiyor &#39; &lt;eventname&gt; &#39; aynı imzaya sahip olmadığı için
Belirtilen olay için tanımlanan parametrelerin ya da belirtilen olay işleyicisi için parametre türleri eşleşmiyor.  
  
 **Hata Kimliği:** BC31029  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Parametre eşleşme listeler olay veya olay işleyicisi için yeniden tanımlayın.
