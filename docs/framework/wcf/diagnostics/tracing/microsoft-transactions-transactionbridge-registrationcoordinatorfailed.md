# <a name="microsofttransactionstransactionbridgeregistrationcoordinatorfailed"></a>Microsoft.Transactions.TransactionBridge.RegistrationCoordinatorFailed
WS-AT protokolü hizmeti ilgili düzenleyiciye bir kayıt iletisi gönderilemedi  
  
## <a name="description"></a>Açıklama  
 Yerel TransactionManager kendi üst TransactionManager nedeniyle bir ileti göndermek için kullanamama kaydı mümkün değilse, izlenen.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 İleti gönderme hataya neden olan özel durum izleme iletisi inceleyin.
