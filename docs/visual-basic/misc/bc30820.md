# <a name="39ltlsetstmtgt39-is-not-declared"></a>&#39;&lt;lsetstmt&gt; &#39; bildirilmemiş olan
'\<lsetstmt >' bildirilmemiş olan. LSet deyimleri artık desteklenmiyor; Bunun yerine Microsoft.VisualBasic.LSet kullanın.  
  
 Önceki sürümlerde Visual Basic'e iç çeşitli işlevleri taşınmıştır <xref:Microsoft.VisualBasic?displayProperty=nameWithType> ad alanı. Bu işlevleri tüm programlama dilleri için daha genel olarak kullanılabilir hale getirir.  
  
 **Hata Kimliği:** BC30820
