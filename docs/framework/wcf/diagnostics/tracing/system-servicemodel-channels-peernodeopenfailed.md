# <a name="systemservicemodelchannelspeernodeopenfailed"></a>System.ServiceModel.Channels.PeerNodeOpenFailed
Komşu bağlantı açma girişimi başarısız oldu.  
  
## <a name="description"></a>Açıklama  
 Bu izleme komşu bağlantı kurmak için girişiminden sonra oluşur. Olası nedenler, yanıt vermeyen bir çözümleyici, bir hatalı bağlama ayarı veya geçici ağ hataları içerebilir.  
  
## <a name="troubleshooting"></a>Sorun giderme  
 Her iki düğüm geçerli adresleri olduğundan ve uygun şekilde yapılandırıldığından emin olun.
