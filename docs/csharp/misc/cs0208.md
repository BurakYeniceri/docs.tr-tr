 <span data-ttu-id="085fd-104">Hatta ile kullanıldığında [güvenli](../../csharp/language-reference/keywords/unsafe.md) anahtar sözcüğü, yönetilen bir nesnenin adresini almak, yönetilen bir nesnenin boyutunu alma veya bildirme bir yönetilen türün bir işaretçiye izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="085fd-104">Even when used with the [unsafe](../../csharp/language-reference/keywords/unsafe.md) keyword, taking the address of a managed object, getting the size of a managed object, or declaring a pointer to a managed type is not allowed.</span></span> <span data-ttu-id="085fd-105">Yönetilen bir türdür:</span><span class="sxs-lookup"><span data-stu-id="085fd-105">A managed type is:</span></span>

## <a name="example"></a><span data-ttu-id="085fd-109">Örnek</span><span class="sxs-lookup"><span data-stu-id="085fd-109">Example</span></span>  
 <span data-ttu-id="085fd-110">Aşağıdaki örnek, CS0208 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="085fd-110">The following sample generates CS0208:</span></span>  
  
```csharp  
// CS0208.cs  
// compile with: /unsafe  
  
class myClass  
{  
    public int a = 98;  
}  
  
struct myProblemStruct  
{  
    string s;  
    float f;  
}  
  
struct myGoodStruct  
{  
    int i;  
    float f;  
}  
  
public class MyClass  
{  
    unsafe public static void Main()  
    {  
        // myClass is a class, a managed type.  
        myClass s = new myClass();    
        myClass* s2 = &s;    // CS0208  
  
        // The struct contains a string, a managed type.  
        int i = sizeof(myProblemStruct); //CS0208  
  
        // The struct contains only value types.  
        i = sizeof(myGoodStruct); //OK  
  
    }  
}  
```
