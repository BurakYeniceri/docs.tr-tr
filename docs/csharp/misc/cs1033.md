# <a name="compiler-error-cs1033"></a>Derleyici Hatası CS1033
Kaynak dosya görüntülenebilecek 16,707,565 satır PDB gösterilebilir sınırını aştı; hata ayıklama bilgisi yanlış olur  
  
 Kaynak kodu dosyası izin verilen maksimum derleyici işleyebilir satır sayısı aşıldı. Bu hatayı gidermek için özgün dosyanın iki veya daha fazla kaynak kodu dosyaları oluşturun. En çok satır sayısını 268,435,454 satırdır. Kullanıyorsanız **/debug**, birden fazla 16,707,566 kullanarak bozuk hata ayıklama bilgilerini neden olur.
