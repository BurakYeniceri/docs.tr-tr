# <a name="compiler-error-cs0310"></a><span data-ttu-id="8962b-102">Derleyici Hatası CS0310</span><span class="sxs-lookup"><span data-stu-id="8962b-102">Compiler Error CS0310</span></span>
<span data-ttu-id="8962b-103">'typename' türü, parametre olarak 'parametresi' genel tür ya da 'genel' yöntemi kullanmak için bir Özet olmayan tür genel bir parametresiz oluşturucuya sahip olmalıdır</span><span class="sxs-lookup"><span data-stu-id="8962b-103">The type 'typename' must be a non-abstract type with a public parameterless constructor in order to use it as parameter 'parameter' in the generic type or method 'generic'</span></span>  
  
 <span data-ttu-id="8962b-104">Genel tür ya da yöntemi yeni bir kısıtlama, where tanımlayan genel türü veya yöntemi için bir tür bağımsız değişkeni olarak kullanılması için herhangi bir tür genel bir parametresiz oluşturucuya sahip olmalıdır şekilde yan tümcesi.</span><span class="sxs-lookup"><span data-stu-id="8962b-104">The generic type or method defines a new constraint in its where clause, so any type must have a public parameterless constructor in order to be used as a type argument for that generic type or method.</span></span> <span data-ttu-id="8962b-105">Bu hatayı önlemek için türü doğru oluşturucuya sahip olduğundan emin olun veya genel türü veya yöntemi sınırlama tümceciğini değiştirin.</span><span class="sxs-lookup"><span data-stu-id="8962b-105">To avoid this error, make sure that the type has the correct constructor, or modify the constraint clause of the generic type or method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8962b-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="8962b-106">Example</span></span>  
 <span data-ttu-id="8962b-107">Aşağıdaki örnek CS0310 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8962b-107">The following sample generates CS0310:</span></span>  
  
```csharp  
// CS0310.cs  
using System;  
  
class G<T> where T : new()  
{  
    T t;  
  
    public G()  
    {  
        t = new T();  
        Console.WriteLine(t);  
    }  
}  
  
class B  
{  
    private B() { }  
    // Try this instead:  
    // public B() { }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
        G<B> g = new G<B>();   // CS0310  
        Console.WriteLine(g.ToString());  
    }  
}  
```
