# <a name="error-embedding-win32-manifest-manifest-file-ltfilegt-cannot-be-found"></a>Win32 bildirimi katıştırma hatası: bildirim dosyası &lt;dosya&gt; bulunamıyor
Belirtilen dosya `/win32manifest` derleyici seçeneği bulunamıyor.  
  
 **Hata Kimliği:** BC31192  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Geçerli bir dosya adı ve yolu sağlayın `/win32manifest` derleyici seçeneği.
