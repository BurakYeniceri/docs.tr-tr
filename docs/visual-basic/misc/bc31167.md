# <a name="expected-3939-for-xml-end-tag"></a>Beklenen &#39; / &#39; XML bitiş etiketi
XML değişmez değeri XML bitiş etiketi, gerekli '/' karakteri eksik.  
  
 **Hata Kimliği:** BC31167  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   '/' Karakterini XML bitiş etiketi başlangıç değişmez değer XML ekleyin.
