# <a name="compiler-error-cs0629"></a><span data-ttu-id="f4acf-102">Derleyici Hatası CS0629</span><span class="sxs-lookup"><span data-stu-id="f4acf-102">Compiler Error CS0629</span></span>
<span data-ttu-id="f4acf-103">Koşullu üyesi 'member' türünde 'Tür adı' 'taban sınıf üyesi' arabirim üyesini uygulayamaz</span><span class="sxs-lookup"><span data-stu-id="f4acf-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="f4acf-104">[Koşullu](https://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) özniteliği bir arabirim uygulaması üzerinde kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="f4acf-104">The [Conditional](https://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="f4acf-105">Aşağıdaki örnek, CS0629 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f4acf-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
