# <a name="39ltfilenamegt39-cannot-be-referenced-because-it-is-not-an-assembly"></a>&#39;&lt;Dosya adı&gt; &#39; bütünleştirilmiş olmadığından başvurulamaz
Bir .NET derlemesi olmayan bir dosya için bir başvuru yapıldı.  
  
 **Hata Kimliği:** BC31519  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Derlemenin konumunu ve adını doğrulayın.  
  
2.  Derleme zarar görmüş veya bozuk olmadığını doğrulayın.
