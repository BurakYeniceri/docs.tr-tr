# <a name="event-name-length-cannot-exceed-1011-characters"></a>Olay adı uzunluğu 1011 karakterden uzun olamaz
Ad veya tanımlayıcı her olayın ile bildirilen bir `Event` deyimi 1011 karakterle sınırlıdır.  
  
 **Hata Kimliği:** BC32204  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tanımlayıcı kısaltın.
