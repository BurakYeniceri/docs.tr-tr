# <a name="expected-beginning-39lt39-for-an-xml-tag"></a>Başlangıç beklenen &#39; &lt; &#39; bir XML etiketi
XML değişmez değeri XML etiketi gerekli başına eksik ' <' karakter.  
  
 **Hata Kimliği:** BC31165  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekle ' <' XML değişmez değeri XML etiketi başlangıç karakteri.
