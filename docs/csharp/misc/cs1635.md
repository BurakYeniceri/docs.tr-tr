# <a name="compiler-warning-level-1-cs1635"></a><span data-ttu-id="29ec0-102">Derleyici Uyarısı (düzey 1) CS1635</span><span class="sxs-lookup"><span data-stu-id="29ec0-102">Compiler Warning (level 1) CS1635</span></span>
<span data-ttu-id="29ec0-103">Genel olarak devre dışı bırakıldığı için uyarı uyarı 'code' geri yüklenemiyor</span><span class="sxs-lookup"><span data-stu-id="29ec0-103">Cannot restore warning 'warning code' because it was disabled globally</span></span>  
  
 <span data-ttu-id="29ec0-104">Kullanırsanız, bu uyarıyı oluşur **/nowarn** komut satırı seçeneği veya bir uyarı için tüm derleme birimi devre dışı bırakma ayarı proje ancak kullandığınız `#pragma warning restore` bu uyarı geri yükleme girişimi için.</span><span class="sxs-lookup"><span data-stu-id="29ec0-104">This warning occurs if you use the **/nowarn** command line option or project setting to disable a warning for the entire compilation unit, but you use `#pragma warning restore` to attempt to restore that warning.</span></span> <span data-ttu-id="29ec0-105">Bu hatayı gidermek için / nowarn komut satırı seçeneği proje ayarı kaldırma veya kaldırma `#pragma warning restore` , devre dışı bırakma proje ayarları ve komut satırı tüm uyarılar için.</span><span class="sxs-lookup"><span data-stu-id="29ec0-105">To resolve this error, remove the /nowarn command line option or project setting, or remove the `#pragma warning restore` for any warnings you are disabling via the command line or project settings.</span></span> <span data-ttu-id="29ec0-106">Daha fazla bilgi için bkz: [#pragma Uyarısı](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) konu.</span><span class="sxs-lookup"><span data-stu-id="29ec0-106">For more information, see the [#pragma warning](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) topic.</span></span>  
  
 <span data-ttu-id="29ec0-107">Aşağıdaki örnek CS1635 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="29ec0-107">The following sample generates CS1635:</span></span>  
  
```csharp  
// CS1635.cs  
// compile with: /w:1 /nowarn:162  
  
enum MyEnum {one=1,two=2,three=3};  
  
class MyClass  
{  
    public static void Main()  
    {  
#pragma warning disable 162  
  
    if (MyEnum.three == MyEnum.two)  
        System.Console.WriteLine("Duplicate");  
  
#pragma warning restore 162  
    }  
}  
```
