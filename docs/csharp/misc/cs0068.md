 <span data-ttu-id="7d953-104">Bir olay arabirimdeki bir başlatıcı sahip olamaz.</span><span class="sxs-lookup"><span data-stu-id="7d953-104">An event in an interface cannot have an initializer.</span></span> <span data-ttu-id="7d953-105">Daha fazla bilgi için bkz: [arabirimleri](../../csharp/programming-guide/interfaces/index.md).</span><span class="sxs-lookup"><span data-stu-id="7d953-105">For more information, see [Interfaces](../../csharp/programming-guide/interfaces/index.md).</span></span>  
  
 <span data-ttu-id="7d953-106">Aşağıdaki örnek CS0068 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="7d953-106">The following sample generates CS0068:</span></span>  
  
```csharp  
// CS0068.cs

{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068

   // event MyDelegate d2;

{  
   event MyDelegate d = new MyDelegate(M.f);  
  
   public static void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
