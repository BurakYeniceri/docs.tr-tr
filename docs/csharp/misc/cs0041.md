# <a name="compiler-error-cs0041"></a>Derleyici Hatası CS0041
'Type' için tam adı, hata ayıklama bilgileri için çok uzun. Olmadan derleme '/ debug' seçeneği.  
  
 Bu hatayı kullanılırken oluşabilir [/debug](../../csharp/language-reference/compiler-options/debug-compiler-option.md) derleyici seçeneği. Bu hatayla karşılaşırsanız, bin dizinindeki PDB dosyaları silmeyi deneyin ve yeniden derlenmesi. Bu hata hala karşılaşılıyorsa, onarmak veya Visual Studio yeniden yüklemek zorunda kalabilirsiniz.
