 <span data-ttu-id="f7a12-104">Genel tür ya da yöntemi bir sınıf türü kısıtlamasına sahip olduğunda, bu kısıtlama listelenmesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="f7a12-104">When a generic type or method has a class type constraint, that constraint must be listed first.</span></span> <span data-ttu-id="f7a12-105">Bu hatayı önlemek için sınıf türü kısıtlaması kısıtlaması listenin başına taşıyın.</span><span class="sxs-lookup"><span data-stu-id="f7a12-105">To avoid this error, move the class type constraint to the beginning of the constraint list.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f7a12-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="f7a12-106">Example</span></span>
