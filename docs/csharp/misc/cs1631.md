# <a name="compiler-error-cs1631"></a><span data-ttu-id="b57b0-102">Derleyici Hatası CS1631</span><span class="sxs-lookup"><span data-stu-id="b57b0-102">Compiler Error CS1631</span></span>
<span data-ttu-id="b57b0-103">Catch yan tümcesinin gövdesinde bir değer verim olamaz</span><span class="sxs-lookup"><span data-stu-id="b57b0-103">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="b57b0-104">Verim deyimini bir catch yan tümcesi gövdesi içinde izin verilmiyor.</span><span class="sxs-lookup"><span data-stu-id="b57b0-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="b57b0-105">Bu hatayı önlemek için catch yan tümcesi gövdesi dışında verim deyimi taşıyın.</span><span class="sxs-lookup"><span data-stu-id="b57b0-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="b57b0-106">Aşağıdaki örnek CS1631 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b57b0-106">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
