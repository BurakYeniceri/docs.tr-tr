# <a name="generic-methods-cannot-be-exposed-to-com"></a>Genel yöntemler COM gösterilemez
A [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] bir veya daha fazla genel yordamlar içeren bileşeni için bir COM bileşeni aktarılır.  
  
 Bileşen Nesne Modeli (COM) genel türleri desteklemez ve bunlarla işleyemez.  
  
 **Hata Kimliği:** BC30943  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Genel yordam veya yordamlardan kaldırmak [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] bileşeni veya COM birlikte çalışma için kullanmayın.
