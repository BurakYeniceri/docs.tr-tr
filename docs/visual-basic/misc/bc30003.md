# <a name="39next39-expected"></a>&#39;Sonraki&#39; bekleniyor
Bir `On Error Resume` yapı oluşur tamamlanırken olmadan `Next`.  
  
 **Hata Kimliği:** BC30003  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme `Next` anahtar sözcüğü sonuna `On Error` deyimi.
