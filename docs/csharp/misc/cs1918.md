 <span data-ttu-id="f4d26-104">Nesne Başlatıcı kendisi bir yapı türü özelliklerini başlatmak için başlatıldığını sınıfının bir özelliği kullanmaya çalıştığınızda bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="f4d26-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>

## <a name="example"></a><span data-ttu-id="f4d26-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="f4d26-108">Example</span></span>  
 <span data-ttu-id="f4d26-109">Aşağıdaki örnek CS1918 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="f4d26-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
