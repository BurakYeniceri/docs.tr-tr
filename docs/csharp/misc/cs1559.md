# <a name="compiler-error-cs1559"></a>Derleyici Hatası CS1559
Alınmış olduğundan 'object' ana yöntemi için kullanılamaz  
  
 Geçersiz bir sınıf için belirtilen [/ana](../../csharp/language-reference/compiler-options/main-compiler-option.md) derleyici seçeneği; sınıfı için bir konum olarak kullanılamaz [ana](../../csharp/programming-guide/main-and-command-args/index.md) yöntemi.
