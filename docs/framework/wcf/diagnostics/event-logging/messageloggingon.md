Kimliği: 116  
  
 Önem derecesi: hata  
  
 Kategori: ServiceModel  
  
## <a name="description"></a>Açıklama  
 İleti günlüğe kaydetme etkinleştirildi. Kablo şifrelenmiş olsa bile, hassas bilgileri günlüğe olabilir. Örneğin, İleti gövdeleri. Olay AppID, özel durum, işlem adı ve işlem kimliği listeler
