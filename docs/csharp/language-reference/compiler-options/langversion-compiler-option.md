NuGet paketlerini veya diğer kitaplıkları.</span><span class="sxs-lookup"><span data-stu-id="22523-137">While the new features definitely require a new compiler update that is also released alongside the C# revision, each specific feature has its own minimum .NET API or common language runtime requirements that may allow it to run on downlevel frameworks by including NuGet packages or other libraries.</span></span>
