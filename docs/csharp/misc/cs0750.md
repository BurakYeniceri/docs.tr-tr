# <a name="compiler-error-cs0750"></a><span data-ttu-id="06826-102">Derleyici Hatası CS0750</span><span class="sxs-lookup"><span data-stu-id="06826-102">Compiler Error CS0750</span></span>
<span data-ttu-id="06826-103">Kısmi bir yöntem erişim değiştiricileri veya yeni, korumalı, sanal, soyut, geçersiz veya extern değiştiricileri sahip olamaz.</span><span class="sxs-lookup"><span data-stu-id="06826-103">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="06826-104">Özel davranışlarını nedeniyle kabul edebilir değiştiricileri sınırlama kısmi yöntemler tabidir.</span><span class="sxs-lookup"><span data-stu-id="06826-104">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>

1.  <span data-ttu-id="06826-106">Yetkisiz değiştirici yöntem bildiriminden kaldırın.</span><span class="sxs-lookup"><span data-stu-id="06826-106">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="06826-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="06826-107">Example</span></span>  
 <span data-ttu-id="06826-108">Aşağıdaki örnek CS0750 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="06826-108">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
