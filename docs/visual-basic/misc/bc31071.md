# <a name="structure-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>Arabirim yapısında bildirilemez &#39; &lt;belirticisi&gt;&#39;
Geçersiz değiştirici bir arabirim, bildirilen bir yapı ile belirtildi.  
  
 **Hata Kimliği:** BC31071  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değiştiriciler gibi kaldırmak `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, veya `Overridable`.
