# <a name="arrays-used-as-attribute-arguments-are-required-to-explicitly-specify-values-for-all-elements"></a>Öznitelik bağımsız değişkenleri olarak kullanılan diziler açıkça tüm öğeler için değerler belirtmeniz gerekir
Öznitelik bağımsız değişken olarak kullanılan bir dizi belirtilmeyen bir değeri olan bir öğe içeriyor.  
  
 **Hata Kimliği:** BC31110  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tüm öğesi değerlerini belirtin.
