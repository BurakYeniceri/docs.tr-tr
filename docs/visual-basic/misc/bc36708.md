# <a name="39group39-not-allowed-in-this-context-identifier-expected"></a>&#39;Grup&#39; bu bağlamda; izin verilmiyor tanımlayıcı bekleniyor
`Group` Anahtar sözcüğü dahil `Into` bölümünü bir `Aggregate` yan tümcesi. `Group` Anahtar sözcüğü geçerli yalnızca `Group By` veya `Group Join` yan tümceleri.  
  
 **Hata Kimliği:** BC36618  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Group` anahtar sözcüğünün `Aggregate` yan tümcesi. Ekleyebileceğiniz bir `Group By` Grup sonuçları için sorgu yan tümcesini.
