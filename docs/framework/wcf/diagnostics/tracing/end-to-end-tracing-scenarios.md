# <a name="end-to-end-tracing-scenarios"></a>Uçtan Uca İzleme Senaryoları
Bu bölüm, izlemenin kullanmak için farklı senaryolar açıklanmaktadır konuları içerir.  
  
 Etkinlik izleme/yanıt için aşağıdakileri kullanarak OneWay ve çift yönlü senaryoları uygulanır  
  
-   Zaman uyumlu veya zaman uyumsuz  
  
-   BasicHttpBinding, netNamedPipeBinding, netTcpBinding, wsHttpBinding veya netMsmqBinding  
  
-   COM hizmeti  
  
-   Kullanıcı Etkinlik Kimliği yayma
