# <a name="operator-39ltoperatornamegt39-is-not-defined-for-types-39lttype1gt39-and-39lttype2gt39"></a>Operator &#39; &lt;operatorname&gt;&#39; türleri &#39; için tanımlı değil&lt; type1&gt;&#39; ve &#39;&lt; type2&gt;&#39;

-   İki öğe denetleyin ve uyumlu olduklarından emin olun.
