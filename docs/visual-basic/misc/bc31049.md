# <a name="initializers-on-structure-members-are-valid-only-for-39shared39-members-and-constants"></a>Başlatıcılar üyeleri yalnızca yapısını &#39; Paylaşılan &#39; Üyeler ve sabitler
Yapı üye değişkeni bildiriminden bir parçası olarak başlatıldı.  
  
 **Hata Kimliği:** BC31049  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir sabit yerine bir değişkeni kullanın.  
  
-   Parametreli bir oluşturucu yapısına ekleyin. Örneğin:  
  
    ```  
    Structure TestStruct  
       Public t As Integer  
       Sub New(ByVal Tval As Integer)  
          t = Tval  
       End Sub  
    End Structure  
    ```
