# <a name="namespace-or-type-39ltnamegt39-has-already-been-imported"></a>Namespace veya türü &#39; &lt;adı&gt; &#39; önceden içeri aktarıldı
Bir ad alanı birden çok kez içeri aktarmak için girişimde bulunuldu.  
  
 **Hata Kimliği:** BC31051  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yedek Kaldır `Imports` deyimleri.
