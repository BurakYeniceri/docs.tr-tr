# <a name="conversion-from-39double39-to-39date39-requires-calling-the-39datefromoadate39"></a>Dönüştürme &#39;çift&#39; için &#39;tarih&#39; arama gerektirir &#39;Date.FromOADate&#39;
Cast denediniz bir `Date` değeri bir `Double` kullanmadan yapılamaz değeri <xref:System.DateTime.FromOADate%2A?displayProperty=nameWithType> yöntemi.  
  
 **Hata Kimliği:** BC30533  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanım <xref:System.DateTime.FromOADate%2A> değerini dönüştürmek için yöntem.
