# <a name="39set39-parameter-cannot-be-declared-39ltspecifiergt39"></a>&#39;Ayarlama&#39; parametresi bildirilemez &#39; &lt;belirticisi&gt;&#39;
Geçersiz değiştirici bir parametresi ile belirtilen bir `Set` özellik yordamı.  
  
 **Hata Kimliği:** BC31065  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçersiz değiştirici gibi kaldırmak `ByRef`, `ParamArray`, ve `Optional`.
