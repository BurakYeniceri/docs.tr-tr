# <a name="compiler-error-cs1623"></a>Derleyici Hatası CS1623
Yineleyiciler sahip ref veya out Parametreleri  
  
 İterator yöntemi sürerse, bu hata oluşur. bir `ref` veya `out` parametresi. Bu hatayı önlemek için kaldırmak `ref` veya `out` yöntemi imza anahtar sözcük.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS1623 oluşturur:  
  
```  
// CS1623.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {  
        yield return i;  
    }  
  
    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {  
        f = 0.0F;  
        yield return f;  
    }  
}  
```
