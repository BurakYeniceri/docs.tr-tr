# <a name="39set39-is-already-declared"></a>&#39; Set &#39; zaten bildirildi
A `Set` deyimi bir kod bloğu içinde birden çok kez görünüyor. `Set` Deyimi bildiren bir `Set` bir özellik için bir değer atamak için kullanılan özellik yordamı.  
  
 **Hata Kimliği:** BC30444  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yedek kaldırmak `Set` deyimi.
