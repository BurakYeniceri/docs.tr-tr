# <a name="evaluation-of-lambda-expressions-is-not-valid-in-the-debugger"></a>Lambda ifadeleri değerlendirme Hata Ayıklayıcısı'ndaki geçerli değil
Lambda ifadesi girilen **izleme** veya **hemen** penceresi bir hata ayıklama oturumu sırasında. Lambda ifadeleri windows hata ayıklama kullanılabilir değil.  
  
 **Hata Kimliği:** BC36627  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Lambda ifadeleri Kod düzenleyicisinde test edin.
