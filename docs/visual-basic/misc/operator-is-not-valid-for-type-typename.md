# <a name="operator-is-not-valid-for-type-39lttypenamegt39"></a>İşleç türü için geçerli değil &#39; &lt;typename&gt;&#39;
Bir işleç türleri için uygun değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İşleç üzerinde çalıştırılan türleri için uygun olduğundan emin olun.
