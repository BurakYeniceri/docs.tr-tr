# <a name="interface-39ltinterfacenamegt39-can-be-implemented-only-once-by-this-type"></a>Arabirim &#39; &lt;InterfaceName&gt;&#39; bu türü tarafından yalnızca bir kez uygulanabilir
Belirtilen arabirim zaten bu türü tarafından uygulanır ve yeniden uygulanır.  
  
 **Hata Kimliği:** BC31033  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bu tür yalnızca bir kez kullanarak bu arabirimi uygular.
