# <a name="-visual-basic-reference"></a>$ (Visual Basic Başvurusu)

Bir dize sabit değeri olarak tanımlayan bir [ilişkilendirilmiş bir dizedir](../../programming-guide/language-features/strings/interpolated-strings.md). Bir aradeğerlendirme dizesinde metin ile birlikte içeren bir şablon benzeri dizedir *ilişkilendirilmiş ifade*. İlişkilendirilmiş dize çözüldüğünde, örneğin bir atama ifadesi veya bir yöntem çağrısının, ilişkilendirilmiş ifadeler bunların dize temsilleri sonuç dizesindeki değiştirilir. İlişkilendirilmiş dizeler için değiştirmeler olan [bileşik biçimli dizeler](../../../standard/base-types/composite-format.md) .NET Framework tarafından desteklenir.
