 <span data-ttu-id="0a0bb-104">Bir sınıftan değil genişletmek uzantı metodu çağırma çalıştığınızda bu hatayı oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="0a0bb-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="0a0bb-105">Burada gösterilen örnekte, türetilmiş sınıf için genişletme yöntemi tanımlanmamışsa `A`, ancak temel sınıfı için `B`.</span><span class="sxs-lookup"><span data-stu-id="0a0bb-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>

1.  <span data-ttu-id="0a0bb-107">Yoksa, çağırma çağrısı varolan yöntemin genişlettiği türünde bir nesneye taşımak için sahip olduğu türü için yeni bir genişletme yöntemi oluşturun.</span><span class="sxs-lookup"><span data-stu-id="0a0bb-107">Create a new extension method for the type where you have to invoke it, or else move the call into an object of the type that the existing method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0a0bb-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="0a0bb-108">Example</span></span>  
 <span data-ttu-id="0a0bb-109">Aşağıdaki kod CS1928 ve CS1929 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0a0bb-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
