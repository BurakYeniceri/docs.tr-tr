# <a name="39ltnamegt39-is-not-a-member-of-39ltclassnamegt39"></a>&#39; &lt;adı&gt;&#39; üyesi &#39; değil&lt; ClassName&gt;&#39;
Sağladığınız üye sınıf üyesi değil.  
  
 **Hata Kimliği:** BC30456  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Doğru olduğundan emin olmak için üyenin adını kontrol edin.  
  
2.  Gerçek bir sınıf üyesi kullanın.
