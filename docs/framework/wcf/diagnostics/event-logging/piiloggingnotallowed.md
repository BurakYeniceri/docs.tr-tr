# <a name="piiloggingnotallowed"></a>PiiLoggingNotAllowed
Kimliği: 108  
  
 Önem derecesi: hata  
  
 Kategori: izleme  
  
## <a name="description"></a>Açıklama  
 Bu olay, bilinen hiçbir PII oturum olduğunu gösterir. Bilinen PII günlüğe izin verilmiyor. Bilinen PII günlüğe izin vermek için ayarlanmış "enableLoggingKnownPii" `true` Machine.Config'deki. İşlem kimliği ve işlem adı olay listeler
