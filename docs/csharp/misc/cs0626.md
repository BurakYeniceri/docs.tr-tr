 <span data-ttu-id="47500-105">Bir yöntem olarak işaretlenmiş `extern` de olan bir öznitelik, örneğin işaretlenmelidir, [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) özniteliği.</span><span class="sxs-lookup"><span data-stu-id="47500-105">A method marked `extern` should also be marked with an attribute, for example, the [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) attribute.</span></span>

   }
