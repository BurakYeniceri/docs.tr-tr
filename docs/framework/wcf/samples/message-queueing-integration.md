# <a name="message-queueing-integration"></a>İleti Kuyruğa Alma Tümleştirme
Bu bölüm, Message Queuing ve Windows Communication Foundation (WCF) arasındaki tümleştirmeden gösteren örnekler içerir.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [Message Queuing’den Windows Communication Foundation'a](../../../../docs/framework/wcf/samples/message-queuing-to-wcf.md)  
 Message Queuing (MSMQ) uygulama bir MSMQ iletisinin bir WCF hizmeti için nasıl gönderebileceğiniz gösterilir.
  
 [Windows Communication Foundation'dan Message Queuing’e](../../../../docs/framework/wcf/samples/wcf-to-message-queuing.md)  
 WCF uygulaması bir Message Queuing (MSMQ) uygulaması için bir ileti nasıl gönderebileceğiniz gösterilir.  
  
 [İleti Bağıntısı](../../../../docs/framework/wcf/samples/message-correlation.md)  
 Message Queuing (MSMQ) uygulama bir MSMQ iletisinin bir WCF hizmeti için nasıl gönderebilir ve iletileri bir istek/yanıt senaryosunda gönderen ve alıcı uygulamalar arasında nasıl eşleştirilebilir gösterir.
