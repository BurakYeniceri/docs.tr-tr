# <a name="default-member-39ltmembernamegt39-is-not-a-property"></a>Varsayılan üye &#39; &lt;membername&gt;&#39; bir özellik değil
Bir üyesi bildirin denediniz `Default`. Ancak, `Default` anahtar sözcüğü varsayılan özelliklerini tanımlamak için kullanılır.  
  
 **Hata Kimliği:** BC30555  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir özellik tanımlama emin olmak için onay `Default`, ya da başka Kaldır `Default` anahtar sözcüğü.
