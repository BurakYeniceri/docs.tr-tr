# <a name="39end-event39-must-be-preceded-by-a-matching-39custom-event39"></a>&#39; Son olay &#39; eşleşen bir &#39;göre gelmelidir; Özel olay &#39;
Bir `End Event` deyimi oluşur karşılık gelen olmadan `Custom Event` deyimi. `End Event`karşılık gelen tarafından gelmelidir `Custom Event` deyimi.  
  
 **Hata Kimliği:** BC31123  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir önceki emin `Custom Event` deyimi geçerli olduğundan ve doğru yazılmış.
