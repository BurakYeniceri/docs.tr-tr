# <a name="parameter-39ltparameternamegt39-already-has-a-matching-argument"></a>Parametre &#39; &lt;parametername&gt; &#39; eşleşen bir bağımsız değişken zaten var
Bir bağımsız değişken adıyla aynı yordam çağrısında birden fazla kez geçirilir.  
  
 **Hata Kimliği:** BC36567  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bağımsız değişken adları birini yanlış yazılmış olup olmadığını denetleyin.  
  
-   Yinelenen bağımsız değişken bağımsız değişken listesinden kaldırır.
