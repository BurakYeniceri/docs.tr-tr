## <a name="example"></a><span data-ttu-id="1e5f6-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="1e5f6-107">Example</span></span>  

 <span data-ttu-id="1e5f6-108">Aşağıdaki örnek, CS0417 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="1e5f6-108">The following example generates CS0417:</span></span>

```csharp
// CS0417

{  
    // The following line causes CS0417.  
    T instance1 = new T(1);
  
    // The following line doesn't cause the error.

}  
```
