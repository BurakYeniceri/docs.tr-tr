# <a name="xml-dtds-are-not-supported"></a>XML DTD desteklenmez.
Bir belge türü tanımı (DTD) bir XML değişmez değer belirtilmedi.  
  
 **Hata Kimliği:** BC31175  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   DTD çıkarın. XML şema tanımı (XSD) bir XML belgesi için şemayı belirtmek için kullanın.
