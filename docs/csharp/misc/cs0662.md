# <a name="compiler-error-cs0662"></a><span data-ttu-id="54959-102">Derleyici Hatası CS0662</span><span class="sxs-lookup"><span data-stu-id="54959-102">Compiler Error CS0662</span></span>

<span data-ttu-id="54959-103">'yöntemi' ref parametresi üzerindeki tek Out özniteliğini belirtemezsiniz.</span><span class="sxs-lookup"><span data-stu-id="54959-103">'method' cannot specify only Out attribute on a ref parameter.</span></span> <span data-ttu-id="54959-104">Hem de ve Out özniteliklerini veya tipleri kullanın.</span><span class="sxs-lookup"><span data-stu-id="54959-104">Use both In and Out attributes, or neither.</span></span>  
  
 <span data-ttu-id="54959-105">Bir arabirim yöntemini kullanan bir parametreye sahip [ref](../../csharp/language-reference/keywords/ref.md) ile yalnızca [çıkışı](xref:System.Runtime.InteropServices.OutAttribute) özniteliği.</span><span class="sxs-lookup"><span data-stu-id="54959-105">An interface method has a parameter that uses [ref](../../csharp/language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="54959-106">A `ref` kullanan parametre **çıkışı** özniteliği de kullanmalıdır [içinde](xref:System.Runtime.InteropServices.InAttribute) özniteliği.</span><span class="sxs-lookup"><span data-stu-id="54959-106">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>  
  
 <span data-ttu-id="54959-107">Aşağıdaki örnek CS0662 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="54959-107">The following sample generates CS0662:</span></span>  
  
```csharp
// CS0662.cs  
using System.Runtime.InteropServices;  
  
interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
