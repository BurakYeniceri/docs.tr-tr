## <a name="example"></a><span data-ttu-id="4f8c4-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="4f8c4-106">Example</span></span>  
 <span data-ttu-id="4f8c4-107">Aşağıdaki örnek CS1642 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="4f8c4-107">The following sample generates CS1642.</span></span>  
  
```  
// CS1642.cs  
// compile with: /unsafe /target:library  
unsafe class C  
{  
   fixed int a[10];   // CS1642  
}

{  
    fixed int a[10];  
}  
  
unsafe class E  
{  
   public int[] a = null;  
}  
```
