# <a name="can39t-open-39ltfilenamegt39-for-writing"></a>Yapabilirsiniz&#39;t açık &#39; &lt;filename&gt; &#39; yazma
Belirtilen dosya, çünkü zaten açıldı, belki de yazmak için açılamıyor.  
  
 **Hata Kimliği:** BC2012  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Dosyayı kapatıp yeniden açın.  
  
2.  Dosyanın izinlerini denetleyin.
