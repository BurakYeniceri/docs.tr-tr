# <a name="argument-39ltargumentnamegt39-is-not-valid-for-the-array"></a>Bağımsız değişken &#39; &lt;argumentname&gt; &#39; dizi için geçerli değil
Bir dizi geçersiz bir bağımsız değişken içeriyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İfade değişkenlerinde yazımını denetleyin. Yanlış yazılmış bir değişken adı örtük olarak sıfır olarak başlatılan sayısal bir değişken oluşturabilirsiniz.
