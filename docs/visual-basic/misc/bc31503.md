# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>&#39;&lt;TypeName&gt; &#39; bir sınıf olmadığından öznitelik olarak kullanılamaz
Bir sınıf özniteliği olarak değil bir türünü kullanmak için girişimde bulunuldu.  
  
 **Hata Kimliği:** BC31503  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Öğesinden türetilen sınıflar olarak özel özniteliklerini tanımlayan `System.Attribute`.
