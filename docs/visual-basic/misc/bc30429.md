# <a name="39end-sub39-must-be-preceded-by-a-matching-39sub39"></a>&#39;Bitiş alt&#39; eşleşen bir gelmelidir &#39;alt&#39;
Bir `End Sub` deyimi görünüyor, kodunuzda hiçbir eşleşen `Sub` , önceki yordamı tanımı.  
  
 **Hata Kimliği:** BC30429  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `End Sub` yedek ise deyimi.  
  
-   Eksik tedarik `Sub` eksikse yordamı.  
  
-   Taşıma `End Sub` uygun yerleştirmek için kod.
