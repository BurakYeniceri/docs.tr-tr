# <a name="compiler-error-cs2035"></a>Derleyici Hatası CS2035
Komut satırı sözdizimi hatası: eksik ':\<numarası >' 'compiler_option' seçeneği için  
  
 Bazı derleyici seçenekleri bir değer gerektirir.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS2035 oluşturur.  
  
```  
// CS2035.cs  
// compile with: /baseaddress  
// CS2035 expected  
```
