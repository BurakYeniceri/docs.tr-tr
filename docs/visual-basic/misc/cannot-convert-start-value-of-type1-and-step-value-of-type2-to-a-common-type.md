# <a name="cannot-convert-start-value-of-type-39lttype1gt39-and-step-value-of-type-39lttype2gt39-to-a-common-type"></a>Başlangıç değeri türü dönüştürülemiyor &#39; &lt;type1&gt; &#39; ve adım türü değeri &#39; &lt;type2&gt; &#39; ortak türü
Belirtilen `For...Next` denetim değerleri farklı türlere sahip ve ortak bir türe dönüştürülemediğinden bu nedenle geçerli değil.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Tüm değerleri ortak bir türe dönüştürülebilir bir veya daha fazla değer türünü değiştirin.
