# <a name="39ltmodifiergt39-is-not-valid-on-a-local-variable-declaration"></a>&#39;&lt;Değiştirici&gt; &#39; bir yerel değişken bildirimi geçerli değil
A `Dim` bir yordam ifadeye içeren bir geçersiz anahtar sözcüğü gibi `Overloads`.  
  
 **Hata Kimliği:** BC30247  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçersiz anahtar sözcüğü kaldırmak `Dim` deyimi.
