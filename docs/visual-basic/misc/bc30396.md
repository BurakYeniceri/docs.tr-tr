# <a name="39ltspecifiergt39-is-not-valid-on-an-enum-declaration"></a>&#39;&lt;belirleyici&gt; &#39; bir Enum bildiriminde geçerli değil
Geçerli olmayan bir belirleyici kullanılan bir `Enum` bildirimi.  
  
 **Hata Kimliği:** BC30396  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Belirleyici kaldırın.
