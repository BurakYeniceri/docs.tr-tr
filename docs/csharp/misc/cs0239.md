<span data-ttu-id="35786-103">'member': devralınan üye 'devralınan üye' korumalı olduğu için geçersiz kılınamaz</span><span class="sxs-lookup"><span data-stu-id="35786-103">'member' : cannot override inherited member 'inherited member' because it is sealed</span></span>  
  
 <span data-ttu-id="35786-104">Üye olamaz [geçersiz kılma](../../csharp/language-reference/keywords/override.md) bir [korumalı](../../csharp/language-reference/keywords/sealed.md) üye devralındı.</span><span class="sxs-lookup"><span data-stu-id="35786-104">A member cannot [override](../../csharp/language-reference/keywords/override.md) a [sealed](../../csharp/language-reference/keywords/sealed.md) inherited member.</span></span> <span data-ttu-id="35786-105">Daha fazla bilgi için bkz: [Checked ve Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="35786-105">For more information, see [Checked and Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="35786-106">Aşağıdaki örnek CS0239 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="35786-106">The following sample generates CS0239:</span></span>  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
