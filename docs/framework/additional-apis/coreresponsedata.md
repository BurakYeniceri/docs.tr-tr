# <a name="coreresponsedata-class"></a>CoreResponseData sınıfı

`CoreResponseData` Sınıfı, HTTP üstbilgileri ve yanıt gövdesi ayrıştırılırken temsil eder.

## <a name="syntax"></a>Sözdizimi
  
```csharp
internal class CoreResponseData
```

> [!WARNING]
> Bu API dahili kullanım içindir ve kodunuzda doğrudan kullanılmak üzere tasarlanmamıştır. Bunun yerine, kullanmanız bir <xref:System.Diagnostics.DiagnosticSource> ağ kod bağlanmanıza. Bkz: [DiagnosticSource Kullanıcı Kılavuzu](https://github.com/dotnet/corefx/blob/master/src/System.Diagnostics.DiagnosticSource/src/DiagnosticSourceUsersGuide.md).
> 
> Microsoft hiçbir koşulda bir üretim uygulamasında bu sınıf kullanımını desteklemez.
