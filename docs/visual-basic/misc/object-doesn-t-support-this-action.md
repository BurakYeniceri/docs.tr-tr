# <a name="object-doesnt-support-this-action-visual-basic"></a>Bu eylem (Visual Basic) nesnesi desteklemiyor
Bir yöntem veya bu nesne tarafından desteklenmeyen özellik başvurulan.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Nesnesi hakkında daha fazla bilgi için nesnenin belgelerini okuyun.  
  
2.  Özellikleri ve yöntemleri adını doğru yazdığınızdan emin olun.
