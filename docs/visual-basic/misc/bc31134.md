# <a name="39addhandler39-and-39removehandler39-method-parameters-cannot-be-declared-39byref39"></a>&#39;AddHandler&#39; ve &#39;RemoveHandler&#39; yöntem parametreleri bildirilemez &#39;ByRef&#39;
Parametreleri `AddHandler` ve `RemoveHandler` yöntemleri ile bildirilemez `ByRef` değiştiricisi.  
  
 **Hata Kimliği:** BC31134  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `ByRef` anahtar sözcüğü parametre.
