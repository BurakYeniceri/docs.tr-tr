# <a name="properties-declared-39readonly39-cannot-have-a-39set39"></a>Bildirilen özellikler &#39;ReadOnly&#39; olamaz bir &#39;ayarlayın&#39;
`Set` Yordamı bir özelliğin değerini yazar. `ReadOnly` Özellikler yazılamaz.  
  
 **Hata Kimliği:** BC30022  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `ReadOnly` anahtar sözcüğünün `Property` deyimi veya kaldırma `Set` özellik gövdesi yordamdan.
