# <a name="error-embedding-win32-manifest-ltmanifestgt"></a>Win32 bildirimi katıştırma hatası: &lt;bildirimi&gt;
Belirtilen dosya `/win32manifest` derleyici seçeneği olamaz katıştırılmış. Bu bildirim dosyası için yeterli güvenlik ayrıcalıkları sonucu olabilir.  
  
 **Hata Kimliği:** BC31191  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Visual Basic derleyici komutunu çalıştırarak kimlik Win32 bildirim dosyası erişimi olduğundan emin olun.
