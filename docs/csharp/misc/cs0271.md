# <a name="compiler-error-cs0271"></a><span data-ttu-id="8175f-102">Derleyici Hatası CS0271</span><span class="sxs-lookup"><span data-stu-id="8175f-102">Compiler Error CS0271</span></span>
<span data-ttu-id="8175f-103">Get erişimcisi erişilemediğinden, özelliği veya Dizin Oluşturucu 'özelliği/Dizin Oluşturucu' Bu bağlamda kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="8175f-103">The property or indexer 'property/indexer' cannot be used in this context because the get accessor is inaccessible</span></span>  
  
 <span data-ttu-id="8175f-104">Bu hata, bir erişilemez erişmeyi denediğinde oluşur `get` erişimcisi.</span><span class="sxs-lookup"><span data-stu-id="8175f-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="8175f-105">Bu hatayı gidermek için erişimci erişilebilirliğini artırmak veya arama konumunu değiştirebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="8175f-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="8175f-106">Daha fazla bilgi için bkz: [erişimci erişilebilirliğini](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) ve [özellikleri](../../csharp/programming-guide/classes-and-structs/properties.md).</span><span class="sxs-lookup"><span data-stu-id="8175f-106">For more information, see [Accessor Accessibility](../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../../csharp/programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="8175f-107">Aşağıdaki örnek CS0271 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8175f-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)   
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
