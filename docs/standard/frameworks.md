## <a name="how-to-specify-target-frameworks"></a><span data-ttu-id="3a543-190">Bir hedef çerçeveyi belirtme</span><span class="sxs-lookup"><span data-stu-id="3a543-190">How to specify target frameworks</span></span>

<span data-ttu-id="3a543-191">Bir hedef çerçeveyi, proje dosyasında belirtilir.</span><span class="sxs-lookup"><span data-stu-id="3a543-191">Target frameworks are specified in your project file.</span></span> <span data-ttu-id="3a543-192">Bir tek hedef framework belirtildiğinde kullanmak **TargetFramework** öğesi.</span><span class="sxs-lookup"><span data-stu-id="3a543-192">When a single target framework is specified, use the **TargetFramework** element.</span></span> <span data-ttu-id="3a543-193">Aşağıdaki konsol uygulama projesi dosyasını .NET Core 2.0 hedef gösterilmiştir:</span><span class="sxs-lookup"><span data-stu-id="3a543-193">The following console app project file demonstrates how to target .NET Core 2.0:</span></span>

```xml
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <OutputType>Exe</OutputType>
    <TargetFramework>netcoreapp2.0</TargetFramework>
  </PropertyGroup>

</Project>
```

<span data-ttu-id="3a543-194">Birden çok hedef çerçeveyi belirttiğinizde, her hedef çerçevesi için derlemeleri koşullu başvurabilir.</span><span class="sxs-lookup"><span data-stu-id="3a543-194">When you specify multiple target frameworks, you may conditionally reference assemblies for each target framework.</span></span> <span data-ttu-id="3a543-195">Kodunuzda, koşullu bu derlemeler karşı önişlemci sembolleriyle kullanarak derleyebilir *IF-then-else* mantığı.</span><span class="sxs-lookup"><span data-stu-id="3a543-195">In your code, you can conditionally compile against those assemblies by using preprocessor symbols with *if-then-else* logic.</span></span>

<span data-ttu-id="3a543-196">Aşağıdaki kitaplığı proje dosyası .NET standart API'leri, hedefler (`netstandard1.4`) ve API'ler .NET Framework'ün (`net40` ve `net45`).</span><span class="sxs-lookup"><span data-stu-id="3a543-196">The following library project file targets APIs of .NET Standard (`netstandard1.4`) and APIs of the .NET Framework (`net40` and `net45`).</span></span> <span data-ttu-id="3a543-197">Çoğul kullanmak **TargetFrameworks** birden çok hedef çerçeveyi sahip öğe.</span><span class="sxs-lookup"><span data-stu-id="3a543-197">Use the plural **TargetFrameworks** element with multiple target frameworks.</span></span> <span data-ttu-id="3a543-198">Not nasıl `Condition` öznitelikler, kitaplık için iki .NET Framework TFMs derlendiğinde uygulamaya özel paketleri içerir:</span><span class="sxs-lookup"><span data-stu-id="3a543-198">Note how the `Condition` attributes include implementation-specific packages when the library is compiled for the two .NET Framework TFMs:</span></span>

```xml
<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <TargetFrameworks>netstandard1.4;net40;net45</TargetFrameworks>
  </PropertyGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.0 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net40' ">
    <Reference Include="System.Net" />
  </ItemGroup>

  <!-- Conditionally obtain references for the .NET Framework 4.5 target -->
  <ItemGroup Condition=" '$(TargetFramework)' == 'net45' ">
    <Reference Include="System.Net.Http" />
    <Reference Include="System.Threading.Tasks" />
  </ItemGroup>

</Project>
```

<span data-ttu-id="3a543-199">Kitaplığı veya uygulama içinde her hedef çerçeve için derlemek için koşullu bir kod yazın:</span><span class="sxs-lookup"><span data-stu-id="3a543-199">Within your library or app, you write conditional code to compile for each target framework:</span></span>

```csharp
public class MyClass

    static void Main()
    {
#if NET40
        Console.WriteLine("Target framework: .NET Framework 4.0");
#elif NET45  
        Console.WriteLine("Target framework: .NET Framework 4.5");

        Console.WriteLine("Target framework: .NET Standard 1.4");
#endif
    }
}
```
