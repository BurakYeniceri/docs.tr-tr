# <a name="iclrhostbindingpolicymanager-interface"></a>ICLRHostBindingPolicyManager Arabirimi
Geçerli bağlama ilkesi değerlendirmek ve ilke değişiklikleri için belirtilen bir derleme iletişim kurmak ana bilgisayar için yöntemleri sağlar.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[EvaluatePolicy yöntemi](../../../../docs/framework/unmanaged-api/hosting/iclrhostbindingpolicymanager-evaluatepolicy-method.md)|Ana bilgisayar adına bağlama ilkeyi değerlendirir.|  
|[ModifyApplicationPolicy yöntemi](../../../../docs/framework/unmanaged-api/hosting/iclrhostbindingpolicymanager-modifyapplicationpolicy-method.md)|Bağlama ilkesi için belirtilen derlemeyi değiştirir ve ilke yeni bir sürümünü oluşturur.|
