<span data-ttu-id="293f9-103">Sabit veya using deyimi bildiriminde bir başlatıcı sağlamalısınız</span><span class="sxs-lookup"><span data-stu-id="293f9-103">You must provide an initializer in a fixed or using statement declaration</span></span>  
  
 <span data-ttu-id="293f9-104">Bildirme ve değişkende başlatma bir [deyimi sabit](../../csharp/language-reference/keywords/fixed-statement.md).</span><span class="sxs-lookup"><span data-stu-id="293f9-104">You must declare and initialize the variable in a [fixed statement](../../csharp/language-reference/keywords/fixed-statement.md).</span></span> <span data-ttu-id="293f9-105">Daha fazla bilgi için bkz: [güvenli olmayan kod ve işaretçiler](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="293f9-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="293f9-106">Aşağıdaki örnek CS0210 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="293f9-106">The following sample generates CS0210:</span></span>  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 <span data-ttu-id="293f9-107">Aşağıdaki örnek de çünkü CS0210 oluşturur [deyimiyle](../../csharp/language-reference/keywords/using-statement.md) hiçbir Başlatıcı sahiptir.</span><span class="sxs-lookup"><span data-stu-id="293f9-107">The following sample also generates CS0210 because the [using statement](../../csharp/language-reference/keywords/using-statement.md) has no initializer.</span></span>  
  
```csharp  
// CS0210b.cs  
  
using System.IO;  
class Test   
{  
   static void Main()   
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))   
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
