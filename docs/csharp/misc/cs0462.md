# <a name="compiler-error-cs0462"></a><span data-ttu-id="ba4df-102">Derleyici Hatası CS0462</span><span class="sxs-lookup"><span data-stu-id="ba4df-102">Compiler Error CS0462</span></span>
<span data-ttu-id="ba4df-103">Bunlar kılınamaması devralınan üyeleri 'Üye1' ve 'üye2' aynı imzaya türü 'type' sahip</span><span class="sxs-lookup"><span data-stu-id="ba4df-103">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="ba4df-104">Bu hata, genel türler başlanmasıyla ortaya çıkar.</span><span class="sxs-lookup"><span data-stu-id="ba4df-104">This error arises with the introduction of generics.</span></span> <span data-ttu-id="ba4df-105">Normalde, aynı imzayla sınıfında bir yöntem iki sürümü sahip olamaz.</span><span class="sxs-lookup"><span data-stu-id="ba4df-105">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="ba4df-106">Ancak genel türler ile belirli bir tür ile örneği varsa, başka bir yöntem yinelenen genel yöntem belirtebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="ba4df-106">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ba4df-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="ba4df-107">Example</span></span>  
 <span data-ttu-id="ba4df-108">Zaman `C<int>` örneği, iki sürümü yöntemi `F` aynı imzayla, bu nedenle geçersiz kılma sınıfında oluşturulan `D` geçersiz kılmanın uygulanacağı hangi bir karar veremez.</span><span class="sxs-lookup"><span data-stu-id="ba4df-108">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="ba4df-109">Aşağıdaki örnek CS0462 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="ba4df-109">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>   
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>   
{  
   public override void F(int t) {}   // CS0462  
}  
```
