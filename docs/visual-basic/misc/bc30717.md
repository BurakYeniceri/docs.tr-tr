# <a name="39try39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Deneyin&#39; ifadeleri geçerli değildir komut penceresi
`Try` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30717  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Try` kodda hata ayıklama gelen deyimleri.
