# <a name="possible-problem-detected-while-building-assembly-lterrorgt"></a>Derleme oluşturulurken olası sorun algılandı: &lt;hata&gt;
Visual Basic derleyici tarafından adlı ALink aracı bütünleştirilmiş kod oluşturma bir hata bildirir. Olası bir nedeni imzalanmamış bir derlemeyi başvuru yapma imzalı bir derlemedir.  
  
 Bu ileti bir uyarıdır. Derleyici derleme oluşturmak devam ediyor. Uyarıları gizleme veya uyarıları hata olarak davranma hakkında daha fazla bilgi için lütfen bkz [yapılandırma uyarılarını Visual Basic'te](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Hata Kimliği:** BC40009  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Tırnak işaretli hata iletisini inceleyin ve uygun eylemi gerçekleştirin.  
  
2.  Hata durumunda yinelenen yeniden görmek için programı derleyin.  
  
3.  Hata tekrar oluşursa, Visual Basic derleyici yeniden yükleyin.  
  
4.  Yeniden yüklenmesinden sonra hata devam ederse, koşullar hakkında bilgi toplayın ve Microsoft Ürün Destek Hizmetleri'ne bildirin.
