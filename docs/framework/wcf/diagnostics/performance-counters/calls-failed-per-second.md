# <a name="calls-failed-per-second"></a>Saniyede Başarısız Olan Çağrı
Sayaç adı: Saniyede başarısız olan çağrılar  
  
## <a name="description"></a>Açıklama  
 Bu ikinci bir işlemde işlenmemiş özel durumlardan sahip çağrı sayısı.  
  
 Bu sayaç, performans sayacı türü [PERF_COUNTER_COUNTER](http://go.microsoft.com/fwlink/?LinkID=94649), değeri, aşağıdaki formül kullanılarak hesaplanır.  
  
 (1 - N 0 N) / ((D 1 - D 0) / F)  
  
 Bu sayaç artırılır her, bu işlem içinde işlenmeyen bir özel yoktur.
