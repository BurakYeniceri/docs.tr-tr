# <a name="targetfilepath-specifies-an-existing-folder"></a>Varolan bir klasörü TargetFilePath belirtir
Parametre `TargetFilePath` varolan bir dizin belirtiyor, ancak yeni dizinin adını belirtmeniz gerekir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değişiklik `TargetFilePath` yeni yerine var olan, dizini belirtmek için.
