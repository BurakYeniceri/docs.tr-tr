 <span data-ttu-id="acb78-104">Bir [soyut](../../csharp/language-reference/keywords/abstract.md) özel öznitelik sınıfı bir özniteliği olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="acb78-104">An [abstract](../../csharp/language-reference/keywords/abstract.md) custom attribute class cannot be used as an attribute.</span></span>

{  
}
