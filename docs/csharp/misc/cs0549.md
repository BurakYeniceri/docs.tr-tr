 <span data-ttu-id="99a12-104">A [korumalı](../../csharp/language-reference/keywords/sealed.md)[sınıfı](../../csharp/language-reference/keywords/class.md) temel sınıf olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="99a12-104">A [sealed](../../csharp/language-reference/keywords/sealed.md)[class](../../csharp/language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="99a12-105">Bu nedenle, sanal bir yöntem korumalı sınıfında sahip gereksizdir.</span><span class="sxs-lookup"><span data-stu-id="99a12-105">Therefore, it is useless to have a virtual method in sealed class.</span></span>
