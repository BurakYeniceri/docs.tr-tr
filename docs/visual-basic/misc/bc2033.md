# <a name="error-embedding-win32-manifest-option-win32manifest-conflicts-with-nowin32manifest"></a>Win32 bildirimi katıştırma hatası: seçeneği/win32manifest/nowin32manifest ile çakışıyor
Visual Basic derleyici her ikisi de geçirildi bir `/win32manifest` derleyici seçeneği ve `/nowin32manifest` derleyici seçeneği. Bu seçenekler yalnızca biri için Visual Basic derleyici geçirilebilir.  
  
 **Hata Kimliği:** BC2033  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ya da kaldırma `/win32manifest` derleyici seçeneği veya `/nowin32manifest` derleyici seçeneği.
