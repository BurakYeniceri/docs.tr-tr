# <a name="xml-comment-tag-39lttaggt39-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>XML açıklama etiketi &#39; &lt;etiketi&gt;&#39; özdeş özniteliklere sahip birden çok kez aynı XML açıklama bloğunda görüntülenir
XML açıklama etiketi \<etiketi > özdeş özniteliklere sahip birden çok kez aynı XML açıklama bloğunda görüntülenir. XML açıklama yoksayılacak.  
  
 XML açıklama aynı öznitelikleri ile birden fazla etiket içerir.  
  
 **Hata Kimliği:** BC42305  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yinelenen etiket kaldırın.
