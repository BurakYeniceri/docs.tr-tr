# <a name="ltreturnsgt-visual-basic"></a>&lt;verir&gt; (Visual Basic)
Özellik veya işlev dönüş değerini belirtir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```xml  
<returns>description</returns>  
```  
  
#### <a name="parameters"></a>Parametreler  
 `description`  
 Dönüş değeri açıklaması.  
  
## <a name="remarks"></a>Açıklamalar  
 Kullanım `<returns>` dönüş değeri tanımlamak bir yöntem bildirimi açıklamasını etiketinde.  
  
 İle derleme [/doc](../../../visual-basic/reference/command-line-compiler/doc.md) bir dosyaya işlem belgesi açıklamaları için.
