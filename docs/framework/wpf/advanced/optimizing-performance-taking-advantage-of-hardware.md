## <a name="software-rendering-pipeline"></a><span data-ttu-id="5c3c1-110">Yazılım işleme ardışık düzeni</span><span class="sxs-lookup"><span data-stu-id="5c3c1-110">Software Rendering Pipeline</span></span>  
 <span data-ttu-id="5c3c1-111">[!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] Yazılım işleme ardışık düzeni olan tamamen CPU'ya bağlıdır.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-111">The [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] software rendering pipeline is entirely CPU bound.</span></span> [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]<span data-ttu-id="5c3c1-112"> bir en iyi duruma getirilmiş, tam özellikli yazılım tarayıcısını uygulamak için CPU içindeki SSE ve SSE2 yönerge yararlanır ayarlar.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-112"> takes advantage of the SSE and SSE2 instruction sets in the CPU to implement an optimized, fully-featured software rasterizer.</span></span> <span data-ttu-id="5c3c1-113">Geri dönüş yazılım için uygulama işlevselliği kullanılarak donanım işleme ardışık düzeni işlenemiyor. istediğiniz zaman sorunsuz olarak gerçekleştirilir.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-113">Fallback to software is seamless any time application functionality cannot be rendered using the hardware rendering pipeline.</span></span>

### <a name="graphics-rendering-tiers"></a><span data-ttu-id="5c3c1-118">Grafik İşleme Katmanları</span><span class="sxs-lookup"><span data-stu-id="5c3c1-118">Graphics Rendering Tiers</span></span>  
 <span data-ttu-id="5c3c1-119">Uygulamanızın üzerinde çalışacağı donanım yapılandırmasını tahmin etmek oldukça zor olabilir.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-119">It may be very difficult to predict the hardware configuration that your application will be running on.</span></span> <span data-ttu-id="5c3c1-120">Ancak, böylece her farklı donanım yapılandırması tam anlamıyla yararlanabilmek sorunsuz bir şekilde özellikleri farklı donanım üzerinde çalışırken geçiş yapmak, uygulamanızın izin veren bir tasarım düşünmek isteyebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-120">However, you might want to consider a design that allows your application to seamlessly switch features when running on different hardware, so that it can take full advantage of each different hardware configuration.</span></span>  
  
 <span data-ttu-id="5c3c1-121">Bunu başarmak için [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] zamanında sistemin grafik yeteneğini belirlemek için işlevsellik sağlar.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-121">To achieve this, [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] provides functionality to determine the graphics capability of a system at runtime.</span></span> <span data-ttu-id="5c3c1-122">Grafik yeteneği, video kartı üç işleme yeteneği katmanından biri olarak sınıflandırarak belirlenir.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-122">Graphics capability is determined by categorizing the video card as one of three rendering capability tiers.</span></span> [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]<span data-ttu-id="5c3c1-123"> kullanıma sunan bir [!INCLUDE[TLA#tla_api](../../../../includes/tlasharptla-api-md.md)] işleme yeteneği katmanını sorgulamak için bir uygulama sağlar.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-123"> exposes an [!INCLUDE[TLA#tla_api](../../../../includes/tlasharptla-api-md.md)] that allows an application to query the rendering capability tier.</span></span> <span data-ttu-id="5c3c1-124">Uygulamanız, daha sonra donanım tarafından desteklenen işleme katmanına bağlı olarak çalışma zamanında farklı kod yollarını alabilir.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-124">Your application can then take different code paths at run time depending on the rendering tier supported by the hardware.</span></span>

-   <span data-ttu-id="5c3c1-126">**Video RAM** birleştirme grafikler için kullanılan arabellek sayısı ve boyutu grafik donanımı üzerindeki video bellek miktarını belirler.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-126">**Video RAM** The amount of video memory on the graphics hardware determines the size and number of buffers that can be used for compositing graphics.</span></span>  
  
-   <span data-ttu-id="5c3c1-127">**Piksel gölgelendirici** piksel gölgelendirici grafik piksel başına temelinde etkileri hesaplar işlevi değil.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-127">**Pixel Shader** A pixel shader is a graphics processing function that calculates effects on a per-pixel basis.</span></span> <span data-ttu-id="5c3c1-128">Görüntülenen grafiklerin çözümleme bağlı olarak, her görüntü çerçevesi için işlenmesi gereken birkaç milyon piksel olabilir.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-128">Depending on the resolution of the displayed graphics, there could be several million pixels that need to be processed for each display frame.</span></span>  
  
-   <span data-ttu-id="5c3c1-129">**Köşe gölgelendirici** köşe gölgelendirici grafik nesnesinin köşe veriler üzerinde matematiksel işlemler gerçekleştiren işlevi değil.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-129">**Vertex Shader** A vertex shader is a graphics processing function that performs mathematical operations on the vertex data of the object.</span></span>  
  
-   <span data-ttu-id="5c3c1-130">**Çoklu doku desteği** 3B grafik nesnesi üzerinde yakınsama işlemi sırasında iki veya daha fazla farklı dokuları uygulama özelliği için çoklu doku desteği başvuruyor.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-130">**Multitexture Support** Multitexture support refers to the ability to apply two or more distinct textures during a blending operation on a 3D graphics object.</span></span> <span data-ttu-id="5c3c1-131">Çoklu doku desteğinin derecesi grafik donanımı üzerindeki çoklu doku birimlerinin sayısı tarafından belirlenir.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-131">The degree of multitexture support is determined by the number of multitexture units on the graphics hardware.</span></span>  
  
 <span data-ttu-id="5c3c1-132">Piksel gölgelendirici, köşe gölgelendirici ve çoklu doku özellikleri belirli tanımlamak için kullanılan [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] hangi sırayla, farklı işleme katmanlarını tanımlamak için kullanılan sürüm düzeylerini [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].</span><span class="sxs-lookup"><span data-stu-id="5c3c1-132">The pixel shader, vertex shader, and multitexture features are used to define specific [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version levels, which, in turn, are used to define the different rendering tiers in [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].</span></span>  
  
 <span data-ttu-id="5c3c1-133">Grafik donanım özellikleri işleme yeteneğini belirlemek bir [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] uygulama.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-133">The features of the graphics hardware determine the rendering capability of a [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] application.</span></span> <span data-ttu-id="5c3c1-134">[!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] Sistem üç işleme katmanı tanımlar:</span><span class="sxs-lookup"><span data-stu-id="5c3c1-134">The [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] system defines three rendering tiers:</span></span>  
  
-   <span data-ttu-id="5c3c1-135">**Katman 0 işleme** hiçbir grafik donanım hızlandırmasını.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-135">**Rendering Tier 0** No graphics hardware acceleration.</span></span> <span data-ttu-id="5c3c1-136">[!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] Sürüm düzeyi sürüm 7.0 azdır.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-136">The [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version level is less than version 7.0.</span></span>  
  
-   <span data-ttu-id="5c3c1-137">**Katman 1 işleme** kısmi grafik donanım hızlandırmasını.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-137">**Rendering Tier 1** Partial graphics hardware acceleration.</span></span> <span data-ttu-id="5c3c1-138">[!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] Sürüm düzeyi 7.0 sürümüne eşit veya daha büyük ve **küçük** 9.0 sürümünden.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-138">The [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version level is greater than or equal to version 7.0, and **lesser** than version 9.0.</span></span>  
  
-   <span data-ttu-id="5c3c1-139">**Katman 2 işleme** çoğu grafik özelliği grafik donanım hızlandırmasını kullanır.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-139">**Rendering Tier 2** Most graphics features use graphics hardware acceleration.</span></span> <span data-ttu-id="5c3c1-140">[!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] Sürüm düzeyi 9.0 sürümüne eşit veya daha büyük.</span><span class="sxs-lookup"><span data-stu-id="5c3c1-140">The [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version level is greater than or equal to version 9.0.</span></span>  
  
 <span data-ttu-id="5c3c1-141">Daha fazla bilgi için [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] işleme katmanları bkz [grafik işleme katmanları](../../../../docs/framework/wpf/advanced/graphics-rendering-tiers.md).</span><span class="sxs-lookup"><span data-stu-id="5c3c1-141">For more information on [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] rendering tiers, see [Graphics Rendering Tiers](../../../../docs/framework/wpf/advanced/graphics-rendering-tiers.md).</span></span>
