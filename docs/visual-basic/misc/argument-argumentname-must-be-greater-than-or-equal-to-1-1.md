# <a name="argument-39ltargumentnamegt39-must-be-greater-than-or-equal-to--1"></a>Bağımsız değişken &#39; &lt;argumentname&gt;&#39; -1 veya daha büyük olmalıdır
Bir bağımsız değişken, -1'den küçük değerdir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bağımsız değişkeninin değeri geçerli aralıkta olduğundan emin olun.  
  
2.  Bağımsız değişkeni bir hesaplama sağlanırsa, hesaplama işlemleri değerlerini denetleyin.
