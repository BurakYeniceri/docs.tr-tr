# <a name="compiler-warning-level-1-cs3001"></a>Derleyici Uyarısı (düzey 1) CS3001
Bağımsız değişken türü 'type' CLS uyumlu değil  
  
 A [ortak](../../csharp/language-reference/keywords/public.md), [korumalı](../../csharp/language-reference/keywords/protected.md), veya `protected internal` yöntemi türü ortak dil belirtimi (CLS) ile uyumlu olan bir parametre kabul etmelisiniz. CLS uyumluluğu hakkında daha fazla bilgi için bkz: [CLS uyumlu kod yazma](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3) ve [dil bağımsızlığı ve dilden bağımsız bileşenler](../../../docs/standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS3001 oluşturur:  
  
```csharp  
// CS3001.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public void bad(ushort i)   // CS3001  
    {  
    }  
  
    private void OK(ushort i)   // OK, private method  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
