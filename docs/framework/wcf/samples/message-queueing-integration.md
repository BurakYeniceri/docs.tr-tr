# <a name="message-queueing-integration"></a>İleti Kuyruğa Alma Tümleştirme
Bu bölüm, Message Queuing ve Windows Communication Foundation (WCF) arasındaki tümleştirmeyi göstermek örnekleri içerir.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [Message Queuing’den Windows Communication Foundation'a](../../../../docs/framework/wcf/samples/message-queuing-to-wcf.md)  
 Message Queuing (MSMQ) uygulama için bir WCF hizmeti bir MSMQ iletisi nasıl gönderebilirsiniz gösterir.  
  
 [Özel Demux](../../../../docs/framework/wcf/samples/custom-demux.md)  
 Böylece kullanan WCF hizmetleri nasıl MSMQ ileti üstbilgilerini farklı hizmet işlemleri eşlenebilir gösteren <xref:System.ServiceModel.MsmqIntegration.MsmqIntegrationBinding> bir hizmet işlemi için kullanılacak sınırlı değildir.  
  
 [Windows Communication Foundation'dan Message Queuing’e](../../../../docs/framework/wcf/samples/wcf-to-message-queuing.md)  
 WCF uygulaması bir Message Queuing (MSMQ) uygulamaya ileti nasıl gönderebilirsiniz gösterir.  
  
 [İleti Bağıntısı](../../../../docs/framework/wcf/samples/message-correlation.md)  
 Message Queuing (MSMQ) uygulama için bir WCF hizmeti bir MSMQ iletisi nasıl gönderebilir ve nasıl iletileri bir istek/yanıt senaryosunda göndereni ve alıcısı uygulamalar arasında ilişkili olabilir gösterir.
