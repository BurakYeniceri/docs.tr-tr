# <a name="peer-channel"></a>Eş Kanal
Bu konu, Windows Communication Foundation (WCF) eş kanal tarafından oluşturulan tüm özel durumları listeler.  
  
## <a name="exception-list"></a>Özel durum listesi  
  
|Kaynak kodu|Kaynak dizesi|  
|-------------------|---------------------|  
|InsufficientCredentials|Belirtilen kimlik bilgileri istenen işlemi gerçekleştirmek yeterli değil. Lütfen belirtilen işlem için geçerli bir değer belirtin|  
|InvalidResolverMode|Belirtilen PeerResolverMode değeri geçersiz. PeerResolveMode.Auto, varsayılan veya PNRP belirtin.|  
|PeerNodeAborted|PeerNode sona erdi olduğundan açılamıyor.|  
|PeerNullRegistrationInfo|Kayıt bilgileri null olamaz. Kayıt işlemi geçerli bir RegistrationInfo nesnesi ile başlatıldığından emin olun.|  
|PeerNullResolveInfo|Bilgi olamaz gidermek `null`. Çözümleme işlemi geçerli bir ResolveInfo nesnesi ile başlatıldığından emin olun.|
