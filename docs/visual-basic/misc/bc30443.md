# <a name="39get39-is-already-declared"></a>&#39; Al &#39; zaten bildirildi
A `Get` deyimi bir kod bloğu içinde birden çok kez görünüyor. `Get` Deyimi bildiren bir `Get` özellik yordamı bir özelliğin değerini almak için kullanılır.  
  
 **Hata Kimliği:** BC30443  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yedek kaldırmak `Get` deyimi.
