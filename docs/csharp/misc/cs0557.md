# <a name="compiler-error-cs0557"></a><span data-ttu-id="e9ee1-102">Derleyici Hatası CS0557</span><span class="sxs-lookup"><span data-stu-id="e9ee1-102">Compiler Error CS0557</span></span>
<span data-ttu-id="e9ee1-103">Kullanıcı tanımlı dönüştürme türü 'sınıfı' Çoğalt</span><span class="sxs-lookup"><span data-stu-id="e9ee1-103">Duplicate user-defined conversion in type 'class'</span></span>  
  
 <span data-ttu-id="e9ee1-104">Yinelenen dönüştürme yordamları bir sınıf içinde izin verilmiyor.</span><span class="sxs-lookup"><span data-stu-id="e9ee1-104">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="e9ee1-105">Aşağıdaki örnek CS0557 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e9ee1-105">The following example generates CS0557:</span></span>  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
