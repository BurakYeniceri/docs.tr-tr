# <a name="icordebugstepperenum-interface1"></a>Icordebugstepperenum Interface1
Icordebugenum yöntemlerini uygular ve ICorDebugStepper diziler numaralandırır.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[Next yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugstepperenum-next-method.md)|Belirtilen sayıda alır `ICorDebugStepper` geçerli konumdan başlayarak numaralandırması örneklerden.|
