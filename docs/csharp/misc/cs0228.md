# <a name="compiler-error-cs0228"></a>Derleyici Hatası CS0228
'type', 'member' tanımı içermiyor veya erişilebilir değil  
  
 Sistem sürümü yerine **System.Hashtable**, **System.String**, veya **System.Array**, ve değiştirme içermediği `member`.  
  
 Aksi takdirde, onarmak veya Visual Studio yeniden yüklemeniz gerekebilir.
