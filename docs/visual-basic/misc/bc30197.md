# <a name="39as39-expected"></a>&#39; Olarak &#39; Beklenen
Bir bildirim değil olarak sağladığınız oluşur bir `As` yan tümcesi bir veya daha fazla bildirilen öğe.  
  
 **Hata Kimliği:** BC30197  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Furnish `As` gerektiği gibi yan tümcesi veya bırakma `Option Strict Off`.
