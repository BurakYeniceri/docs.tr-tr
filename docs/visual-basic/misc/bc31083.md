# <a name="methods-in-a-module-cannot-implement-interface-members"></a>Bir modüldeki yöntemleri arabirim üyeleri uygulayamaz
Arabirim üyeleri sınıfları veya yapıları tarafından uygulanmalıdır.  
  
 **Hata Kimliği:** BC31083  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Arabirim üyeleri sınıfları veya yapıları uygulayın.
