# <a name="parameter-39ltparameternamegt39-in-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39-already-has-a-matching-argument"></a>Parametre &#39; &lt;parametername&gt; &#39; uzantı yöntemindeki &#39; &lt;methodname&gt; &#39; tanımlanan &#39; &lt;typename&gt; &#39; zaten eşleşen bir bağımsız değişkenini içeriyor
Bir bağımsız değişken adıyla aynı yordam çağrısında birden çok kez geçirilir.  
  
 **Hata Kimliği:** BC36584  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bağımsız değişken adları birini yanlış yazılmış olup olmadığını denetleyin.  
  
-   Yinelenen bağımsız değişken bağımsız değişken listesinden kaldırır.
