# <a name="debugging-on-the-client"></a>İstemcide Hata Ayıklama
İstemci uygulamaları yazmak kullanıcıların kolaylaştırmak için [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] hizmeti ekleyebilirsiniz [ \<serviceDebug >](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) hizmet hizmetinizin yapılandırma dosyasına davranışı. Bu davranış, Yardım sayfalarına yayımlamak ve istemciye döndürülen SOAP hatalarının ayrıntıları yönetilen özel durum bilgileri döndürmek için kullanılabilir.
