# <a name="goto-to-label-39ltlabelnamegt39-cannot-be-used-within-lambda-expressions"></a>GoTo etiket &#39; &lt;etiketadı&gt;&#39; içinde lambda ifadeleri kullanılamaz
Lambda ifadesi gövdesi hesaplar ve tek bir değer döndüren bir ifade olmalıdır. Hiçbir ifadeleri de dahil olmak üzere izin verilen `GoTo` deyimleri.  
  
 **Hata Kimliği:** BC36597
