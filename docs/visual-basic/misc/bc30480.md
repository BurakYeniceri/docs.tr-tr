# <a name="shared-39sub-new39-cannot-be-declared-39ltspecifiergt39"></a>Paylaşılan &#39;Sub New&#39; bildirilemez &#39; &lt;belirticisi&gt;&#39;
Bir belirleyici kullanma girişiminde bulundunuz bir `Shared Sub New` bildirimi geçerli değil.  
  
 **Hata Kimliği:** BC30480  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirleyici kaldırın.
