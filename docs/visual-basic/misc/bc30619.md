# <a name="statement-cannot-appear-within-an-enum-body"></a>Enum gövdesi içinde deyimi yer alamaz
Deyimi Enum gövdesi içinde gerçekleşemez. Kabul Enum sonu.  
  
 Beklenmeyen dil yapısı karşılaşıldı. Varsayılır bir `End Enum` yapı eksik olduğundan ve bu noktadan sonra herhangi bir kaynak kod dışında numaralandırması.  
  
 **Hata Kimliği:** BC30619  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Numaralandırma içinde kullanılan kod söz dizimini doğrulayın.  
  
2.  Arabirim tanımı ile biten emin olun bir `End Enum` oluşturun.
