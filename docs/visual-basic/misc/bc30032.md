# <a name="events-cannot-have-a-return-type"></a>Olayları bir dönüş türüne sahip olamaz
Olaylar bağımsız değişkenleri kabul edebilir, ancak doğrudan değerler döndüremez.  
  
 **Hata Kimliği:** BC30032  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Dönüş türünden kaldırmak `Event` deyimi.
