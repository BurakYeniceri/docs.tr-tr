# <a name="web-methods-cannot-be-evaluated-in-debug-windows"></a>Hata ayıklama pencerelerinde Web yöntemleri değerlendirilemez
XML Web hizmeti yöntemleri hata ayıklama penceresi içinde sınanamıyor.  
  
 **Hata Kimliği:** BC30732  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML Web hizmeti yöntemleri Kod Düzenleyicisi'nden sınayın.
