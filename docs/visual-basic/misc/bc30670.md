# <a name="39redim39-statements-require-a-parenthesized-list-of-the-new-bounds-of-each-dimension-of-the-array"></a>&#39;ReDim&#39; deyimleri gerektiren her boyut dizinin yeni sınırları parantez içine alınmış listesi
Bir parçası olarak bir dizi yeni boyutunu belirtmeli bir `ReDim` deyimi.  
  
 **Hata Kimliği:** BC30670  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Dizinin her derece boyutunu sağlayın; Örneğin:  
  
    ```  
    ReDim arr(5, 6)  
    ```
