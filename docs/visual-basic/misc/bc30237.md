# <a name="parameter-already-declared-with-name-39ltargumentnamegt39"></a>Parametre adı &#39;bildirimi zaten; &lt;argumentname&gt;&#39;
Aynı ada sahip iki bağımsız değişken bir yordam bildirimi tanımlar. Bir yordamı bildirimi içindeki tüm bağımsız değişken adları benzersiz olmalıdır.  
  
 **Hata Kimliği:** BC30237  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bağımsız değişkenlerden biri adını değiştirin.
