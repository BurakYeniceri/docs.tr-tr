# <a name="compiler-warning-level-4-cs1610"></a>Derleyici Uyarısı (düzey 4) CS1610
Varsayılan Win32 kaynağı--kaynak için kullanılan'file ' geçici dosya silinemiyor  
  
 Kullanırken [/win32res](../../../csharp/language-reference/compiler-options/win32res-compiler-option.md) derleyici seçeneği ve ne zaman, **% TEMP %** dizin silme izni yok, bu uyarı derleyici oluşturulduğu geçici bir dosya silinemedi gösterir.  
  
 Okuma/yazma/silme izniniz olduğundan emin olun **% TEMP %** dizin.  
  
 Gerekirse, bu dosyaları el ile silebilirsiniz ve C# hiçbir zarar veya programlarınızı hiçbirini olacaktır.
