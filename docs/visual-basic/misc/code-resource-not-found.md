# <a name="code-resource-not-found"></a>Kod kaynak bulunamadı
Kod kaynak bir yordam için bir çağrı yapıldı, ancak kod kaynağı bulunamadı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaynağın kullanılabilir olduğunu ve düzgün şekilde başvurulan olup olmadığını denetleyin.
