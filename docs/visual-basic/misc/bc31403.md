# <a name="imports-alias-39ltnamegt39-conflicts-with-39ltname2gt39-declared-in-the-root-namespace"></a>İçeri aktarmalar diğer &#39; &lt;adı&gt; &#39; çakışıyor &#39; &lt;ad2&gt; &#39; kök ad alanında bildirilen
Kullanılan diğer ad `Imports` deyimi, kök ad alanında bildirilen başka bir ad çakışıyor.  
  
 **Hata Kimliği:** BC31403  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tarafından kullanılan diğer adı için benzersiz bir ad seçin `Imports` deyimi.
