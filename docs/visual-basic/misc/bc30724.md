# <a name="39error39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Hata &#39; deyimleri Komut penceresi içinde geçerli değildir
`Error`deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30724  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Error` kodda hata ayıklama gelen deyimleri.
