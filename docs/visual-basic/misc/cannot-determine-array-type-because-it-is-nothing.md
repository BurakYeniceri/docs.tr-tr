# <a name="cannot-determine-array-type-because-it-is-nothing"></a>Hiçbir şey olduğundan dizi türü belirlenemiyor
Değerine ayarlandığından, dizi türü belirlenemiyor `Nothing`.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Dışında bir değer sağlayın `Nothing` dizi.
