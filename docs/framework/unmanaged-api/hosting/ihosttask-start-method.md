# <a name="ihosttaskstart-method"></a>IHostTask::Start Yöntemi
Taşıma geçerli tarafından temsil edilen görevi konak istekleri [Ihosttask](../../../../docs/framework/unmanaged-api/hosting/ihosttask-interface.md) askıya alınmış bir örnekten, kod yürütülebilir canlı bir duruma.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT Start ();  
```  
  
## <a name="return-value"></a>Dönüş Değeri  
  
|HRESULT|Açıklama|  
|-------------|-----------------|  
|S_OK|Başlangıç başarıyla döndürüldü.|  
|E_FAIL|Bilinmeyen yıkıcı bir hata oluştu. Ortak dil çalışma zamanı (CLR), artık bir yöntem E_FAIL döndüğünde, işlemi içinde kullanılamaz. Yöntemleri barındırma sonraki çağrılar HOST_E_CLRNOTAVAILABLE döndürür.|  
  
## <a name="remarks"></a>Açıklamalar  
 `Start` her zaman bir HRESULT değerini S_OK, dışında yıkıcı bir hata oluştuğu durumlarda döndürür.
