# <a name="3939-is-not-allowed"></a>&#39;: &#39; izin verilmiyor
':' verilmez. XML nitelenmiş adlar bu bağlamda kullanılamaz.  
  
 XML değişmez değer adıyla bir örneğin, bir hedef işleme yönergesi değişmez değeri olarak geçersiz bir konumda belirtilmedi.  
  
 **Hata Kimliği:** BC31186  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tam adı ile değişmez değer XML kaldırın veya geçerli bir konuma taşıyın.  
  
     Karakter olduğu konusunda XML adlarında geçerli daha fazla bilgi için bkz [Genişletilebilir İşaretleme Dili (XML) 1.0](http://go.microsoft.com/fwlink/?LinkId=73927).
