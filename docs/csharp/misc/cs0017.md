<span data-ttu-id="e3627-103">Program 'çıktı dosyası adını' tanımlı birden fazla giriş noktası yok.</span><span class="sxs-lookup"><span data-stu-id="e3627-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="e3627-104">Giriş noktası içeren türünü belirtmek için/Main ile derleyin.</span><span class="sxs-lookup"><span data-stu-id="e3627-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="e3627-105">Bir programı yalnızca bir bulunabilir [ana](../../csharp/programming-guide/main-and-command-args/index.md) yöntemi.</span><span class="sxs-lookup"><span data-stu-id="e3627-105">A program can only have one [Main](../../csharp/programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="e3627-106">Bu hatayı gidermek için kodunuzda biri dışında tüm ana yöntemleri silebilirsiniz veya kullanabilirsiniz [/ana](../../csharp/language-reference/compiler-options/main-compiler-option.md) derleyici seçeneği hangi Main yöntemini kullanmak istediğinizi belirtin.</span><span class="sxs-lookup"><span data-stu-id="e3627-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="e3627-107">Aşağıdaki örnek CS0017 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e3627-107">The following sample generates CS0017:</span></span>  
  
```csharp  
// CS0017.cs  
// compile with: /target:exe  
public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
