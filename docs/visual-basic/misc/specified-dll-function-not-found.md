# <a name="specified-dll-function-not-found"></a>Belirtilen DLL işlev bulunamadı
Bir kullanıcı Kitaplık Başvurusu'nda dinamik bağlantı kitaplığı (DLL) bulundu, ancak belirtilen DLL işlevi DLL bulunamadı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Geçerli bir sıra işlevi bildiriminde belirtin.  
  
2.  DLL adını ve diğer doğru olduğundan emin olun.
