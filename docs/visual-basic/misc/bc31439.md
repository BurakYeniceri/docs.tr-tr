# <a name="partial-method-39ltmethodnamegt39-cannot-use-the-39implements39-keyword"></a>Kısmi yöntemi &#39; &lt;methodname&gt; &#39; kullanamazsınız &#39;uygulayan&#39; anahtar sözcüğü
Kısmi yöntemi tanımı bir arabirim veya arabirim üyesini uygulayamaz.  
  
 **Hata Kimliği:** BC31439  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kaldırma `Implements` kısmi yöntemi tanımından deyimi.
