# <a name="alias-39ltnamespacenamegt39-is-already-declared"></a>Diğer ad &#39; &lt;namespacename&gt;&#39; zaten bildirildi
Kullanma girişiminde bulundunuz `Imports` zaten tanımlanmış bir ad alanı içe aktarmak için.  
  
 **Hata Kimliği:** BC30572  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İçe aktarmaya çalıştığınız ad alanı denetleyin.
