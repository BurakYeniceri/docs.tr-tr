 <span data-ttu-id="b3791-104">`New` Kısıtlaması anlamına gelir, tür parametresi için sağlanan tür bağımsız değişkeni nesneleri oluşturduğu koduna erişilebilir parametresiz bir kurucusu kullanıma gerekir.</span><span class="sxs-lookup"><span data-stu-id="b3791-104">The `New` constraint means that the type argument supplied to that type parameter must expose a parameterless constructor accessible to the code that creates objects from it.</span></span> <span data-ttu-id="b3791-105">Tüm değer türleri parametresiz oluşturucular sahip, ancak tüm başvuru türleri yapın.</span><span class="sxs-lookup"><span data-stu-id="b3791-105">All value types have parameterless constructors, but not all reference types do.</span></span> <span data-ttu-id="b3791-106">Bu nedenle `Structure` kısıtlamasına `New` kısıtlaması, ancak `Class` kısıtlaması ya da bir sınıf veya arabirim adı içermiyor.</span><span class="sxs-lookup"><span data-stu-id="b3791-106">Therefore the `Structure` constraint satisfies the `New` constraint, but the `Class` constraint, or a class or interface name, does not.</span></span>
