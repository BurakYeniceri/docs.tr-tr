# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-base-class-39ltclassnamegt39"></a>Modül &#39;için gereken başvuru; &lt;modulename&gt;&#39; temel sınıfı &#39; içeren&lt; ClassName&gt;&#39;
Modül için gereken başvuru '\<modulename >' temel sınıfını içeren\<classname >'. Projenize ekleyin.  
  
 Sınıfı, doğrudan projenizde başvurulmuyor bir modüldeki tanımlanır. [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] Derleyici sınıfı birden fazla modülde tanımlanmış durumda Karışıklığı önlemek için bir başvuru gerektirir.  
  
 **Hata Kimliği:** BC30008  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Başvurulmayan modül adı, proje başvuruları içerir.
