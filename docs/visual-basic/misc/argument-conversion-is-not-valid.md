# <a name="argument-39conversion39-is-not-valid"></a>Bağımsız değişken &#39; Dönüştürme &#39; geçerli değil
İçin bir değer `VbStrConv` numaralandırma için geçerli değil, büyük olasılıkla, küçük sıfır. Numaralandırma değerleri değerinden büyük veya sıfıra eşit olmalıdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Değeri sıfırdan büyük veya sıfıra eşit olduğundan emin olun.
