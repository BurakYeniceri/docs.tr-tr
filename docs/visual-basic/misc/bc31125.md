# <a name="39end-removehandler39-must-be-preceded-by-a-matching-39removehandler39-declaration"></a>&#39;End RemoveHandler&#39; eşleşen bir gelmelidir &#39;RemoveHandler&#39; bildirimi
Bir `End RemoveHandler` deyimi oluştu karşılık gelen olmadan `RemoveHandler` deyimi. `End RemoveHandler` karşılık gelen tarafından gelmelidir `RemoveHandler` deyimi.  
  
 **Hata Kimliği:** BC31125  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Denetleyin önceki `RemoveHandler` deyimi geçerli olduğundan ve doğru yazılmış.
