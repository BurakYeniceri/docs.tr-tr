# <a name="cannot-convert-start-value-of-type-39lttype1gt39-and-step-value-of-type-39lttype2gt39-to-a-common-type"></a>Başlangıç değeri türü &#39;dönüştüremezsiniz; &lt;type1&gt;&#39; ve adım değeri türü &#39;&lt; type2&gt;&#39; ortak bir türü
Belirtilen `For...Next` denetim değerleri farklı olan ve ortak türüne dönüştürülemez bu nedenle geçerli değildir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Tüm değerler için ortak bir türü dönüştürülebilir şekilde bir veya daha fazla değeri türünü değiştirin.
