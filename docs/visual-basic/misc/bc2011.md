# <a name="unable-to-open-response-file-39ltfilenamegt39"></a>Yanıt dosyası açılamıyor &#39; &lt;dosya adı&gt;&#39;
Erişmeye çalıştığınız dosya kullanılamıyor.  
  
 **Hata Kimliği:** BC2011  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ağ bağlantınızın hala etkin olduğundan emin olun.  
  
-   Dosya üzerindeki izinlerinizi denetleyin.
