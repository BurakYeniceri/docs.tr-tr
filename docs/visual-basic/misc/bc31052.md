# <a name="modules-cannot-be-declared-39ltspecifiergt39"></a>Modülleri bildirilemez &#39; &lt;belirticisi&gt;&#39;
Bir modül geçersiz erişim değiştiricisi ile bildirildi.  
  
 **Hata Kimliği:** BC31052  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Erişim değiştiricisi geçerli erişim değiştiricisi ile aşağıdaki gibi değiştirin `Public` veya `Friend`.
