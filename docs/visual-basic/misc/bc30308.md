# <a name="39ltmethod1gt39-cannot-override-39ltmethod2gt39-because-they-differ-by-optional-parameters"></a>&#39;&lt;method1&gt; &#39; geçersiz kılamaz &#39; &lt;method2&gt; &#39; isteğe bağlı parametreleri farklı olduğundan
Birinciden farklı başka bir yöntemle yöntemini geçersiz kılın denediniz kendi imzaları farklı olduğu anlamına gelir, isteğe bağlı parametrelerinin değerleri. Bir tür aynı ad ve imza yöntemiyle bildirme ve bildirimiyle işaretleme devralınan geçersiz kılınabilir bir yöntemi geçersiz kılabilir `Overrides` değiştiricisi.  
  
 **Hata Kimliği:** BC30308  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İki yöntem aynı imzaya sahip olduğundan emin olun.
