# <a name="compiler-error-cs1946"></a><span data-ttu-id="ff04c-102">Derleyici Hatası CS1946</span><span class="sxs-lookup"><span data-stu-id="ff04c-102">Compiler Error CS1946</span></span>

<span data-ttu-id="ff04c-103">Anonim yöntem ifadesi ifade ağacına dönüştürülemiyor.</span><span class="sxs-lookup"><span data-stu-id="ff04c-103">An anonymous method expression cannot be converted to an expression tree.</span></span>  
  
<span data-ttu-id="ff04c-104">Anonim bir yöntem deyimleri kümesini temsil eder, ancak bir ifade ağacı bir deyimi içermemelidir.</span><span class="sxs-lookup"><span data-stu-id="ff04c-104">An anonymous method represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="ff04c-105">Bu nedenle tarafından bir ifade ağacı anonim bir yöntem gösterilemez.</span><span class="sxs-lookup"><span data-stu-id="ff04c-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="ff04c-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="ff04c-106">To correct this error</span></span>

1. <span data-ttu-id="ff04c-107">Anonim yöntem, lambda ifadesi olarak değiştirin.</span><span class="sxs-lookup"><span data-stu-id="ff04c-107">Change the anonymous method to a lambda expression.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ff04c-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="ff04c-108">Example</span></span>

<span data-ttu-id="ff04c-109">Aşağıdaki örnek, CS1946 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ff04c-109">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
