## <a name="example"></a><span data-ttu-id="ce552-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="ce552-106">Example</span></span>  
 <span data-ttu-id="ce552-107">Aşağıdaki örnek CS0535 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="ce552-107">The following sample generates CS0535.</span></span>  
  
```  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
