# <a name="compiler-warning-level-1-cs0420"></a><span data-ttu-id="a9122-102">Derleyici Uyarısı (düzey 1) CS0420</span><span class="sxs-lookup"><span data-stu-id="a9122-102">Compiler Warning (level 1) CS0420</span></span>
<span data-ttu-id="a9122-103">'tanımlayıcısı': geçici bir alana başvuru geçici kabul edilmez</span><span class="sxs-lookup"><span data-stu-id="a9122-103">'identifier': a reference to a volatile field will not be treated as volatile</span></span>  
  
 <span data-ttu-id="a9122-104">Geçici bir alan normalde kullanarak geçirilmemelidir bir `ref` veya **çıkışı** işlevi kapsamında geçici olarak değerlendirilmez beri parametresi.</span><span class="sxs-lookup"><span data-stu-id="a9122-104">A volatile field should not normally be passed using a `ref` or **out** parameter, since it will not be treated as volatile within the scope of the function.</span></span> <span data-ttu-id="a9122-105">İstisnalar bunun gibi bir ınterlocked API çağrılırken.</span><span class="sxs-lookup"><span data-stu-id="a9122-105">There are exceptions to this, such as when calling an interlocked API.</span></span> <span data-ttu-id="a9122-106">Herhangi bir uyarıyı ile kullanmanızı gibi [#pragma Uyarısı](../../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) burada kasıtlı olarak kullandığınız geçici bir alana başvuru parametre olarak bu nadir durumlarda bu uyarıyı devre dışı bırakmak için.</span><span class="sxs-lookup"><span data-stu-id="a9122-106">As with any warning, you may use the [#pragma warning](../../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) to disable this warning in those rare cases where you are intentionally using a volatile field as a reference parameter.</span></span>  
  
 <span data-ttu-id="a9122-107">Aşağıdaki örnek CS0420 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a9122-107">The following sample generates CS0420:</span></span>  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420   
   }  
}  
```
