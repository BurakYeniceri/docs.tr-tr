# <a name="file-io-with-type-39lttypenamegt39-is-not-valid"></a>Dosya g/ç türü &#39; &lt;typename&gt;&#39; geçerli değil
Desteklenmeyen bir tür bir dosyaya yazmak çalışıyorsunuz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Desteklenmeyen tür nerede oluştuğunu belirlemek için yapısının denetleyin ve geçerli bir tür olarak değiştirin.
