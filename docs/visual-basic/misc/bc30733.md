# <a name="39ltmembernamegt39-is-already-declared-by-39ltprocedurenamegt39-which-was-generated-for-this-ltcontainernamegt"></a>&#39; &lt;membername&gt;&#39; tarafından bildirilen &#39;&lt; procedurename&gt;&#39; bunun için oluşturulan &lt;kapsayıcı adı&gt;
Comom dil çalışma zamanı tarafından bildirilen bir üye olarak aynı ada sahip bir üye bildirildi.  
  
 **Hata Kimliği:** BC30733  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Çakışma kaldırmak için üyenin adını değiştirin.
