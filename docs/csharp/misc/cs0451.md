# <a name="compiler-error-cs0451"></a><span data-ttu-id="3f73a-102">Derleyici Hatası CS0451</span><span class="sxs-lookup"><span data-stu-id="3f73a-102">Compiler Error CS0451</span></span>
<span data-ttu-id="3f73a-103">'New()' kısıtlaması 'struct' kısıtlaması ile kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="3f73a-103">The 'new()' constraint cannot be used with the 'struct' constraint</span></span>  
  
 <span data-ttu-id="3f73a-104">Bir genel tür kısıtlamaları belirtirken `new()` kısıtlaması, yalnızca sınıf türü kısıtlamaları, arabirim türü kısıtlamaları, başvuru türü kısıtlamaları ve türü parametresi kısıtlamaları, ancak olmayan değer türü kısıtlamaları ile kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="3f73a-104">When specifying constraints on the type of a generic, the `new()` constraint may only be used with class type constraints, interface type constraints, reference type constraints, and type parameter constraints, but not with value type constraints.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3f73a-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="3f73a-105">Example</span></span>  
 <span data-ttu-id="3f73a-106">Aşağıdaki örnek CS0451 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="3f73a-106">The following sample generates CS0451.</span></span>  
  
```csharp  
// CS0451.cs  
using System;  
public class C4   
{  
   public void F4<T>() where T : struct, new() {}   // CS0451  
}  
  
// OK  
public class C5  
{  
   public void F5<T>() where T : struct {}  
}  
  
public class C6  
{  
   public void F6<T>() where T : new() {}  
}  
```
