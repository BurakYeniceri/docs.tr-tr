# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-base-class-39ltclassnamegt39"></a>Modül için gereken başvuru &#39; &lt;modulename&gt; &#39; temel sınıfını içeren &#39; &lt;classname&gt;&#39;
Modül için gereken başvuru '\<modulename >' temel sınıfını içeren\<classname >'. Projenize ekleyin.  
  
 Sınıfı, doğrudan projenizde başvurulmuyor bir modüldeki tanımlanır. Visual Basic derleyici sınıfı birden fazla modülde tanımlanmış durumda Karışıklığı önlemek için bir başvuru gerektirir.  
  
 **Hata Kimliği:** BC30008  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Başvurulmayan modül adı, proje başvuruları içerir.
