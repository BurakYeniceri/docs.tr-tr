# <a name="-nowin32manifest-visual-basic"></a>-nowin32manifest (Visual Basic)
Herhangi bir uygulama bildirimi yürütülebilir dosyasına katıştırma değil derleyiciye.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
-nowin32manifest  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu seçenek kullanıldığında, bir uygulama bildirimi Win32 kaynak dosyasını ya da daha yeni bir derleme adımı sırasında sağladığınız sürece uygulama Windows Vista sanallaştırma tabi olacaktır. Sanallaştırma hakkında daha fazla bilgi için bkz: [Windows Vista'da ClickOnce dağıtımı](/visualstudio/deployment/clickonce-deployment-on-windows-vista).  
  
 Bildirim oluşturma hakkında daha fazla bilgi için bkz: [-win32manifest (Visual Basic)](../../../visual-basic/reference/command-line-compiler/win32manifest.md).
