# <a name="expected-closing-3939-for-xml-entity"></a>Beklenen kapatma &#39; &#39; XML varlık için
Bir XML varlık başvurusu (örneğin, `>`) kapanış noktalı virgül (;) eksik.  
  
 **Hata Kimliği:** BC31178  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML varlık başvurusu sonunda noktalı içerir.
