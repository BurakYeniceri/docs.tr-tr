## <a name="example"></a><span data-ttu-id="1bc07-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="1bc07-108">Example</span></span>  
 <span data-ttu-id="1bc07-109">Aşağıdaki kod, CS0738 oluşturur, çünkü sınıf yöntemi döndürür `void` ve aynı ada sahip bir arabirim üyesini döndürür `int`:</span><span class="sxs-lookup"><span data-stu-id="1bc07-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>  
  
```csharp

interface ITest  
{  
    int TestMethod();  
}  
public class Test: ITest  
{  
    public void TestMethod() { } // CS0738  
    // Try the following line instead.  
    // public int TestMethod();  
}  
```
