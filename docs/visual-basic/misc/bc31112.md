# <a name="statement-cannot-appear-within-an-event-body"></a>Bir olay gövdesinde deyimi yer alamaz
Deyimi, bir olay gövdesinde yer alamaz. Son olay olduğu varsayılır.  
  
 Bir olay gövdesi içinde geçerli olmayan bir deyim bir içinde görüntülenir.  
  
 **Hata Kimliği:** BC31112  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme bir `End Event` deyim önce.
