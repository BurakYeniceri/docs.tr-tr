# <a name="operator-39ltoperatornamegt39-is-not-defined-for-type-39lttypenamegt39"></a>İşleç &#39; &lt;operatorname&gt; &#39; türü için tanımlı değil &#39; &lt;typename&gt;&#39;
Birli işleç kullanma girişiminde bulundunuz (`+`, `-`, veya `Not`) bir türündeki gibi bir `String`, hangi tanımlanmamış için.  
  
 **Hata Kimliği:** BC30487  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Birli işleç ifade için uygun bir işleç değiştirin.
