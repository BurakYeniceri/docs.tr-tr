 <span data-ttu-id="563a1-109">Daha sonra:</span><span class="sxs-lookup"><span data-stu-id="563a1-109">Then:</span></span>

```  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="563a1-110">Daha sonra:</span><span class="sxs-lookup"><span data-stu-id="563a1-110">Then:</span></span>
