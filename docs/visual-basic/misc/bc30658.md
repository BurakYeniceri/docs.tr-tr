# <a name="property-39ltpropertynamegt39-with-no-parameters-cannot-be-found"></a>Özellik &#39; &lt;propertyname&gt; &#39; parametresiz bulunamıyor
Kullanılan öznitelik, bir veya daha fazla parametre gerektiren bir özelliğe sahiptir.  
  
 **Hata Kimliği:** BC30658  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Özelliği için parametre belirtin.
