# <a name="39readonly39-variable-cannot-be-the-target-of-an-assignment"></a>&#39;Salt okunur&#39; değişkeni atama işleminin hedefi olamaz
A `ReadOnly` özelliği için değer atayan bir bağlamda bulundu. Yalnızca yazılabilir değişkenleri, özellikleri ve dizi öğeleri yürütme sırasında atanmış değerlere sahip olabilir.  
  
 **Hata Kimliği:** BC30064  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `ReadOnly` anahtar sözcüğünün `Dim` değişken bildirme deyimi veya değer atayan deyimi kaldırın.
