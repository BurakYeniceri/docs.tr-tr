# <a name="option-win32manifest-ignored"></a>Seçenek/win32manifest göz ardı
Seçenek/win32manifest yoksayıldı. Yalnızca hedef bir derlemeyi olduğunda belirtilebilir.  
  
 Visual Basic derleyici geçirilen `/win32manifest` derleyici seçeneği `/target` seçeneği `module`.  
  
 **Hata Kimliği:** BC2034  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kaldırma `/win32manifest` derleyici seçeneği veya kümesi `/target` için seçenek `exe`, `winexe`, veya `library`.
