# <a name="39end-sub39-expected"></a>&#39;Bitiş alt&#39; bekleniyor
Bir `End` deyimi sonlandırır blok eşleşmesi gerekir. A `Sub` yordamı tanımı değil düzgün bir şekilde sonlandırıldı.  
  
 **Hata Kimliği:** BC30026  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme bir `End Sub` yordamı sonuna deyimi.
