 <span data-ttu-id="15bb9-104">Türü var. başka bir tür içinde iç içe geçmiş bir tür başvurusu, ancak hiçbir gibi iç içe olduğunda bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="15bb9-104">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="15bb9-105">İç içe geçmiş tür adını yanlış yazarsanız, bu durum oluşabilir.</span><span class="sxs-lookup"><span data-stu-id="15bb9-105">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="15bb9-106">Kullanılan adları yazımını denetleyin ve kapsayan türdeki beklenen üyesi olduğunu doğrulayın.</span><span class="sxs-lookup"><span data-stu-id="15bb9-106">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="15bb9-107">Aşağıdaki örnek, CS0426 oluşturur, çünkü iç içe geçmiş tür y: C sınıfı yok</span><span class="sxs-lookup"><span data-stu-id="15bb9-107">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```csharp  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.     
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```
