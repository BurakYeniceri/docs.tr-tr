# <a name="warning-number-39ltnumbergt39-for-the-option-39ltoptionnamegt39-is-either-not-configurable-or-not-valid"></a>Uyarı numarasını &#39; &lt;numarası&gt;&#39; seçeneği &#39;&lt; SeçenekAdı&gt;&#39; yapılandırılamaz veya geçerli değil
Bir uyarı kimliği geçerli değil. Kimlikleri tamsayı değerleri 0'dan büyük olmalıdır.  
  
 **Hata Kimliği:** BC2026  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçerli olduğundan emin olmak için sağlanan uyarı kimliği denetleyin.
