<span data-ttu-id="9139f-103">' Class 'sınıfı, birden çok taban sınıfı olamaz: 'class_1' ve 'class_2'</span><span class="sxs-lookup"><span data-stu-id="9139f-103">Class 'class' cannot have multiple base classes: 'class_1' and 'class_2'</span></span>  
  
 <span data-ttu-id="9139f-104">Bu hata iletisinin en yaygın nedeni, birden çok devralma kullanmayı deniyor.</span><span class="sxs-lookup"><span data-stu-id="9139f-104">The most common cause of this error message is attempting to use multiple inheritance.</span></span> <span data-ttu-id="9139f-105">C# ' ta bir sınıf yalnızca doğrudan bir sınıftan devralabilir.</span><span class="sxs-lookup"><span data-stu-id="9139f-105">A class in C# may only inherit directly from one class.</span></span> <span data-ttu-id="9139f-106">Ancak, bir sınıf herhangi bir sayıda arabirim uygulayabilir.</span><span class="sxs-lookup"><span data-stu-id="9139f-106">However, a class can implement any number of interfaces.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9139f-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="9139f-107">Example</span></span>  
 <span data-ttu-id="9139f-108">Aşağıdaki örnek, hangi CS1721 oluşturulur ve ardından hatayı önlemek için iki olası yolları gösterir yollarından biri gösterilmektedir.</span><span class="sxs-lookup"><span data-stu-id="9139f-108">The following example shows one way in which CS1721 is generated, and then shows two possible ways to avoid the error.</span></span>  
  
```csharp  
// CS1721.cs  
public class A {}  
public class B {}  
public class MyClass : A, B {}   // CS1721  
  
// One possible fix is to use the following approach instead:  
public class A {}  
public class B : A {}  
public class C : B {}  
  
// Another possible fix is to use interfaces instead of base classes:  
public class A {}  
public interface B {}  
public class C : A, B {}  
```
