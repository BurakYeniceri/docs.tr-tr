# <a name="39ltmembernamegt39-is-not-declared-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>&#39;&lt;membername&gt; &#39; bildirilmemiş olan veya onu içeren modülü hata ayıklama oturumunda yüklü değil
Belirtilen üye bulunamadı.  
  
 **Hata Kimliği:** BC30699  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Üye tanımlı ve doğru yazıldığından emin olun.  
  
2.  Modülde tanımlanmış üyelerinden erişin. Bazı durumlarda, burada bildirilen modülleri değil henüz yüklenmiş olmadığından hata ayıklama ortamı üyeleri bulunamıyor.
