# <a name="compiler-error-cs0520"></a>Derleyici Hatası CS0520
Önceden tanımlanmış türü 'name' yanlış bildirilmiş  
  
 Derleyici tarafından gerekli bir veya daha fazla dosya bulunamadı. .NET Framework ortak dil çalışma zamanı yeniden yükleyin.
