# <a name="compiler-error-cs0448"></a><span data-ttu-id="eabde-102">Derleyici Hatası CS0448</span><span class="sxs-lookup"><span data-stu-id="eabde-102">Compiler Error CS0448</span></span>
<span data-ttu-id="eabde-103">Dönüş türü için ++ veya--işleci kapsayan türden türetilmiş veya kapsayan tür olmalıdır</span><span class="sxs-lookup"><span data-stu-id="eabde-103">The return type for ++ or -- operator must be the containing type or derived from the containing type</span></span>  
  
 <span data-ttu-id="eabde-104">Kıldığınızda `++` veya `--` işleçleri, bunlar gerekir aynı türde içeren dönüş veya kapsayan türden türetilmiş bir tür döndürür.</span><span class="sxs-lookup"><span data-stu-id="eabde-104">When you override the `++` or `--` operators, they must return the same type as the containing type, or return a type that is derived from the containing type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="eabde-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="eabde-105">Example</span></span>  
 <span data-ttu-id="eabde-106">Aşağıdaki örnek CS0448 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="eabde-106">The following sample generates CS0448.</span></span>  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="eabde-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="eabde-107">Example</span></span>  
 <span data-ttu-id="eabde-108">Aşağıdaki örnek CS0448 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="eabde-108">The following sample generates CS0448.</span></span>  
  
```csharp  
// CS0448_b.cs

{  
   public static S? operator ++(S s) { return new S(); }   // CS0448  
   public static S? operator --(S s) { return new S(); }   // CS0448

   public static T operator --(T t) { return new T(); }  
   public static T operator ++(T t) { return new T(); }  
  
   public static T? operator --(T? t) { return new T(); }  
   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}  
}  
```
