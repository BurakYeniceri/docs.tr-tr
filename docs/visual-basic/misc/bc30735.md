# <a name="type-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Bir modül türü bildirilemez &#39; &lt;belirticisi&gt;&#39;
Geçersiz değiştirici türüyle içinde bir modüle bildirildi.  
  
 **Hata Kimliği:** BC30735  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanmayın `Protected` veya `Protected Friend` türlerinde değiştiricileri modülü içinde bildirilen.
