# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-ltnumbergt-characters"></a>Ad alanlarında türleri için tam olarak nitelenmiş adlar daha uzun olamaz &lt;numarası&gt; karakterleri
Türünün tam adını, belirtilen sayıda karakteri uzun olamaz.  
  
 **Hata Kimliği:** BC30031  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Türün adını kısaltın.
