 <span data-ttu-id="db063-104">`New` Sınırlama, bu tür parametresi için sağlanan tür bağımsız değişkeni erişilebilir nesneleri oluşturduğu kod parametresiz bir oluşturucu kullanıma açmalıdır anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="db063-104">The `New` constraint means that the type argument supplied to that type parameter must expose a parameterless constructor accessible to the code that creates objects from it.</span></span> <span data-ttu-id="db063-105">Tüm türlerin parametresiz oluşturucular sahiptir, ancak tüm başvuru türleri yapın.</span><span class="sxs-lookup"><span data-stu-id="db063-105">All value types have parameterless constructors, but not all reference types do.</span></span> <span data-ttu-id="db063-106">Bu nedenle `Structure` kısıtlamasına `New` kısıtlaması, ancak `Class` kısıtlaması veya bir sınıf veya arabirim adı yok.</span><span class="sxs-lookup"><span data-stu-id="db063-106">Therefore the `Structure` constraint satisfies the `New` constraint, but the `Class` constraint, or a class or interface name, does not.</span></span>
