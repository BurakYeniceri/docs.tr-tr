# <a name="39inherits39-not-valid-in-modules"></a>&#39;Devralınan&#39; modüllerinde geçerli değil
Bir `Inherits` deyimi içinde bir modül oluşur. Modülleri sınıflardan devralan olamaz.  
  
 **Hata Kimliği:** BC30230  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Inherits` deyimi veya modülü bir sınıf olarak yeniden yazın.
