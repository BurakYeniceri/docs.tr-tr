 <span data-ttu-id="b5ae1-104">Bir sınıftaki bir alan iç içe bir sınıf için otomatik olarak kullanılabilir değil.</span><span class="sxs-lookup"><span data-stu-id="b5ae1-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="b5ae1-105">İç içe bir sınıf için kullanılabilir olması için alan olmalıdır [statik](../../../csharp/language-reference/keywords/static.md).</span><span class="sxs-lookup"><span data-stu-id="b5ae1-105">To be available to a nested class, the field must be [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="b5ae1-106">Aksi durumda, dış sınıfının bir örneğini oluşturmanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="b5ae1-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="b5ae1-107">Daha fazla bilgi için bkz: [iç içe geçmiş türler](../../../csharp/programming-guide/classes-and-structs/nested-types.md).</span><span class="sxs-lookup"><span data-stu-id="b5ae1-107">For more information, see [Nested Types](../../../csharp/programming-guide/classes-and-structs/nested-types.md).</span></span>  
  
 <span data-ttu-id="b5ae1-108">Aşağıdaki örnek CS0038 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b5ae1-108">The following sample generates CS0038:</span></span>  
  
```csharp  
// CS0038.cs  
class OuterClass  
{  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;  
         int count2 = count;   // CS0038  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
