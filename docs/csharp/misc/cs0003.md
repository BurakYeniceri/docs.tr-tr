# <a name="compiler-error-cs0003"></a>Derleyici Hatası CS0003
Bellek yetersiz  
  
 Derleyici derleme tamamlamak için yeterli sanal bellek ayıramadı. Tüm gereksiz uygulamaları kapatın ve yeniden derleyin.  
  
 Disk belleği dosyası boyutunu artırın ve boş disk alanı olduğundan emin olmak isteyebilirsiniz.  
  
 Bu hata ayrıca sürümleri eşleşmeyen sonucu görünebilir [!INCLUDE[winsdklong](~/includes/winsdklong-md.md)] ve C# Derleyici veya C# Derleyici desteği; Visual Studio yeniden bir veya daha fazla bozuk dosya.
