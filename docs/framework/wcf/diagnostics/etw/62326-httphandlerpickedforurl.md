# <a name="62326---httphandlerpickedforurl"></a>62326 - HttpHandlerPickedForUrl
## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|Kimlik|62326|  
|Anahtar Sözcükler|WebHost|  
|Düzey|Bilgiler|  
|Kanal|Microsoft Windows uygulaması sunucu-uygulamalar/hata ayıklama|  
  
## <a name="description"></a>Açıklama  
 HTTP işleyicisi bu URL'sine yapılan tüm isteklere hizmet vermek için çekilir olduğunda bu olay yayınlanır.  
  
## <a name="message"></a>İleti  
 '%1' URL'si konakları XAML belge kök öğesi olan '%2' yazın. HTTP işleyicisi türü '%3', bu url için yapılan tüm isteklere hizmet vermek için çekilir.
