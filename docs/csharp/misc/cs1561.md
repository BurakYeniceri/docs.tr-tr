# <a name="compiler-error-cs1561"></a>Derleyici Hatası CS1561
Çıkış dosya adı çok uzun veya geçersiz  
  
 Çıktı dosyası adı 256 karakterden uzun olamaz ve Avrupa simgesi, soru işareti veya diğerlerinin yanı sıra ters eğik çizgi gibi geçersiz karakterler içeremez.
