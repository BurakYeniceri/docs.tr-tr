# <a name="evaluation-of-expression-or-statement-stopped"></a>İfade veya deyimin değerinin değerlendirme durduruldu
**Durdurmak değerlendirme** menü öğesi seçildiğinde, durdurmak değerlendirme neden olur.  
  
 **Hata Kimliği:** BC30721  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Seçim yapmadan devam etmek yürütülmesine izin **durdurmak değerlendirme**.
