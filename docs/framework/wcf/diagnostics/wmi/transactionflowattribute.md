# <a name="transactionflowattribute"></a>TransactionFlowAttribute
TransactionFlowAttribute  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
class TransactionFlowAttribute : Behavior  
{  
  string TransactionFlowOption;  
};  
```  
  
## <a name="methods"></a>Yöntemler  
 TransactionFlowAttribute sınıfı herhangi bir yöntem tanımlamıyor.  
  
## <a name="properties"></a>Özellikler  
 TransactionFlowAttribute sınıfı aşağıdaki özelliklere sahiptir:  
  
### <a name="transactionflowoption"></a>TransactionFlowOption  
 Veri türü: dize  
  
 Erişim türüne: salt okunur  
  
 İşlemler akış olup olmadığını gösterir.
