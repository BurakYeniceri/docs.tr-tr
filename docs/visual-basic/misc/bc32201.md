# <a name="property-parameters-cannot-have-the-name-39value39"></a>Özellik parametrelerinin adı &#39; olamaz Değer &#39;
İçindeki bir parametre bir `Property` deyimi adlandırılan `Value`. Bu ad özelliğinde atanmasına yeni değer için bir özellik tanımı içinde ayrılmış `Set` yordamı.  
  
 **Hata Kimliği:** BC32201  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Parametrenin adını değiştirmek `Property` deyimi.
