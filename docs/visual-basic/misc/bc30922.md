# <a name="39ltderivedtypenamegt39-cannot-inherit-from-lttypegt-39ltconstructedbasetypenamegt39-because-it-expands-the-access-of-type-39ltinternaltypenamegt39-outside-the-assembly"></a><span data-ttu-id="2ee26-102">&#39;&lt;derivedtypename&gt; &#39; devralınmalıdır olamaz &lt;türü&gt; &#39; &lt;constructedbasetypename&gt; &#39; türü erişimini genişlettiğinden &#39; &lt; internaltypename&gt; &#39; derleme dışına</span><span class="sxs-lookup"><span data-stu-id="2ee26-102">&#39;&lt;derivedtypename&gt;&#39; cannot inherit from &lt;type&gt; &#39;&lt;constructedbasetypename&gt;&#39; because it expands the access of type &#39;&lt;internaltypename&gt;&#39; outside the assembly</span></span>
<span data-ttu-id="2ee26-103">Tür bağımsız değişkeni bir temel sınıf veya arabirim olarak kullanarak bir kısıtlanmış türdeki erişim düzeyini genişletmek bir türetilmiş sınıfta veya arabirimde çalışır.</span><span class="sxs-lookup"><span data-stu-id="2ee26-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="2ee26-104">Aşağıdaki kod bu hata oluşmasına neden olabilir.</span><span class="sxs-lookup"><span data-stu-id="2ee26-104">The following code can generate this error.</span></span>  
  
```  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="2ee26-105">Kod derleme dışına erişmek için izin verilmeyen `restrictedStructure`.</span><span class="sxs-lookup"><span data-stu-id="2ee26-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="2ee26-106">Ancak, `derivedClass` başvurusu yapabilir herhangi koddan erişilebilir.</span><span class="sxs-lookup"><span data-stu-id="2ee26-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="2ee26-107">Bu nedenle, `derivedClass` devral olamaz `baseClass` onu kullanıyorsa `restrictedStructure` bir tür bağımsız değişkeni olarak maruz bırakabileceğinden çünkü `restrictedStructure` herhangi bütünleştirilmiş kodunda.</span><span class="sxs-lookup"><span data-stu-id="2ee26-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="2ee26-108">**Hata Kimliği:** BC30922</span><span class="sxs-lookup"><span data-stu-id="2ee26-108">**Error ID:** BC30922</span></span>
