# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-without-a-narrowing-conversion-ltlistgt"></a>Hayır erişilebilir aşırı yüklenmiş &#39; &lt;methodname&gt;&#39; daraltma dönüşümü olmadan bu bağımsız değişkenlerle çağrılamaz: &lt;listesi&gt;
Aşırı yüklenmiş yöntemin çağrıldı, ancak sağlanan bağımsız değişkenler olmadan daraltma dönüşümü listesiyle yöntemi eşleştirilemiyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Belirtin `Option``Strict` `Off`.  
  
2.  Bağımsız değişkenlerini aşırı yüklenmiş yöntemin imzası eşleşecek şekilde değiştirin.
