# <a name="a-log-has-already-been-created-with-this-name-on-this-machine"></a>Bir günlük bu makinede bu adla zaten oluşturuldu
Zaten kullanımda olan bir ada sahip bir günlük oluşturma girişiminde bulunuldu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Çakışan günlük kaldırın.  
  
-   Günlük için farklı bir ad kullanın.
