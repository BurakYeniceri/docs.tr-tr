# <a name="visual-basic-language-specification"></a>Visual Basic Dil Belirtimi
Visual Basic dil belirtimi, Visual Basic dil bilgisi ve sözdizimi hakkında tüm soruların yanıtları için yetkili kaynağıdır. Visual Basic başvurusu belgelerinde kapsanmayan çok puan içeren dili hakkında ayrıntılı bilgiler içerir.  
  
 Belirtimi kullanılabilir [Microsoft Download Center](http://go.microsoft.com/fwlink/?LinkId=188623).  
  
Bu sitedeki [VB 11 belirtimi](../../../../_vblang/spec/introduction.md). İçinde yer alan Markdown dosyalarını yerleşik [dotnet/vblang GitHub deposuna](https://github.com/dotnet/vblang/blob/master/spec/README.md).

Belirtimi sorunları oluşturulmalıdır [dotnet/vblang](https://github.com/dotnet/vblang/issues) deposu. Veya, bulduğunuz hataları düzelttikten ilgilendiğiniz varsa, siz gönderebilirsiniz bir [çekme isteği](https://github.com/dotnet/vblang/pulls) aynı deponuza.
