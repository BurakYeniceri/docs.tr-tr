# <a name="39raiseevent39-is-already-declared"></a>&#39; RaiseEvent &#39; zaten bildirildi
Birden fazla `RaiseEvent` bildirimi özel olay bildiriminde görüldü. A `RaiseEvent` bildirimi olayı oluşturmak için kullanılan bir yordam bildirir.  
  
 **Hata Kimliği:** BC31129  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yedek kaldırmak `RaiseEvent` deyimi.
