# <a name="compiler-error-cs1611"></a>Derleyici Hatası CS1611
Params parametresi ref veya out olarak bildirilemez  
  
 Anahtar sözcükler [ref](../../csharp/language-reference/keywords/ref.md) veya [çıkışı](../../csharp/language-reference/keywords/out.md) kullanılamaz [params](../../csharp/language-reference/keywords/params.md) anahtar sözcüğü.  
  
 Aşağıdaki örnek CS1611 oluşturur:  
  
```  
// CS1611.cs  
public class MyClass  
{  
   public static void Test(params ref int[] a)   // CS1611, remove ref  
   {  
   }  
  
   public static void Main()  
   {  
      Test(1);  
   }  
}  
```
