# <a name="systemservicemodelserviceoperationmissingreply"></a>System.ServiceModel.ServiceOperationMissingReply
System.ServiceModel.ServiceOperationMissingReply  
  
## <a name="description"></a>Açıklama  
 İstek/yanıt işlemi yanıt iletisi yok.  
  
 Bu iki yönlü bir işlemdir; Ancak, istemciye gönderilecek yanıt yoktur. Bu izleme, geliştirme sırasında hata ayıklama için kullanabilirsiniz.
