## <a name="example"></a><span data-ttu-id="2de35-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="2de35-106">Example</span></span>  
 <span data-ttu-id="2de35-107">Aşağıdaki örnek CS0115 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="2de35-107">The following sample generates CS0115.</span></span> <span data-ttu-id="2de35-108">CS0115 iki yoldan biriyle çözümleyebilirsiniz:</span><span class="sxs-lookup"><span data-stu-id="2de35-108">You can resolve CS0115 in one of two ways:</span></span>  
  
-   <span data-ttu-id="2de35-109">Kaldırma `override` yönteminde anahtar sözcüğünün `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="2de35-109">Remove the `override` keyword from the method in `MyClass2`.</span></span>  
  
-   <span data-ttu-id="2de35-110">Kullanım `MyClass1` için temel sınıf olarak `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="2de35-110">Use `MyClass1` as a base class for `MyClass2`.</span></span>  
  
```csharp  
// CS0115.cs  
namespace MyNamespace  
{  
    abstract public class MyClass1  
    {  
        public abstract int f();  
    }  
  
    abstract public class MyClass2  
    {  
        public override int f()   // CS0115  
        {  
            return 0;  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
