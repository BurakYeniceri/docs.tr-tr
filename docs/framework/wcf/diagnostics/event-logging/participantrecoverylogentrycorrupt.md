# <a name="participantrecoverylogentrycorrupt"></a>ParticipantRecoveryLogEntryCorrupt
Kimliği: 138  
  
 Önem derecesi: hata  
  
 Kategori: TransactionBridge  
  
## <a name="description"></a>Açıklama  
 Bu olay, katılımcı kurtarma günlük girişi bozuktu ve seri durumdan çıkarılmış açılamadı gösterir. Bu hatadan veri kaybına neden olabilir. Olay listeleri işlem kimliği, kurtarma verileri (Base64 ile kodlanmış), özel durum, işlem adı ve işlem kimliği
