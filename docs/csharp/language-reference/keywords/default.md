# <a name="default-c-reference"></a>default (C# Başvurusu)

`default` Anahtar sözcüğü.

`default` Anahtar sözcüğü kullanılabilir `switch` deyimi veya varsayılan değer ifadesi:

- [Switch deyimi](switch.md): varsayılan etiket belirtir.

- [Varsayılan değer ifadeleri](../../programming-guide/statements-expressions-operators/default-value-expressions.md): varsayılan değer türü üretir. Başvuru türleri ve değer türleri için sıfır ve yapılar için 0 bit düzeni için null olur.
