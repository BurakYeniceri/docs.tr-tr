# <a name="39ltstatementnamegt39-statement-requires-an-array"></a>&#39;&lt;statementname&gt; &#39; deyim için bir dizi gerekir
`Erase` Ve `ReDim` deyimleri dizi veri türü gerektirir. Sağlanan bir dizi türünde değil.  
  
 **Hata Kimliği:** BC30049  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Veri türü için bir dizi türünü değiştirin.
