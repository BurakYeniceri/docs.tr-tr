# <a name="variable-declaration-statements-are-not-valid-in-the-immediate-window"></a>Değişken bildirimi deyimleri Komut penceresi içinde geçerli değildir
Değişken bildirimleri kullanılamaz **hemen** penceresi.  
  
 **Hata Kimliği:** BC30709  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değişkenleri bildirme **hemen** penceresi.
