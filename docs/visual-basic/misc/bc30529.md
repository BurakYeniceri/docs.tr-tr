# <a name="all-parameters-must-be-explicitly-typed-if-any-are"></a>Varsa tüm parametreleri açıkça yazılmalıdır
Farklı türde bir işlev türü bildirimlerinde karma varsa.  
  
 **Hata Kimliği:** BC30529  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tüm parametreleri açıkça yazdığınızdan emin olun.
