# <a name="compiler-error-cs0690"></a>Derleyici Hatası CS0690
Giriş dosyası 'dosya', geçersiz meta veriler içeriyor.  
  
 Meta veri dosyası açabilirsiniz, ancak bazı yerelleştirme sorunu nedeniyle meta verileri bozuk. Bu hata için benzer [CS0009](../../csharp/misc/cs0009.md), meta veri dosyası açabilmelisiniz dışında.  
  
 PE'nin (işlem yürütülebilir) yürütülebilir bir dosyadır.
