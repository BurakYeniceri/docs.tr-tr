# <a name="compiler-error-cs1007"></a><span data-ttu-id="30c30-102">Derleyici Hatası CS1007</span><span class="sxs-lookup"><span data-stu-id="30c30-102">Compiler Error CS1007</span></span>

 <span data-ttu-id="30c30-104">Ne zaman bildirdiğiniz bir [özelliği](../../csharp/programming-guide/classes-and-structs/using-properties.md), erişimci yöntemlerini bildirmeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="30c30-104">When you declare a [property](../../csharp/programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="30c30-105">Ancak, bir özelliği birden fazla olamaz `get` erişimci yöntemi veya birden fazla `set` erişimci yöntemi.</span><span class="sxs-lookup"><span data-stu-id="30c30-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="30c30-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="30c30-106">Example</span></span>  
 <span data-ttu-id="30c30-107">Aşağıdaki örnek CS1007 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="30c30-107">The following sample generates CS1007:</span></span>  
  
```csharp  
// CS1007.cs

    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method

            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
