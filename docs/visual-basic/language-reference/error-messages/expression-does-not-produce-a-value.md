# <a name="expression-does-not-produce-a-value"></a>İfade bir değer üretmiyor
Bir değer arama gibi değeri oluşturan bir bağlamda üretmez bir ifade kullanmaya çalışmış bir `Sub` bağlamda burada bir `Function` beklenir.  
  
 **Hata Kimliği:** BC30491  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İfade bir değer üreten bir değiştirin.
