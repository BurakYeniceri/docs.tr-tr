# <a name="xml-comment-tag-39lttaggt39-is-not-permitted-on-a-39ltelementgt39-language-element"></a>XML açıklama etiketi &#39; &lt;etiketi&gt; &#39; üzerinde izin verilmiyor bir &#39; &lt;öğesi&gt; &#39; language öğesi
XML açıklama etiketi \<etiketi > üzerinde izin verilmiyor bir '\<öğesi >' öğesi. XML açıklama yoksayılacak.  
  
 Bir XML açıklama etiketi, uygulandığı dil öğesi tarafından desteklenmiyor.  
  
 **Hata Kimliği:** BC42306  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Desteklenmeyen XML açıklama etiketi kaldırın.
