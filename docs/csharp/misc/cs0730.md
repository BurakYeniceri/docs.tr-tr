## <a name="example"></a><span data-ttu-id="bfc81-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="bfc81-105">Example</span></span>  
 <span data-ttu-id="bfc81-106">Aşağıdaki örnek CS0730 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="bfc81-106">The following sample generates CS0730.</span></span> <span data-ttu-id="bfc81-107">İki kaynak dosyaları içerir.</span><span class="sxs-lookup"><span data-stu-id="bfc81-107">It consists of two source files.</span></span> <span data-ttu-id="bfc81-108">İlk olarak, kitaplık dosyası derleme `CS0730a.cs`ve dosyanın derleme `CS0730.cs` kitaplık dosyasını başvuruyor.</span><span class="sxs-lookup"><span data-stu-id="bfc81-108">First, compile the library file `CS0730a.cs`, and the compile the file `CS0730.cs` referencing the library file.</span></span>  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```
