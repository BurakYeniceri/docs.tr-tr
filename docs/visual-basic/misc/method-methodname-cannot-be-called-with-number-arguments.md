# <a name="method-39ltmethodnamegt39-cannot-be-called-with-ltnumbergt-arguments"></a>Yöntem &#39; &lt;methodname&gt;&#39; ile çağrılamaz &lt;numarası&gt; bağımsız değişkenleri
Yöntem çağrısı yanlış sayıda bağımsız değişken içeriyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Verdiğiniz bağımsız değişken sayısı.  
  
2.  Aramak istediğiniz yöntemi parametre sayısı karşı numarası aynı.  
  
3.  İki sayı değil kabul etmesi durumunda bağımsız değişken sayısı ayarlayın.
