# <a name="white-space-may-not-appear-here"></a>Beyaz alan burada görünmeyebilir
Boşluklardan bir XML öğesi adına değişmez değer dahil edilir.  
  
 **Hata Kimliği:** BC31177  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Boşluk veya boşluklar XML öğesi değişmez değeri adından kaldırın.
