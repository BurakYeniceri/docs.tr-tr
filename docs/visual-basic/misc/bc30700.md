# <a name="side-effects-not-valid-during-expression-evaluation-in-this-context"></a>Yan etkiler bu bağlamda ifade değerlendirme sırasında geçerli değil
Bir nesnenin durumu değiştirme deyimleri bu bağlamda izin verilmiyor.  
  
 **Hata Kimliği:** BC30700  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kodunuzu diğer öğeleri durumunu değiştirmeden deyimleri değerlendirilir şekilde değiştirin.
