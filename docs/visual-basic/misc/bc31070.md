# <a name="class-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>Arabirim sınıfında bildirilemez &#39; &lt;belirticisi&gt;&#39;
Geçersiz değiştirici bir arabirim, bildirilen sınıfı ile belirtildi.  
  
 **Hata Kimliği:** BC31070  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değiştiriciler gibi kaldırmak `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`, veya `Overridable`.
