# <a name="compiler-error-cs0052"></a><span data-ttu-id="c3968-102">Derleyici Hatası CS0052</span><span class="sxs-lookup"><span data-stu-id="c3968-102">Compiler Error CS0052</span></span>
<span data-ttu-id="c3968-103">Tutarsız erişilebilirlik: alan türü 'type' alanı 'field' az erişilebilir</span><span class="sxs-lookup"><span data-stu-id="c3968-103">Inconsistent accessibility: field type 'type' is less accessible than field 'field'</span></span>  
  
 <span data-ttu-id="c3968-104">Tüm ortak yapılar genel olarak erişilebilir bir nesneye dönüştürüldüğünden bir alan türü alan az erişilebilir olamaz.</span><span class="sxs-lookup"><span data-stu-id="c3968-104">The type of a field cannot be less accessible than the field itself because all public constructs must return a publicly accessible object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c3968-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="c3968-105">Example</span></span>  
 <span data-ttu-id="c3968-106">Aşağıdaki örnek CS0052 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="c3968-106">The following sample generates CS0052:</span></span>  
  
```csharp  
// CS0052.cs  
public class MyClass2  
{  
    // The following line causes an error because the field, M, is declared  
    // as public, but the type, MyClass, is private. Therefore the type is   
    // less accessible than the field.  
    public MyClass M;   // CS0052  
  
    private class MyClass  
    {  
    }  
    // One way to resolve the error is to change the accessibility of the type  
    // to public.   
    //public class MyClass  
    // Another solution is to change the accessibility of the field to private.  
    // private MyClass M;  
}  
  
    public class MainClass  
    {  
        public static void Main()  
        {  
        }  
    }  
```
