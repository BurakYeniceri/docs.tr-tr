# <a name="unable-to-apply-security-attributes-to-39ltnamegt39-lterrorgt"></a>Güvenlik öznitelikleri &#39;uygulanamaz; &lt;adı&gt;&#39;: &lt;hata&gt;
Güvenlik öznitelikleri uygulanırken bir hata oluştu. Bu güvenlik öznitelikleri yanlış kullanmanın sonucu olabilir veya derleyici ile ilgili bir sorun olduğunu gösteriyor olabilir.  
  
 **Hata Kimliği:** BC31520  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Doğrulayın sözdizimi ve öznitelik bağımsız değişkenleri.  
  
2.  Hata tekrar oluşuyorsa, yeniden görmek için programı derleyin.  
  
3.  Hata devam ederse, çalışmanızı kaydedin ve yeniden [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
4.  Hata yeniden oluşursa, yeniden [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
5.  Yeniden yüklenmesinden sonra hata devam ederse, Microsoft Ürün Destek Hizmetleri'ne bildirin.
