# <a name="file-not-found-visual-basic-run-time-error"></a>Dosya bulunamadı (Visual Basic Çalışma Süresi Hatası)
Dosyanın belirtilen konumda bulunamadı. Hata aşağıdaki olası nedenleri vardır:  
  
-   Bir deyim var olmayan bir dosyaya başvuruyor.  
  
-   Bir dinamik bağlantı kitaplığı (DLL) bir yordam çağırma çalışıldı, ancak kitaplığı belirtilen `Lib` yan tümcesi `Declare` deyimi bulunamıyor.  
  
-   Bir proje açın veya var olmayan bir metin dosyası yükleme girişiminde bulunuldu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Dosya adını ve yolunu yazımını denetleyin.
