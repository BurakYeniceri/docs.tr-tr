# <a name="39redim39-preserve-operand-cannot-be-nothing"></a>&#39;ReDim&#39; Koru işleneni hiçbir şey olamaz

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değişiklik `Preserve` işleneni geçerli bir değer.
