# <a name="39ltfunctionnamegt39-is-not-declared-visual-basic-error"></a>&#39;&lt;FunctionName&gt; &#39; (Visual Basic hatası) bildirilmedi
'\<functionname >' bildirilmemiş olan. Dosya g/ç işlevselliği Microsoft.VisualBasic ad alanında kullanılabilir.  
  
 **Hata Kimliği:** BC30815  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tanımlanan dosya g/ç işlevlerini `My.Computer.FileSystem`, `System.IO` ad alanı, Visual Basic çalışma zamanı işlevleri veya `Microsoft.VisualBasic.FileSystem` nesnesi.
