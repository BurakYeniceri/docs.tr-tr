# <a name="overloading-methods-declared-in-multiple-base-interfaces-is-not-valid"></a>Aşırı yükleme yöntemleri birden fazla temel arabirimde bildirilen geçerli değil
Birden çok devralınan arabirimi örtük olarak aynı yöntemi aşırı yükleme.  
  
 **Hata Kimliği:** BC31410  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanım `Shadows` yerine değiştiricisi `Overloads` değiştiricisi.
