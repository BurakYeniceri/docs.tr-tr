# <a name="bad-record-number"></a>Hatalı kayıt numarası
Kayıt sayısı `a FileGet`, `FilePut`, `FileGetObject`, veya `FilePutObject` açıklamadır sıfıra eşit veya daha az.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kayıt sayısı oluşturmak için kullanılan hesaplama denetleyin. Kayıt sayısını içeren veya kayıt sayıları hesaplamak için kullanılan değişken yazımını doğrulayın. Yanlış yazılmış bir değişken adı örtük olarak bildirilen ve kullandığınız sürece, sıfıra başlatıldı `Option Explicit On` modülünde.
