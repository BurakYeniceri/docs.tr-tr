 <span data-ttu-id="d1998-104">Ne zaman bildirdiğiniz bir [özelliği](../../csharp/programming-guide/classes-and-structs/using-properties.md), erişimci yöntemlerini bildirmeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="d1998-104">When you declare a [property](../../csharp/programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="d1998-105">Ancak, bir özelliği birden fazla olamaz `get` erişimci yöntemi veya birden fazla `set` erişimci yöntemi.</span><span class="sxs-lookup"><span data-stu-id="d1998-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d1998-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="d1998-106">Example</span></span>

```  
            return 0;

    }

}  
```
