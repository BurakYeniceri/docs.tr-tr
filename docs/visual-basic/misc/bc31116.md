# <a name="39removehandler39-declaration-must-end-with-a-matching-39end-removehandler39"></a>&#39;RemoveHandler&#39; bildirimi eşleşen ile bitmelidir &#39;End RemoveHandler&#39;
A `RemoveHandler` bildirimi ile bitmelidir bir `End RemoveHandler` deyimi.  
  
 **Hata Kimliği:** BC31116  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Denetleyin `RemoveHandler` bildirimi ile biten bir `End RemoveHandler` deyimi.
