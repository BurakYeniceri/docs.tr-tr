# <a name="compiler-error-cs0591"></a>Derleyici Hatası CS0591
Bağımsız değişkeni 'öznitelik' özniteliği için geçersiz değer  
  
 Bir öznitelik geçersiz bağımsız değişken veya iki birbirini dışlayan bağımsız değişken geçirildi.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS0591 oluşturur:  
  
```csharp  
// CS0591.cs  
using System;  
  
[AttributeUsage(0)]   // CS0591  
class I: Attribute  
{  
}  
  
public class a  
{  
    public static void Main()  
    {  
    }  
}  
```
