# <a name="ltclaimtyperequirementsgt-for-ltmessagegt"></a>&lt;ileti&gt; için &lt;claimTypeRequirements&gt;
Gerekli talep türleri koleksiyonunu belirtir.  
  
 Koleksiyon, istemcinin hizmete erişmek için kullandığı verilen belirteç olması gereken tüm gerekli ve isteğe bağlı talepleri belirtmek için hizmet tarafından kullanılır. Hizmet meta veri gerekli talep türlerini WSDL yayımlama etkinleştirildi, ancak verilen belirteç içeren belirtilen talep türü WCF gerektirmez, kullanıma sunar. Gerekli talep türlerini mevcut zorlamak isteyen Hizmetleri Yetkilendirme İlkesi'ni kullanarak yapmanız gerekir.  
  
 Federasyon istemcilerde bu koleksiyon için güvenlik belirteci hizmeti istemci isteği için verilen bir belirteç gönderdiği gerekli ve isteğe bağlı talep listesini içerir.
