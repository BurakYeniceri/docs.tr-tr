## <a name="choice-of-passing-mechanism"></a><span data-ttu-id="05954-113">Seçim mekanizması geçirme</span><span class="sxs-lookup"><span data-stu-id="05954-113">Choice of Passing Mechanism</span></span>  
 <span data-ttu-id="05954-114">Her değişken için dikkatle geçirme mekanizması seçmeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="05954-114">You should choose the passing mechanism carefully for each argument.</span></span>  
  
-   <span data-ttu-id="05954-115">**Koruma**.</span><span class="sxs-lookup"><span data-stu-id="05954-115">**Protection**.</span></span> <span data-ttu-id="05954-116">Seçerken iki iletme mekanizmaları en önemli değişkenlerini değiştirmek için arama Etkilenme ölçüttür.</span><span class="sxs-lookup"><span data-stu-id="05954-116">In choosing between the two passing mechanisms, the most important criterion is the exposure of calling variables to change.</span></span> <span data-ttu-id="05954-117">Bağımsız değişken geçirme avantajı `ByRef` yordamı çağıran kodu bağımsız aracılığıyla bir değer döndürmesi emin olan.</span><span class="sxs-lookup"><span data-stu-id="05954-117">The advantage of passing an argument `ByRef` is that the procedure can return a value to the calling code through that argument.</span></span> <span data-ttu-id="05954-118">Bağımsız değişken geçirme avantajı `ByVal` yordamı tarafından değiştirilen bir değişken koruduğu emin olan.</span><span class="sxs-lookup"><span data-stu-id="05954-118">The advantage of passing an argument `ByVal` is that it protects a variable from being changed by the procedure.</span></span>  
  
-   <span data-ttu-id="05954-119">**Performans**.</span><span class="sxs-lookup"><span data-stu-id="05954-119">**Performance**.</span></span> <span data-ttu-id="05954-120">Geçirme mekanizması kodunuzu performansını etkileyebilir olsa da, genellikle anlamsız bir farktır.</span><span class="sxs-lookup"><span data-stu-id="05954-120">Although the passing mechanism can affect the performance of your code, the difference is usually insignificant.</span></span> <span data-ttu-id="05954-121">Bunun tek istisnası geçirilen değer türüdür `ByVal`.</span><span class="sxs-lookup"><span data-stu-id="05954-121">One exception to this is a value type passed `ByVal`.</span></span> <span data-ttu-id="05954-122">Bu durumda, [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] bağımsız değişkeni, tüm veri içeriğini kopyalar.</span><span class="sxs-lookup"><span data-stu-id="05954-122">In this case, [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] copies the entire data contents of the argument.</span></span> <span data-ttu-id="05954-123">Bu nedenle, bir yapı gibi büyük değer türü için bunu geçirileceğini daha etkili olabilir `ByRef`.</span><span class="sxs-lookup"><span data-stu-id="05954-123">Therefore, for a large value type such as a structure, it can be more efficient to pass it `ByRef`.</span></span>  
  
     <span data-ttu-id="05954-124">Referans türleri için yalnızca verileri işaretçisine kopyalanan (dört 32-bit platformlarda 64 bit platformlarda sekiz bayt) bayttır.</span><span class="sxs-lookup"><span data-stu-id="05954-124">For reference types, only the pointer to the data is copied (four bytes on 32-bit platforms, eight bytes on 64-bit platforms).</span></span> <span data-ttu-id="05954-125">Bu nedenle, tür bağımsız değişkenler geçirebilirsiniz `String` veya `Object` performans zarar olmadan değeriyle.</span><span class="sxs-lookup"><span data-stu-id="05954-125">Therefore, you can pass arguments of type `String` or `Object` by value without harming performance.</span></span>  
  
## <a name="determination-of-the-passing-mechanism"></a><span data-ttu-id="05954-126">Geçirme mekanizması olarak belirleme</span><span class="sxs-lookup"><span data-stu-id="05954-126">Determination of the Passing Mechanism</span></span>

 <span data-ttu-id="05954-129">Bir parametre ile bildirilirse `ByRef`, çağrıyı yapan kod mekanizmasına zorlayabilirsiniz `ByVal` çağrısında parantez içinde bağımsız değişken adı kapsayan tarafından.</span><span class="sxs-lookup"><span data-stu-id="05954-129">If a parameter is declared with `ByRef`, the calling code can force the mechanism to `ByVal` by enclosing the argument name in parentheses in the call.</span></span> <span data-ttu-id="05954-130">Daha fazla bilgi için bkz: [nasıl yapılır: bağımsız değişkeni değere göre bayraklarıdır için zorlama](./how-to-force-an-argument-to-be-passed-by-value.md).</span><span class="sxs-lookup"><span data-stu-id="05954-130">For more information, see [How to: Force an Argument to Be Passed by Value](./how-to-force-an-argument-to-be-passed-by-value.md).</span></span>  
  
 <span data-ttu-id="05954-131">Varsayılan olarak [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] bağımsız değişkenleri değere göre geçirmektir.</span><span class="sxs-lookup"><span data-stu-id="05954-131">The default in [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] is to pass arguments by value.</span></span>  
  
## <a name="when-to-pass-an-argument-by-value"></a><span data-ttu-id="05954-132">Bir bağımsız değişken değeri tarafından geçirmek ne zaman</span><span class="sxs-lookup"><span data-stu-id="05954-132">When to Pass an Argument by Value</span></span>  
  
-   <span data-ttu-id="05954-133">Bağımsız değişken temel arama code öğesi değiştirilemez bir öğeyse, karşılık gelen bir parametre bildirme [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md).</span><span class="sxs-lookup"><span data-stu-id="05954-133">If the calling code element underlying the argument is a nonmodifiable element, declare the corresponding parameter [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md).</span></span> <span data-ttu-id="05954-134">Kod değiştirilemez öğesinin değeri değiştirebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="05954-134">No code can change the value of a nonmodifiable element.</span></span>  
  
-   <span data-ttu-id="05954-135">Temel öğesi değiştirilebilir, ancak değerini değiştirmek için yordamı istemiyorsanız, parametre bildirme `ByVal`.</span><span class="sxs-lookup"><span data-stu-id="05954-135">If the underlying element is modifiable, but you do not want the procedure to be able to change its value, declare the parameter `ByVal`.</span></span> <span data-ttu-id="05954-136">Yalnızca çağıran kodu değeri tarafından geçirilen değiştirilebilir bir öğenin değerini değiştirebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="05954-136">Only the calling code can change the value of a modifiable element passed by value.</span></span>  
  
## <a name="when-to-pass-an-argument-by-reference"></a><span data-ttu-id="05954-137">Başvuruya göre bağımsız değişken geçirmek ne zaman</span><span class="sxs-lookup"><span data-stu-id="05954-137">When to Pass an Argument by Reference</span></span>  
  
-   <span data-ttu-id="05954-138">Yordamı çağıran kodu temel alınan öğe değiştirmek için orijinal gereksinimi varsa, karşılık gelen bir parametre bildirme [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md).</span><span class="sxs-lookup"><span data-stu-id="05954-138">If the procedure has a genuine need to change the underlying element in the calling code, declare the corresponding parameter [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md).</span></span>  
  
-   <span data-ttu-id="05954-139">Çağrıyı yapan kod temel alınan öğe değiştirme yordamı doğru kodunun yürütülmesini bağımlı olması durumunda, parametre bildirme `ByRef`.</span><span class="sxs-lookup"><span data-stu-id="05954-139">If the correct execution of the code depends on the procedure changing the underlying element in the calling code, declare the parameter `ByRef`.</span></span> <span data-ttu-id="05954-140">Değere göre başarılı olursa ya da çağıran kodu geçersiz kılmaları `ByRef` parantez içinde bağımsız değişken kapsayan tarafından mekanizması geçirme yordam çağrısı beklenmeyen sonuçlara neden.</span><span class="sxs-lookup"><span data-stu-id="05954-140">If you pass it by value, or if the calling code overrides the `ByRef` passing mechanism by enclosing the argument in parentheses, the procedure call might produce unexpected results.</span></span>
