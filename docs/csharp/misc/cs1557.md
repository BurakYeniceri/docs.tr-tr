# <a name="compiler-error-cs1557"></a>Derleyici Hatası CS1557
Farklı çıkış dosyasında olduğundan 'sınıfı' Main yöntemi için kullanılamaz  
  
 [/Ana](../../csharp/language-reference/compiler-options/main-compiler-option.md) bir çok çıktı dosyası derleme dosyasına çıkış için derleyici seçeneği belirtildi. Ancak, sınıf / Main için kaynak kodundaki bulunamadı derleme; derlemedeki diğer çıktı dosyaları biri için kaynak kod dosyasında bulunamadı.
