<span data-ttu-id="889cd-103">Devralma yapısının bir iç içe geçmiş sınıflar arasında döngüsel bağımlılık diğer bir deyişle, iki sınıf birbirinden devralma sonuçlanır.</span><span class="sxs-lookup"><span data-stu-id="889cd-103">An inheritance structure results in circular dependency among nested classes, that is, two classes inheriting from each other.</span></span>

    End Class
