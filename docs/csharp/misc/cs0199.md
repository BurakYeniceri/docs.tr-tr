 <span data-ttu-id="6f460-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) değişken aynı olması gerekir [statik](../../csharp/language-reference/keywords/static.md) kullanım olarak geçirmek istediğiniz oluşturucusu olarak bir [ref](../../csharp/language-reference/keywords/ref.md) veya [çıkışı](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametresi.</span><span class="sxs-lookup"><span data-stu-id="6f460-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to pass it as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="6f460-105">Daha fazla bilgi için bkz: [geçirme parametreleri](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="6f460-105">For more information, see [Passing Parameters](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="6f460-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="6f460-106">Example</span></span>  
 <span data-ttu-id="6f460-107">Aşağıdaki örnek CS0199 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6f460-107">The following sample generates CS0199:</span></span>  
  
```csharp  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
