# <a name="compiler-error-cs0669"></a><span data-ttu-id="3d6d2-102">Derleyici Hatası CS0669</span><span class="sxs-lookup"><span data-stu-id="3d6d2-102">Compiler Error CS0669</span></span>

<span data-ttu-id="3d6d2-103">Bir sınıf ComImport özniteliğine sahip bir kullanıcı tarafından tanımlanan Oluşturucu sahip olamaz.</span><span class="sxs-lookup"><span data-stu-id="3d6d2-103">A class with the ComImport attribute cannot have a user-defined constructor.</span></span>

<span data-ttu-id="3d6d2-104">Ortak dil çalışma zamanı COM birlikte çalışma katmanda Oluşturucusu sağlayan [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) sınıfları.</span><span class="sxs-lookup"><span data-stu-id="3d6d2-104">The COM interop layer in the common language runtime supplies the constructor for [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) classes.</span></span> <span data-ttu-id="3d6d2-105">Sonuç olarak, COM nesnesi çalışma zamanında yönetilen bir nesne olarak kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="3d6d2-105">Consequently, a COM object can be used as a managed object in the runtime.</span></span>

<span data-ttu-id="3d6d2-106">Aşağıdaki örnek CS0669 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="3d6d2-106">The following sample generates CS0669:</span></span>

```csharp
// CS0669.cs
using System.Runtime.InteropServices;
[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
