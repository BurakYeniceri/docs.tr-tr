# <a name="compiler-error-cs0507"></a><span data-ttu-id="10c21-102">Derleyici Hatası CS0507</span><span class="sxs-lookup"><span data-stu-id="10c21-102">Compiler Error CS0507</span></span>
<span data-ttu-id="10c21-103">'function1': ' erişim ' üyesi 'function2' devralınan erişim değiştiricileri değiştirilemiyor</span><span class="sxs-lookup"><span data-stu-id="10c21-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="10c21-104">Bir yöntemi geçersiz kılmada erişim belirtimini değiştirmek için girişimde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="10c21-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="10c21-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="10c21-105">Example</span></span>  
 <span data-ttu-id="10c21-106">Aşağıdaki örnek CS0507 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="10c21-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="10c21-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="10c21-107">Example</span></span>  
 <span data-ttu-id="10c21-108">CS0507 da gerçekleşebilir bir sınıf olarak işaretli bir yöntem geçersiz kılma girişiminde `protected internal` başvurulan meta verilerde tanımlanan.</span><span class="sxs-lookup"><span data-stu-id="10c21-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="10c21-109">Bu durumda, geçersiz kılma yöntemi olarak işaretlenmelidir `protected`.</span><span class="sxs-lookup"><span data-stu-id="10c21-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs

abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="10c21-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="10c21-110">Example</span></span>  
 <span data-ttu-id="10c21-111">Aşağıdaki örnek CS0507 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="10c21-111">The following sample generates CS0507.</span></span>
