# <a name="the-folder-cannot-be-created-since-a-file-already-exists-with-the-same-path"></a>Bir dosya ile aynı yol zaten var. bu yana klasörü oluşturulamıyor
Bir dizin oluşturmak için bir girişimde bulunuldu, ancak bir dosya zaten var. Bu yoluna sahip. Bu yolları, özellikle göreli yollar ile çalışırken, yanlış ayrıştırma sonucunu olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Doğru yolu sağladığını denetleyin.  
  
-   Artık gerekli değilse yedek dosyayı kaldırın.
