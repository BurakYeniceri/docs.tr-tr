# <a name="developing-client-applications-with-the-net-framework"></a>.NET Framework ile istemci uygulamaları geliştirme

.NET Framework ile Windows tabanlı uygulamalar geliştirmek için birkaç yolu vardır. Bu araçlar ve çerçeveleri herhangi birini kullanabilirsiniz: 

* [Evrensel Windows Platformu (UWP)](https://developer.microsoft.com/windows/apps)
* [Windows Presentation Foundation (WPF)](../../docs/framework/wpf/index.md)
* [Windows Forms](../../docs/framework/winforms/index.md)

Bu bölüm, Windows Presentation Foundation kullanarak veya Windows formları kullanarak Windows tabanlı uygulamalar oluşturmak nasıl açıklayan konuları içerir. Ancak, .NET Framework ile web uygulamaları ve bilgisayarlar veya Microsoft Store kullanılabilir hale aygıtlar için istemci uygulamaları da oluşturabilirsiniz.

Windows Forms kullanarak uygulamaları geliştirme hakkında bilgi sağlar.

İstemci uygulamaları geliştirirken kullanılan ek teknolojiler hakkında bilgi sağlar.
