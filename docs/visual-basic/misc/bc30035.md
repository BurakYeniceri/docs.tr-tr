# <a name="syntax-error-visual-basic-error"></a>Sözdizimi hatası (Visual Basic hatası)
Kaynak kodu satır belirli olmayan bir sözdizimi hatası oluşturur.  
  
 **Hata Kimliği:** BC30035  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Her kaynak kod satırını kullanılan anahtar sözcükleri belgelerini inceleyin. Bu hata oluşturmadan kaynak satırı karşı belgelere örnek kodda karşılaştırın.  
  
2.  Hatanın nedenini tanımlayamıyor, koşullar hakkında bilgi toplamak ve Microsoft Ürün Destek Hizmetleri'ne bildirin.
