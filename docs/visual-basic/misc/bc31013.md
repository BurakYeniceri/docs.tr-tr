# <a name="unable-to-load-dll-39ltfilenamegt39-lterrorgt"></a>DLL yüklenemiyor &#39; &lt;filename&gt;&#39;: &lt;hata&gt;

 **Hata Kimliği:** BC31013  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Dosyanın belirtilen konumda olduğundan ve başka bir uygulama tarafından kilitlenmediğinden emin olun.
