<span data-ttu-id="6fc24-103">'member' extern olmalı ve gövde bildirme</span><span class="sxs-lookup"><span data-stu-id="6fc24-103">'member' cannot be extern and declare a body</span></span>  
  
 <span data-ttu-id="6fc24-104">Sınıf üyesine işaretli olduğunda [extern](../../csharp/language-reference/keywords/extern.md), üyenin tanımı başka bir dosyaya bulunur anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="6fc24-104">When a class member is marked [extern](../../csharp/language-reference/keywords/extern.md), it means that the member's definition is located in another file.</span></span> <span data-ttu-id="6fc24-105">Bu nedenle, bir sınıf üyesi olarak işaretlenmiş **extern** sınıfında tanımlanamaz.</span><span class="sxs-lookup"><span data-stu-id="6fc24-105">Therefore, a class member marked as **extern** cannot be defined in the class.</span></span> <span data-ttu-id="6fc24-106">Kaldırın veya `extern` anahtar sözcüğü ya da delete tanımı.</span><span class="sxs-lookup"><span data-stu-id="6fc24-106">Either remove the `extern` keyword or delete the definition.</span></span> <span data-ttu-id="6fc24-107">Daha fazla bilgi için bkz: [yöntemleri](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="6fc24-107">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="6fc24-108">Aşağıdaki örnek CS0179 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6fc24-108">The following sample generates CS0179:</span></span>

```csharp  
   // try the following line instead

   }

```
