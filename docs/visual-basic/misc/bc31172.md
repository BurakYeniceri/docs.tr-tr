# <a name="an-embedded-expression-may-not-be-used-here"></a>Katıştırılmış bir ifade burada kullanılamaz.
Katıştırılmış bir ifade değişmez değer XML desteklenmeyen bir konumda eklenmiştir. Katıştırılmış ifadeler bir XML ad alanı değeri belirtmek için kullanılamayacağını unutmayın.  
  
 **Hata Kimliği:** BC31172  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Katıştırılmış ifade desteklenmeyen konumdan kaldırın.
