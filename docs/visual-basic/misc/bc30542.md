# <a name="division-by-zero-occurred-while-evaluating-this-expression"></a>Bu ifade hesaplanırken sıfıra bölme oluştu
Bu ifade değerlendirmede bir sayı sıfıra bölünme girişiminde bulunuldu.  
  
 **Hata Kimliği:** BC30542  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Deyim içindeki payda sıfır nasıl gerçekleştiğini belirlemek için sabit ve değişken değerlerini denetleyin.
