## <a name="example"></a><span data-ttu-id="b3b15-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="b3b15-107">Example</span></span>  
 <span data-ttu-id="b3b15-108">Aşağıdaki örnek CS0191 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="b3b15-108">The following sample generates CS0191.</span></span>  
  
```csharp  
// CS0191.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  // OK to assign to readonly field in declaration  
  
    MyClass()  
    {  
        TestInt = 11; // OK to assign to readonly field in constructor  
    }  
  
    public void TestReadOnly()  
    {  
        TestInt = 19;                  // CS0191  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
