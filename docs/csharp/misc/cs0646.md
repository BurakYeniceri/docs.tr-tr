# <a name="compiler-error-cs0646"></a><span data-ttu-id="893a7-102">Derleyici Hatası CS0646</span><span class="sxs-lookup"><span data-stu-id="893a7-102">Compiler Error CS0646</span></span>
<span data-ttu-id="893a7-103">Bir dizin oluşturucu içeren bir tür üzerinde DefaultMember özniteliği belirtilemez</span><span class="sxs-lookup"><span data-stu-id="893a7-103">Cannot specify the DefaultMember attribute on a type containing an indexer</span></span>  
  
 <span data-ttu-id="893a7-104">Bir sınıf veya diğer tür belirtiyorsa **System.Reflection.DefaultMemberAttribute**, bir dizin oluşturucu içeremez.</span><span class="sxs-lookup"><span data-stu-id="893a7-104">If a class or other type specifies **System.Reflection.DefaultMemberAttribute**, it cannot contain an indexer.</span></span> <span data-ttu-id="893a7-105">Daha fazla bilgi için bkz: [özellikleri](../../csharp/programming-guide/classes-and-structs/properties.md).</span><span class="sxs-lookup"><span data-stu-id="893a7-105">For more information, see [Properties](../../csharp/programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="893a7-106">Aşağıdaki örnek CS0646 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="893a7-106">The following sample generates CS0646:</span></span>  
  
```csharp  
// CS0646.cs  
// compile with: /target:library  
[System.Reflection.DefaultMemberAttribute("x")]   // CS0646  
class MyClass  
{  
   public int this[int index]   // an indexer  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public int x = 0;  
}  
  
// OK  
[System.Reflection.DefaultMemberAttribute("x")]  
class MyClass2  
{  
   public int prop  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public int x = 0;  
}  
  
class MyClass3  
{  
   public int this[int index]   // an indexer  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public int x = 0;  
}  
```
