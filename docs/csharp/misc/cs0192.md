# <a name="compiler-error-cs0192"></a><span data-ttu-id="cb43a-102">Derleyici Hatası CS0192</span><span class="sxs-lookup"><span data-stu-id="cb43a-102">Compiler Error CS0192</span></span>
<span data-ttu-id="cb43a-103">Statik salt okunur alan 'name' alanlarının ref geçirilen olamaz veya out (statik bir oluşturucu hariç)</span><span class="sxs-lookup"><span data-stu-id="cb43a-103">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="cb43a-104">Bir alan (değişken) işaretlenmiş [readonly](../../csharp/language-reference/keywords/readonly.md) anahtar sözcüğü olamaz bayraklarıdır ya da bir [ref](../../csharp/language-reference/keywords/ref.md) veya [çıkışı](../../csharp/language-reference/keywords/out-parameter-modifier.md) yapıcı içinde dışında parametre.</span><span class="sxs-lookup"><span data-stu-id="cb43a-104">A field (variable) marked with the [readonly](../../csharp/language-reference/keywords/readonly.md) keyword cannot be passed either to a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter except inside a constructor.</span></span> <span data-ttu-id="cb43a-105">Daha fazla bilgi için bkz: [alanları](../../csharp/programming-guide/classes-and-structs/fields.md).</span><span class="sxs-lookup"><span data-stu-id="cb43a-105">For more information, see [Fields](../../csharp/programming-guide/classes-and-structs/fields.md).</span></span>  
  
 <span data-ttu-id="cb43a-106">CS0192 ayrıca sonuçları, `readonly` alandır [statik](../../csharp/language-reference/keywords/static.md) ve Oluşturucusu işaretlenmemiş `static`.</span><span class="sxs-lookup"><span data-stu-id="cb43a-106">CS0192 also results if the `readonly` field is [static](../../csharp/language-reference/keywords/static.md) and the constructor is not marked `static`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cb43a-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="cb43a-107">Example</span></span>  
 <span data-ttu-id="cb43a-108">Aşağıdaki örnek CS0192 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="cb43a-108">The following sample generates CS0192.</span></span>  
  
```csharp 
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
