# <a name="39ltpropertynamegt39-implicitly-defines-39ltmembernamegt39-which-conflicts-with-a-member-of-the-same-name-in-lttypegt-39lttypenamegt39"></a>&#39;&lt;PropertyName&gt; &#39; örtük olarak tanımlayan &#39; &lt;membername&gt;&#39;, aynı ada sahip bir üyeyle çakışan &lt;türü&gt; &#39; &lt; TypeName&gt;&#39;
Örtük olarak bir özellik için oluşturulmuş bir üyenin adını adıyla türü üye çakışıyor. Özellikleri örtük olarak tanımlanan üyeleriyle çakışabilir birkaç örtük değişkenleri oluşturun.  
  
 **Hata Kimliği:** BC31060  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Adlandırma çakışması kaldırmak için açıkça bildirilen üye yeniden adlandırın.
