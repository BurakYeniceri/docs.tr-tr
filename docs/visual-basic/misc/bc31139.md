# <a name="39ltnamegt39-cannot-refer-to-itself-through-its-default-instance-use-39me39-instead"></a>&#39; &lt;adı&gt;&#39; varsayılan örneği, kullanımı &#39; kendisine başvuramaz &#39; Bunun yerine
Bir gelen bir form içinde bu formda varsayılan bir örnek olarak başvurmak için girişimde bulunuldu. Bu, formun kendisi özyinelemeli olarak aramak neden olabilir.  
  
 Çoğu durumda, kullanmanız gereken `Me` geldiğinde varsayılan örnek yerine formun geçerli örneğine başvurma.  
  
 **Hata Kimliği:** BC31139  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanım `Me` nesneye başvurmak için.
