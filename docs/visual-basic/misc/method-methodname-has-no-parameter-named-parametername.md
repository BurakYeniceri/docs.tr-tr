# <a name="method-39ltmethodnamegt39-has-no-parameter-named-39ltparameternamegt39"></a>Yöntem &#39; &lt;methodname&gt; &#39; sahip adlı bir parametre &#39; &lt;parametername&gt;&#39;
Adlandırılmış bağımsız değişkeni sağlanan hedef yöntemin bağımsız değişkenlerden biri eşleşmiyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Adlandırılmış bağımsız değişkeni doğru sağlandı emin olun.
