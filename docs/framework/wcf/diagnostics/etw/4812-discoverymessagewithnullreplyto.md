|||  
|-|-|  
|Kimlik|4812|  
|Anahtar Sözcükler|Bulma|  
|Düzey|Uyarı|  
|Kanal|Microsoft Windows uygulaması sunucu-uygulamalar/hata ayıklama|  
  
## <a name="description"></a>Açıklama  
 Bu olay, bir ReplyTo adresi içermediğinden bulma ileti DiscoveryClient tarafından bırakıldı zaman yayınlanır.  
  
## <a name="message"></a>İleti  
 Bir bulma isteğine MessageID = '%1' ReplyTo adresine sahip olmadığından bırakıldı.
