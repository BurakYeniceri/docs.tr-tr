# <a name="compiler-error-cs0601"></a><span data-ttu-id="0427e-102">Derleyici Hatası CS0601</span><span class="sxs-lookup"><span data-stu-id="0427e-102">Compiler Error CS0601</span></span>
<span data-ttu-id="0427e-103">Bir 'static' olarak işaretlenmiş bir yöntem ve 'extern' DllImport özniteliği belirtilmelidir</span><span class="sxs-lookup"><span data-stu-id="0427e-103">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="0427e-104">`DllImport` Özniteliği doğru erişim anahtar olmayan bir yöntem kullanıldı.</span><span class="sxs-lookup"><span data-stu-id="0427e-104">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="0427e-105">Aşağıdaki örnek CS0601 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="0427e-105">The following sample generates CS0601:</span></span>  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
