# <a name="isymunmanagedvariablegetsignature-method"></a>ISymUnmanagedVariable::GetSignature Metodu
Bu değişken imzasını alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT GetSignature(  
    [in]  ULONG32  cSig,  
    [out] ULONG32  *pcSig,  
    [out, size_is(cSig),  
        length_is(*pcSig)] BYTE sig[]);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `cSig`  
 [in] Tarafından için arabellek uzunluğu işaret `sig` parametresi.  
  
 `pcSig`  
 [out] Bir işaretçi bir `ULONG32` karakter imza içermesi gerekir arabellek boyutunu alır.  
  
 `sig`  
 [out] İmza depolayan bir arabellek.  
  
## <a name="return-value"></a>Dönüş Değeri  
 Yöntem başarılı olursa S_OK; Aksi takdirde E_FAIL veya başka bir hata kodu.  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** CorSym.idl, CorSym.h
