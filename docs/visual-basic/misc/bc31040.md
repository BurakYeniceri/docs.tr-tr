# <a name="39ltmembernamegt39-exists-in-multiple-base-interfaces"></a>&#39; &lt;membername&gt;&#39; birden fazla temel arabirimde var.
'\<membername >' birden fazla temel arabirimde bulunuyor. Bildirir arabirimi adını kullanmak '\<membername >' türetilmiş bir arabirim adı yerine 'Implements' yan tümcesinde.  
  
 Bu arabirim üyeleri aynı ada sahip birden çok arabirimlerden belirsizlik oluşturma, devralır.  
  
 **Hata Kimliği:** BC31040  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tanımlama arabirimi adında kullanmak `Implements` yan tümceleri türetilmiş bir arabirim adı yerine.
