# <a name="compiler-error-cs0424"></a><span data-ttu-id="03402-102">Derleyici Hatası CS0424</span><span class="sxs-lookup"><span data-stu-id="03402-102">Compiler Error CS0424</span></span>
<span data-ttu-id="03402-103">'class': ComImport özniteliğine sahip bir sınıf bir taban sınıf belirtilemez</span><span class="sxs-lookup"><span data-stu-id="03402-103">'class': a class with the ComImport attribute cannot specify a base class</span></span>  
  
 <span data-ttu-id="03402-104">Belirtme <xref:System.Runtime.InteropServices.ComImportAttribute> özniteliği gelir sınıfı için uygulama COM modülünden içe aktarılmasını olduğunu.</span><span class="sxs-lookup"><span data-stu-id="03402-104">Specifying the <xref:System.Runtime.InteropServices.ComImportAttribute> attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="03402-105">Ek yöntemleri veya temel sınıfından devralınan alanları COM modülde tanımlanmış uygulamaya eklenecek izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="03402-105">Additional methods or fields inherited from the base class are not allowed to be added to the implementation defined in the COM module.</span></span>  
  
 <span data-ttu-id="03402-106">Aşağıdaki örnek CS0424 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="03402-106">The following sample generates CS0424:</span></span>

