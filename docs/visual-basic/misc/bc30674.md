# <a name="39end-synclock39-must-be-preceded-by-a-matching-39synclock39"></a>&#39;Son SyncLock&#39; eşleşen bir gelmelidir &#39;SyncLock&#39;
`SyncLock` ile başlayan blokları `SyncLock` anahtar sözcüğü ve sonu `End SyncLock` oluşturun.  
  
 **Hata Kimliği:** BC30674  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Emin olun `SyncLock` blok ile başlayan bir `SyncLock` deyimi.
