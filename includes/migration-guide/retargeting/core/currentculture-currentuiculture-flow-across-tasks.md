### <a name="currentculture-and-currentuiculture-flow-across-tasks"></a>Görevler arasında CurrentCulture ve CurrentUICulture akışı

|   |   |
|---|---|
|Ayrıntılar|.NET Framework 4.6 başlayarak <xref:System.Globalization.CultureInfo.CurrentCulture?displayProperty=name> ve <xref:System.Globalization.CultureInfo.CurrentUICulture?displayProperty=name> iş parçacığı içinde depolanan <xref:System.Threading.ExecutionContext?displayProperty=name>, zaman uyumsuz işlemler arasındaki akar. Değişikliklerini Bunun anlamı <xref:System.Globalization.CultureInfo.CurrentCulture?displayProperty=name> veya <xref:System.Globalization.CultureInfo.CurrentUICulture?displayProperty=name> daha sonra zaman uyumsuz olarak çalışan görevler yansıtılır. Bu önceki .NET Framework sürümü davranışından farklıdır (sıfırlama <xref:System.Globalization.CultureInfo.CurrentCulture?displayProperty=name> ve <xref:System.Globalization.CultureInfo.CurrentUICulture?displayProperty=name> tüm zaman uyumsuz görevler).|
|Öneri|Bu değişiklikten etkilenen uygulamalar iş çevresinde açıkça istenen ayarlayarak <xref:System.Globalization.CultureInfo.CurrentCulture?displayProperty=name> veya <xref:System.Globalization.CultureInfo.CurrentUICulture?displayProperty=name> zaman uyumsuz Görev ilk işlemde olarak. Alternatif olarak, eski davranışı (değil boyunca <xref:System.Globalization.CultureInfo.CurrentCulture?displayProperty=name> / <xref:System.Globalization.CultureInfo.CurrentUICulture?displayProperty=name>) içine aşağıdaki uyumluluk anahtarını ayarlayarak seçti:<pre><code class="lang-csharp">AppContext.SetSwitch(&quot;Switch.System.Globalization.NoAsyncCurrentCulture&quot;, true);&#13;&#10;</code></pre>.NET Framework 4.6.2 WPF'de tarafından bu sorun düzeltilmiştir. .NET Framework 4. 6, 4.6.1 aracılığıyla da düzeltilmiştir [KB 3139549](https://support.microsoft.com/kb/3139549). .NET Framework 4.6 veya sonrasını hedefleyen uygulamalar otomatik olarak elde edersiniz doğru davranış WPF uygulamalarında - <xref:System.Globalization.CultureInfo.CurrentCulture?displayProperty=name> / <xref:System.Globalization.CultureInfo.CurrentUICulture?displayProperty=name>) dağıtıcısı işlemler arasındaki korunması.|
