# <a name="compiler-warning-level-2-cs1668"></a>Derleyici Uyarısı (Düzey 2) CS1668
'Path' 'yol dizesi'--'sistem hatası iletisi' belirtilen geçersiz Aram yolu  
  
 Sağlanan yol [/lib](../../csharp/language-reference/compiler-options/lib-compiler-option.md) komut satırı geçerli değildi veya LIB ortam değişkeni olarak bir yolu geçersiz. Yolun var olduğunu ve erişilebilir olduğunu doğrulamak için kullanılan denetleyin. Tek tırnak işaretleri içindeki hata işletim sisteminden döndürülen hata iletisidir.
