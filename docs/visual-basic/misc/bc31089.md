# <a name="types-declared-39private39-must-be-inside-another-type"></a>Bildirilen türleri &#39;özel&#39; içinde başka bir tür olmalıdır
`Private` Değiştiricisi türünde başka bir tür içinde değil kullanıldı.  
  
 **Hata Kimliği:** BC31089  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Daha az kısıtlayıcı bir erişim değiştiricisi gibi kullandığınız `Friend`.  
  
2.  Başka bir tür içindeki tür bildirin.
