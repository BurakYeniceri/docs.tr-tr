 <span data-ttu-id="50d17-104">Anonim bir yöntem parametresi türü değiştiricisi yöntemi atama temsilci bildiriminde kullanılan eşleşmiyor Bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="50d17-104">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>

 <span data-ttu-id="50d17-106">Aşağıdaki örnek CS1677 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="50d17-106">The following sample generates CS1677:</span></span>

```csharp  
    static void Main()

        // D d = delegate(int j){};  
    }  
}  
```
