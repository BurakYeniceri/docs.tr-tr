# <a name="compiler-error-cs1563"></a>Derleyici Hatası CS1563
Çıktı 'çıktı dosyası' kaynak dosya yok  
  
 [/Out](../../csharp/language-reference/compiler-options/out-compiler-option.md) derleyici seçeneği belirtildi, ancak hiçbir kaynak kodu dosyaları izleyin. Ya da kullanılamaz **/out**, ya da aşağıdaki kaynak kodu dosyaları belirtmelisiniz **/out**.
