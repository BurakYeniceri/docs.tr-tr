# <a name="file-is-too-large-to-read-into-a-byte-array"></a>Dosya bir bayt dizisine okunamayacak kadar büyük
Bir bayt dizisine okunamayacak kadar çalıştığınız dosya boyutu 4 GB aşıyor. `My.Computer.FileSystem.ReadAllBytes` Yöntemi bu boyuttan daha büyük bir dosya okunamaz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanım bir <xref:System.IO.StreamReader> dosya okunamıyor.
