# <a name="labels-are-not-valid-in-the-immediate-window"></a>Etiketler komut penceresi içinde geçerli değildir
Çizgi etiketleri dallanma etkinleştirmek ve bir hata ayıklama bağlamında izin verilmez.  
  
 **Hata Kimliği:** BC30134  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir satır etiket girmeyin **hemen** penceresi.
