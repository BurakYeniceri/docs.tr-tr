# <a name="argument-not-specified-for-parameter-39ltparameternamegt39-of-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Parametresi için bağımsız değişken belirtilmemiş &#39; &lt;parametername&gt; &#39; genişletme yöntemi, &#39; &lt;methodname&gt; &#39; tanımlanan &#39; &lt;typename&gt;&#39;
Bir genişletme yöntemi, gerekli parametre için bir bağımsız değişken sağlanmadı.  
  
 **Hata Kimliği:** BC36586  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirtilen parametre için bir bağımsız değişken sağlayın.
