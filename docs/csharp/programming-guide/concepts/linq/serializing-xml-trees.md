# <a name="serializing-xml-trees-c"></a>Biçimlendiricisi XML ağaçları (C#)
Bir XML ağacı seri hale getirme XML ağacından XML oluşturma anlamına gelir. Bir dosyaya somut bir uyarlamasını serileştirebilen <xref:System.IO.TextWriter> sınıfı veya somut bir uygulaması için bir <xref:System.Xml.XmlWriter>.  
  
 Seri hale getirme çeşitli yönlerini kontrol edebilir. Örneğin, serileştirilmiş XML girinti mı yoksa ve bir XML bildirimi yazılıp yazılmayacağını kontrol edebilirsiniz.
