# <a name="overflow-visual-basic-error"></a>Taşma (Visual Basic Hatası)
Bir hazır değer bu atandığı veri türünün sınırları dışındaki bir değeri temsil eder.  
  
 **Hata Kimliği:** BC30036  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Veri türü ve sabit bu aralığa uyacak şekilde yeniden hedefi için değer aralığı başvurun.
