 <span data-ttu-id="ecd1f-104">C# dil belirtimi güvenli olmayan kod içinde yineleyiciler izin vermiyor.</span><span class="sxs-lookup"><span data-stu-id="ecd1f-104">The C# language specification does not allow unsafe code in iterators.</span></span>  
  
 <span data-ttu-id="ecd1f-105">Aşağıdaki örnek CS1629 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="ecd1f-105">The following sample generates CS1629:</span></span>  
  
```csharp  
// CS1629.cs

class C   
{  
   IEnumerator<int> IteratorMeth() {  
      int i;

      }  
   }  
}
```
