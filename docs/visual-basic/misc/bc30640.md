# <a name="39option-explicit39-can-be-followed-only-by-39on39-or-39off39"></a>&#39; seçeneği açık &#39; yalnızca izlenebilir &#39; üzerinde &#39; ya &#39; kapalı &#39;
`Option Explicit`deyimleri ya da belirtmeniz gerekir `On` veya `Off`. Diğer tüm değerler geçersizdir.  
  
 **Hata Kimliği:** BC30640  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirtin `Option Explicit On` veya `Option Explicit Off`.
