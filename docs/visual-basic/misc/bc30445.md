# <a name="const-declaration-cannot-have-an-array-initializer"></a>Const bildirimi bir dizi Başlatıcı sahip olamaz
Bir dizi Başlatıcı kullanma girişiminde bir `Const` bildirimi.  
  
 **Hata Kimliği:** BC30445  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Başlatıcı çevresindeki küme ayraçları kaldırarak için geçerli bir ifade değiştirin.
