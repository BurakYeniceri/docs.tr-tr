# <a name="methods-declared-in-structures-cannot-have-39handles39-clauses"></a>Yapıların içinde bildirilen yöntemlerde olamaz &#39;işleme&#39; yan tümceleri
Yapı yöntemlerini kullanamıyor `Handles` olayları işlemek için anahtar sözcüğü.  
  
 **Hata Kimliği:** BC30728  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir sınıf olarak yapınızı yeniden göz önünde bulundurun.  
  
     Kullanabileceğiniz `AddHandler` yapısı bir arabiriminde tanımlanan olay işleyicisi uygular koşuluyla, bir olay bir yapı, olay işleyici ilişkilendirmek için.
