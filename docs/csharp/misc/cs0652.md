 <span data-ttu-id="294da-104">Derleyici bir sabit ve sabiti değişkeni aralık dışında olduğu bir değişken arasında bir karşılaştırma algıladı.</span><span class="sxs-lookup"><span data-stu-id="294da-104">The compiler detected a comparison between a constant and a variable where the constant is out of the range of the variable.</span></span>  
  
 <span data-ttu-id="294da-105">Aşağıdaki örnek CS0652 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="294da-105">The following sample generates CS0652:</span></span>  
  
```csharp  
// CS0652.cs  
// compile with: /W:2  
public class Class1  
{  
   private static byte i = 0;  
   public static void Main()  
   {  
      short j = 256;  
      if (i == 256)   // CS0652, 256 is out of range for byte  
         i = 0;  
   }  
}  
```
