# <a name="icordebugappdomainenum-interface1"></a>Icordebugappdomainenum Interface1
Sağlar `Next` belirtilen sayıda döndürür yöntemi `ICorDebugAppDomainEnum` listedeki sonraki konumunda başlayan değer. Bu arabirim "ICorDebugEnum" sınıfıdır.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[Next yöntemi](../../../../docs/framework/unmanaged-api/debugging/icordebugappdomainenum-next-method.md)|Uygulama etki alanları belirtilen sayıda geçerli İmleç konumuna başlangıç koleksiyondan alır.|  
  
## <a name="remarks"></a>Açıklamalar  
  
> [!NOTE]
>  Bu arabirim, makineler arası veya çapraz işlem uzaktan çağrılan desteklemez.
