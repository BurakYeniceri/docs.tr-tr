# <a name="39notoverridable39-cannot-be-specified-on-methods-that-do-not-override-another-method"></a>&#39;NotOverridable&#39; başka bir yöntem geçersiz kılmaz yöntemlerde belirtilemez
Özellikleri ve yöntemleri `NotOverridable` varsayılan olarak. `NotOverridable` Değiştiricisi, yalnızca başka bir özellik geçersiz kılma yöntemleri veya yöntem kullanılabilir.  
  
 **Hata Kimliği:** BC31088  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `NotOverridable` özellikleri ve başka bir üye geçersiz kılmaz yöntemleri değiştiricisi.
