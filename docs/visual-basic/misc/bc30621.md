# <a name="39end-structure39-must-be-preceded-by-a-matching-39structure39"></a>&#39; Son yapısını &#39; eşleşen bir &#39;göre gelmelidir; Yapı &#39;
`End Structure` Deyimi yalnızca kullanılabilir ile başlayan bir yapı bloğu sonunda bir `Structure` deyimi.  
  
 **Hata Kimliği:** BC30621  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçerli Yapı bloğu ile başladığından emin olun bir `Structure` deyimi.
