# <a name="developing-client-applications-with-the-net-framework"></a>.NET Framework ile İstemci Uygulamaları Geliştirme
Kullanıcıların bilgisayarları veya cihazları yerel olarak çalışan Windows tabanlı uygulamalar .NET framework ile geliştirmek için birden çok yolu vardır. Bu bölüm, Windows Presentation Foundation (WPF) kullanarak veya Windows formları kullanarak Windows tabanlı uygulamalar oluşturmak nasıl açıklayan konuları içerir. Ancak, .NET Framework kullanarak web uygulamaları ve istemci uygulamalarını bilgisayarları veya Windows mağazası veya Windows Phone mağazası kullanılabilir hale de oluşturabilirsiniz.
