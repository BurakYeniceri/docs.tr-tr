# <a name="connectiongroup-class"></a>ConnectionGroup sınıfı

`ConnectionGroup` Sınıfı grupları içinde bağlantılarının bir listesini <xref:System.Net.ServicePoint> bağlamı ve ağ kaynakları (örneğin, proxy ve ayrı istemcilerin) için içerik korumak için kullanılır.

## <a name="syntax"></a>Sözdizimi
  
```csharp  
internal class ConnectionGroup
```

> [!WARNING]
> `ConnectionGroup` Sınıfı, iç ve kodunuzda doğrudan kullanılmak üzere yüksetlmesi.
> 
> Microsoft hiçbir koşulda bir üretim uygulamasında bu sınıf kullanımını desteklemez.

## <a name="requirements"></a>Gereksinimler

**Namespace:** <xref:System.Net>

**Derleme:** sisteminde (System.dll)

**.NET framework sürümleri:** 2.0 sürümünden itibaren kullanılabilir.
