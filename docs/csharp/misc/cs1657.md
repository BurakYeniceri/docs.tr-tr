# <a name="compiler-error-cs1657"></a><span data-ttu-id="619f2-102">Derleyici Hatası CS1657</span><span class="sxs-lookup"><span data-stu-id="619f2-102">Compiler Error CS1657</span></span>
<span data-ttu-id="619f2-103">'Parametresi' bir ref veya out bağımsız değişkeni çünkü geçirilemez ' nedeni ''</span><span class="sxs-lookup"><span data-stu-id="619f2-103">Cannot pass 'parameter' as a ref or out argument because 'reason''</span></span>  
  
 <span data-ttu-id="619f2-104">Bir değişken olarak geçirilen bu hata oluşur. bir [ref](../../csharp/language-reference/keywords/ref.md) veya [çıkışı](../../csharp/language-reference/keywords/out-parameter-modifier.md) bağımsız değişken salt okunur olan bağlamda.</span><span class="sxs-lookup"><span data-stu-id="619f2-104">This error occurs when a variable is passed as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) argument in a context in which that variable is readonly.</span></span> <span data-ttu-id="619f2-105">Salt okunur bağlamları dahil [foreach](../../csharp/language-reference/keywords/foreach-in.md) yineleme değişkenleri [kullanarak](../../csharp/language-reference/keywords/using-statement.md) değişkenleri ve `fixed` değişkenleri.</span><span class="sxs-lookup"><span data-stu-id="619f2-105">Readonly contexts include [foreach](../../csharp/language-reference/keywords/foreach-in.md) iteration variables, [using](../../csharp/language-reference/keywords/using-statement.md) variables, and `fixed` variables.</span></span> <span data-ttu-id="619f2-106">Bu hatayı gidermek için almayan işlevleri çağırmayın `foreach`, `using` veya `fixed` değişken olarak bir `ref` veya `out` parametresinde `using` blokları, `foreach` deyimleri ve `fixed` deyimleri.</span><span class="sxs-lookup"><span data-stu-id="619f2-106">To resolve this error, do not call functions that take the `foreach`, `using` or `fixed` variable as a `ref` or `out` parameter in `using` blocks, `foreach` statements, and `fixed` statements.</span></span>  
  
## <a name="example"></a><span data-ttu-id="619f2-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="619f2-107">Example</span></span>  
 <span data-ttu-id="619f2-108">Aşağıdaki örnek CS1657 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="619f2-108">The following sample generates CS1657:</span></span>  
  
```csharp  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```
