# <a name="the-connectiontimeout-must-be-greater-than-0"></a>ConnectionTimeout 0'dan büyük olmalıdır
Karşıya yükleme ve indirme zaman dosyaları ile [My.Computer.Network](xref:Microsoft.VisualBasic.Devices.Network), belirtmeniz gerekir bir `connectionTimeout` büyük `0`.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tedarik bir `connectionTimeout` daha büyük `0`.
