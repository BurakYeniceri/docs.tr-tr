# <a name="expected-closing-39gt39-for-xml-processor-instruction"></a>Beklenen kapanış &#39;? &gt;&#39; XML işlemci yönergesi
XML işleme yönergesi değişmez değeri gerekli kapanış eksik `?>` karakter.  
  
 **Hata Kimliği:** BC31160  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gerekli kapanış eklemek `?>` XML işleme yönergesi değişmez değeri sonuna karakter.
