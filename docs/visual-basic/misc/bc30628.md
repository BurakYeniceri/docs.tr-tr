# <a name="structures-cannot-have-39inherits39-statements"></a>Yapıları olamaz &#39; Inherits &#39; deyimleri
Sınıflar, yapılar devralma desteklemez.  
  
 **Hata Kimliği:** BC30628  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Inherits` yapısından deyimi.  
  
-   Uygulamanızı sınıfları yerine kullanacak şekilde yeniden tasarlamanız.
