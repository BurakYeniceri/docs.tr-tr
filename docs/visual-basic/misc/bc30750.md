# <a name="information-for-the-type-of-39lttypenamegt39-has-not-been-loaded-into-the-runtime"></a>Bilgi türü &#39; &lt;typename&gt;&#39; çalışma zamanına yüklendi değil
Bir tür, çalışma zamanı tarafından yüklenmemiş başvuruldu.  
  
 **Hata Kimliği:** BC30750  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Türü için geçerli kapsamda tanımlı böylece kodunuzu yeniden yapılandırabilirsiniz.  
  
2.  Üye tanımlandı ve, üye adının doğru yazıldığından olduğunu doğrulayın.  
  
3.  Modülde tanımlanmış üyelerinden erişmeyi deneyin. Bazı durumlarda, burada bildirilen modülleri yüklenmedi olmadığından üyeleri hata ayıklama ortamı bulunamıyor.
