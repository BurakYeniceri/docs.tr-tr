# <a name="file-already-exists"></a>Dosya zaten var.
Bu hata, varolan bir dosya adı ile yeni bir dosya adlandırma veya kullanarak sonucunda oluşur **Kaydet** yüklü bir proje zaten mevcut bir adla kaydetme komutu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ad ve ya da değil çakışma veya artık gerekli değilse önceden var olan dosya veya proje silme, değiştirme denetleyin.
