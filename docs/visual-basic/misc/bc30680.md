# <a name="39end-region39-must-be-preceded-by-a-matching-39region39"></a>&#39;#End region&#39; eşleşen bir gelmelidir &#39;#Region&#39;
İle `#Region` genişletmek veya Visual Studio Kod Düzenleyicisi'nin anahat özelliğini kullanırken daraltmak için kod bloğunu belirtebilirsiniz. Başlangıcı ve sonu `#Region` deyimleri, aynı kod bloğunda olmalıdır.  
  
 **Hata Kimliği:** BC30680  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   INSERT `#Region` karşılık gelen önce uygun yerde `#End``Region` deyimi.
