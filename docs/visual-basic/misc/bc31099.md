# <a name="property-accessors-cannot-be-declared-39ltkeywordgt39"></a>Özellik erişimcisi bildirilemez &#39; &lt;anahtar sözcüğü&gt;&#39;
A `Get` veya `Set` yordamı bildirimi bir özellik yordamı üzerinde geçerli değil bir anahtar sözcük içerir.  
  
 [Alma deyimi](../../visual-basic/language-reference/statements/get-statement.md) ve [Set deyimi](../../visual-basic/language-reference/statements/set-statement.md) yalnızca bir erişim değiştiricisi izin ver (`Public`, `Protected`, `Friend`, `Protected Friend`, `Private`).  
  
 **Hata Kimliği:** BC31099  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçersiz anahtar sözcüğü kaldırmak `Get` veya `Set` deyimi.
