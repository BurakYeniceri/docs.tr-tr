# <a name="xml-documentation-parse-error-start-tag-39lttaggt39-doesn39t-have-a-matching-end-tag"></a>XML belgesi ayrıştırma hatası: Etiket &#39; Başlat &lt;etiketi&gt;&#39; belirlemeden #39; eşleşen bitiş etiketi sahip t
XML belgesi ayrıştırma hatası: başlangıç etiketi \<etiketi > eşleşen bitiş etiketi yok. XML açıklama yoksayılacak.  
  
 XML açıklama bir başlangıç etiketi içeriyor ancak eşleşen bir bitiş etiketi içermiyor.  
  
 **Hata Kimliği:** BC42316  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Başlangıç etiketi ile eşleşen bir bitiş etiketi ekleyin.  
  
     — veya —  
  
-   Etiket yok iç metni gibi içeriyorsa [ \<seealso >](../../visual-basic/language-reference/xmldoc/seealso.md), eğik kapanış açılı ayraç önce belirtin.
