# <a name="properties-declared-39writeonly39-cannot-have-a-39get39"></a>Bildirilen özellikler &#39;WriteOnly&#39; olamaz bir &#39;Al&#39;
`Get` Yordamı bir özelliğin değerini okur. `WriteOnly` özellikleri okunamıyor.  
  
 **Hata Kimliği:** BC30023  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `WriteOnly` anahtar sözcüğünün `Property` deyimi veya kaldırma `Get` özellik gövdesi yordamdan.
