 <span data-ttu-id="1226d-179">[ \<Yönergeleri >](../../../docs/framework/net-native/directives-element-net-native.md) sıfır veya bir öğe içerebilir `<Application>` öğesi.</span><span class="sxs-lookup"><span data-stu-id="1226d-179">The [\<Directives>](../../../docs/framework/net-native/directives-element-net-native.md) element can contain zero or one `<Application>` element.</span></span> <span data-ttu-id="1226d-180">Birden çok `<Application>` bir tek yansıma yönergeleri dosyasındaki öğeleri desteklenmiyor.</span><span class="sxs-lookup"><span data-stu-id="1226d-180">Multiple `<Application>` elements in a single reflection directives file are not supported.</span></span>
