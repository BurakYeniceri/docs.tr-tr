# <a name="compiler-error-cs1519"></a><span data-ttu-id="b7a13-102">Derleyici Hatası CS1519</span><span class="sxs-lookup"><span data-stu-id="b7a13-102">Compiler Error CS1519</span></span>
<span data-ttu-id="b7a13-103">Geçersiz belirteç 'belirteci sınıf, yapı veya arabirim üyesi bildirimi</span><span class="sxs-lookup"><span data-stu-id="b7a13-103">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="b7a13-104">Bu hata, bir belirteç ait bir konumda karşılaşıldığında oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="b7a13-104">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="b7a13-105">A *belirteci* bir anahtar sözcük; bir tanımlayıcı (sınıf, yapı, yöntem ve benzeri adı); bir dize, karakter veya 108 gibi sayısal değişmez değer 'Hello", veya ' bir'; veya bir işleç veya noktalama gibi işaretçisi `==` veya `;`.</span><span class="sxs-lookup"><span data-stu-id="b7a13-105">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="b7a13-106">Tüm [sınıfı](../../../csharp/language-reference/keywords/class.md), yapı veya türünden önce geçersiz değiştirici içeren arabirim üyesi bildirimi bu hatayı üretir.</span><span class="sxs-lookup"><span data-stu-id="b7a13-106">Any [class](../../../csharp/language-reference/keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="b7a13-107">Hatayı düzeltmek için geçersiz değiştirici kaldırın.</span><span class="sxs-lookup"><span data-stu-id="b7a13-107">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="b7a13-108">Aşağıdaki örnek, geçerli olduğu yerlerde olmayan konumlara belirteçleri yerleştirildiğinden CS1519 beş yerde oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b7a13-108">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42   
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;   
// Generates CS1519 because of "checked" on void method:  
    checked void f4();     
  
// Generates CS1519 because of "num":  
    void f5(int a num){}        
  
// Generates CS1519 because of namespace inside class:  
    namespace;             
  
}  
```
