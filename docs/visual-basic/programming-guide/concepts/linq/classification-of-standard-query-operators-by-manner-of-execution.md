## <a name="manners-of-execution"></a><span data-ttu-id="c1c5e-108">Yolla yürütme</span><span class="sxs-lookup"><span data-stu-id="c1c5e-108">Manners of Execution</span></span>  
  
### <a name="immediate"></a><span data-ttu-id="c1c5e-109">Hemen</span><span class="sxs-lookup"><span data-stu-id="c1c5e-109">Immediate</span></span>  
 <span data-ttu-id="c1c5e-110">Hemen bir yürütme veri kaynağı okuyun ve işlem kodda bir noktada sorgu burada bildirilmiş gerçekleştirilir anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="c1c5e-110">Immediate execution means that the data source is read and the operation is performed at the point in the code where the query is declared.</span></span> <span data-ttu-id="c1c5e-111">Numaralandırılabilir olmayan, tek bir sonuç tüm standart sorgu işleçleri hemen yürütün.</span><span class="sxs-lookup"><span data-stu-id="c1c5e-111">All the standard query operators that return a single, non-enumerable result execute immediately.</span></span>

 <span data-ttu-id="c1c5e-113">Ertelenmiş yürütme işlemi kodda bir noktada sorgu burada bildirilmiş gerçekleştirildiğini değil anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="c1c5e-113">Deferred execution means that the operation is not performed at the point in the code where the query is declared.</span></span> <span data-ttu-id="c1c5e-114">Yalnızca sorgu değişkeni, örneğin kullanarak numaralandırılır işlemi gerçekleştirilir bir `For Each` deyimi.</span><span class="sxs-lookup"><span data-stu-id="c1c5e-114">The operation is performed only when the query variable is enumerated, for example by using a `For Each` statement.</span></span> <span data-ttu-id="c1c5e-115">Bu sorgu, sorgu zaman tanımlanan yerine yürütüldüğünde sorgu yürütülürken sonuçlarını'nın veri kaynağını içeriğine bağlı anlamına gelir.</span><span class="sxs-lookup"><span data-stu-id="c1c5e-115">This means that the results of executing the query depend on the contents of the data source when the query is executed rather than when the query is defined.</span></span> <span data-ttu-id="c1c5e-116">Sorgu değişkeni birden çok kez listelenebilir, sonuçları her zaman farklı olabilir.</span><span class="sxs-lookup"><span data-stu-id="c1c5e-116">If the query variable is enumerated multiple times, the results might differ every time.</span></span> <span data-ttu-id="c1c5e-117">Dönüş türü olan neredeyse tüm standart sorgu işleçleri <xref:System.Collections.Generic.IEnumerable%601> veya <xref:System.Linq.IOrderedEnumerable%601> ertelenmiş bir biçimde yürütün.</span><span class="sxs-lookup"><span data-stu-id="c1c5e-117">Almost all the standard query operators whose return type is <xref:System.Collections.Generic.IEnumerable%601> or <xref:System.Linq.IOrderedEnumerable%601> execute in a deferred manner.</span></span>  
  
 <span data-ttu-id="c1c5e-118">Ertelenmiş yürütme kullanmak sorgu işleçleri ayrıca akış veya akış dışı olarak sınıflandırılabilir.</span><span class="sxs-lookup"><span data-stu-id="c1c5e-118">Query operators that use deferred execution can be additionally classified as streaming or non-streaming.</span></span>  
  
#### <a name="streaming"></a><span data-ttu-id="c1c5e-119">Akış</span><span class="sxs-lookup"><span data-stu-id="c1c5e-119">Streaming</span></span>
