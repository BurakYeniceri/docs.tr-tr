# <a name="assembly-attribute-39ltattributenamegt39-is-not-valid-lterrorgt"></a>Derleme özniteliği &#39; &lt;attributename&gt; &#39; geçerli değil: &lt;hata&gt;
Ortak dil çalışma zamanı bütünleştirilmiş öznitelikte kabul etmiyor.

**Hata Kimliği:** BC30129

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

1. Öznitelik bildiriminden kaldırın veya bağlamı için geçerli olan özniteliği ile değiştirin.

2. Tırnak işaretli hata iletisini inceleyin ve uygun eylemi gerçekleştirin.
