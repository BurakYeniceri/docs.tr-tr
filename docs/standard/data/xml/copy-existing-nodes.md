# <a name="copy-existing-nodes"></a>Varolan düğümleri kopyalayın
Birçok yöntemleri ve özellikleri de XML belge nesne modeli (gibi bir düğümü seçmek için kullanabileceğiniz DOM) **SelectSingleNode**, **ChildNodes [int t]**, **öznitelikleri [int t]**. Düğüm seçildikten sonra bu belirli düğüm türü için iş Ekle yöntemlerden birini kullanarak ağaca ekleyebilirsiniz. Ağacına bir düğüm eklemek için yalnızca düğüm eklendikten sonra belgenin hala doğru biçimlendirilmiş olması gerektiğini kısıtlamadır. Varolan bir düğümü DOM ağacına takıldığında, özgün konumundan kaldırılır ve hedef konumuna eklendi.
