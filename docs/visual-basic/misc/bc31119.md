# <a name="39end-removehandler39-must-be-the-first-statement-on-a-line"></a>&#39; End RemoveHandler &#39; satırındaki ilk deyim olmalıdır
Bir `End RemoveHandler` deyimi izleyen iki nokta üst üste (:) deyimi ayırıcı. `End RemoveHandler`Kaynak satırında yegane deyim olmalıdır.  
  
 **Hata Kimliği:** BC31119  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Birden çok deyime farklı satırlarına ayırın.
