# <a name="compiler-warning-level-1-cs1645"></a>Derleyici Uyarısı (düzey 1) CS1645
Özellik 'özelliği standartlaştırılmış ISO C# dil belirtimi parçası olmayan ve diğer derleyiciler tarafından kabul edilmedi  
  
 Kullanmakta olduğunuz özellik ISO standart bir parçası değil. Bu özelliği kullanarak kod diğer derleyiciler derleme değil.  
  
```csharp  
// CS1645.cs  
// compile with: /W:1 /t:module /langversion:ISO-1  
[assembly:System.CLSCompliant(false)]  
// To supress the warning use the switch: /nowarn:1645  
[module:System.CLSCompliant(false)]   // CS1645  
class Test  
{  
}  
```
