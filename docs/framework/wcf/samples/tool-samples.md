# <a name="tool-samples"></a>Araç Örnekleri
Bu bölüm, Windows Communication Foundation (WCF) araçları içerir.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 [ConfigurationCodeGenerator](../../../../docs/framework/wcf/samples/configurationcodegenerator.md)  
 Yapılandırma sistemi için özel kanal uygulamalarını kullanıma sunmak için ConfigurationCodeGenerator.exe Aracı'nı kullanmayı açıklar.  
  
 [CustomChannelsTester](../../../../docs/framework/wcf/samples/customchannelstester.md)  
 Bir dizi önceden tanımlanmış hizmet sözleşmelerini, özel kanal uygulamaları test etmek için CustomChannelsTester.exe Aracı'nı kullanmayı açıklar.  
  
 [FindPrivateKey](../../../../docs/framework/wcf/samples/findprivatekey.md)  
 Belirli bir X.509 sertifikası sertifika deposunda ilişkili özel anahtar dosyasının adını ve konumunu bulma işlemini kolaylaştırmak için FindPrivateKey.exe Aracı'nı kullanmayı açıklar.
