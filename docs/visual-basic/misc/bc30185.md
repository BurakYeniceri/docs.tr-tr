# <a name="39enum39-must-end-with-a-matching-39end-enum39"></a>&#39;Enum&#39; eşleşen ile bitmelidir &#39;Enum sonu&#39;
Bir `Enum` deyimi oluşur karşılık gelen olmadan `End Enum` deyimi. Bir `End Enum` deyimi, numaralandırma sona erdirmek için kullanılmalıdır.  
  
 **Hata Kimliği:** BC30185  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Denetleyin `Enum` üyeleri doğru biçimlendirilmiş.  
  
2.  Ekleme bir `End Enum` sabit bitmesini deyimi.
