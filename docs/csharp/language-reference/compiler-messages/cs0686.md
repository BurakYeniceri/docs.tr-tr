## <a name="example"></a><span data-ttu-id="88a3c-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="88a3c-110">Example</span></span>  
 <span data-ttu-id="88a3c-111">Aşağıdaki örnek CS0686 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="88a3c-111">The following sample generates CS0686:</span></span>  
  
```csharp  
// CS0686.cs  
interface I  
{  
    int get_P();  
}  
  
class C : I  
{  
    public int P  
    {  
        get { return 1; }  // CS0686  
    }  
}  
// But the following is valid:  
class D : I  
{  
    int I.get_P() { return 1; }  
    public static void Main() {}  
}  
```
