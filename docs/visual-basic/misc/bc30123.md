# <a name="39end39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Bitiş &#39; deyimleri Komut penceresi içinde geçerli değildir
`Stop` Ve `End` deyimleri yürütme askıya alma ve hata ayıklama bağlamda izin verilmiyor.  
  
 **Hata Kimliği:** BC30123  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sorun değil bir `End` veya `Stop` deyiminde **hemen** penceresi.
