# <a name="39ltspecifiergt39-is-not-valid-on-an-interface-property-declaration"></a>&#39;&lt;belirleyici&gt; &#39; bir arabirim özelliği bildiriminde geçerli değil
A `Property` bir arabirim ifadeye içeren bir geçersiz anahtar sözcüğü gibi `Implements`. Arabirim üyeleri yalnızca tanımlayın, bunları uygulama değil.  
  
 **Hata Kimliği:** BC30273  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçersiz anahtar sözcüğü bildirimi deyimden kaldırın.  
  
-   Arabirim üyeleri uyarlamasını arabirimini uygulayan bir sınıf taşıyın.
