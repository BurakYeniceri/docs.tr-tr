# <a name="range-variable-name-cannot-match-the-name-of-a-member-of-the-39object39-class"></a>Aralık değişkeni adı üyesi adını eşleşemez &#39; nesnesi &#39; sınıfı
Aralık değişkeni bir LINQ Sorgu üyesi adıyla eşleşen <xref:System.Object> sınıfı. Bu bir çakışma LINQ sorgusu için Visual Basic tarafından oluşturulan nesneler oluşturur.  
  
 **Hata Kimliği:** BC36606  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Herhangi bir üyenin adını eşleşmiyor aralık değişkeni için yeni bir ad seçin <xref:System.Object> sınıfı.
