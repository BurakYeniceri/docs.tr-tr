# <a name="argument-not-specified-for-parameter-39ltparameternamegt39-of-39ltmethodnamegt39"></a>Bağımsız değişken parametre &#39;belirtilmemiş; &lt;parametername&gt;&#39; &#39;&lt; MethodName&gt;&#39;
Gerekli parametre için bir bağımsız değişken sağladınız değil.  
  
 **Hata Kimliği:** BC30455  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirtilen parametre için bir bağımsız değişken sağlayın.
