## <a name="example"></a><span data-ttu-id="05234-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="05234-106">Example</span></span>  
 <span data-ttu-id="05234-107">Aşağıdaki örnek CS1609 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="05234-107">The following sample generates CS1609.</span></span>

   public event Del MyEvent
