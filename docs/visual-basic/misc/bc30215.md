# <a name="39sub39-or-39function39-expected"></a>&#39;Sub&#39; veya &#39;işlevi&#39; bekleniyor
A `Declare` deyimi değil belirtin ya da `Sub` veya `Function`. `Declare` dış bir yordam tanımlar ve yordamı türünü belirtmeniz gerekir.  
  
 **Hata Kimliği:** BC30215  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleyip ya da `Sub` veya `Function` anahtar `Declare` deyimi.
