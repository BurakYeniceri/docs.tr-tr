# <a name="39mybase39-cannot-be-used-with-method-39ltmethodnamegt39-because-it-is-declared-39mustoverride39"></a>&#39; MyBase &#39; Yöntem &#39;kullanılamaz; &lt;methodname&gt;&#39; bunu bildirildiğinden &#39; MustOverride &#39;
Kullanma girişiminde bulundunuz `MyBase` bildirilmiş bir yöntem `MustOverride`.  
  
 **Hata Kimliği:** BC30399  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `MustOverride` bildirimi.
