# <a name="structures-cannot-declare-a-non-shared-39sub-new39-with-no-parameters"></a>Yapıları bir paylaşılmayan &#39;bildiremezsiniz; Yeni alt &#39; Parametresiz
`Sub New`içindeki yapıları bildirilen oluşturucular bağımsız değişkenlerini kabul eder ya da gerekir veya ile bildirilmesi `Shared` değiştiricisi.  
  
 **Hata Kimliği:** BC30629  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değişiklik `Sub New` bağımsız değişkenleri kabul edecek şekilde Oluşturucusu.  
  
-   Uygulama `Shared` paylaşılan Oluşturucusu yapmak için değiştiricisi.
