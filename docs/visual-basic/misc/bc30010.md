# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-implemented-interface-39ltinterfacenamegt39"></a>Modül &#39;için gereken başvuru; &lt;modulename&gt;&#39; uygulanan arabirimi &#39; içeren&lt; InterfaceName&gt;&#39;
Modül için gereken başvuru '\<modulename >' uygulanmış arabirimini içeren '\<InterfaceName >'. Projenize ekleyin.  
  
 Arabirim, doğrudan projenizde başvurulmuyor bir modüldeki tanımlanır. [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] Derleyici arabirimi birden fazla modülde tanımlanmış durumda Karışıklığı önlemek için bir başvuru gerektirir.  
  
 **Hata Kimliği:** BC30010  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Başvurulmayan modül adı, proje başvuruları içerir.
