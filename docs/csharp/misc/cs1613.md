# <a name="compiler-error-cs1613"></a><span data-ttu-id="699d5-102">Derleyici Hatası CS1613</span><span class="sxs-lookup"><span data-stu-id="699d5-102">Compiler Error CS1613</span></span>
<span data-ttu-id="699d5-103">Yönetilen coclass sarmalayıcı sınıfı arabirimi 'arabirim' için ' sınıfı' bulunamadı (bir derleme başvurusu eksik?)</span><span class="sxs-lookup"><span data-stu-id="699d5-103">The managed coclass wrapper class 'class' for interface 'interface' cannot be found (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="699d5-104">Bir arabirimden bir COM nesnesinin örneği oluşturmak için girişimde bulunuldu.</span><span class="sxs-lookup"><span data-stu-id="699d5-104">An attempt was made to instantiate a COM object from an interface.</span></span> <span data-ttu-id="699d5-105">Arabirim sahip **ComImport** ve `CoClass` öznitelikler, ancak derleyici için verilen türü bulamıyor `CoClass` özniteliği.</span><span class="sxs-lookup"><span data-stu-id="699d5-105">The interface has the **ComImport** and `CoClass` attributes, but the compiler cannot find the type given for the `CoClass` attribute.</span></span>  
  
 <span data-ttu-id="699d5-106">Bu hatayı gidermek için şunlardan birini deneyin:</span><span class="sxs-lookup"><span data-stu-id="699d5-106">To resolve this error, you can try one of the following:</span></span>  
  
-   <span data-ttu-id="699d5-107">(Çoğu zaman arabirim ve coclass'ı aynı bütünleştirilmiş kodda olmalıdır) coclass sahip derlemesine başvuru ekleyin.</span><span class="sxs-lookup"><span data-stu-id="699d5-107">Add a reference to the assembly that has the coclass (most of the time the interface and coclass should be in the same assembly).</span></span> <span data-ttu-id="699d5-108">Bkz: [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) veya [başvuru iletişim kutusunu](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) bilgi.</span><span class="sxs-lookup"><span data-stu-id="699d5-108">See [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) or [Add Reference Dialog Box](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) for information.</span></span>  
  
-   <span data-ttu-id="699d5-109">Düzeltme `CoClass` arabirimde özniteliği.</span><span class="sxs-lookup"><span data-stu-id="699d5-109">Fix the `CoClass` attribute on the interface.</span></span>  
  
 <span data-ttu-id="699d5-110">Aşağıdaki örnek doğru kullanımını gösteren **CoClassAttribute**:</span><span class="sxs-lookup"><span data-stu-id="699d5-110">The following sample demonstrates correct usage of **CoClassAttribute**:</span></span>  
  
```csharp  
// CS1613.cs  
using System;  
using System.Runtime.InteropServices;  
  
[Guid("1FFD7840-E82D-4268-875C-80A160C23296")]  
[ComImport()]  
[CoClass(typeof(A))]  
public interface IA{}  
public class A : IA {}  
  
public class AA  
{  
   public static void Main()  
   {  
      IA i;  
      i = new IA(); // This is equivalent to new A().  
                    // because of the CoClass attribute on IA  
   }  
}  
```
