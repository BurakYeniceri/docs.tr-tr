# <a name="attribute-constant-39ltconstantnamegt39-cannot-be-the-target-of-an-assignment"></a>Öznitelik sabiti &#39; &lt;constantname&gt;&#39; atama işleminin hedefi olamaz
Öznitelik içinde bildirilen bir sabit bir değer atamak için girişimde bulunuldu.  
  
 **Hata Kimliği:** BC31510  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Hatalı atamasını kaldırın.  
  
-   Geliştirilmiş özel öznitelikleri kullanılıyorsa, öznitelik üye bir alan olarak yeniden tanımlayın.
