# <a name="error-number-must-be-within-the-range-0-and-65535"></a>Hata numarası 0 ile 65535 aralığında olmalıdır
Bir hata numarası dışında verilen 0 ile 65535 arasında döner.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sayının geçerli aralıkta emin olun.
