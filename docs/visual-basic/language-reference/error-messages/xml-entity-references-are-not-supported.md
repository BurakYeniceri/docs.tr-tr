# <a name="xml-entity-references-are-not-supported"></a>XML varlık başvuruları desteklenmiyor
Bir varlık başvurusu (örneğin, `©`) tanımlı olmayan XML 1.0 belirtimi XML değişmez değeri için bir değer olarak dahil. Yalnızca `&`, `"`, `<`, `>`, ve `'` XML varlık başvuruları XML değişmez değerlerine desteklenir.  
  
 **Hata Kimliği:** BC31180  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Desteklenmeyen varlık başvurusunu kaldırın.
