# <a name="ltmethodgt-39ltmethodname1gt39-must-be-declared-39overloads39-because-another-39ltmethodname2gt39-is-declared-39overloads39"></a>&lt;yöntem&gt; &#39; &lt;methodname1&gt; &#39; bildirilmelidir &#39;Overloads&#39; olduğundan başka bir &#39; &lt;methodname2&gt; &#39; bildirilmiş &#39; Aşırı yüklemeler&#39;
Başka bir özellik veya yöntem aynı ada sahip olan bildirilmiş `Overloads` değiştiricisi.  
  
 **Hata Kimliği:** BC31409  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ekleme `Overloads` değiştirici özelliği veya yöntemi.  
  
2.  Kaldırma `Overloads` değiştiricisi tüm özellikleri veya yöntemleri aynı ada sahip.
