İçine dallanamıyor bir `SyncLock` bloğu.  
  
 **Hata Kimliği:** BC30755  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Etiket önündeki böylece kodunuzu yeniden yapılandırmayı `SyncLock` bloğu.
