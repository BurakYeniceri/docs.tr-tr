# <a name="only-one-of-39notoverridable39-39mustoverride39-or-39overridable39-can-be-specified"></a>Yalnızca tek bir &#39;NotOverridable&#39;, &#39;MustOverride&#39;, veya &#39;Overridable&#39; belirtilebilir
Çakışan geçersiz kılma belirtimleri öğesi bildiriminde oluşur. Geçersiz kılma anahtar sözcükleri karşılıklı olarak birbirini dışlar.  
  
 **Hata Kimliği:** BC30177  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir geçersiz kılma tanımlayıcısı seçin ve diğer anahtar sözcükler bildiriminden kaldırın.
