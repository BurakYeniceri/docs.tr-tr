# <a name="module-attribute-39ltattributenamegt39-is-not-valid-lterrorgt"></a>Modül özniteliği &#39; &lt;attributename&gt;&#39; geçerli değil: &lt;hata&gt;
Ortak Dil Çalışma Zamanı Modülü öznitelikte kabul etmiyor.

**Hata Kimliği:** BC30130

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için

1. Öznitelik bildiriminden kaldırın veya bağlamı için geçerli olan özniteliği ile değiştirin.

2. Tırnak işaretli hata iletisini inceleyin ve uygun eylemi gerçekleştirin.
