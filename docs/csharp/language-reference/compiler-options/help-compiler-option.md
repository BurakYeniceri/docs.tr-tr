# <a name="-help---c-compiler-options"></a>-help,-? (C# Derleyici Seçenekleri)
Bu seçenek Stdout'a derleyici seçenekleri listesini ve her seçeneğin kısa bir açıklamasını gönderir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```console  
-help  
-?  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Bu seçenek bir derlemede eklenirse, herhangi bir çıktı dosyası oluşturulur ve hiçbir derleme gerçekleşir.  
  
 Bu derleyici seçeneği Visual Studio'da kullanılamıyor ve programlı olarak değiştirilemez.
