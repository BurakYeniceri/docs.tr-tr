 <span data-ttu-id="38b58-105">Genel bir tür kısıtlı bir tür parametresi kullandığında, yeni sınıfın de bu aynı kısıtlamalara karşılaması gerekir.</span><span class="sxs-lookup"><span data-stu-id="38b58-105">When a generic type uses a type parameter that is constrained, the new class must also satisfy those same constraints.</span></span>

## <a name="example"></a><span data-ttu-id="38b58-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="38b58-108">Example</span></span>  
 <span data-ttu-id="38b58-109">Aşağıdaki kod CS0314 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="38b58-109">The following code generates CS0314:</span></span>  
  
```csharp  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.  
public class C<T> : A<T> where T : ClassConstraint  
{ }  
```
