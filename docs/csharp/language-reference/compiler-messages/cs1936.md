## <a name="example"></a><span data-ttu-id="cddc4-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="cddc4-110">Example</span></span>

<span data-ttu-id="cddc4-111">Aşağıdaki örnek, CS1936 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="cddc4-111">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs

using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```
