# <a name="39sub39-or-39function39-expected-after-39delegate39"></a>&#39; Alt &#39; ya &#39; İşlev &#39; sonra beklenen &#39; temsilci &#39;
A `Delegate` deyimi belirtmiyor bir `Sub` veya `Function` yordamı. `Sub` Veya `Function` anahtar sözcüğü hemen uymalıdır `Delegate` anahtar sözcüğü.  
  
 **Hata Kimliği:** BC30278  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ekleme `Sub` veya `Function` anahtar sözcüğü hemen sonra `Delegate`.  
  
2.  Bağımsız değişken listesi bir yordam adı belirtin ve uygun şekilde türü döndürür.
