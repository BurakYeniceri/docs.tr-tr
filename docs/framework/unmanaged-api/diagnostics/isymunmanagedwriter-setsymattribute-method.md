# <a name="isymunmanagedwritersetsymattribute-method"></a>ISymUnmanagedWriter::SetSymAttribute Yöntemi
Adını dayalı özel bir öznitelik tanımlar. Bu öznitelikler meta verileri özel öznitelikler aksine sembol deposu tutulur.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT SetSymAttribute(  
    [in] mdToken parent,  
    [in] const WCHAR *name,  
    [in] ULONG32 cData,  
    [in, size_is(cData)] unsigned char data[]);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `parent`  
 [in] Öznitelik tanımlanıyorsa meta veri simgesi.  
  
 `name`  
 [in] Bir işaretçi bir `WCHAR` öznitelik adı içerir.  
  
 `cData`  
 [in] A `ULONG32` boyutunu gösterir `data` dizi.  
  
 `data`  
 [in] Öznitelik değeri.  
  
## <a name="return-value"></a>Dönüş Değeri  
 Yöntem başarılı olursa S_OK; Aksi takdirde E_FAIL veya başka bir hata kodu.  
  
## <a name="requirements"></a>Gereksinimler  
 **Başlık:** CorSym.idl, CorSym.h
