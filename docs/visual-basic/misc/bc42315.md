# <a name="xml-comment-tag-39returns39-is-not-permitted-on-a-39declare-sub39-language-element"></a>XML açıklama etiketi &#39;döndürür&#39; üzerinde izin verilmiyor bir &#39;declare sub&#39; language öğesi
XML açıklama etiketi 'returns' bir 'declare sub' dil öğesinde izin verilmez. XML açıklama yoksayılacak.  
  
 XML açıklama için bir `Declare Sub` bildirimi içeremez bir `<`döndürür`>` etiketi.  
  
 **Hata Kimliği:** BC42315  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Desteklenmeyen kaldırmak `<`döndürür`>` etiketi.
