# <a name="compiler-error-cs1639"></a>Derleyici Hatası CS1639
Yönetilen coclass kapsayıcı sınıfı imzası arabirimi 'arabirim' için ' imzası' geçerli bir sınıf adı imzası değildir  
  
 Meta veri arabirimi için geçerli değilse bu hata oluşur. Arabirim C# veya desteklenen bir .NET dili kullanılarak oluşturulduysa bu asla meydana. Arabirimin satıcısına başvurun veya arabirim derleme başarıyla oluşturuldu doğrulayın.
