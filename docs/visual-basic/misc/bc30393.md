# <a name="39exit-try39-can-only-appear-inside-a-39try39-statement"></a>&#39;Exit try&#39; yalnızca içinde görünebilir bir &#39;deneyin&#39; deyimi
`Exit Try` içinde yalnızca görünebilir bir `Try` deyimi engelleyin. Ya da bir yedek sahip `Exit Try` deyimi veya `Exit Try` deyimi görünüyor ilgili sınırları dışında `Try` bloğu.  
  
 **Hata Kimliği:** BC30393  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bulun ve gereksiz kaldırmak `Exit Try` deyimi.  
  
2.  Taşıma `Exit Try` kodunuzu içinde uygun yere deyimi.
