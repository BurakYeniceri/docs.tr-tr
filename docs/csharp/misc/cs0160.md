# <a name="compiler-error-cs0160"></a><span data-ttu-id="81a70-102">Derleyici Hatası CS0160</span><span class="sxs-lookup"><span data-stu-id="81a70-102">Compiler Error CS0160</span></span>
<span data-ttu-id="81a70-103">Önceki bir catch yan tümcesi zaten bu veya süper türünde ('type') tüm özel durumları yakalar</span><span class="sxs-lookup"><span data-stu-id="81a70-103">A previous catch clause already catches all exceptions of this or of a super type ('type')</span></span>  
  
 <span data-ttu-id="81a70-104">Bir dizi **catch** deyimleri türetilmemiş azalan düzende olması gerekir.</span><span class="sxs-lookup"><span data-stu-id="81a70-104">A series of **catch** statements needs to be in decreasing order of derivation.</span></span> <span data-ttu-id="81a70-105">Örneğin, en çok türetilen nesneler ilk görünmesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="81a70-105">For example, the most derived objects must appear first.</span></span>  
  
 <span data-ttu-id="81a70-106">Daha fazla bilgi için bkz: [özel durum işleme deyimleri](../../csharp/language-reference/keywords/exception-handling-statements.md) ve [özel durumlar ve özel durum işleme](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="81a70-106">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="81a70-107">Aşağıdaki örnek CS0160 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="81a70-107">The following sample generates CS0160:</span></span>  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
