# <a name="compiler-error-cs0586"></a>Derleyici Hatası CS0586
Derleyici iç hatası: 'aşama' aşama  
  
 Derleyici beklenmeyen sözdizimi ayrıştırmak için kendi sorunları nedeniyle başarısız olup olmadığını belirlemek için deneyin. Bu durumda değilse, deneyin [konuşun bize](/visualstudio/ide/talk-to-us).
