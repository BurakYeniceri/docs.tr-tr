# <a name="39synclock39-statement-must-end-with-a-matching-39end-synclock39"></a>&#39; SyncLock &#39; deyimi, eşleşen bir &#39;ile bitmelidir; Son SyncLock &#39;
`SyncLock`blokları ile başlar `SyncLock` anahtar sözcüğü ve ile biten `End SyncLock` oluşturun.  
  
 **Hata Kimliği:** BC30675  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Olduğundan emin olun `SyncLock` blok sona erdiği ile bir `End SyncLock` oluşturun.
