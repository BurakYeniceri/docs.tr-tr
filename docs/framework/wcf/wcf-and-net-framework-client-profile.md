# <a name="wcf-and-net-framework-client-profile"></a>WCF ve .NET Framework İstemci Profili
[!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)] hafif bir sürümüdür tam .net Framework tüm framework gerekmeyen istemciler için tasarlanmıştır. Tüm Windows Communication Foundation istemcisi framework tarafından desteklenir.  
  
## <a name="wcf-features-supported-by-the-net-framework-client-profile"></a>WCF özellikleri tarafından .net Framework istemci profili desteklenir.  
 Aşağıdaki Windows Communication Foundation özellikleri tarafından desteklenen [!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)]:  
  
-   Tüm WCF Cardspace ve web barındırma dışında desteklenir.  
  
-   TCP/IP'yi uzaktan iletişim kanalları desteklenir.  
  
-   Asmx (Web Hizmetleri) desteklenmiyor.
