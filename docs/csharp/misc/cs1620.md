# <a name="compiler-error-cs1620"></a>Derleyici Hatası CS1620
Bağımsız değişken 'numara' with 'anahtar sözcüğü' anahtar sözcüğü geçirilmelidir  
  
 Alan bir işleve bağımsız değişken geçirme değilse bu hata oluşur bir [ref](../../csharp/language-reference/keywords/ref.md) veya [çıkışı](../../csharp/language-reference/keywords/out.md) yok parametre ve dahil `ref` veya `out` anahtar sözcüğü çağrısı veya noktasında içerir yanlış anahtar sözcük. Hata metni kullanmak için uygun anahtar sözcüğü gösterir ve hangi bağımsız değişkeninin hatasına neden oldu.  
  
 Aşağıdaki örnek CS1620 oluşturur:  
  
```  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;  
        f(out x);  // CS1620 – f takes a ref parameter, not an out parameter  
        // Try this line instead:  
        // f(ref x);  
    }  
}  
```
