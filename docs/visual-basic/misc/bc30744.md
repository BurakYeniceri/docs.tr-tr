# <a name="referenced-array-element-has-a-value-of-39nothing39"></a>Başvurulan dizi öğesinin değerine sahip &#39;hiçbir şey&#39;
Başvurulan dizi öğesi ayarlanmamış ya da daha önce ayarlandığı `Nothing`.  
  
 **Hata Kimliği:** BC30744  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Dizideki ayarlamak doğru doğrulayın.
