# <a name="attribute-value-is-not-valid-expecting-39value39"></a>Öznitelik değeri geçerli değil; bekleniyor &#39; değeri &#39;
Bir XML belgesi değişmez değer özniteliği için geçersiz değer belirtildi. Bir XML belgesi değişmez değeri geçerli öznitelik değerlerini XML 1.0 belirtimi içinde belirtilir.  
  
 **Hata Kimliği:** BC31181  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML 1.0 belirtimi belirtildiği gibi geçerli bir değer için öznitelik değerini güncelleştirin.
