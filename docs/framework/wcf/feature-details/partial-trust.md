# <a name="partial-trust"></a>Kısmi Güven
İle başlayarak [!INCLUDE[netfx35_long](../../../../includes/netfx35-long-md.md)], kısmen güvenilen arayanlara genel türleri erişebilir ve yöntemleri uygulanan <xref:System.ServiceModel>, <xref:System.Runtime.Serialization>, ve <xref:System.ServiceModel.Web>. Bu bölümde kullanmak için desteklenen senaryolar açıklanmaktadır [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] kısmen güvenilir uygulama gibi sınırlı alt kümesi içinde [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] kod erişim güvenliği (CAS) izinleri azaltılmış işlevsellik ile çalışan uygulamalar için kullanılabilir.
