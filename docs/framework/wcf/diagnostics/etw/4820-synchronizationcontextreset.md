# <a name="4820---synchronizationcontextreset"></a>4820 - SynchronizationContextReset
## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|Kimlik|4820|  
|Anahtar Sözcükler|Bulma|  
|Düzey|Bilgiler|  
|Kanal|Microsoft Windows uygulaması sunucu-uygulamalar/hata ayıklama|  
  
## <a name="description"></a>Açıklama  
 SynchronizationContext tarafından DiscoveryClientChannel özgün değerine sıfırlandı, bu olay yayınlanır.  
  
## <a name="message"></a>İleti  
 SynchronizationContext özgün %1 değerine göre DiscoveryClientChannel sıfırlandı.
