# <a name="specifiers-valid-only-at-the-beginning-of-a-declaration"></a>Yalnızca bir bildirimi başında geçerli tanımlayıcıları
Anahtar sözcükler erişilebilirlik ve diğer ölçülerine belirtme görünmelidir bildirimi anahtar sözcüğü önce gibi `Dim` veya `Function`.  
  
 **Hata Kimliği:** BC30181  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tüm tanımlayıcıları öznitelik listesi (varsa) ve bildirim anahtar sözcüğü arasında taşıyın.
