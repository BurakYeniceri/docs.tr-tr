# <a name="xml-comment-parameter-must-have-a-39name39-attribute"></a>XML açıklama parametresi olmalıdır bir &#39;adı&#39; özniteliği
XML açıklama parametresinde 'name' özniteliği olmalıdır. XML açıklama yoksayılacak.  
  
 XML etiketi gerekli eksik `name` özniteliği.  
  
 **Hata Kimliği:** BC42308  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gerekli sağlamak `name` özniteliği etiketi.
