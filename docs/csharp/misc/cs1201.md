# <a name="compiler-warning-level-1-cs1201"></a>Derleyici Uyarısı (düzey 1) CS1201
Özellik 'geçersiz özelliği kullanım dışı bırakılmıştır. Bunun yerine 'geçerli özelliğini' kullanın.  
  
 Kullanmaya çalıştığınız özellik artık kullanılmıyor. Bunun yerine geçerli özelliğini kullanmak için kodunuzu güncelleştirin.
