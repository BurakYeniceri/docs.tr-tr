# <a name="39end-interface39-must-be-preceded-by-a-matching-39interface39"></a>&#39; End Interface &#39; eşleşen bir &#39; arabirimi &#39;tarafından gelmelidir;
Bir `End Interface` deyimi oluşur karşılık gelen olmadan `Interface` deyimi. `End Interface`karşılık gelen tarafından gelmelidir `Interface` deyimi.  
  
 **Hata Kimliği:** BC30252  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bir önceki olmadığını görmek için onay `Interface` deyimi yanlış yazılmış veya geçersiz.  
  
2.  Emin `Interface` üyeleri doğru biçimlendirilmiş.
