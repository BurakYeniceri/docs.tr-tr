# <a name="xml-comment-tag-39include39-must-have-a-39ltattributegt39-attribute"></a>XML açıklama etiketi &#39;dahil&#39; olmalıdır bir &#39; &lt;özniteliği&gt; &#39; özniteliği
XML açıklama etiketi 'include' olmalıdır bir \<özniteliği > özniteliği. XML açıklama yoksayılacak.  
  
 A `file` veya `path` özniteliği, geçirilen `<`dahil`>` etiketi, ya da eksik.  
  
 **Hata Kimliği:** BC42310  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme ya da gerekli öznitelik tamamlamak \<dahil > etiketi.
