# <a name="type-39lttypenamegt39-cannot-be-used-as-an-attribute-because-its-container-39ltcontainernamegt39-is-not-declared-39public39"></a>Tür &#39; &lt;typename&gt; &#39; öznitelik olarak kullanılamaz, çünkü kapsayıcısı &#39; &lt;containername&gt; &#39; bildirilmemiş olan &#39;ortak&#39;
Sınıf veya bu öznitelik tanımlandığı modülü kullanılarak bildirilmedi `Public` değiştiricisi. Sınıfları ve bir erişim değiştiricisi belirtmeyin modülleri olarak bildirildiğinde `Friend` varsayılan olarak.  
  
 **Hata Kimliği:** BC31517  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Ekleme `Public` sınıfına veya bu öznitelik tanımlandığı modülü değiştiricisi.
