 <span data-ttu-id="b2acb-105">Bulunan derleyici bir `override` olarak işaretlenmiş bir yönteme `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="b2acb-105">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="b2acb-106">Ancak, geçersiz kılma yöntemi kendisini kullanımdan kaldırılmış olarak işaretlenmiş değildi.</span><span class="sxs-lookup"><span data-stu-id="b2acb-106">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="b2acb-107">Geçersiz kılma yöntemi hala oluşturacak [CS0612](../../csharp/misc/cs0612.md), çağrılır.</span><span class="sxs-lookup"><span data-stu-id="b2acb-107">The overriding method will still generate [CS0612](../../csharp/misc/cs0612.md), if called.</span></span>

 <span data-ttu-id="b2acb-109">Aşağıdaki örnek CS0672 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b2acb-109">The following sample generates CS0672:</span></span>  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}

{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}

{  
   static public void Main()  
   {  
   }  
}  
```
