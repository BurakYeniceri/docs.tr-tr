# <a name="multi-dimensional-array-cannot-be-converted-to-an-expression-tree"></a><span data-ttu-id="f004e-102">Çok boyutlu bir dizi bir ifade ağacına dönüştürülemiyor</span><span class="sxs-lookup"><span data-stu-id="f004e-102">Multi-dimensional array cannot be converted to an expression tree</span></span>
<span data-ttu-id="f004e-103">Çoğu ifadeler için ifade ağaçları dönüştürülebilir, ancak çok boyutlu diziler olamaz.</span><span class="sxs-lookup"><span data-stu-id="f004e-103">Most expressions can be converted to expression trees, but multi-dimensional arrays cannot be.</span></span> <span data-ttu-id="f004e-104">Örneğin, aşağıdaki kod bu hataya neden olur:</span><span class="sxs-lookup"><span data-stu-id="f004e-104">For example, the following code causes this error:</span></span>  
  
```vb  
Module Module1  
    Sub Main()  
        '' A multi-dimensional array cannot be converted.  
        'Dim expTree As Expressions.Expression(Of Func(Of Object)) = _  
        '    Function() New Integer(1, 1) {{1, 2}, {2, 3}}  
  
        ' A one-dimensional array can be converted.  
        Dim expTree2 As Expressions.Expression(Of Func(Of Object)) = _  
            Function() New Integer() {1, 2, 3}  
    End Sub  
End Module  
```
