# <a name="1223---switchcasenotfound"></a>1223 - SwitchCaseNotFound
## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|Kimlik|1223|  
|Anahtar Sözcükler|WFActivities|  
|Düzey|Bilgiler|  
|Kanal|Microsoft Windows uygulaması sunucu-uygulamalar/hata ayıklama|  
  
## <a name="description"></a>Açıklama  
 Eşleşen bir ifade sonuç servis talebi etkinlikle anahtar etkinliğinde bulunamadı gösterir.  
  
## <a name="message"></a>İleti  
 Geçiş etkinliği '%1' ifadesi sonucu eşleşen bir servis talebi etkinlik bulunamadı.
