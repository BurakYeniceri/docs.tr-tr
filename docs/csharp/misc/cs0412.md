 <span data-ttu-id="dc67c-104">Genel yöntem tür parametresi yöntemi veya yöntemin parametrelerinden biri yerel bir değişkende arasındaki ad çakışması yoktur.</span><span class="sxs-lookup"><span data-stu-id="dc67c-104">There is a name conflict between the type parameter of a generic method and a local variable in the method or one of the method's parameters.</span></span> <span data-ttu-id="dc67c-105">Bu hatayı önlemek için herhangi bir çakışan parametreleri veya yerel değişkenler yeniden adlandırın.</span><span class="sxs-lookup"><span data-stu-id="dc67c-105">To avoid this error, rename any conflicting parameters or local variables.</span></span>  
  
## <a name="example"></a><span data-ttu-id="dc67c-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="dc67c-106">Example</span></span>  
 <span data-ttu-id="dc67c-107">Aşağıdaki örnek CS0412 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="dc67c-107">The following sample generates CS0412:</span></span>  
  
```csharp  
// CS0412.cs  
using System;  
  
class C  
{  
    // Parameter name is the same as method type parameter name  
    public void G<T>(int T)  // CS0412  
    {  
    }  
    public void F<T>()  
    {  
        // Method local variable name is the same as method type  
        // parameter name  
        double T = 0.0;  // CS0412  
        Console.WriteLine(T);  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
