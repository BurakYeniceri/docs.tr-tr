# <a name="compiler-error-cs2022"></a>Derleyici Hatası CS2022
Seçenekler ' / out' ve '/ hedef' kaynak dosya adlarını önce görünmesi gerekir  
  
 [/Out (çıktı dosyası adını ayarlayın)](../../csharp/language-reference/compiler-options/out-compiler-option.md) ve [/target (çıktı dosyası biçimini belirtin)](../../csharp/language-reference/compiler-options/target-compiler-option.md) derleyici seçenekleri, komut satırında belirtildiğinde, kaynak kodu dosyaları gelmelidir.
