# <a name="39stop39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Stop &#39; deyimleri Komut penceresi içinde geçerli değildir
`Stop` Ve `End` deyimleri yürütme askıya alma ve hata ayıklama bağlamda izin verilmiyor.  
  
 **Hata Kimliği:** BC30122  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sorun değil bir `Stop` veya `End` deyiminde **hemen** penceresi.
