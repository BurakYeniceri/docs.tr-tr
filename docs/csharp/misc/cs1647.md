# <a name="compiler-error-cs1647"></a>Derleyici Hatası CS1647
Çok uzun veya karmaşık 'code' derlemek için ifade  
  
 Kodunuzu işleme derleyicide yığın taşması oluştu. Bu hatayı gidermek için kodunuzu basitleştirin. Kodunuzu geçerliyse, kişi [ürün desteği](https://msdn.microsoft.com/library/77e75b8b-817d-45bf-9c38-458930d873b4).
