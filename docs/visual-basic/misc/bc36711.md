# <a name="argument-matching-parameter-39ltparameternamegt39-narrows-to-39lttypenamegt39"></a>Bağımsız değişken parametre eşleştirme &#39; &lt;parametername&gt; &#39; için daraltır &#39; &lt;typename&gt;&#39;
Aşırı yüklenmiş bir yönteme bir çağrı yapıldı, ancak derleyici daraltma dönüşümü olmadan adlı bir yöntem bulunamıyor. A *dönüştürme daraltma* bazı olası değerleri tutun mümkün olmayabilir veri türü için bir değer değiştirir.
