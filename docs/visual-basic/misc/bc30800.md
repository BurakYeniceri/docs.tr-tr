# <a name="method-arguments-must-be-enclosed-in-parentheses"></a>Yöntem bağımsız değişkenleri parantez içine alınmalıdır
Yordam çağrıları yöneten kurallar, daha yeni sürümlerinde daha basit [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]. Tüm bağımsız değişkenler tarafından parantez içine alınması gerekir.  
  
 **Hata Kimliği:** BC30800  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bağımsız değişken listesi parantez içine alın; Örneğin:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
