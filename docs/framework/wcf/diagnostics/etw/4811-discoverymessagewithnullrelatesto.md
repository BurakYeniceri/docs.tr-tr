# <a name="4811---discoverymessagewithnullrelatesto"></a>4811 - DiscoveryMessageWithNullRelatesTo
## <a name="properties"></a>Özellikler  
  
|||  
|-|-|  
|Kimlik|4811|  
|Anahtar Sözcükler|Bulma|  
|Düzey|Uyarı|  
|Kanal|Microsoft Windows uygulaması sunucu-uygulamalar/hata ayıklama|  
  
## <a name="description"></a>Açıklama  
 İleti üstbilgisi gerekli RelatesTo özelliği içermediğinden bulma ileti DiscoveryClient tarafından bırakıldı olduğunda bu olay yayınlanır.  
  
## <a name="message"></a>İleti  
 MessageID ile bir %1 ileti = '%2', DiscoveryClient tarafından bırakıldı, çünkü ileti üstbilgisi gerekli RelatesTo özelliği içermiyordu.
