# <a name="compiler-warning-level-2-cs0252"></a><span data-ttu-id="30bb9-102">Derleyici Uyarısı (Düzey 2) CS0252</span><span class="sxs-lookup"><span data-stu-id="30bb9-102">Compiler Warning (level 2) CS0252</span></span>
<span data-ttu-id="30bb9-103">Olası istenmeyen başvuru karşılaştırması; değer karşılaştırması almak için 'type' türü için sol taraftaki atayın</span><span class="sxs-lookup"><span data-stu-id="30bb9-103">Possible unintended reference comparison; to get a value comparison, cast the left hand side to type 'type'</span></span>  
  
 <span data-ttu-id="30bb9-104">Derleyici başvurusu karşılaştırma yapıyor.</span><span class="sxs-lookup"><span data-stu-id="30bb9-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="30bb9-105">Dize değeri karşılaştırma yapmak isterseniz, sol tarafındaki ifade cast `type`.</span><span class="sxs-lookup"><span data-stu-id="30bb9-105">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="30bb9-106">Aşağıdaki örnek CS0252 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="30bb9-106">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
