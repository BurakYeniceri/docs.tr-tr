# <a name="compiler-error-cs1001"></a><span data-ttu-id="b6cdf-102">Derleyici Hatası CS1001</span><span class="sxs-lookup"><span data-stu-id="b6cdf-102">Compiler Error CS1001</span></span>
<span data-ttu-id="b6cdf-103">Tanımlayıcı bekleniyor</span><span class="sxs-lookup"><span data-stu-id="b6cdf-103">Identifier expected</span></span>  
  
 <span data-ttu-id="b6cdf-104">Bir tanımlayıcı sağlamadınız.</span><span class="sxs-lookup"><span data-stu-id="b6cdf-104">You did not supply an identifier.</span></span> <span data-ttu-id="b6cdf-105">Bir sınıf, yapı, ad alanı, yöntemi, değişkenin adını ve sağladığınız benzeri bir tanımlayıcıdır.</span><span class="sxs-lookup"><span data-stu-id="b6cdf-105">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>  
  
 <span data-ttu-id="b6cdf-106">Aşağıdaki örnek, basit bir sınıf bildirir ancak sınıfı bir ad sağlamaz:</span><span class="sxs-lookup"><span data-stu-id="b6cdf-106">The following example declares a simple class but does not give the class a name:</span></span>  
  
```csharp  
//cs1001.cs  
public class              //CS1001  
    {  
        public int Num {get; set;}  
        void MethodA(){}  
    }  
```  
  
 <span data-ttu-id="b6cdf-107">Enum bildirirken üyeleri belirtmeniz gerekir çünkü aşağıdaki örnek CS1001 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b6cdf-107">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>  
  
```csharp  
// CS1001.cs  
public class clx  
{  
   enum Colors : int  
   {  
      'a', 'b' // CS1001, 'a' is not a valid int identifier  
       // The following line shows examples of valid identifiers:  
       // Blue, Red, Orange  
   };  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="b6cdf-108">Derleyici, örneğin, bir arabirim tanımı bile kullanmıyorsa parametre adları gereklidir.</span><span class="sxs-lookup"><span data-stu-id="b6cdf-108">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="b6cdf-109">Bu parametreler, böylece arabirimi tüketen programcıları şeyler hakkında parametrelerin söylemek gereklidir.</span><span class="sxs-lookup"><span data-stu-id="b6cdf-109">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>  
  
```csharp  
// CS1001-2.cs  
// compile with: /target:library  
interface IMyTest  
{  
   void TestFunc1(int, int);  // CS1001  
   // Use the following line instead:  
   // void TestFunc1(int a, int b);  
}  
  
class CMyTest : IMyTest  
{  
   void IMyTest.TestFunc1(int a, int b)  
   {  
   }  
}  
```
