# <a name="39ltspecifiergt39-is-not-valid-on-an-interface-method-declaration"></a>&#39;&lt;belirleyici&gt; &#39; bir arabirim yöntemi bildiriminde geçerli değil
A `Function` veya `Sub` bir arabirim ifadeye içeren bir geçersiz anahtar sözcüğü gibi `Implements`. Arabirim üyeleri yalnızca tanımlayın, bunları uygulama değil.  
  
 **Hata Kimliği:** BC30270  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Geçersiz anahtar sözcüğü bildirimi deyimden kaldırın.  
  
2.  Arabirim üyeleri uyarlamasını arabirimini uygulayan bir sınıf taşıyın.
