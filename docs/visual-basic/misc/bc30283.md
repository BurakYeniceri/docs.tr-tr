# <a name="39sub-new39-cannot-be-declared-39overrides39"></a>&#39;Sub New&#39; bildirilemez &#39;geçersiz kılar&#39;
Bir oluşturucu devralınan Oluşturucusu geçersiz kılmaları gösterir. Oluşturucular değiştirilemiyor.  
  
 **Hata Kimliği:** BC30283  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Overrides` anahtar sözcüğünün `Sub` bildirimi.
