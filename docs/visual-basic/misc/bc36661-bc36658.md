## <a name="to-correct-this-error"></a><span data-ttu-id="1cf79-111">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="1cf79-111">To correct this error</span></span>  
  
-   <span data-ttu-id="1cf79-112">Aşağıdaki kodda gösterildiği gibi bir veya daha fazla bağımsız değişken açıkça uyumlu bir türe dönüştürmek mümkün olabilir:</span><span class="sxs-lookup"><span data-stu-id="1cf79-112">You might be able to convert one or more arguments explicitly to a compatible type, as shown in the following code:</span></span>
