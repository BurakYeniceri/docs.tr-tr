 <span data-ttu-id="97009-104">Aynı ada sahip bir etiket iki kez görüldü.</span><span class="sxs-lookup"><span data-stu-id="97009-104">A label with the same name appeared twice.</span></span> <span data-ttu-id="97009-105">Daha fazla bilgi için bkz: [goto](../../csharp/language-reference/keywords/goto.md).</span><span class="sxs-lookup"><span data-stu-id="97009-105">For more information, see [goto](../../csharp/language-reference/keywords/goto.md).</span></span>  
  
 <span data-ttu-id="97009-106">Aşağıdaki örnek CS0140 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="97009-106">The following sample generates CS0140:</span></span>  
  
```csharp  
// CS0140.cs

{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         label1: int i = 0;

      }  
   }  
}  
```
