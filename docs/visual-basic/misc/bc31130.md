# <a name="39addhandler39-definition-missing-for-event-39lteventnamegt39"></a>&#39; AddHandler &#39; Olay &#39;eksik tanımı; &lt;eventname&gt;&#39;
Bir olay olarak bildirilirse `Custom`, olay işleyici ekleme için bir yordam sağlamanız gerekir.  
  
 **Hata Kimliği:** BC31130  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Dahil bir `AddHandler` bildirimi arasında `Custom Event` deyimi ve `End Event` deyimi.  
  
2.  Olay bildirimi son diğer yordamlar doğru doğrulayın.
