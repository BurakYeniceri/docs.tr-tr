# <a name="xml-comment-tag-39lttaggt39-is-not-permitted-on-a-39ltelementgt39-language-element"></a>XML açıklama etiketine &#39; &lt;etiketi&gt; &#39; üzerinde izin verilmiyor bir &#39; &lt;öğesi&gt; &#39; dil öğesi
XML açıklama etiketine \<etiket > üzerinde izin verilmiyor bir '\<öğesi >' öğesi. XML açıklaması yoksayılacak.  
  
 Bir XML açıklama etiketi, uygulandığı dil öğesi tarafından desteklenmiyor.  
  
 **Hata Kimliği:** BC42306  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Desteklenmeyen XML açıklama etiketine kaldırın.
