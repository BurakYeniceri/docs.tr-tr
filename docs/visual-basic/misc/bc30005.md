# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>Derleme &#39;için gereken başvuru; &lt;assemblyname&gt;&#39; olay &#39; tanımını içeren&lt; eventname&gt;&#39;
Derleme için gerekli referans ' <`assemblyname`>' olay tanımını içeren ' <`eventname`>'. Projeniz için bir başvuru ekleyin.  
  
 Olay bir dinamik bağlantı kitaplığı (DLL) tanımlanan veya doğrudan projenizde başvurulmuyor derleme. [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] Derleyici olay birden fazla DLL veya derleme tanımlı durumda Karışıklığı önlemek için bir başvuru gerektirir.  
  
 **Hata Kimliği:** BC30005  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Başvurulmayan DLL veya derleme adını, proje başvuruları içerir.
