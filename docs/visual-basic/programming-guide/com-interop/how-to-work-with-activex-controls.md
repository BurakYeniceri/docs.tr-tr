# <a name="how-to-work-with-activex-controls-visual-basic"></a>Nasıl yapılır: ActiveX Denetimleriyle Çalışma (Visual Basic)
ActiveX denetimleri şunlardır: COM bileşenlerini ya da nesneleri bir Web sayfası veya başka birinin programlanmış paketlenmiş işlevselliği yeniden kullanmak için başka bir uygulama yerleştirin. Visual Basic 6.0 ve önceki sürümler için geliştirilen ActiveX denetimlerini özellikleri eklemek için kullanabileceğiniz **araç** Visual Studio.  
  
### <a name="to-add-activex-controls-to-the-toolbox"></a>Araç kutusuna ActiveX denetimleri ekleme  
  
1.  Üzerinde **Araçları** menüsünde tıklatın **araç kutusu öğelerini Seç**.  
  
     **Seçin araç** iletişim kutusu görüntülenir.  
  
2.  Tıklatın **COM bileşenlerini** sekmesi.  
  
3.  Kullanın ve ardından istediğiniz ActiveX denetimi yanındaki onay kutusunu işaretleyin **Tamam**.  
  
     Diğer araçları ile yeni denetim görünür **araç**.  
  
    > [!NOTE]
    >  Birlikte çalışma bütünleştirilmiş ActiveX denetimleri için el ile oluşturmak için Aximp yardımcı programını kullanabilirsiniz. Daha fazla bilgi için bkz: [Aximp.exe (Windows Forms ActiveX denetimi alıcısı)](http://msdn.microsoft.com/library/482c0d83-7144-4497-b626-87d2351b78d0).
