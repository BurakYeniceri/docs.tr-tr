# <a name="compiler-error-cs1621"></a><span data-ttu-id="5ac5b-102">Derleyici Hatası CS1621</span><span class="sxs-lookup"><span data-stu-id="5ac5b-102">Compiler Error CS1621</span></span>
<span data-ttu-id="5ac5b-103">Verim deyimi adsız yöntem veya lambda ifadesi içinde kullanılamaz</span><span class="sxs-lookup"><span data-stu-id="5ac5b-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="5ac5b-104">Verim deyimi yineleyici anonim yöntemi bloğunda olamaz.</span><span class="sxs-lookup"><span data-stu-id="5ac5b-104">The yield statement cannot be in an anonymous method block in an iterator.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5ac5b-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="5ac5b-105">Example</span></span>  
 <span data-ttu-id="5ac5b-106">Aşağıdaki örnek CS1621 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="5ac5b-106">The following sample generates CS1621:</span></span>  
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
