# <a name="39case39-cannot-follow-a-39case-else39-in-the-same-39select39-statement"></a>&#39; Servis talebi &#39; izlenemiyor bir &#39; Servis talebi başka &#39; aynı &#39; &#39;seçin; deyimi
A `Case Else` deyimi tanıtır ilk değeri için eşleşme bulunamazsa, yürütülecek deyimleri `Case`. A `Case` deyimi, sonra bulundu bir `Case Else` aynı `Select` bloğu.  
  
 **Hata Kimliği:** BC30321  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Taşıma `Case Else` sonra uygun konuma `Case` deyimi.
