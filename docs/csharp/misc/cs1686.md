# <a name="compiler-error-cs1686"></a><span data-ttu-id="1fb88-102">Derleyici Hatası CS1686</span><span class="sxs-lookup"><span data-stu-id="1fb88-102">Compiler Error CS1686</span></span>
<span data-ttu-id="1fb88-103">Adresleri alınmış olamaz ve anonim yöntemi veya lambda ifadesi içinde kullanılan yerel 'değişkeni' veya üyeleri</span><span class="sxs-lookup"><span data-stu-id="1fb88-103">Local 'variable' or its members cannot have their address taken and be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="1fb88-104">Bu hata, bir değişken kullanın ve adresini ele geçirmeye ve şu eylemlerden birini anonim yönteminin içinde yapılır oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="1fb88-104">This error is generated when you use a variable, and attempt to take its address, and one of these actions is done inside an anonymous method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1fb88-105">Örnek</span><span class="sxs-lookup"><span data-stu-id="1fb88-105">Example</span></span>  
 <span data-ttu-id="1fb88-106">Aşağıdaki örnek CS1686 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="1fb88-106">The following sample generates CS1686.</span></span>  
  
```csharp  
// CS1686.cs  
// compile with: /unsafe /target:library  
class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
