# <a name="39ltnamegt39-necessary-for-compiling-this-construct-cannot-be-found"></a>&#39;&lt;ad&gt;&#39;, bu yapıyı derleme bulunamıyor için gerekli
Derleme için gerekli bir öğe bulunamıyor.  
  
 **Hata Kimliği:** BC30930  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Gerekli yardımcı öğe sağlayın.  
  
2.  Kod geçerli yönetilen kod olduğundan emin olun.
