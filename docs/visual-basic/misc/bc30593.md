# <a name="variables-in-modules-cannot-be-declared-39ltspecifiergt39"></a>Modül içindeki değişkenler bildirilemez &#39; &lt;belirticisi&gt;&#39;
Bir belirleyici gibi kullandığınız `MustInherit`, bir değişkende üzerinde bir `Module` deyimi. Modüller hiçbir zaman oluşturulabilir, devralma desteklemez ve arabirimleri uygulayamaz.  
  
 **Hata Kimliği:** BC30593  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirleyici kaldırın.
