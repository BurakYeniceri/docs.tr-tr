# <a name="39end-using39-must-be-preceded-by-a-matching-39using39"></a>&#39;Kullanarak son&#39; eşleşen bir gelmelidir &#39;kullanma&#39;
Bir `End Using` deyimi görünüyor eşleşme ile `Using` kendisinden bildirimi.  
  
 **Hata Kimliği:** BC36007  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `End Using` yedek ise deyimi.  
  
-   Eksik tedarik [kullanarak deyimi](../../visual-basic/language-reference/statements/using-statement.md) eksikse.  
  
-   Taşıma `End Using` kodu uygun bir yerinde ifadesine.
