# <a name="compiler-error-cs0569"></a>Derleyici Hatası CS0569
'method2': dil tarafından desteklenmediğinden 'method1' geçersiz kılınamaz  
  
 Bu hata, başka bir dilde ve derleyici geçersiz kılmak için çalışıyorsunuz yöntemi desteklemediğinde yazılmış bir taban sınıftan türetilen oluşur.
