# <a name="39global39-not-allowed-in-handles-local-name-expected"></a>&#39;Genel&#39; içinde tanıtıcıları; izin verilmiyor Yerel ad bekleniyor
A `Handles` yan tümcesi, yerel bir olaya başvurmalıdır. `Global` Anahtar sözcüğü genel programlama öğeleri ile erişim sağlar.  
  
 **Hata Kimliği:** BC36002  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değişiklik `Handles` olay genel örneği yerine yerel bir örneğini başvurmak için yan tümcesi.
