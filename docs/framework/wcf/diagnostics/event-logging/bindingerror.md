# <a name="bindingerror"></a>BindingError
Kimliği: 159  
  
 Önem derecesi: hata  
  
 Kategori: SMSvcHost  
  
## <a name="description"></a>Açıklama  
 Bu olay, belirli site belirli bağlamasının ayrıştırılırken bir hata oluştu, böylece Protokolü site için geçici olarak devre dışıdır gösterir. Olay bağlama, kaynak, özel durum, işlem adı ve işlem kimliği listeler
