<span data-ttu-id="00fdd-103">Anonim yöntem bloğu bazı bloğundaki dönüş türleri temsilci dönüş türü örtük olarak dönüştürülebilir olmadığından temsilci türü 'type' temsilci dönüştürülemiyor</span><span class="sxs-lookup"><span data-stu-id="00fdd-103">Cannot convert anonymous method block to delegate type 'delegate type' because some of the return types in the block are not implicitly convertible to the delegate return type</span></span>  
  
 <span data-ttu-id="00fdd-104">Temsilci dönüş türüne örtük olarak dönüştürülebilir değildi türü anonim yöntemi bloğun return deyimi sahipse, bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="00fdd-104">This error occurs if the anonymous method block's return statement had a type that was not implicitly convertible to the return type of the delegate.</span></span>  
  
 <span data-ttu-id="00fdd-105">Aşağıdaki örnek CS1662 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="00fdd-105">The following sample generates CS1662:</span></span>  
  
```csharp  
// CS1662.cs  
  
delegate int MyDelegate(int i);  
  
class C  
{  
  
  public static void Main()  
  {  
     MyDelegate d = delegate(int i) { return 1.0; };  // CS1662  
     // Try this instead:  
     // MyDelegate d = dekegate(int i) { return (int)1.0; };  
  }  
}  
```
