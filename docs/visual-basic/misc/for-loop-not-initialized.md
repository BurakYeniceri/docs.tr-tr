# <a name="for-loop-not-initialized-visual-basic"></a>For döngüsü başlatılmadı (Visual Basic)

## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Döngü içine atlama kaldırın.
