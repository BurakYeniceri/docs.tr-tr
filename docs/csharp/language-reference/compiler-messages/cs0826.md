 <span data-ttu-id="39a9d-104">Dizi öğeleri tüm aynı türde veya dolaylı olarak derleyici tarafından kullanılan türü çıkarım kurallarına göre aynı türüne dönüştürülebilir olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="39a9d-104">Array elements must all be the same type or implicitly convertible to the same type according to the type inference rules used by the compiler.</span></span> <span data-ttu-id="39a9d-105">En iyi türü dizi ifadesinde bulunan mevcut türlerden biri olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="39a9d-105">The best type must be one of the types present in the array expression.</span></span> <span data-ttu-id="39a9d-106">Öğeleri dönüştürülmeyecek yeni bir türe gibi `object`.</span><span class="sxs-lookup"><span data-stu-id="39a9d-106">Elements will not be converted to a new type such as `object`.</span></span> <span data-ttu-id="39a9d-107">Türü örtük olarak belirlenmiş bir dizi için derleyici dizi Infer gerekir türüne bağlı olarak kendisine atanmış öğelerin türü.</span><span class="sxs-lookup"><span data-stu-id="39a9d-107">For an implicitly typed array, the compiler must infer the array type based on the type of elements assigned to it.</span></span>

## <a name="example"></a><span data-ttu-id="39a9d-112">Örnek</span><span class="sxs-lookup"><span data-stu-id="39a9d-112">Example</span></span>  
 <span data-ttu-id="39a9d-113">Öğeleri olan bir dizi türünde değil aynı olduğundan aşağıdaki kodu CS0826 oluşturur ve derleyicinin türü çıkarım mantığı tek bir en iyi tür bulmaz:</span><span class="sxs-lookup"><span data-stu-id="39a9d-113">The following code generates CS0826 because the array elements are not all the same type, and the compiler's type inference logic does not find a single best type:</span></span>

```csharp  
public class C  
{  
    public static int Main()  
    {  
        var x = new[] { 1, "str" }; // CS0826  
  
        char c = 'c';  
        short s1 = 0;  
        short s2 = -0;  
        short s3 = 1;  
        short s4 = -1;  
  
        var array1 = new[] { s1, s2, s3, s4, c, '1' }; // CS0826  
        return 1;  
    }  
}  
```
