# <a name="initialization-error"></a>Başlatma hatası
Program başlatılırken bir hata oluştu. Bunun bir sonucu hala devam ederken bir yükleme iptal ediliyor olabilir.  
  
 **Hata Kimliği:** BC2000  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yüklemeyi yeniden çalıştırmayı deneyin.
