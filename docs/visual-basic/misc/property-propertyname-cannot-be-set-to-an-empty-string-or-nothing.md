# <a name="property-ltpropertynamegt-cannot-be-set-to-an-empty-string-or-nothing"></a>Özellik &lt;propertyname&gt; boş bir dize veya hiçbir şey ayarlanamaz
Özellik boş bir dize dışında bir değer gerektirir ("") veya `Nothing`. Bu değeri yanlış hesaplıyor bir sonucu olabilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçerli bir değer sağlayın.
