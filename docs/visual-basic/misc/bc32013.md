# <a name="option-strict-on-disallows-operands-of-type-object-for-operator-39ltoperatornamegt39"></a>Option Strict On, operatöre nesne türündeki işlenenler izin vermez &#39; &lt;operatorname&gt;&#39;
Option Strict On, operatöre nesne türündeki işlenenler izin vermez '\<operatorname >'. Kullanım için nesne kimliğini sınamak için işlecidir.  
  
 Aritmetik karşılaştırma işleci gibi `=` bir veya daha fazla nesne değişkenleri ile kullanılan zaman `Option Strict` olan `On`.  
  
 **Hata Kimliği:** BC32013  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kapatma `Option Strict Off` nesne değişkenleri sayısal değerler içeriyorsa ve bir aritmetik karşılaştırma düşündüğünüz.  
  
2.  Kullanım `Is` nesne kimliğini Karşılaştırılacak işleci.
