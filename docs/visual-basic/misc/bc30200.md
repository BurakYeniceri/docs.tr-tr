# <a name="39new39-is-not-valid-in-this-context"></a>&#39;Yeni&#39; bu bağlamda geçerli değil
`New` Anahtar sözcüğü bir tür adı gerekli olduğu oluşur. Birleştirilemez bir değer türü için içerik çağırır `New`.  
  
 **Hata Kimliği:** BC30200  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `New` anahtar sözcüğü.
