# <a name="39inherits39-statements-must-precede-all-declarations-in-an-interface"></a>&#39; devralır &#39; deyimleri tüm bildirimlerden bir arabirim, önce olmalıdır
Bir `Inherits` deyimi bir sınıf tanımı ilk boş olmayan ve yorum olmayan satır olması gerekir.  
  
 **Hata Kimliği:** BC30357  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Taşıma, `Inherits` olan tüm bildirimleri önündeki şekilde deyimi.
