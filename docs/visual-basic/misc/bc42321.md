# <a name="badly-formed-xml-file-39ltcommentfilegt39-cannot-be-included"></a>Hatalı biçimlendirilmiş XML. Dosya &#39; &lt;commentfile&gt;&#39; eklenemiyor
Bir `<include>` XML açıklamaları birini etiketine başvuruyor var olmayan bir dosya ya da hatalı içeren bir dosyayı biçimlendirilmiş XML.  
  
 **Hata Kimliği:** BC42321  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Dosyanın var olduğunu doğrulayın.  
  
2.  XML dosyasında geçerli olduğunu doğrulayın.
