 <span data-ttu-id="30a98-104">Erişimci değiştirilemiyor çünkü bir özellik için erişimci yöntemi geçersiz kılma girişimi başarısız oldu.</span><span class="sxs-lookup"><span data-stu-id="30a98-104">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="30a98-105">Bu hata oluşabilir:</span><span class="sxs-lookup"><span data-stu-id="30a98-105">This error can occur if:</span></span>

 <span data-ttu-id="30a98-108">Temel sınıf özelliğini geçersiz kılmak istemiyorsanız kullanabileceğiniz [yeni](../../csharp/language-reference/keywords/new.md) anahtar sözcüğü türetilmiş bir sınıf özelliği önce.</span><span class="sxs-lookup"><span data-stu-id="30a98-108">If you do not want to override the base class property, you can use the [new](../../csharp/language-reference/keywords/new.md) keyword before the property in derived class.</span></span>

## <a name="example"></a><span data-ttu-id="30a98-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="30a98-110">Example</span></span>  
 <span data-ttu-id="30a98-111">Aşağıdaki örnek, temel sınıf özelliği için ayarlama erişimcisine bildirmiyor çünkü CS0546 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="30a98-111">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
