# <a name="constants-unmanaged-api-reference"></a>Sabitler (Yönetilmeyen API Başvurusu)
Bu konu, dil türü, dil satıcı ve CorSym.idl içinde tanımlanan belge türü sabitleri açıklar.  
  
## <a name="language-type-constants"></a>Dil türü sabitleri  
 Aşağıdaki tabloda dil programlama dillerini belirleyin GUID'ler temsil türü sabitleri gösterir.  
  
|Simgesi|Açıklama|  
|------------|-----------------|  
|CorSym_LanguageType_C|C dili gösterir.|  
|CorSym_LanguageType_CPlusPlus|C++ dili belirtir.|  
|CorSym_LanguageType_CSharp|C# dili belirtir.|  
|CorSym_LanguageType_Basic|Temel dili gösterir.|  
|CorSym_LanguageType_Java|Java dili gösterir.|  
|CorSym_LanguageType_Cobol|COBOL dili gösterir.|  
|CorSym_LanguageType_Pascal|Pascal dili gösterir.|  
|CorSym_LanguageType_ILAssembly|Microsoft Ara dili (MSIL) bütünleştirilmiş kodu gösterir.|  
|CorSym_LanguageType_JScript|JScript dili gösterir.|  
|CorSym_LanguageType_SMC|SMC dili gösterir.|  
|CorSym_LanguageType_MCPlusPlus|.NET Framework için etkin C++ dili belirtir.|  
  
## <a name="language-vendor-constants"></a>Dil satıcı sabitleri  
 Aşağıdaki tabloda dil programlama dili satıcıları tanımlamak GUID'ler temsil eden satıcı sabitleri gösterir.  
  
|Simgesi|Açıklama|  
|------------|-----------------|  
|CorSym_LanguageVendor_Microsoft|Microsoft gösterir.|  
  
## <a name="document-type-constants"></a>Belge türü sabitleri  
 Aşağıdaki tabloda belge belge türlerini tanımlamak GUID'ler temsil türü sabitleri gösterir.  
  
|Simgesi|Açıklama|  
|------------|-----------------|  
|CorSym_DocumentType_Text|Bir metin belgesi gösterir.|  
|CorSym_DocumentType_MC|Bir metin dışı belge gösterir.|
