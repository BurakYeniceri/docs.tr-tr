# <a name="type-39lttypenamegt39-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>Türü &#39; &lt;typename&gt;&#39; tanımlı değil veya onu içeren modülü hata ayıklama oturumunda yüklü değil
Bir türü mevcut değil ya da tanımlı değil başvuruldu.  
  
 **Hata Kimliği:** BC30743  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Tür adının doğru yazıldığından emin olun.  
  
2.  Geçerli sınıf veya modülü içinde türünü tanımlayın.
