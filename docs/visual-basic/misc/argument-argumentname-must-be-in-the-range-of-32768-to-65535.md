# <a name="argument-39ltargumentnamegt39-must-be-in-the-range-of--32768-to-65535"></a>Bağımsız değişken &#39; &lt;argumentname&gt;&#39; -32768 ile 65535 aralığında olmalıdır
Bir değişkenin değeri izin verilen -32.768 65.535 arasında aralığının dışında.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bağımsız değişkeninin değeri geçerli aralıkta olduğundan emin olun.  
  
2.  Bağımsız değişkeni bir hesaplama sağlanırsa, hesaplama işlemleri değerlerini denetleyin.
