# <a name="xml-namespace-prefix-39prefix39-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>XML ad alanı öneki &#39;önek&#39; kullanılmak üzere XML ve URI değiştirilemez ad tarafından ayrılmıştır
Bir XML ad alanı öneki gibi ayrılmış `xmlns`, bir XML değişmez değeri kullanılır.  
  
 **Hata Kimliği:** BC31152  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML ad alanı öneki ayrılmamış bir değerle değiştirin.
