# <a name="myapplicationinfo-object"></a>My.Application.Info Nesnesi
Sürüm numarası, açıklama, yüklenen derlemeler ve benzerleri gibi uygulama hakkında bilgi almak için özellikleri sağlar.  
  
## <a name="remarks"></a>Açıklamalar  
 Yöntemleri ve özellikleri hakkında bilgi için `My.Application.Info` nesne için bkz: <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>.  
  
> [!NOTE]
>  Özelliklerini kullanabilirsiniz <xref:System.Diagnostics.FileVersionInfo?displayProperty=nameWithType> disk üzerindeki bir dosya hakkında bilgi edinmek için sınıf.

 **Sınıfı:**<xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>  
  
 **Derleme:** Visual Basic çalışma zamanı kitaplığı (Microsoft.VisualBasic.dll içinde)
