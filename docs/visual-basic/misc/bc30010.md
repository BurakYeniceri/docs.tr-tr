# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-implemented-interface-39ltinterfacenamegt39"></a>Modül için gereken başvuru &#39; &lt;modulename&gt; &#39; uygulanan arabirimi içeren &#39; &lt;InterfaceName&gt;&#39;
Modül için gereken başvuru '\<modulename >' uygulanmış arabirimini içeren '\<InterfaceName >'. Projenize ekleyin.  
  
 Arabirim, doğrudan projenizde başvurulmuyor bir modüldeki tanımlanır. Visual Basic derleyici arabirimi birden fazla modülde tanımlanmış durumda Karışıklığı önlemek için bir başvuru gerektirir.  
  
 **Hata Kimliği:** BC30010  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Başvurulmayan modül adı, proje başvuruları içerir.
