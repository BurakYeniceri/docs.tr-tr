# <a name="property-access-must-assign-to-the-property-or-use-its-value"></a>Özellik erişimi özelliğine atayın veya değerini kullanın
Atanarak veya değerini kullanarak bir özelliğe erişmek çalıştınız.
  
 **Hata Kimliği:** BC30545  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir değer özelliğine atayın.  
  
     \-veya -  
  
-   Özelliğin değerini kullanın.
