# <a name="compiler-error-cs0123"></a><span data-ttu-id="04f69-102">Derleyici Hatası CS0123</span><span class="sxs-lookup"><span data-stu-id="04f69-102">Compiler Error CS0123</span></span>
<span data-ttu-id="04f69-103">Hiçbir aşırı yüklemesi 'yönteminin' temsilci 'temsilci' ile eşleşir</span><span class="sxs-lookup"><span data-stu-id="04f69-103">No overload for 'method' matches delegate 'delegate'</span></span>  
  
 <span data-ttu-id="04f69-104">Doğru imza kullanılmadığı için bir temsilci oluşturma girişimi başarısız oldu.</span><span class="sxs-lookup"><span data-stu-id="04f69-104">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="04f69-105">Bir temsilci örneklerini temsilci bildirimi aynı imzayla bildirilmelidir.</span><span class="sxs-lookup"><span data-stu-id="04f69-105">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>  
  
 <span data-ttu-id="04f69-106">Bu hatayı gidermek yöntemi ya da ayarlayarak veya imza temsilci.</span><span class="sxs-lookup"><span data-stu-id="04f69-106">You can resolve this error by adjusting either the method or delegate signature.</span></span> <span data-ttu-id="04f69-107">Daha fazla bilgi için bkz: [Temsilciler](../../csharp/programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="04f69-107">For more information, see [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="04f69-108">Aşağıdaki örnek CS0123 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="04f69-108">The following sample generates CS0123.</span></span>  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
