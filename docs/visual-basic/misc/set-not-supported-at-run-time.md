# <a name="set-not-supported-at-run-time"></a>Çalışma zamanında desteklenmeyen ayarlayın
Ayarlama veya bir özellik değeri yalnızca tasarım zamanında ayarlanabilir değiştirme çalışıldı.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Özellik referansı kodunuzdan kaldırın.  
  
2.  Yalnızca çalışma zamanında özelliğinin değeri döndürmek için başvuru değiştirin.
