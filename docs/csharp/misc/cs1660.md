# <a name="compiler-error-cs1660"></a><span data-ttu-id="e1c46-102">Derleyici Hatası CS1660</span><span class="sxs-lookup"><span data-stu-id="e1c46-102">Compiler Error CS1660</span></span>

 <span data-ttu-id="e1c46-104">Atama veya aksi halde adsız yöntem bloğunun bir temsilci türü olmayan bir tür dönüştürme çalışırsanız, bu hata oluşur.</span><span class="sxs-lookup"><span data-stu-id="e1c46-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="e1c46-105">Aşağıdaki örnek CS1660 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e1c46-105">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
