# <a name="mycomputerports-object"></a>My.Computer.Ports Nesnesi
Bir özellik ve bilgisayar seri bağlantı noktalarına erişmek için bir yöntem sağlar.  
  
## <a name="remarks"></a>Açıklamalar  
 Yöntemleri ve özellikleri hakkında bilgi için `My.Computer.Ports` nesne için bkz: <xref:Microsoft.VisualBasic.Devices.Ports>.  
  
 Daha fazla bilgi için bkz: [bilgisayarın bağlantı noktalarına erişme](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-computer-s-ports.md).  
  
> [!NOTE]
>  Özellikleri ve yöntemleri de kullanabilirsiniz <xref:System.IO.Ports.SerialPort?displayProperty=nameWithType> bilgisayar seri bağlantı noktalarına erişmek için sınıf.
