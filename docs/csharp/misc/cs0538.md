# <a name="compiler-error-cs0538"></a><span data-ttu-id="bc685-102">Derleyici Hatası CS0538</span><span class="sxs-lookup"><span data-stu-id="bc685-102">Compiler Error CS0538</span></span>
<span data-ttu-id="bc685-103">Açık arabirim bildirimindeki ' name' bir arabirim değil</span><span class="sxs-lookup"><span data-stu-id="bc685-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="bc685-104">Açıkça bildirmek için girişimde bulunuldu bir [arabirimi](../../csharp/language-reference/keywords/interface.md), ancak bir arabirim belirtilmedi.</span><span class="sxs-lookup"><span data-stu-id="bc685-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="bc685-105">Aşağıdaki örnek CS0538 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="bc685-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
