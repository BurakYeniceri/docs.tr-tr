# <a name="compiler-error-cs2012"></a>Derleyici Hatası CS2012
'Yazmak için dosya' açılamıyor  
  
 Kullanırken [/bugreport:](../../csharp/language-reference/compiler-options/bugreport-compiler-option.md) `file` derleyici seçeneği, dosya yazma için açılamadı. Geçerli bir dosya adı belirtilen ve dosya salt okunur olmadığından emin olun.
