# <a name="compiler-warning-level-1-cs0602"></a>Derleyici Uyarısı (düzey 1) CS0602
'Old_feature' özelliği kaldırıldı. Lütfen bunun yerine 'new_feature' kullanın  
  
 Kod içinde kullanılan dil özelliği (*old_feature*) destek gelecekteki bir sürümde kaldırılmış olabilir ancak bu hala desteklenir. Bunun yerine, önerilen sözdizimini kullanmanız gerekir (*new_feature*).
