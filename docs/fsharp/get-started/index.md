F # ile çalışmaya başlamak için birden çok yolu vardır.  Her ana yol için bir kılavuz sağlamak üzere birden çok makalelerinin sahibiz.  Bir kararı de yardımcı olmak için aşağıdaki tabloyu kullanın.

| İŞLETİM SİSTEMİ | Visual Studio tercih | Visual Studio Code tercih | Bir komut satırı tercih |
| -- |------------------------|--------------------------|-----------------------------|-------------------------|
| Windows | [Visual Studio ile çalışmaya başlama](get-started-visual-studio.md) | [VSCode ve Ionide kullanmaya başlama](get-started-vscode.md) | [.NET Core CLI ile çalışmaya başlama](get-started-command-line.md) |
| MacOS | [Mac için VS ile çalışmaya başlama](get-started-with-visual-studio-for-mac.md) | [VSCode ve Ionide kullanmaya başlama](get-started-vscode.md) | [.NET Core CLI ile çalışmaya başlama](get-started-command-line.md) |
| Linux | Yok | [VSCode ve Ionide kullanmaya başlama](get-started-vscode.md) | [.NET Core CLI ile çalışmaya başlama](get-started-command-line.md) |

Genel olarak, kalan iyi olan başlamak için belirli yolu yoktur.  F # şeyleri görmek için makinenizde kullanmak için tüm yolları çalışırken öneririz en iyi!
