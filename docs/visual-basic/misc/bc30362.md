# <a name="39ltpropertyname1gt39-cannot-override-39ltpropertyname2gt39-because-they-differ-by-39readonly39-or-39writeonly39"></a>&#39;&lt;propertyname1&gt; &#39; geçersiz kılamaz &#39; &lt;propertyname2&gt; &#39; kullanıcılar tarafından farklı olduğundan &#39;ReadOnly&#39; veya &#39;WriteOnly&#39;
Yalnızca ilk farklı ikinci bir özelliğe sahip bir özellik geçersiz kılma girişiminde bulundunuz `ReadOnly` veya `WriteOnly`.  
  
 **Hata Kimliği:** BC30362  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Özellikleri emin olun Ayrıştırılan tarafından birden fazla `ReadOnly` ve `WriteOnly`.
