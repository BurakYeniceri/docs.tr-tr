# <a name="queued-rejected-messages"></a>Kuyruğa Alınan Reddedilmiş İletiler
Sayaç adı: Kuyruğa alınan reddedilen ileti.  
  
## <a name="description"></a>Açıklama  
 Bu hizmet, sıraya alınan aktarım tarafından reddedilen ileti sayısı.  
  
 Bkz: [Poison ileti işleme](http://go.microsoft.com/fwlink/?LinkID=96233) iletileri ne zaman reddedilir hakkında daha fazla bilgi.
