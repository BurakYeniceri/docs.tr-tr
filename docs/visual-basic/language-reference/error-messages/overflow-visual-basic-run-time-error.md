# <a name="overflow-visual-basic-run-time-error"></a>Taşma (Visual Basic Çalışma Süresi Hatası)
Atamanın hedef sınırlarını aşıyor atama çalıştığınızda taşma sonuçlanır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Dönüşümler değişkenleri bu türde bir değer için izin verilen aralıktaki gösterilemeyecek kadar büyük olmayan ve bir türün bir değişkene değer atamak atamaları, hesaplama ve veri türü sonuçlarını daha büyük bir değer aralığı tutabilir emin olun , gerekirse.  
  
2.  Özelliklere atamalar yapılan özellik aralığını uygun olduğundan emin olun.  
  
3.  Tamsayıları yüklenen hesaplamalarda kullanılan numaraları sonuçları tamsayılar büyük olmadığından emin olun.
