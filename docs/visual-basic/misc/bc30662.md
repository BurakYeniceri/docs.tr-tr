# <a name="attribute-39ltattributenamegt39-cannot-be-applied-to-39ltmembernamegt39-because-the-attribute-is-not-valid-on-this-declaration-type"></a>Öznitelik &#39; &lt;attributename&gt; &#39; uygulanamaz &#39; &lt;membername&gt; &#39; özniteliği bu bildirim türünde geçerli olmadığından
Kullanmakta olduğunuz öznitelik kullanmakta olduğunuz öğe için uygun değil.  
  
 **Hata Kimliği:** BC30662  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kullanmakta olduğunuz öğe için amaçlanan bir öznitelik seçin. Örneğin, yöntemini kullanıyorsanız, yöntemleri ile kullanılmak üzere tasarlanmış bir öznitelik seçin.  
  
2.  Geliştirilmiş özel öznitelikler kullanıyorsanız, değiştirme `AttributeUsage` kullanmakta olduğunuz öğe türü eşleşecek şekilde özniteliği.
