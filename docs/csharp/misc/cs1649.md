# <a name="compiler-error-cs1649"></a><span data-ttu-id="b55b5-102">Derleyici Hatası CS1649</span><span class="sxs-lookup"><span data-stu-id="b55b5-102">Compiler Error CS1649</span></span>
<span data-ttu-id="b55b5-103">Salt okunur alan 'tanımlayıcısı' üyeleri ref geçirilen olamaz veya out (dışındaki bir oluşturucu)</span><span class="sxs-lookup"><span data-stu-id="b55b5-103">Members of readonly field 'identifier' cannot be passed ref or out (except in a constructor)</span></span>  
  
 <span data-ttu-id="b55b5-104">Bir değişken üyesi olan bir işleve geçirirseniz, bu hata oluştuğunda bir `readonly` olarak alan bir `ref` veya `out` bağımsız değişkeni.</span><span class="sxs-lookup"><span data-stu-id="b55b5-104">This error occurs if you pass a variable to a function that is a member of a `readonly` field as a `ref` or `out` argument.</span></span> <span data-ttu-id="b55b5-105">Bu yana `ref` ve `out` parametreleri değiştirilmesi işlev tarafından bu verilmez.</span><span class="sxs-lookup"><span data-stu-id="b55b5-105">Since `ref` and `out` parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="b55b5-106">Bu hatayı gidermek için kaldırma `readonly` alan anahtar sözcüğü veya üyeleri geçmeyin `readonly` işlevi alanı.</span><span class="sxs-lookup"><span data-stu-id="b55b5-106">To resolve this error, remove the `readonly` keyword on the field, or do not pass the members of the `readonly` field to the function.</span></span> <span data-ttu-id="b55b5-107">Örneğin, bir geçici oluşturma deneyebilirsiniz değiştirilen ve geçici olarak geçirme olabilen değişkeni bir `ref` bağımsız değişkeni, aşağıdaki örnekte gösterildiği gibi.</span><span class="sxs-lookup"><span data-stu-id="b55b5-107">For example, you might try creating a temporary variable which can be modifed and passing the temporary as a `ref` argument, as shown in the following example.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b55b5-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="b55b5-108">Example</span></span>  
 <span data-ttu-id="b55b5-109">Aşağıdaki örnek CS1649 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b55b5-109">The following sample generates CS1649:</span></span>  
  
```csharp  
// CS1649.cs  
public struct Inner  
    {  
        public int i;  
    }  
  
class Outer  
{  
    public readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void f(ref int iref)  
    {  
    }  
  
    static void Main()  
    {  
        Outer outer = new Outer();   
        f(ref outer.inner.i);  // CS1649  
        // Try this code instead:  
        // int tmp = outer.inner.i;  
        // f(ref tmp);  
    }  
}  
```
