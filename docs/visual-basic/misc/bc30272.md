# <a name="39ltargumentnamegt39-is-not-a-parameter-of-39ltmethoddeclarationgt39"></a>&#39;&lt;argumentname&gt; &#39; , bir parametre değil &#39; &lt;methoddeclaration&gt;&#39;
Bir yordam çağrısı tarafından yordamı tanımlı değil bir ek bağımsız değişken sağlar.  
  
 **Hata Kimliği:** BC30272  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gereksiz bağımsız değişken bağımsız değişken listesinden kaldırır.
