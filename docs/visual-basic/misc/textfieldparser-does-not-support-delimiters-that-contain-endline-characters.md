# <a name="textfieldparser-does-not-support-delimiters-that-contain-endline-characters"></a>TextFieldParser endline karakterden sınırlayıcılar desteklemiyor
Endline karakter içeren bir sınırlayıcı belirtilmiş, ancak `TextFieldParser` endline karakterden sınırlayıcılar desteklemez.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Doğru sınırlayıcı sağlayın.
