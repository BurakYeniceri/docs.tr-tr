# <a name="compiler-error-cs0177"></a>Derleyici Hatası CS0177
Denetim geçerli yöntemi terk etmeden önce 'parametresi' out parametresi atanmalıdır  
  
 Bir parametre olarak [çıkışı](../../csharp/language-reference/keywords/out.md) anahtar sözcüğü yöntemi gövdesinde bir değer değil atandı. Daha fazla bilgi için bkz: [parametreleri geçirme](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)  
  
 Aşağıdaki örnek CS0177 oluşturur:  
  
```  
// CS0177.cs  
public class MyClass  
{  
   public static void Foo(out int i)   // CS0177  
   {  
   // uncomment the following line to resolve this error  
   //   i = 0;  
   }  
  
   public static void Main()  
   {  
       int x = -1;  
       Foo(out x);  
   }  
}  
```
