# <a name="loop-control-variable-cannot-be-a-property-or-a-late-bound-indexed-array"></a>For döngüsü denetim değişkeni bir özellik ya da geç bağlama sıralı dizinin olamaz
Yinelemek için kullanılan değişken bir `For` döngü sayısal veri türünde olması gerekir.  
  
 **Hata Kimliği:** BC30039  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   For döngüsü denetim değişkeni belirtmek için bildirimi değiştirme `Integer`, `Short`, `Long`, `Byte`, `Single`, `Double`, veya `Decimal`.
