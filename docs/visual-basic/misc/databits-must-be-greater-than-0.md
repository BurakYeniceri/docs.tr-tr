# <a name="databits-must-be-greater-than-0"></a>DataBits 0'dan büyük olmalıdır
`DataBits` İçin belirtilen bağımsız değişken `My.Computer.Ports.OpenSerialPort` yöntemi sıfırdan büyük olmalıdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değerini değiştirme `DataBits` pozitif bir sayı bağımsız değişkeni.
