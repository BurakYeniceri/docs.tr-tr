# <a name="array-dimensions-do-not-match-those-specified-in-the-39vbfixedarray39-attribute"></a>Dizi boyutları belirtilen eşleşmiyor &#39;VBFixedArray&#39; özniteliği
Dizinin boyutları içinde belirttiğiniz boyutları eşleşmiyor `VBFixedArray` özniteliği.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Dizi boyutları dizideki olun `Dim` deyimle eşleşecek tarafından belirtilen boyutları `VBFixedArray` özniteliği.
