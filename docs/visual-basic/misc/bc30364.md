# <a name="39sub-new39-cannot-be-declared-39ltmodifiergt39"></a>&#39; Yeni alt &#39; bildirilemez &#39; &lt;değiştiricisi&gt;&#39;
Bildirme girişiminde bulundunuz bir `Sub New` bir değiştirici ile gibi `Overrides`, yani geçerli değil.  
  
 **Hata Kimliği:** BC30364  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değiştirici kaldırın.
