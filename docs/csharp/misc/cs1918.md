## <a name="example"></a><span data-ttu-id="a39d6-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="a39d6-108">Example</span></span>  
 <span data-ttu-id="a39d6-109">Aşağıdaki örnek, CS1918 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="a39d6-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
