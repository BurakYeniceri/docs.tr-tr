<span data-ttu-id="4342a-103">'Base' anahtar sözcüğü geçerli bağlamda mevcut değil</span><span class="sxs-lookup"><span data-stu-id="4342a-103">Keyword 'base' is not available in the current context</span></span>  
  
 <span data-ttu-id="4342a-104">[Temel](../../csharp/language-reference/keywords/base.md) anahtar sözcüğü bir yöntemi, özelliği veya oluşturucusu dışında kullanıldı.</span><span class="sxs-lookup"><span data-stu-id="4342a-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>  
  
 <span data-ttu-id="4342a-105">Aşağıdaki örnek CS1512 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="4342a-105">The following example generates CS1512:</span></span>  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
