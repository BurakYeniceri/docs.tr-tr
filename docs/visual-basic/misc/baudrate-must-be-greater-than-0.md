# <a name="baudrate-must-be-greater-than-0"></a>Baud hızı 0'dan büyük olmalıdır
`BaudRate` İçin belirtilen bağımsız değişken `My.Computer.Ports.OpenSerialPort` yöntemi sıfırdan büyük olmalıdır.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Değerini değiştirme `BaudRate` pozitif bir sayı bağımsız değişkeni.
