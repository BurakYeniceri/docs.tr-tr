# <a name="method-in-a-structure-cannot-be-declared-39protected39-or-39protected-friend39"></a>Yapı içindeki yöntem bildirilemez &#39;korumalı&#39; veya &#39;Protected Friend&#39;
Yapı içindeki bir yöntem olarak bildirildi `Protected` veya `Protected Friend`.  
  
 **Hata Kimliği:** BC31067  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yöntemleri olarak yapılar içinde bildirme `Public` veya `Private`.
