<span data-ttu-id="0545e-103">Bir başvuru dönüştürme, kutulama dönüştürme, kutudan çıkarma dönüştürme, sarmalama dönüştürmesi veya null tür dönüştürme yoluyla 'type2' için 'type1' türü dönüştürülemiyor</span><span class="sxs-lookup"><span data-stu-id="0545e-103">Cannot convert type 'type1' to 'type2' via a reference conversion, boxing conversion, unboxing conversion, wrapping conversion, or null type conversion</span></span>  
  
 <span data-ttu-id="0545e-104">Dönüştürme işlemine [olarak](../../../csharp/language-reference/keywords/as.md) devralma, başvuru dönüşümleri ve kutulamayı dönüşümleri işlece izin verilir.</span><span class="sxs-lookup"><span data-stu-id="0545e-104">A conversion with the [as](../../../csharp/language-reference/keywords/as.md) operator is allowed by inheritance, reference conversions, and boxing conversions.</span></span> <span data-ttu-id="0545e-105">Daha fazla bilgi için bkz: [dönüştürme işleçleri](../../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="0545e-105">For more information, see [Conversion Operators](../../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="0545e-106">Örnek</span><span class="sxs-lookup"><span data-stu-id="0545e-106">Example</span></span>  
 <span data-ttu-id="0545e-107">Aşağıdaki örnek CS0039 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="0545e-107">The following example generates CS0039.</span></span>  
  
```csharp  
// CS0039.cs  
using System;  
class A  
{  
}  
class B: A  
{  
}  
class C: A  
{  
}  
class M  
{  
    static void Main()  
    {  
        A a = new C();  
        B b = new B();  
        C c;  
  
        // This is valid; there is a built-in reference  
        // conversion from A to C.  
        c = a as C;    
  
        //The following generates CS0039; there is no  
        // built-in reference conversion from B to C.  
        c = b as C;  // CS0039  
    }  
}  
```
