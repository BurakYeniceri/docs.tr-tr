# <a name="39case39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Servis talebi&#39; ifadeleri geçerli değildir komut penceresi
`Case` deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30719  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Case` gelen deyimleri **hemen** penceresi.
