# <a name="drive-39ltdrivenamegt39-not-found"></a>Sürücü &#39; &lt;drivename&gt; &#39; bulunamadı
Sürücü bulunamıyor. Ağ bağlantısı kaybedildi ya da erişmeye çalıştığınız sürücü çevrimdışı veya yok.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Sürücünün gücü denetleyin.  
  
2.  Bilgisayarınız ağa bağlanma kabloları düzgün şekilde bağlandığından emin olun.  
  
3.  Bir ağ üzerinden bir sürücüye erişmek için bir mantıksal bağlantı bilgisayarınız ve sürücü arasında olduğundan emin olun.  
  
4.  Ağa bağlayın ve yeniden deneyin.
