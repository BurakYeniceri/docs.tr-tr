# <a name="expression-cannot-be-converted-into-an-expression-tree"></a>İfade bir ifade ağacına dönüştürülemiyor
Çoğu ifadenin ifade ağaçları dönüştürülebilir, ancak bazıları olamaz. Bu hata, girişiminde dönüştürülemeyecek bir ifadeyi dönüştürmek için yapılan gösterir.  
  
 **Hata Kimliği:** BC36534
