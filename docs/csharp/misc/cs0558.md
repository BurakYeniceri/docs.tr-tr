 <span data-ttu-id="9f155-104">Her iki **statik** ve **ortak** erişim [değiştiricileri](../../csharp/language-reference/keywords/modifiers.md) kullanıcı tanımlı işleçler belirtilmesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="9f155-104">Both the **static** and **public** access [modifiers](../../csharp/language-reference/keywords/modifiers.md) must be specified on user-defined operators.</span></span>

      public class iii  
      {

         }  
      }

      {  
      }  
   }  
}
