# <a name="no-method-39ltmethodnamegt39-can-accept-an-argument-of-type-39lttypenamegt39-for-parameter-39ltparameternamegt39"></a>Bir yöntem &#39; &lt;methodname&gt; &#39; türünde bir bağımsız değişken kabul edebilir &#39; &lt;typename&gt; &#39; parametresi için &#39; &lt;parametername&gt;&#39;
Yöntem çağrısı geçerli değil türünde bir bağımsız değişken içeriyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bağımsız değişken türleri kontrol edin.  
  
2.  Buna agreement olduklarından emin olmak için bağımsız değişken türleriyle yöntemi parametre türleri eşleştirin. Türleri gerektiği gibi değiştirin.
