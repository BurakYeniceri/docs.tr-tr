# <a name="compiler-error-cs0013"></a>Derleyici Hatası CS0013
Meta verileri 'ModelStore.dll'--dosyasına yazılırken beklenmeyen bir hata ' mantıksal boşluk sol daha fazla kullanıcı dizesi oluşturmak için.  
  
 .NET Framework ortak dil çalışma zamanı meta veri yayma değil. Yolun doğru olduğunu ve diskin dolu olmadığından emin olmak için kontrol edin. Sorun devam ederse, onarın veya Visual Studio ve/veya .NET Framework yeniden yüklemeniz gerekebilir.
