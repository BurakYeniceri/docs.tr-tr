### <a name="sslstream-supports-tls-alerts"></a>TLS uyarıları SslStream destekler

|   |   |
|---|---|
|Ayrıntılar|Başarısız bir TLS el sıkışma sonra bir <xref:System.IO.IOException?displayProperty=name> bir iç ile <xref:System.ComponentModel.Win32Exception?displayProperty=name> özel ilk g/ç okuma/yazma işlemi tarafından durum. <xref:System.ComponentModel.Win32Exception.NativeErrorCode?displayProperty=name> İçin kod <xref:System.ComponentModel.Win32Exception?displayProperty=name> bu kullanarak uzak taraftan TLS uyarısı eşlenebilir [Schannel belgelerine](https://msdn.microsoft.com/library/windows/desktop/dd721886%28v=vs.85%29.aspx). Daha fazla bilgi için bkz: [RFC 2246'yı: Bölüm 7.2.2 hata uyarıları](https://tools.ietf.org/html/rfc2246#section-7.2.2).NET 4.6.2 ve aşağıda taşıma kanalı (TCP bağlantısı genellikle) olacağı zaman aşımı yazma veya okuma sırasında diğer taraf başarısız olursa davranıştır El Sıkışma ve hemen ardından bağlantıyı reddetti.|
|Öneri|Çağıran uygulamalara ağ g/ç API'leri gibi <xref:System.IO.Stream.Read(System.Byte[],System.Int32,System.Int32)> / <xref:System.IO.Stream.Write(System.Byte[],System.Int32,System.Int32)> işlemesi gereken <xref:System.IO.IOException> veya <xref:System.TimeoutException?displayProperty=name>. TLS uyarılar özelliği .NET 4.7 ile başlayarak varsayılan olarak etkindir. .NET 4.0 - .NET 4.6.2 hedefleme uygulamaları .NET 4.7 veya yüksek sistem üzerinde çalışan uyumluluğu korumak için devre dışı özelliği olacaktır. Aşağıdaki yapılandırma API'si etkinleştirmek veya .NET 4.6 ve .NET 4.7 veya daha yüksek framework üzerinde çalışan uygulamalar yukarıda özelliğini devre dışı bırakmak kullanılabilir.<ul><li>Program aracılığıyla:</li></ul>ServicePointManager yalnızca bir kez başlatacak bu yana uygulama yapacağı çok ilk şey olması gerekir:<pre><code class="language-C#">AppContext.SetSwitch(&quot;TestSwitch.LocalAppContext.DisableCaching&quot;, true);&#13;&#10;AppContext.SetSwitch(&quot;Switch.System.Net.DontEnableTlsAlerts&quot;, true); // Set to &#39;false&#39; to enable the feature in .NET 4.6 - 4.6.2.&#13;&#10;</code></pre><ul><li>AppConfig:</li></ul><pre><code class="language-XML">&lt;runtime&gt;&#13;&#10;&lt;AppContextSwitchOverrides value=&quot;Switch.System.Net.DontEnableTlsAlerts=true&quot;/&gt;&#13;&#10;&lt;!-- Set to &#39;false&#39; to enable the feature in .NET 4.6 - 4.6.2. --&gt;&#13;&#10;&lt;/runtime&gt;&#13;&#10;</code></pre><ul><li>Kayıt defteri anahtarı (makine genel):</li></ul>.NET 4. 6 - 4.6.2 özelliğini etkinleştirmek için 'false' değerine ayarlayın.<pre><code>Key = HKLM\SOFTWARE\[Wow6432Node\]Microsoft\.NETFramework\AppContext\Switch.System.Net.DontEnableTlsAlerts&#13;&#10;Type = String&#13;&#10;Value = &quot;true&quot;&#13;&#10;</code></pre>|
