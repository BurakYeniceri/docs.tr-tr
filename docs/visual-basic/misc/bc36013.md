# <a name="39on-error39-statements-are-not-valid-within-39using39-statements"></a>&#39; hata &#39; deyimleri içinde geçersiz &#39; &#39;kullanarak; deyimleri
Bir `On Error` deyimi görünüyor içinde bir `Using` deyimi ancak bu bağlamda geçerli değil.  
  
 **Hata Kimliği:** BC36013  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Yapılandırılmış hata gibi işleme kullanan bir `Try…Catch` yerine blok `On Error` deyimi.
