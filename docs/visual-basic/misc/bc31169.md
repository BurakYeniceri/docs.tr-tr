# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-at-the-beginning-of-an-xml-name"></a>Karakter &#39; karakter &#39; (0 x&lt;onaltılık&gt;) bir XML adı başında izin verilmiyor
Desteklenmeyen bir karakter, bir XML değişmez değer adının başında ya da bir XML axis özelliği bulunur.  
  
 **Hata Kimliği:** BC31169  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML değişmez adını başından itibaren desteklenmeyen bir karakter veya XML axis özelliği kaldırın.
