# <a name="xml-comment-has-a-tag-with-a-39cref39-attribute-39ltattributegt39-that-could-not-be-resolved"></a>XML açıklama sahip bir etiketle bir &#39;cref&#39; özniteliği &#39; &lt;özniteliği&gt; &#39; , çözümlenemedi
XML açıklama sahip bir etiketle 'cref' özniteliği \<özniteliği >, çözümlenemedi. XML açıklama yoksayılacak.  
  
 Etiketler olabilir bir `cref` tanımlayıcısı göreli adını belirterek XML başka bir öğe için bir bağlantı atar özniteliği. Derleme zamanında derleyici değeri tam bir XML tanımlayıcısı kullanıcı tarafından işaret değeri ile değiştirir. Derleme, tür veya üye bulmak için normal çözümleme kurallarını kullanır.  
  
 **Hata Kimliği:** BC42309  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Doğrulama `cref` geçerli kod öğesine işaret eden özniteliği.
