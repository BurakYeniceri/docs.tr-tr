# <a name="39ltelementnamegt39-is-not-a-parameter-of-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>&#39;&lt;ElementName&gt; &#39; genişletme yöntemi, bir parametre değil &#39; &lt;methodname&gt; &#39; tanımlanan &#39; &lt;typename&gt;&#39;
Bir yordam çağrısı genişletme yöntemi tarafından tanımlanmamış bir ek bağımsız değişken sağlar.  
  
 **Hata Kimliği:** BC36585  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Gereksiz bağımsız değişken bağımsız değişken listesinden kaldırır.
