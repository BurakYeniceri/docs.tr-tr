# <a name="object-required-visual-basic"></a>Gerekli nesne (Visual Basic)
Özellikleri ve yöntemleri başvuruları genellikle açık nesne niteleyicisi gerektirir. Böyle bir durumda budur.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Nesne özelliği veya yöntemi başvurular geçerli nesne niteleyicisi olup olmadığını denetleyin. Bir nesne niteleyicisi bir sağlamadı belirtin.  
  
2.  Nesne niteleyicisi yazımını denetleyin ve nesneyi bu başvurduğunuzdan programı bölümünde görünür olduğundan emin olun.  
  
3.  Bir konak uygulamanın için bir yol sağlamadığını **Dosya Aç** komutu, değişkenlerinde doğru olup olmadığını denetleyin.  
  
4.  Nesnenin belgelerini inceleyin ve eylemi geçerli olduğundan emin olun.
