## <a name="example"></a><span data-ttu-id="6e31f-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="6e31f-108">Example</span></span>  
 <span data-ttu-id="6e31f-109">Aşağıdaki örnek CS0688 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="6e31f-109">The following sample generates CS0688.</span></span> <span data-ttu-id="6e31f-110">Taban sınıf değiştirmeden uyarıyı çözümlemek için geçersiz kılma yönteminden güvenlik özniteliğini kaldırın.</span><span class="sxs-lookup"><span data-stu-id="6e31f-110">To resolve the warning without modifying the base class, remove the security attribute from the overriding method.</span></span> <span data-ttu-id="6e31f-111">Bu güvenlik sorunu çözmek için değil.</span><span class="sxs-lookup"><span data-stu-id="6e31f-111">This will not solve the security problem.</span></span>  
  
```csharp  
// CS0688.cs  
// compile with: /W:1  
using System;  
using System.Security.Permissions;  
  
class Base   
{  
    //Uncomment the following line to close the security hole  
    //[FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")]  
    public virtual void DoScaryFileStuff()  
    {  
    }  
}  
  
class Derived: Base  
{  
    [FileIOPermission(SecurityAction.LinkDemand, All=@"C:\\")] // CS0688  
    public override void DoScaryFileStuff()  
    {  
    }  
    static void Main()  
    {  
    }  
}  
```
