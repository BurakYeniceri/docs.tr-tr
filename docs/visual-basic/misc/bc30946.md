# <a name="stop-request-is-pending"></a>Durdurma isteği bekliyor
Visual Studio hata ayıklayıcısında, yordam çağrısı bir ifade belirtir, ancak iş parçacığı durdurma isteği yok.  
  
 Hata ayıklayıcı etkin olmayan bir iş parçacığında bir yordam çağırma denemez.  
  
 **Hata Kimliği:** BC30946  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Mümkünse, böylece yeniden oluşmasını engellemek iş parçacığı durdurma isteği kaynağı belirler.  
  
-   Sonlandırma ve hata ayıklamayı yeniden başlatın.
