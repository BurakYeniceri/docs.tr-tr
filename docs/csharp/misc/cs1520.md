# <a name="compiler-error-cs1520"></a><span data-ttu-id="e6e33-102">Derleyici Hatası CS1520</span><span class="sxs-lookup"><span data-stu-id="e6e33-102">Compiler Error CS1520</span></span>
<span data-ttu-id="e6e33-103">Yöntemi, dönüş türü olmalıdır</span><span class="sxs-lookup"><span data-stu-id="e6e33-103">Method must have a return type</span></span>  
  
 <span data-ttu-id="e6e33-104">Bir sınıf, yapı veya arabirim bildirilmiş bir yöntem açık bir dönüş türü olmalıdır.</span><span class="sxs-lookup"><span data-stu-id="e6e33-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="e6e33-105">Aşağıdaki örnekte, kare yöntemin dönüş değerine sahip [dize](../../csharp/language-reference/keywords/string.md):</span><span class="sxs-lookup"><span data-stu-id="e6e33-105">In the following example, the Square method has a return value of [string](../../csharp/language-reference/keywords/string.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="e6e33-106">Aşağıdaki örnek CS1520 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="e6e33-106">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520     
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="e6e33-107">Alternatif olarak, bir oluşturucu kişinin adı harf, aşağıdaki örnekteki gibi sınıf veya yapı bildirimi farklıdır, bu hata ile.</span><span class="sxs-lookup"><span data-stu-id="e6e33-107">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="e6e33-108">Adı tam olarak aynı sınıf adı olduğundan, derleyici, bir oluşturucu gibi normal bir yöntem olarak yorumlar ve hata üretir:</span><span class="sxs-lookup"><span data-stu-id="e6e33-108">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
