 <span data-ttu-id="b67d7-104">Çağıramazsınız bir [soyut](../../csharp/language-reference/keywords/abstract.md) yöntemi çünkü bir yöntem gövdesi yok.</span><span class="sxs-lookup"><span data-stu-id="b67d7-104">You cannot call an [abstract](../../csharp/language-reference/keywords/abstract.md) method because it does not have a method body.</span></span> <span data-ttu-id="b67d7-105">Daha fazla bilgi için bkz: [soyut ve korumalı sınıflar ve sınıf üyeleri](../../csharp/programming-guide/classes-and-structs/abstract-and-sealed-classes-and-class-members.md).</span><span class="sxs-lookup"><span data-stu-id="b67d7-105">For more information, see [Abstract and Sealed Classes and Class Members](../../csharp/programming-guide/classes-and-structs/abstract-and-sealed-classes-and-class-members.md).</span></span>  
  
 <span data-ttu-id="b67d7-106">Aşağıdaki örnek CS0205 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="b67d7-106">The following sample generates CS0205:</span></span>  
  
```csharp  
// CS0205.cs

   abstract public void M();  
}  
  
public class MyClass2 : MyClass  
{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
