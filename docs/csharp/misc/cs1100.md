 <span data-ttu-id="4de26-104">`this` Değiştiricisi, derleyicinin yöntemin bir genişletme yöntemi olduğunu belirten bir yöntem yalnızca ilk parametresinin verilir.</span><span class="sxs-lookup"><span data-stu-id="4de26-104">The `this` modifier is allowed only on the first parameter of a method, which indicates to the compiler that the method is an extension method.</span></span>
