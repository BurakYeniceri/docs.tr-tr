# <a name="compiler-error-cs0637"></a><span data-ttu-id="6fd5a-102">Derleyici Hatası CS0637</span><span class="sxs-lookup"><span data-stu-id="6fd5a-102">Compiler Error CS0637</span></span>

<span data-ttu-id="6fd5a-103">Statik veya sabit alanlarda FieldOffset özniteliğine izin verilmiyor.</span><span class="sxs-lookup"><span data-stu-id="6fd5a-103">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="6fd5a-104">[FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) özniteliği ile işaretlenmiş alanlar kullanılamaz [statik](../../csharp/language-reference/keywords/static.md) veya [const](../../csharp/language-reference/keywords/const.md).</span><span class="sxs-lookup"><span data-stu-id="6fd5a-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../../csharp/language-reference/keywords/static.md) or [const](../../csharp/language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="6fd5a-105">Aşağıdaki örnek CS0637 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="6fd5a-105">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
