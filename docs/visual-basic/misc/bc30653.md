# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-type-39ltmembernamegt39"></a>Modül &#39;için gereken başvuru; &lt;modulename&gt;&#39; türündeki &#39; içeren&lt; membername&gt;&#39;
Modül için gereken başvuru '\<modulename >' türünü içeren '\<membername >'. Projenize ekleyin.  
  
 Projesi içinde proje başvuruları dahil edilmeyen bir derlemeyi başka bir türden bağımlı bir türü içerir.  
  
 **Hata Kimliği:** BC30653  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bağımlı türleri için proje başvurusu projeye ekleyin.
