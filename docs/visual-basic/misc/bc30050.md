# <a name="paramarray-parameter-must-be-an-array"></a>ParamArray parametresi bir dizi olmalıdır
A `ParamArray` bağımsız değişkeni bir tek boyutlu dizi türü bildirilmelidir. Bildirim sonra parantez içermez `ParamArray` bağımsız değişken adı.  
  
 **Hata Kimliği:** BC30050  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sonra parantez çifti eklemek `ParamArray` bağımsız değişken adı.
