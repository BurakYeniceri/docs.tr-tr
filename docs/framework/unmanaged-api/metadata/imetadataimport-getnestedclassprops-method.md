# <a name="imetadataimportgetnestedclassprops-method"></a>IMetaDataImport::GetNestedClassProps Metodu
TypeDef için üst belirtecini alır <xref:System.Type> türü belirtilen iç içe geçmiş.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT GetNestedClassProps (  
   [in]   mdTypeDef      tdNestedClass,  
   [out]  mdTypeDef      *ptdEnclosingClass  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `tdNestedClass`  
 [in] TypeDef belirteci temsil eden bir <xref:System.Type> üst sınıfın döndürülecek için belirteç.  
  
 `ptdEnclosingClass`  
 [out] TypeDef belirteci için bir işaretçi <xref:System.Type> , `tdNestedClass` iç içe yerleştirilmiş.
