# <a name="compiler-error-cs0549"></a><span data-ttu-id="8b969-102">Derleyici Hatası CS0549</span><span class="sxs-lookup"><span data-stu-id="8b969-102">Compiler Error CS0549</span></span>
<span data-ttu-id="8b969-103">'function' korumalı sınıfı 'sınıfı' yeni sanal üyesi olduğu</span><span class="sxs-lookup"><span data-stu-id="8b969-103">'function' is a new virtual member in sealed class 'class'</span></span>  
  
 <span data-ttu-id="8b969-104">A [korumalı](../../csharp/language-reference/keywords/sealed.md)[sınıfı](../../csharp/language-reference/keywords/class.md) temel sınıf olarak kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="8b969-104">A [sealed](../../csharp/language-reference/keywords/sealed.md)[class](../../csharp/language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="8b969-105">Bu nedenle, sanal bir yöntem korumalı sınıfında sahip gereksizdir.</span><span class="sxs-lookup"><span data-stu-id="8b969-105">Therefore, it is useless to have a virtual method in sealed class.</span></span>  
  
 <span data-ttu-id="8b969-106">Aşağıdaki örnek CS0549 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="8b969-106">The following sample generates CS0549:</span></span>  
  
```csharp

{  
   virtual public void TestMethod() {}   // CS0549

}  
```
