# <a name="array-exceeds-the-limit-of-32-dimensions"></a>Dizi 32 boyut sınırını aşıyor
Diziler 32'den fazla boyutlar sahip olamaz.  
  
 **Hata Kimliği:** BC30052  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bildiriminde dizinin boyut sayısını azaltın.
