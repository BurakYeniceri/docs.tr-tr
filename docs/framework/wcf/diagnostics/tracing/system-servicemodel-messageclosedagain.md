# <a name="systemservicemodelmessageclosedagain"></a>System.ServiceModel.MessageClosedAgain
System.ServiceModel.MessageClosedAgain  
  
## <a name="description"></a>Açıklama  
 Bir ileti yeniden kapatıldı.  
  
 Bir ileti yalnızca bir kez kapatılması gerekiyor. Bu izleme kullanıcı uzantısı kodunda yayılan, kullanıcı uzantı kodu zaten kapatılmış bir ileti kapatılıyor gösterir. Bu izleme ürün kodu yayılan, kullanıcı uzantı kodu potansiyel olarak bir ileti çok erken kapatma olması olduğunu gösterir.
