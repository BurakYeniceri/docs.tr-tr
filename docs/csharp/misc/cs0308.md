# <a name="compiler-error-cs0308"></a><span data-ttu-id="700d4-102">Derleyici Hatası CS0308</span><span class="sxs-lookup"><span data-stu-id="700d4-102">Compiler Error CS0308</span></span>
<span data-ttu-id="700d4-103">Genel olmayan türü-veya-method 'tanımlayıcısı' tür bağımsız değişkenleri ile kullanılamaz.</span><span class="sxs-lookup"><span data-stu-id="700d4-103">The non-generic type-or-method 'identifier' cannot be used with type arguments.</span></span>  
  
 <span data-ttu-id="700d4-104">Metot veya tür genel değil, ancak tür bağımsız değişkenleri ile kullanıldı.</span><span class="sxs-lookup"><span data-stu-id="700d4-104">The method or type is not generic, but it was used with type arguments.</span></span> <span data-ttu-id="700d4-105">Bu hatayı önlemek için açılı ayraç kaldırıp türü bağımsız değişkenleri veya metot veya tür bir genel yöntem veya türü redeclare.</span><span class="sxs-lookup"><span data-stu-id="700d4-105">To avoid this error, remove the angled brackets and type arguments, or redeclare the method or type as a generic method or type.</span></span>  
  
 <span data-ttu-id="700d4-106">Aşağıdaki örnek CS0308 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="700d4-106">The following example generates CS0308:</span></span>  
  
```csharp  
// CS0308a.cs  
class MyClass  
{  
   public void F() {}  
   public static void Main()  
   {  
      F<int>();  // CS0308 – F is not generic.  
      // Try this instead:  
      // F();  
   }  
}  
```  
  
 <span data-ttu-id="700d4-107">Aşağıdaki örnek, aynı zamanda CS0308 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="700d4-107">The following example also generates CS0308.</span></span> <span data-ttu-id="700d4-108">Hatayı gidermek için "System.Collections.Generic kullanıyor." yönergesini kullanın.</span><span class="sxs-lookup"><span data-stu-id="700d4-108">To resolve the error, use the directive "using System.Collections.Generic."</span></span>  
  
```csharp  
// CS0308b.cs  
// compile with: /t:library  
using System.Collections;  
// To resolve, uncomment the following line:  
// using System.Collections.Generic;  
public class MyStack<T>  
{  
    // Store the elements of the stack:  
    private T[] items = new T[100];  
    private int stack_counter = 0;  
  
    // Define the iterator block:  
    public IEnumerator<T> GetEnumerator()   // CS0308  
    {  
        for (int i = stack_counter - 1 ; i >= 0; i--)  
        yield return items[i];  
    }  
}  
```
