# <a name="39ltmodulenamegt39-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>&#39; &lt;modulename&gt;&#39; bir modül, derleme olarak başvurulamaz
Bir modül derleme olarak kullanmak için girişimde bulunuldu.  
  
 **Hata Kimliği:** BC31076  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bir derleme başvurusu değiştirin.
