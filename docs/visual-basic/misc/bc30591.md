# <a name="39withevents39-variable-does-not-raise-any-events"></a>&#39; WithEvents &#39; değişken herhangi bir olayı oluşturmaz
Nesne değişkeni ile bildirilen `WithEvents` anahtar sözcüğü olayları yükseltebilirsiniz bir sınıf örneği başvurmuyor.  
  
 **Hata Kimliği:** BC30591  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Nesne değişkeni olayları yükseltebilirsiniz bir sınıf örneği başvurduğundan emin olun.
