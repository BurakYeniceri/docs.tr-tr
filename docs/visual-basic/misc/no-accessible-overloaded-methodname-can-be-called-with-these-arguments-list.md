# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-ltlistgt"></a>Hayır erişilebilir aşırı yüklenmiş &#39; &lt;methodname&gt;&#39; bu bağımsız değişkenlerle çağrılamaz: &lt;listesi&gt;
Aşırı yüklenmiş yöntemin çağrıldı, ancak hiçbir yöntemi sağlanan bağımsız değişken listesiyle eşleştirilir.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bağımsız değişken listesine göre çağırmak için hedeflenen aşırı eşleşmesi.  
  
-   Bağımsız değişken listesi aşırı yük imzayı eşleşen şekilde ayarlayın.
