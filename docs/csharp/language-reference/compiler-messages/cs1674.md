# <a name="compiler-error-cs1674"></a>Derleyici Hatası CS1674
'T': using deyiminde kullanılan tür deyimi 'System.IDisposable' örtük olarak dönüştürülebilir olmalıdır  
  
 [Deyimiyle](../../../csharp/language-reference/keywords/using-statement.md) sonunda bir nesnenin elden emin olmak için kullanılması hedeflenen `using` engellemek, bu nedenle, bu tür bir deyimde yalnızca atılabilir türler kullanılabilir. Örneğin, değer türleri tek kullanımlık değildir ve sınıfları kısıtlı olmayan tür parametreleri atılabilir olarak kabul edilebilir değil.  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS1674 oluşturur.  
  
```csharp  
// CS1674.cs  
class C  
{  
   public static void Main()  
   {  
      int a = 0;  
      a++;  
  
      using (a) {}   // CS1674  
   }  
}  
```  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS1674 oluşturur.  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }  
}  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   public static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```  
  
## <a name="example"></a>Örnek  
 Aşağıdaki durum bilinmeyen tür bir parametrenin harcanabilir olduğunu güvence altına almak bir sınıf türü kısıtlaması gereksinimini gösterilmektedir. Aşağıdaki örnek CS1674 oluşturur.  
  
```csharp  
// CS1674_c.cs  
// compile with: /target:library  
using System;  
public class C<T>  
// Add a class type constraint that specifies a disposable class.  
// Uncomment the following line to resolve.  
// public class C<T> where T : IDisposable  
{  
   public void F(T t)  
   {  
      using (t) {}   // CS1674  
   }  
}  
```
