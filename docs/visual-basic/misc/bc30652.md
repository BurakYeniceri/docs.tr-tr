# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-type-39ltmembernamegt39"></a>Derleme &#39;için gereken başvuru; &lt;assemblyname&gt;&#39; türündeki &#39; içeren&lt; membername&gt;&#39;
Derleme için gerekli referans '\<assemblyname >' türünü içeren '\<membername >'. Projenize ekleyin.  
  
 Projesi içinde proje başvuruları dahil edilmeyen bir derlemeyi başka bir türden bağımlı bir türü içerir.  
  
 **Hata Kimliği:** BC30652  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bağımlı türleri için proje başvurusu projeye ekleyin.
