# <a name="39option-infer39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Option Infer&#39; yalnızca izlenebilir &#39;üzerinde&#39; veya &#39;devre dışı&#39;
`Option Infer` deyimleri ya da belirtmeniz gerekir `On` veya `Off`. Diğer değerler geçerli değildir.  
  
 **Hata Kimliği:** BC30979  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Belirtin `Option Infer On` veya `Option Infer Off`.
