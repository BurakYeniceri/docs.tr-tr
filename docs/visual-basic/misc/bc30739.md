# <a name="39raiseevent39-statements-are-not-valid-in-the-immediate-window"></a>&#39;RaiseEvent&#39; ifadeleri geçerli değildir komut penceresi
`RaiseEvent` ifadeleri kullanılamaz **hemen** penceresi.  
  
 **Hata Kimliği:** BC30739  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Test `RaiseEvent` deyimleri koddan **Kod düzenleyicisinde**.
