# <a name="xml-axis-properties-do-not-support-late-binding"></a>XML eksen özellikleri geç bağlamayı desteklemiyor
Yazılmamış bir nesne için bir XML axis özelliği başvuruldu.  
  
 **Hata Kimliği:** BC31168  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Nesne bir güçlü yazılmış olduğundan emin olun <xref:System.Xml.Linq.XElement> XML eksen özelliğe başvurma önce nesne.
