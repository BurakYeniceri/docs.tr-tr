# <a name="ivalidator-interface"></a>IValidator Arabirimi
Taşınabilir yürütülebilir (PE) görüntüler doğrulama ve doğrulama hatalarını raporlama için yöntemleri sağlar.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|Doğrula|Belirtilen PE veya Microsoft Ara dili (MSIL) dosya doğrular.|  
|Formateventınfo|Belirtilen doğrulama hatası karşılık gelen hata iletisi alır.|
