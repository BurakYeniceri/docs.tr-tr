# <a name="xml-comment-tag-39returns39-is-not-permitted-on-a-39writeonly39-property"></a>XML açıklama etiketi &#39; döndürür &#39; üzerinde izin verilmiyor bir &#39; WriteOnly &#39; Özelliği
XML açıklama etiketi 'returns' bir 'WriteOnly' özellikte izin verilmez. XML açıklama yoksayılacak.  
  
 Salt yazılır özellik bildirimi için bir XML yorum içeremez bir \<döndürür > etiketi.  
  
 **Hata Kimliği:** BC42313  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Desteklenmeyen kaldırmak \<döndürür > etiketi.
