# <a name="constructor-must-be-declared-as-a-sub-not-as-a-function"></a>Oluşturucu bir işlevi olarak değil, Sub olarak bildirilmesi gerekir
Bildirme girişiminde bulundunuz bir `Function New`. Oluşturucular bildirilen, olarak `Sub New`.  
  
 **Hata Kimliği:** BC30493  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kullanım `Sub` yerine `Function`.
