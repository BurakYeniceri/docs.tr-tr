# <a name="property-not-found"></a>Özellik bulunamadı
Bu nesne belirtilen bir özelliği desteklemiyor.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Özelliğin adının yazımını denetleyin.  
  
2.  Nesne "Açıklama" veya benzer ada özellik gerçekte desteklediğinde "metin" özellik benzeri erişim çalışılırken değil emin olmak için nesnenin belgelerine bakın.
