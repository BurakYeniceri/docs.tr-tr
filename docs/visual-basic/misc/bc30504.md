# <a name="property-39ltpropertynamegt39-cannot-be-declared-39default39-because-it-overrides-a-property-on-the-base-lttypegt-39lttypenamegt39-that-is-not-default"></a>Özellik &#39; &lt;propertyname&gt;&#39; bildirilemez &#39; varsayılan &#39; temel bir özellik kıldığından &lt;türü&gt; &#39;&lt; TypeName&gt;& varsayılan olmayan #39;
Bir özelliği bildirme girişiminde bulundunuz `Default`, olmayan bir geçerli bildirimi bildirimi yapılmamış bir özellik kıldığından `Default` temel sınıfı.  
  
 **Hata Kimliği:** BC30504  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Default` bildirimi.
