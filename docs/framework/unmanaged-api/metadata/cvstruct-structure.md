# <a name="cvstruct-structure"></a>CVStruct Yapısı
Bir modül veya bileşik görüntü yüklenirken kullanılan bilgileri içerir.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
typedef struct {  
    short Major;  
    short Minor;  
    short Sub;  
    short Build;  
} CVStruct;  
```
