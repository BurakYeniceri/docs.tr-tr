# <a name="attributes-visual-basic"></a>Öznitelikler (Visual Basic)
Visual Basic nesneleri yönetilmeyen kod ile birlikte çalışmak izin birkaç öznitelik ve modül adı erişilmesine modülü üyeleri sağlayan bir öznitelik sağlar. Aşağıdaki tabloda, Visual Basic tarafından kullanılan öznitelikleri listeler.  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.ComClassAttribute>|Bir sınıf bir COM nesnesi olarak gösterilmesine izin veren meta verileri eklemek için derleyicisi bildirir.|  
|<xref:Microsoft.VisualBasic.HideModuleNameAttribute>|Modül üyelerinin modülü için gerekli nitelik kullanarak erişilmesine olanak tanır.|  
|<xref:Microsoft.VisualBasic.VBFixedArrayAttribute>|Bir dizi yapısı ya da yerel olmayan değişkeni bir sabit uzunluk dizisi olarak ele alınması gerektiğini gösterir.|  
|<xref:Microsoft.VisualBasic.VBFixedStringAttribute>|Uzunluk düzeltilen gibi bir dize ele alınması gerektiğini gösterir.|
