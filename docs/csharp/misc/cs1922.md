# <a name="compiler-error-cs1922"></a><span data-ttu-id="18b6f-102">Derleyici Hatası CS1922</span><span class="sxs-lookup"><span data-stu-id="18b6f-102">Compiler Error CS1922</span></span>
<span data-ttu-id="18b6f-103">Koleksiyon Başlatıcısı System.Collections.IEnumerable uygulamak için ' type' türü gerektirir.</span><span class="sxs-lookup"><span data-stu-id="18b6f-103">Collection initializer requires its type 'type' to implement System.Collections.IEnumerable.</span></span>  
  
 <span data-ttu-id="18b6f-104">Koleksiyon Başlatıcısı ile bir türü kullanmak için türü uygulanması gereken `IEnumerable`.</span><span class="sxs-lookup"><span data-stu-id="18b6f-104">In order to use a collection initializer with a type, the type must implement `IEnumerable`.</span></span> <span data-ttu-id="18b6f-105">Nesne Başlatıcı kullanılacak amacı, koleksiyon Başlatıcısı sözdizimi yanlışlıkla kullanırsanız, bu hata oluşabilir.</span><span class="sxs-lookup"><span data-stu-id="18b6f-105">This error can occur if you accidentally use collection initializer syntax when you meant to use an object initializer.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="18b6f-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="18b6f-106">To correct this error</span></span>  
  
-   <span data-ttu-id="18b6f-107">Bir koleksiyon türünü temsil etmez, nesne Başlatıcısı sözdizimi koleksiyon Başlatıcısı sözdizimi yerine kullanın.</span><span class="sxs-lookup"><span data-stu-id="18b6f-107">If the type does not represent a collection, use object initializer syntax instead of collection initializer syntax.</span></span>

## <a name="example"></a><span data-ttu-id="18b6f-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="18b6f-110">Example</span></span>  
 <span data-ttu-id="18b6f-111">Aşağıdaki kod CS1922 üretir:</span><span class="sxs-lookup"><span data-stu-id="18b6f-111">The following code produces CS1922:</span></span>  
  
```csharp  
// cs1922.cs

{  
    public static void Main()  
    {  
        // Collection initializer.  
        var tc = new TestClass  {1,"hello"} ; // CS1922

        var tc2 = new TestClass { memberA = 1, memberB = "hello" }; // OK  
    }

{  
    public int memberA { get; set; }  
    public string memberB { get; set; }  
}  
```
