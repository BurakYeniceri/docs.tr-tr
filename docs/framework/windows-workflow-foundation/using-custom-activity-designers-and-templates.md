## <a name="in-this-section"></a>Bu Bölümde  
 [Nasıl Yapılır: Özel Etkinlik Tasarımcısı Oluşturma](../../../docs/framework/windows-workflow-foundation/how-to-create-a-custom-activity-designer.md)
