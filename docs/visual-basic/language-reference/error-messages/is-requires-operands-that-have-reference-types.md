# <a name="39is39-requires-operands-that-have-reference-types-but-this-operand-has-the-value-type-39lttypenamegt39"></a>&#39; olan &#39; işlenen gerekiyor başvuru türleri vardır, ancak bu işlenen değer türü &#39; var &lt;typename&gt;&#39;
`Is` Karşılaştırma işlecinin iki nesne değişkenleri aynı örneğine başvuru olup olmadığını belirler. Bu karşılaştırma değer türleri için tanımlı değil.  
  
 **Hata Kimliği:** BC30020  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Uygun aritmetik karşılaştırma işlecini kullanın veya `Like` iki değer türleri Karşılaştırılacak işleci.
