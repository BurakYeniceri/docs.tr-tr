# <a name="ltpropertygt-parameters-cannot-be-declared-39byref39"></a>&lt;Özellik&gt; parametreleri bildirilemez &#39;ByRef&#39;
Özellikler için parametreleri kullanılarak bildirilmelidir `ByVal` değiştiricisi.  
  
 **Hata Kimliği:** BC30651  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Özellik parametre olarak tanımlamak `ByVal`.
