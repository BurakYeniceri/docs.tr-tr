 <span data-ttu-id="51b1d-116">Aşağıdaki örnekte ilk parametre yalnızca göstergesidir, `Print` yöntemin genişlettiği `String` veri türü.</span><span class="sxs-lookup"><span data-stu-id="51b1d-116">The first parameter in the following example is the only indication that the `Print` method extends the `String` data type.</span></span> <span data-ttu-id="51b1d-117">Bu nedenle, isteğe bağlı olamaz.</span><span class="sxs-lookup"><span data-stu-id="51b1d-117">Therefore, it cannot be optional.</span></span>
