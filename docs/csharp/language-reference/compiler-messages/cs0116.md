# <a name="compiler-error-cs0116"></a><span data-ttu-id="1c375-102">Derleyici Hatası CS0116</span><span class="sxs-lookup"><span data-stu-id="1c375-102">Compiler Error CS0116</span></span>

 <span data-ttu-id="1c375-104">İçinde bir [ad alanı](../../../csharp/language-reference/keywords/namespace.md), derleyicinin yalnızca sınıflar, yapılar, birleşimler, numaralandırmalar, arabirimler ve temsilciler kabul eder.</span><span class="sxs-lookup"><span data-stu-id="1c375-104">Inside a [namespace](../../../csharp/language-reference/keywords/namespace.md), the compiler only accepts classes, structures, unions, enumerations, interfaces, and delegates.</span></span> <span data-ttu-id="1c375-105">Bu hata genellikle C/C++ arka plan'nden C# ' ta yöntemler ve değişkenler gerekir olarak bildirilen ve bir yapı ya da sınıfı içinde tanımlanan olduğunu unutmayın geliştiriciler tarafından oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="1c375-105">This error is often generated by developers from a C/C++ background who forget that in C#, methods and variables must be declared and defined within a struct or class.</span></span> <span data-ttu-id="1c375-106">Daha fazla bilgi için [C# programının genel yapısı](../../../csharp/programming-guide/inside-a-program/general-structure-of-a-csharp-program.md).</span><span class="sxs-lookup"><span data-stu-id="1c375-106">For more information, see [General Structure of a C# Program](../../../csharp/programming-guide/inside-a-program/general-structure-of-a-csharp-program.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="1c375-107">Örnek</span><span class="sxs-lookup"><span data-stu-id="1c375-107">Example</span></span>

 <span data-ttu-id="1c375-108">Aşağıdaki örnek, CS0116 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="1c375-108">The following sample generates CS0116:</span></span>  

```csharp
// CS0116.cs  
namespace x  
{  
    using System;  
  
    // method must be in class/struct  
    void Method(string str) // CS0116  
    {  
        Console.WriteLine(str);  
    }  
    // To fix the error, you must  
    // enclose a method in a class:  
    class Program  
    {  
        void Method2(string str)  
        {  
            Console.WriteLine(str);  
        }  
    }  
}  
```
