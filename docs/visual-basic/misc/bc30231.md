# <a name="39implements39-not-valid-in-modules"></a>&#39; Implements &#39; Modüller içinde geçerli değil
Bir `Implements` deyimi veya yan tümcesi içinde bir modül oluşur. Modülleri arabirimleri uygulayamaz.  
  
 **Hata Kimliği:** BC30231  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Implements` deyimi veya yan tümcesi veya modülü bir sınıf olarak yeniden yazın.
