# <a name="linq-to-xml-c"></a>LINQ-XML (C#)
LINQ-XML .NET Language-Integrated sorgu (LINQ) Framework yararlanır bir bellek içi XML programlama arabirimi sağlar. LINQ-XML en son .NET Framework dil yeteneklerini kullanır ve bir güncelleştirilmiş, yeniden tasarlanan belge nesne modeli (DOM) XML programlama arabirimine karşılaştırılabilir.  
  
 Teknolojileri LINQ ailesi nesneleri (LINQ nesnelere), ilişkisel veritabanları (LINQ-SQL) ve XML (LINQ-XML) için bir tutarlı sorgu deneyimi sağlar.
