# <a name="xml-comment-type-parameter-ltparametergt-does-not-match-a-type-parameter-on-the-corresponding-ltkeywordgt-statement"></a>XML açıklama türü parametresi &lt;parametresi&gt; karşılık gelen üzerinde bir tür parametresi eşleşmiyor &lt;anahtar sözcüğü&gt; deyimi
XML açıklama parametresi \<parametresi > karşılık gelen üzerinde bir tür parametresi eşleşmiyor \<anahtar sözcüğü > deyimi. XML açıklama yoksayılacak.  
  
 Bir XML açıklama etiketi için bir yöntem, ancak bazı değil tüm tür parametrelerinde belirtilmedi.  
  
 **Hata Kimliği:** BC42317  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   XML açıklama etiketleri yöntemi tüm tür parametrelerinde sağlar.
