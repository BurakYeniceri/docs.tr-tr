# <a name="full-width-characters-are-not-valid-as-xml-delimiters"></a>Tam genişlikli karakter XML sınırlayıcı olarak geçerli değildir.
XML değişmez değer ayırıcı olarak tam genişlikli karakteri içeren tanımlandı. Tam genişlikli karakter, geniş veya çok baytlı karakter olarak da adlandırılır.  
  
 **Hata Kimliği:** BC31197  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Tam genişlikli karakter XML değişmez değer tanımdan kaldırın ve geçerli bir ANSI sınırlayıcı karakteri ile değiştirin. Geçerli sınırlayıcı karakterler şunlardır: `<`, `>`, `=`, `:`, `/`.
