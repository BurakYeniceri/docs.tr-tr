# <a name="unrecognized-option-ltoptionnamegt-ignored"></a>Tanınmayan seçenek &lt;SeçenekAdı&gt;; yoksayıldı

 **Hata Kimliği:** BC2007  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçerli olanı bulmak için belirtilen seçenekler denetleyin.
