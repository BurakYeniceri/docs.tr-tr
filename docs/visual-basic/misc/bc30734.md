# <a name="39ltnamegt39-is-already-declared-as-a-parameter-of-this-method"></a>&#39; &lt;adı&gt;&#39; bu yöntem parametresi olarak zaten bildirildi
Belirtilen parametre adı, bu yöntem için zaten tanımlanmış.  
  
 **Hata Kimliği:** BC30734  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Parametrenin benzersiz şekilde yeniden adlandırın.
