# <a name="39interface39-must-end-with-a-matching-39end-interface39"></a>&#39;Arabirim&#39; eşleşen ile bitmelidir &#39;uç arabirimi&#39;
Bir `Interface` deyimi oluşur karşılık gelen olmadan `End Interface` deyimi. Kullanmalısınız bir `End Interface` arabirim tanımı sonuna deyimi.  
  
 **Hata Kimliği:** BC30253  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Denetleyin `Interface` üyeleri doğru biçimlendirilmiş.  
  
2.  Ekleme bir `End Interface` deyimi tot arabirim tanımı sonu.
