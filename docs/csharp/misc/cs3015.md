# <a name="compiler-warning-level-1-cs3015"></a>Derleyici Uyarısı (düzey 1) CS3015
'yöntem imzası' yalnızca CLS uyumlu türler kullanan erişilebilir oluşturucusu yok  
  
 Ortak dil belirtimi (CLS) ile uyumlu olması için bir öznitelik sınıfı bağımsız değişken listesi bir dizi içeremez. CLS uyumluluğu hakkında daha fazla bilgi için bkz: [CLS uyumlu kod yazma](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) ve [dil bağımsızlığı ve dilden bağımsız bileşenler](../../../docs/standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek C3015 oluşturur.  
  
```  
// CS3015.cs  
// compile with: /target:library  
using System;  
  
[assembly:CLSCompliant(true)]  
public class MyAttribute : Attribute  
{  
   public MyAttribute(int[] ai) {}   // CS3015  
   // try the following line instead  
   // public MyAttribute(int ai) {}  
}  
```
