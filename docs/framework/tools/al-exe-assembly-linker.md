# <a name="alexe-assembly-linker"></a>Al.exe (Derleme Bağlayıcı)

Assembly Linker, modül veya kaynak dosyalar olan bir veya daha fazla dosyadan, derleme bildirimi içeren bir dosya oluşturur. Modül, derleme bildirimi içermeyen bir ara dil (IL) dosyasıdır.

> [!NOTE]
> İle başlayarak [!INCLUDE[vs_orcas_long](../../../includes/vs-orcas-long-md.md)], C# ve Visual Basic derleyicileri derlemeye otomatik olarak bir Win32 Bildirim katıştırma. Daha fazla bilgi için bkz: [/win32manifest (C# Derleyici Seçenekleri)](~/docs/csharp/language-reference/compiler-options/win32manifest-compiler-option.md).

Bu araç, Visual Studio ile birlikte otomatik olarak yüklenir. Aracı çalıştırmak için, Geliştirici Komut İstemi (veya Windows 7'de Visual Studio Komut İstemi) kullanın. Daha fazla bilgi için bkz: [komut istemlerini](../../../docs/framework/tools/developer-command-prompt-for-vs.md).

Komut satırına şunu yazın:

## <a name="syntax"></a>Sözdizimi

```console
al sources options
```

## <a name="remarks"></a>Açıklamalar

Tüm Visual Studio derleyicileri derlemeler üretir. Ancak, bir veya daha fazla modül (meta verileri bildirim olmadan) varsa, kullanabileceğiniz *Al.exe* ayrı bir dosya bildiriminde ile bir derlemeyi oluşturmak için.

Önbellekte derlemelerini yüklemek için önbelleğinden derlemeleri kaldırma veya önbelleğinin içeriğini listesinde, kullanmak [Genel Derleme Önbelleği Aracı (*Gacutil.exe*)](../../../docs/framework/tools/gacutil-exe-gac-tool.md).

## <a name="example"></a>Örnek

Aşağıdaki komut yürütülebilir bir dosya oluşturur *t2a.exe* bir derlemeye ait ile `t2.netmodule` modülü. Giriş noktası `Main` yönteminde `MyClass`.

```console
al t2.netmodule /target:exe /out:t2a.exe /main:MyClass.Main
```
