# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>&#39;Modül&#39; deyimi, eşleşen bir ile bitmelidir &#39;son Modülü&#39;
Modül blokları ile bitmelidir bir `End``Module` deyimi.  
  
 **Hata Kimliği:** BC30625  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Ekleme bir `End Module` deyimi sonuna `Module` bloğu.
