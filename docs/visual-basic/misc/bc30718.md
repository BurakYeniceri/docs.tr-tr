# <a name="39select39-statements-are-not-valid-in-the-immediate-window"></a>&#39; &#39;seçin; deyimleri Komut penceresi içinde geçerli değildir
`Select`deyimleri yalnızca kaynak kodunda kullanılabilir.  
  
 **Hata Kimliği:** BC30718  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Select` kodda hata ayıklama gelen deyimleri.
