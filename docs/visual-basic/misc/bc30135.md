# <a name="39synclock39-statements-are-not-valid-in-the-immediate-window"></a>&#39;SyncLock&#39; ifadeleri geçerli değildir komut penceresi
`SyncLock` Deyimi iş parçacıklarını eşitleme yöntemini ve hata ayıklama bağlamda izin verilmiyor.  
  
 **Hata Kimliği:** BC30135  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Sorun değil bir `SyncLock` deyiminde **hemen** penceresi.
