# <a name="the-file-is-already-open"></a>Dosya zaten açık olan
Zaten açık bir dosyayı açma girişiminde bulunuldu.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Dosyayı açmak denemeden önce kapatın.  
  
     veya  
  
-   Yedek kaldırmak `FileOpen` kodunuzdan deyimi.
