<span data-ttu-id="77de2-103">'Oluşturucusu': statik Oluşturucu parametresiz olmalıdır</span><span class="sxs-lookup"><span data-stu-id="77de2-103">'constructor' : a static constructor must be parameterless</span></span>  
  
 <span data-ttu-id="77de2-104">A [statik](../../csharp/language-reference/keywords/static.md) Oluşturucusu ile bir veya daha fazla parametre bildirilemez.</span><span class="sxs-lookup"><span data-stu-id="77de2-104">A [static](../../csharp/language-reference/keywords/static.md) constructor cannot be declared with one or more parameters.</span></span> <span data-ttu-id="77de2-105">Daha fazla bilgi için bkz: [oluşturucular](../../csharp/programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="77de2-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="77de2-106">Aşağıdaki örnek CS0132 oluşturur:</span><span class="sxs-lookup"><span data-stu-id="77de2-106">The following sample generates CS0132:</span></span>  
  
```csharp  
// CS0132.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public MyClass(int i)  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass  
   {  
      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
