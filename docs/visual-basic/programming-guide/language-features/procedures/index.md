-   <span data-ttu-id="0ec34-140">[İşleç yordamları](./operator-procedures.md) birini veya her ikisini işlenenler olduğunda yeni tanımlı sınıf veya yapı standart işleci davranışını tanımlayın.</span><span class="sxs-lookup"><span data-stu-id="0ec34-140">[Operator Procedures](./operator-procedures.md) define the behavior of a standard operator when one or both of the operands is a newly-defined class or structure.</span></span>
