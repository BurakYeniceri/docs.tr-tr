# <a name="events-in-interfaces-cannot-be-declared-39ltimplementsgt39"></a>Arabirimlerdeki olaylar bildirilemez &#39; &lt;uygular&gt;&#39;
Arabirimlerde bildirilen olaylarda diğer arabirimleri olayları uygulayamaz.  
  
 **Hata Kimliği:** BC30688  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Kaldırma `Implements` deyimi.  
  
2.  Bir sınıf veya yapı içinde olayın uygulayın.
