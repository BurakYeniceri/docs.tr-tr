# <a name="reference-linq-to-xml"></a>Başvuru (LINQ-XML)
Bu konuda LINQ XML başvurusu konulara bağlantılar içerir.  
  
## <a name="in-this-section"></a>Bu Bölümde  
 LINQ-XML sınıfları için başvuru belgeleri için bkz: <xref:System.Xml.Linq>.  
  
 Başvuru için bir XSD dosyası karşı XML ağaçları yardımcı genişletme yöntemleri doğrulamak için belgelere bakın <xref:System.Xml.Schema.Extensions?displayProperty=nameWithType>.  
  
 XPath sorguları bir XML ağaç değerlendirmek etkinleştirmeniz genişletme yöntemleri için başvuru belgeleri için bkz <xref:System.Xml.XPath.Extensions?displayProperty=nameWithType>.
