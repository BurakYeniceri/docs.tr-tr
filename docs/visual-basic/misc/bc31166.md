# <a name="expected-closing-39gt39-for-xml-tag"></a>Beklenen kapanış &#39; &gt;&#39; XML etiketi
XML değişmez değeri XML etiketi gerekli kapanış eksik ' >' karakter.  
  
 **Hata Kimliği:** BC31166  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kapatma Ekle ' >' XML değişmez değeri XML etiketi sonuna karakter.
