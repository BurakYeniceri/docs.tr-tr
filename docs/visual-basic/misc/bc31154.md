# <a name="xml-declaration-does-not-allow-attribute-39attributename39"></a>XML bildirimi özniteliği &#39; attributeName &#39;izin verme;
Geçersiz bir öznitelik için bir XML belgesi değişmez değeri belirtildi. Bir XML belgesi değişmez değeri geçerli öznitelik değerlerini XML 1.0 belirtimi temel alır.  
  
 **Hata Kimliği:** BC31154  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçersiz öznitelik değişmez değer XML belgesinden kaldırın.
