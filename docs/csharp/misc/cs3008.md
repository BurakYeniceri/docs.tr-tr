# <a name="compiler-warning-level-1-cs3008"></a>Derleyici Uyarısı (düzey 1) CS3008
'Yalnızca durumda farklı tanımlayıcı tanımlayıcısı' CLS uyumlu değil  
  
 A [ortak](../../csharp/language-reference/keywords/public.md), [korumalı](../../csharp/language-reference/keywords/protected.md), veya `protected internal` tanımlayıcısı keser uyumluluk ortak dil belirtimi (CLS) ile bir alt çizgi karakteriyle başlıyorsa (\_). CLS uyumluluğu hakkında daha fazla bilgi için bkz: [CLS uyumlu kod yazma](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) ve [dil bağımsızlığı ve dilden bağımsız bileşenler](../../../docs/standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Örnek  
 Aşağıdaki örnek CS3008 oluşturur:  
  
```  
// CS3008.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public class a  
{  
    public static int _a = 0;  // CS3008  
    // OK, private  
    // private static int _a1 = 0;  
  
    public static void Main()  
    {  
    }  
}  
```
