# <a name="compiler-error-cs0128"></a>Derleyici Hatası CS0128
'Değişken' adlı bir yerel değişken bu kapsamda zaten tanımlandı  
  
 Derleyici aynı ada sahip iki yerel değişken bildirimlerini algıladı. Daha fazla bilgi için bkz: [yöntemleri](../../csharp/programming-guide/classes-and-structs/methods.md).  
  
 Aşağıdaki örnek CS0128 oluşturur:  
  
```csharp  
// CS0128.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         char i;  
         int i;   // CS0128  
      }  
   }  
}  
```
