# <a name="replacements-too-long"></a>Değişiklik çok uzun
İzin verilen uzunluğu aşıyor yenisini belirttiğiniz.  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
1.  Bileşen için length kısıtlama belgelerine bakın.
