# <a name="compiler-error-cs1619"></a>Derleyici Hatası CS1619
Geçici dosya 'filename'--neden oluşturulamıyor  
  
 Derleyici belirli bir nedenle geçici bir dosya oluşturamadı (örneğin, disk dolu olduğundan).
