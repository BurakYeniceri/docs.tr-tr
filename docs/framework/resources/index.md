## <a name="retrieving-resources"></a><span data-ttu-id="70733-135">Kaynakları Alma</span><span class="sxs-lookup"><span data-stu-id="70733-135">Retrieving Resources</span></span>  
 <span data-ttu-id="70733-136">Çalışma zamanında bir uygulama, <xref:System.Globalization.CultureInfo.CurrentUICulture%2A?displayProperty=nameWithType> özelliği tarafından belirtilen kültürü temel alan, iş parçacığını düzeyinde uygun yerelleştirilmiş kaynakları yükler.</span><span class="sxs-lookup"><span data-stu-id="70733-136">At run time, an app loads the appropriate localized resources on a per-thread basis, based on the culture specified by the <xref:System.Globalization.CultureInfo.CurrentUICulture%2A?displayProperty=nameWithType> property.</span></span> <span data-ttu-id="70733-137">Bu özellik değeri aşağıdaki şekilde türetilir:</span><span class="sxs-lookup"><span data-stu-id="70733-137">This property value is derived as follows:</span></span>
