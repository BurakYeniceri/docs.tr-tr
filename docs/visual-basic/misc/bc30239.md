# <a name="relational-operator-expected"></a>İlişkisel işleç bekleniyor
A `Case` deyimi içeren bir `Is` yan tümcesi, ancak hiçbir karşılaştırma işleci gibi `=` veya `>`. Varsa bir `Case` deyimi bir işleç içermez `=` varsayılır ve `Is` kullanılmaz.  
  
 **Hata Kimliği:** BC30239  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Kaldırma `Is` anahtar sözcüğü veya bir karşılaştırma işleci ile izleyin.
