# <a name="compiler-error-cs0426"></a><span data-ttu-id="43eb4-102">Derleyici Hatası CS0426</span><span class="sxs-lookup"><span data-stu-id="43eb4-102">Compiler Error CS0426</span></span>
<span data-ttu-id="43eb4-103">Tür adı 'tanımlayıcı' türü 'type' mevcut değil</span><span class="sxs-lookup"><span data-stu-id="43eb4-103">The type name 'identifier' does not exist in the type 'type'</span></span>  
  
 <span data-ttu-id="43eb4-104">Bu hata içinde başka bir tür iç içe geçmiş bir türe başvuruda, ancak hiçbir gibi iç içe türü mevcut oluşur.</span><span class="sxs-lookup"><span data-stu-id="43eb4-104">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="43eb4-105">İç içe geçmiş tür adı yanlış yazdınız bu durum ortaya çıkabilir.</span><span class="sxs-lookup"><span data-stu-id="43eb4-105">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="43eb4-106">Kullanılan adların yazımını denetleyin ve kendilerini kapsayan türle beklenen üyesi olduğunu doğrulayın.</span><span class="sxs-lookup"><span data-stu-id="43eb4-106">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="43eb4-107">Aşağıdaki örnek, CS0426 oluşturur, çünkü iç içe geçmiş tür A: C sınıfı yok</span><span class="sxs-lookup"><span data-stu-id="43eb4-107">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```csharp  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.     
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```
