# <a name="for-loop-control-variable-39ltvariablenamegt39-already-in-use-by-an-enclosing-for-loop"></a>For döngüsü denetim değişkeni için &#39; &lt;variablename&gt; &#39; aldığı For döngüsü tarafından zaten kullanılmakta
İç içe geçmiş `For` döngüler tüm benzersiz denetim değişkenleri kullanmanız gerekir.  
  
 **Hata Kimliği:** BC30069  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Denetim değişken adları biriyle değiştirin.
