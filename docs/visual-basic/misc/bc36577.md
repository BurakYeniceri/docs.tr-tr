# <a name="anonymous-type-initializers-are-not-valid-in-debug-windows"></a>Anonim tür başlatıcıları hata ayıklama pencerelerinde geçerli değil
Anonim tür başlatıcısı girilen **izleme** veya **hemen** penceresi bir hata ayıklama oturumu sırasında. Anonim tür başlatıcıları windows hata ayıklama kullanılabilir değil.  
  
 **Hata Kimliği:** BC36577  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Anonim tür başlatıcıları Kod düzenleyicisinde test edin.
