# <a name="compiler-error-cs0752"></a><span data-ttu-id="4a649-102">Derleyici Hatası CS0752</span><span class="sxs-lookup"><span data-stu-id="4a649-102">Compiler Error CS0752</span></span>
<span data-ttu-id="4a649-103">Kısmi bir yöntemin out Parametreleri olamaz</span><span class="sxs-lookup"><span data-stu-id="4a649-103">A partial method cannot have out parameters</span></span>  
  
 <span data-ttu-id="4a649-104">Kısmi bir yöntemin olamaz bir [kullanıma](../../csharp/language-reference/keywords/out-parameter-modifier.md) parametresi.</span><span class="sxs-lookup"><span data-stu-id="4a649-104">A partial method cannot have an [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="4a649-105">Out Parametreleri olduğundan kısmi yöntem derleyici tarafından kaldırıldıktan sonra out parametresi hiç atanan bir garanti yoktur izin verilmez.</span><span class="sxs-lookup"><span data-stu-id="4a649-105">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4a649-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="4a649-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="4a649-107">Out değiştiricisi parametresini kaldırın ve yöntemin dönüş değerini kullanın, aksi takdirde yöntem bildiriminden partial değiştiricisi kaldırın.</span><span class="sxs-lookup"><span data-stu-id="4a649-107">Remove the out modifier from the parameter and use the return value of the method instead, or else remove the partial modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4a649-108">Örnek</span><span class="sxs-lookup"><span data-stu-id="4a649-108">Example</span></span>

    }
