# <a name="39ltnamegt39-is-declared-in-project-39ltprojectname1gt39-which-is-not-referenced-by-project-39ltprojectname2gt39"></a>&#39;&lt;ad&gt; &#39; projesinde bildirilmiş &#39; &lt;projectname1&gt;&#39;, hangi başvurulmadı projesi tarafından &#39; &lt;projectname2&gt;&#39;
Bir programlama öğesi için geçerli projenin bir başvurusu bulunmayan bir proje ile tanımlanır.  
  
 **Hata Kimliği:** BC32004  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Geçerli projede İstenen öğe içeriyor projesine bir başvuru ekleyin.
