# <a name="icorprofilerinfogetclassfromobject-method"></a>ICorProfilerInfo::GetClassFromObject Metodu
Alır `ClassID` verilen nesnenin kendi `ObjectID`.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT GetClassFromObject(  
    [in]  ObjectID objectId,  
    [out] ClassID *pClassId);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `objectId`  
 [in] Almak istediğiniz için nesnesinin kimliği `ClassID`.  
  
 `pClassId`  
 [out] Bir işaretçi döndürülen `ClassID`.  
  
## <a name="remarks"></a>Açıklamalar  
 Bir null `pClassId` belirten `objectId` kaldırdığı bir türe sahip.
