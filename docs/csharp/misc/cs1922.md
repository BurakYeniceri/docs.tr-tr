 <span data-ttu-id="48a0c-104">Koleksiyon Başlatıcısı ile bir türü kullanmak için türü uygulanması gereken `IEnumerable`.</span><span class="sxs-lookup"><span data-stu-id="48a0c-104">In order to use a collection initializer with a type, the type must implement `IEnumerable`.</span></span> <span data-ttu-id="48a0c-105">Nesne Başlatıcı kullanılacak amacı, koleksiyon Başlatıcısı sözdizimi yanlışlıkla kullanırsanız, bu hata oluşabilir.</span><span class="sxs-lookup"><span data-stu-id="48a0c-105">This error can occur if you accidentally use collection initializer syntax when you meant to use an object initializer.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="48a0c-106">Bu hatayı düzeltmek için</span><span class="sxs-lookup"><span data-stu-id="48a0c-106">To correct this error</span></span>  
  
-   <span data-ttu-id="48a0c-107">Bir koleksiyon türünü temsil etmez, nesne Başlatıcısı sözdizimi koleksiyon Başlatıcısı sözdizimi yerine kullanın.</span><span class="sxs-lookup"><span data-stu-id="48a0c-107">If the type does not represent a collection, use object initializer syntax instead of collection initializer syntax.</span></span>

## <a name="example"></a><span data-ttu-id="48a0c-110">Örnek</span><span class="sxs-lookup"><span data-stu-id="48a0c-110">Example</span></span>

```csharp  
{  
    public static void Main()  
    {  
        // Collection initializer.  
        var tc = new TestClass  {1,"hello"} ; // CS1922

    public string memberB { get; set; }  
}  
```
