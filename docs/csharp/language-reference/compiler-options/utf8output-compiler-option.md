# <a name="-utf8output-c-compiler-options"></a>-utf8output (C# Derleyici Seçenekleri)
**-Utf8output** seçenek UTF-8 kodlaması kullanarak derleyici çıkışı görüntüler.  
  
## <a name="syntax"></a>Sözdizimi  
  
```console  
-utf8output  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Uluslararası bazı yapılandırmalarda Derleyici çıktısı doğru konsolda görüntülenemiyor. Bu yapılandırmalarda kullanmak **-utf8output** ve Derleyici çıktısı bir dosyaya yönlendirin.  
  
 Bu derleyici seçeneği Visual Studio'da kullanılamıyor ve programlı olarak değiştirilemez.
