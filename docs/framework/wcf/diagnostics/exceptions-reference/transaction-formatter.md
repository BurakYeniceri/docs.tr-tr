# <a name="transaction-formatter"></a>İşlem Biçimlendirici
Bu konu, işlem biçimlendirici tarafından oluşturulan tüm özel durumları listeler.  
  
## <a name="exception"></a>Özel Durum  
  
|Kaynak kodu|Kaynak dizesi|  
|-------------------|---------------------|  
|TMCommunicationError|Dağıtılmış İşlem Yöneticisi ile iletişim kurulurken bir hata oluştu.|  
|WhereaboutsNoHostName|MSDTC konum bilgileri, bir ana bilgisayar adı içermiyor.|  
|WhereaboutsReadFailed|MSDTC konum bilgileri serisi yok.|  
|WhereaboutsSignatureMissing|Standart konumu imza MSDTC konum bilgileri eksik.|  
|WsatUriCreationFailed|Bir kayıt hizmeti adresi MSDTC konum bilgileri oluşturulamadı.|
