# <a name="array-subscript-expression-missing"></a>Dizi alt simge ifadesi eksik
Bir dizi başlatma, bir veya daha fazla dizi sınırları tanımlamak indisleri bırakır. Örneğin, deyim ifade içerebilir `myArray (5,5,,10)`, üçüncü alt simge bırakır.  
  
 **Hata Kimliği:** BC30306  
  
## <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Eksik alt simge sağlayın.
