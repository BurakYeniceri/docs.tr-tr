# <a name="c-compiler-options"></a>C# Derleyici Seçenekleri
Derleyici yürütülebilir dosyanın (.exe) dosyaları, dinamik bağlantı kitaplıkları (.dll) veya kod modülleri üretir (.netmodule).  
  
 Her derleyici seçeneği iki biçimde sağlanır: **-seçeneği** ve **/seçeneği**. Belgelere yalnızca gösterir **-seçeneği** formu.  
  
 Visual Studio'da, web.config dosyasında derleyici seçeneklerini ayarlayın. Daha fazla bilgi için bkz: [ \<derleyici > öğesi](../../../framework/configure-apps/file-schema/compiler/compiler-element.md).
