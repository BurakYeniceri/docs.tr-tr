<span data-ttu-id="11711-103">Tutarsız erişilebilirlik: temel arabirim '1 arabirim' arabirim 'arabirim 2' den daha az erişilebilir</span><span class="sxs-lookup"><span data-stu-id="11711-103">Inconsistent accessibility: base interface 'interface 1' is less accessible than interface 'interface 2'</span></span>  
  
 <span data-ttu-id="11711-104">A [ortak](../../csharp/language-reference/keywords/public.md) yapı genel olarak erişilebilen bir nesne döndürmesi gerekir.</span><span class="sxs-lookup"><span data-stu-id="11711-104">A [public](../../csharp/language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>

 <span data-ttu-id="11711-107">Aşağıdaki örnek CS0061 oluşturur.</span><span class="sxs-lookup"><span data-stu-id="11711-107">The following sample generates CS0061.</span></span>  
  
```csharp  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
